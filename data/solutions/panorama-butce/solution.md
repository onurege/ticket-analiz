## Ekran Tanımı

Bütçenin tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Sistem tarafından takip edilen artışlı numara bilgisidir. |
| Durum | Ürün durum bilgisidir. Aktif, Pasif değerlerini alır. |
| Distribütör Kodu | Distribütör kodunun seçildiği sahadır. Tanımlanacak bütçenin hangi distribütöre bağlı olduğu belirlenir. Girilmesi zorunlu alandır. |
| Müşteri Kriter | Müşteri Kriter bilgisidir. Değer olarak “Kod”, ”Grup Kodu”, ”Ek Grup Kodu” ve “Dinamik Grup” olarak değer alır. |
| Müşteri Kodu | Müşteri Kodunun görüldüğü ekrandır. |
| Bütçe Dönemi | Önceden tanımlanmış olan Bütçe Dönem listesinden seçilir. |
| Sözleşme Kod | Bütçe tanımında yapılacak sözleşmenin kod bilgisinin yazıldığı alandır. |
| Hedef Ürün Kriter | Bütçe tanımının hangi ürün alımlarında geçerli olacağını belirlemek için kullanılır. Kriter tipi Kod, Grup ve Ek Grup tiplerinden seçilir. |
| Aktivite Grup Kod | Aktivite grup kodu bilgisidir. |
| Bütçe Maliyeti | Bütçe tanımının maliyetinin yazıldığı alan. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Hedef Giriş | Seçilen kriterlere göre otomatik olarak gelen listedir. |
| F.A Iskonto | Bütçe tanım istonto promosyon ekranıdır. |
| Kapat | Ekranı Kapatma işlemi yapar. |
