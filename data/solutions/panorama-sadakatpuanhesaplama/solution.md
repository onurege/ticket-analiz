# Sadakat Puan Hesaplama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriteri → Sadakat İşlemleri → Sadakat Puan Hesaplama`

## Ekran Tanımı

Sadakat puan hesaplama ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Tanımlanması yapılmak istenilen fiyat onaylamanın hangi distribütöre ait olduğunun bilgisidir. |
| Hedef Kodu | Hedef kodu bilgisidir. |
| Müşteri Kodu | Müşteri kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İşlemi Başlat | İşlemi başlatmak için kullanılır. |
| Listele | Listelemek için kullanılır. |
| Temizle | Temizlemek için kullanılır. |
| Kapat | Ekranı kapatır. |
