# Fatura Kapatma

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → İşlemler → Fatura Kapatma`

## Ekran Tanımı

Fatura kapatma işlemlerinin gerçeklerştirildiği ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Müşteri Kodu | Müşteri kodu bilgisinin bulunduğu alandır. |
| Filtre | Filtre bilgisidir. “Açık”, “Kapatılmış”, “Tümü” seçenekleri yapılabilir. |
| Borç Toplamı | Toplam borç bilgisidir. |
| Alacak Toplamı | Toplam alacak bilgisidir. |
| Bakiye | Bakiye bilgisidir. |
| Vade Tarihi | Fatura Vade tarihi bilgisidir. |
| İşlem Tarihi | Fatura İşlem tarihi bilgisidir. |
| Belge Tür | Belge türü bilgisidir. |
| Matbu No | Matbu no bilgisidir. |
| Belge Kod | Belge kodu bilgisidir. |
| Yıl | Yıl bilgisidir. |
| Net Tutar | Net tutar bilgisidir. |
| Kalan Tutar | Kalan tutar bilgisidir. |
| Kapatılan Tutar | Kapatılan tutar bilgisidir. |
| Belge Tür | Belge türü bilgisidir. |
| Kapatma Durum | Kapatma durumu bilgisidir. |
| Kapatma Tipi | Kapatma tipi bilgisidir. |
| ST | Satış Temsilcisi bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Belge Kapat | Belge üzerinde yapılan işlemleri kaydedip kapatmaya yarar. |
| FIFO Kapatma Uygula | Fifo kapatma uygulama butonudur. |
| Kapat | Ekranı kapatmak için kullanılır. |
| Belge İzle | Detayda seçilen belgeyi incelemek için kullanılır. |
| Geri Al | Detayda seçilen belge üzerindeki işlemi geri almak için kullanılır. |
