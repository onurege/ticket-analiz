# Duyuru / Katılım Bilgilendirme Tanımlama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ve Pazarlama Kriterleri → İskonto Promosyon → Duyuru / Katılım Bilgilendirme Tanımlama`

## Ekran Tanımı

Duyuru / Katılım Bilgilendirme Tanımlama ekradınır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Kod bilgisidir. |
| Durum | Belge Durumu bilgisidir. |
| Distribütör Kriter Tipi | Distribütör Kriter Tipi bilgisidir. |
| Distribütör Kriteri | Distribütör kriteri bilgisidir. |
| Başlangıç Tarihi | Başlangıç tarihi bilgisidir. |
| Bitiş Tarihi | Bitiş tarihi bilgisidir. |
| Onay Bilgi Maili | Onay bilgisi maili ekranıdır. |
| Distribütör Not Girişi Yapabilir | Distribütör not girişi yapılsın istenirse seçilir. |
| Bilgi Maili Gönderilsin | Bilgi maili gönderilsin istenirse seçilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Belgeyi kaydedip yeni belge oluşturma işlemini yapmaktadır. |
| Kaydet ve Kapat | Belgeyi kaydedip ekranı kapatmaya yarar. |
| Kapat | Ekranı kaydetmeden kapatır. |
