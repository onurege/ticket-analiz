# Veri Alma

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ İşlemler → Veri Alma`

## Ekran Tanımı

Excel üzerinden Panoramaya veri (fatura,sipariş,fiyat,iskonto... gibi) alma işlemidir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Veri aktarımı yapılacak distribütör bilgisi seçilir. |
| Görev Kod | Veri aktarımı yapılacak görev kodu seçilir. |
| İşlem No | Sistem tarafından otomatik olarak atanan işlem numarası gösterilir. |
| Ay | Aktarım yapılacak ay bilgisi seçilir. |
| Yıl | Aktarım yapılacak yıl bilgisi seçilir. |
| Mükerrer Satış Faturası Kontrolü Yapılsın | İlgili kutucuk işaretlenir ise aynı matbu noya ait satış faturası kontrolü yapılarak sisteme aktarılması engellenir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Dosya Yükle | Panorama’ya aktarılacak excel dosyası seçimi için kullanılır. |
| Dosya Aktar | Seçilen excel dosyasının sisteme aktarılması için kullanılır. |
| İşlem Durum | Aktarım başlatılan işlemin durumunu takip etmek için kullanılır. |
| Kapat | Ekran kapatılır. |
| Seç | Sistem üzerinde yaratılan , Panorama’ya aktarılacak excel doyası seçimi yapılır. |
| Yükle | Seçilen excel dosyasının sisteme aktarılması için kullanılır. |
| Kapat | Ekran kapatılır. |
