## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |

| Giriş Tipi | Belgenin oluşturulma yeri bilgisidir. “FA İskontolari”, “Bayi Müşteri Desteği”, “Bölge Bütçe Fon”, “Alış Çıkış Oranı” değerlerini alabilir. |
| Yıl | Yıl bilgisidir. |
| Ay | Ay bilgisidir. |

| Bitiş Tarihi | Kaydın(Dönemin) biteceği zorunlu Tarih alanıdır. |
| Distribütör / Bayi | Distribütör / Bayi bilgisidir. |
| Seviyeli Ürün-2 | Seviyeli ürün bilgisidir. |
| Merkez Müşteri Kodu | Merkez müşteri kodu bilgisidir. |
| Müşteri Kodu | Müşteri kodu bilgisidir. |
| Distribütör Grup | Distribütör kodu bilgisidir. |
| Kampanya % | Kampanya iskonto oranıdır. |
| Aksiyon İskonto % | Aksiyon iskonto oranıdır. |
| Toplam İskonto % | Toplam iskonto oranıdır. |
| Bayi Desteği Oranı % | Bayi Desteği iskonto oranıdır. |
| Bütçe Bölge Fon Oranı % | Bütçe Bölge Fon iskonto oranıdır. |
| Bayi Alış Çıkış Oranı | Bayi alış çıkış iskonto oranıdır. |
