## Ekran Tanımı

Depozitolu ürün miktar kısıtlama ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Durum | Evrak durum bilgisidir. Aktif/Pasif değerlerini alır. |
| Kontrol Tipi | Kontrol Tipi bilgisidir. |
| Başlangıç Tarihi | Başlangıç tarhihi bilgisidir. |
| Bitiş Tarhihi | Bitiş tarihi bilgisidir. |
| Distribütör Kodu | Distribütör Kodu bilgisidir. |
| Müşteri Kriter Tipi | Müşteri Kriter Tipi bilgisidir. |
| Müşteri Kodu | Müşteri Kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Ekranı kaydetmek için kullanılan butondur. |
| Kapat | Ekranı Kapatma işlemi yapar. |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Detayda yeni tanım yapmak için kullanılır. |
| Düzenle | Detaydaki kaydı düzenlemek için kullanılır. |
| İzle | Detaydaki kaydı görüntülemek için kullanılır. |
| Sil | Detaydaki kaydın silinmesi için kullanılır. |
