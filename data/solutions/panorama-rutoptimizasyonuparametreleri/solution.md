# Rut Optimizasyonu Parametreleri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri → Dağıtım İşlemleri → Rut Optimizasyonu → Rut Optimizasyonu Parametreleri`

## Ekran Tanımı

Rut optimizasyon parametreleri ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| Araç | Araç bilgisidir. |
| Optimizayon Hedefi | Optimizasyon hedefinin seçildiği alandır. |
| Kapasite Hesaplama Tipi | Kapasite hesaplama tipi bilgisidir. |
| Maksimum Yol Süre | Maksimum yol süre bilgisidir. |
| Günlük Çalışma Süresi | Günlük çalışma süresi bilgisidir. |
| Maksimum Rota Sayısı | Maksimum rota sayısı bilgisidir. |
| Rota Başına Maksimum Durak | Rota başına maksimum durak sayısı bilgisidir. |
| Gün Sonunda Aracın Gideceği İstasyon | Gün sonunda aracın gideceği istasyon seçineği yapılacak olan alandır. |
| Minimum Duraklama Süresi | Minimum duraklama süresidir. (saat:dakika olarak girilmektedir.) |
| Maksimum Rota Zamanı | Maksimum duraklama süresidir. (saat:dakika olarak girilmektedir.) |
| Rota Başına Maksimum Mesafe | Rota başına maksimum mesafe bilgisidir. |
| Bir Bölgeye Aynı Araçla Hizmet | Bir bölgeye aynı araçla hizmet edilmesi istenirse seçilir. |
| Araçları Sınıflandırma | Araçları sınıflandırmak istenirse seçilir. |
| Ücretli Girişleri Kullanma | Ücretli girişleri kullanmasın istenirse seçinilir. |
| Depolara Aynı Araçla Birden Fazla Ziyaret | Depolara aynı araçla birden fazla ziyaret yapılacaksa seçilir. |
| Otoyolları Kullanma | Otoyolların kullanılması istenirse seçilir. |
| İstasyonları Bir Kere Ziyaret Et | İstasyonları bir kere ziyaret etsin istenirse seçilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Belgeyi kaydedip yeni bir belge açmayı sağlar. |
| Kaydet ve Kapat | Yapılan işlemin kaydedilmesini ve kapatması sağanır. |
| Kapat | Ekranı kapatır. |
