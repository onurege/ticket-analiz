# Bayi Risk Limit

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → DBS İşlemleri → Bayi Risk Limit`

## Ekran Tanımı

Banka bazında distribütör DBS limit bilgilerinin listeleneceği ekrandır. Banka seçimi sadece Panorama sistem yöneticisi ekranında yer alan DBS Entegrasyon ekranında kullanılan bankalar üzerinden listelenmekte ve işlem yapılmaktadır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Banka | Banka bilgisinin seçildiği alandır. |
| Distribütör Bölge | Merkez kullanıcı için distribütör bölge kriterinin verilebildiği alandır. |
| Distribütör Kodu | Distribütör kriterinin verildiği alandır. |
| Limit Yüzdesi | Sütunlarında yer alan harcanan rakamın limite bölümünden bulunan limit kullanım yüzde değerinin üzerindeki limit değerlerinin listelenmesi için kullanılmaktadır. |
| Kalan Limit Sayısı | Kullanılabilir limit kolonu artan veya azalan seçeneklerine göre sıralanmaktadır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Girilen kriterlere göre verilen listelenmesini sağlar. |
| Excel’e Aktar | Ekranda listelenen verilerin excel formatında kayıt işleminin yapılmasını sağlar |
| Pdf’e Aktar | Ekranda listelenen verilerin pdf formatında kayıt işleminin yapılmasını sağlar |
| Kapat | Açılan ekran kapanır. |
