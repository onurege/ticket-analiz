# Satış Temsilcisi Şablon Uygulama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Parametreler → Satış Temsilcisi Şablon Uygulama`

## Ekran Tanımı

Satış temsilcisi şablon uygulama ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kriter Tipi | Distribütörlerin seçileceği kriterlerin girildiği alandır değer olarak (Kod, Grup, Ek Grup, Dinamik Grup) değerlerini almaktadır. |
| Uygulama Yeri | Fiyat tanımının uygulama yeri bilgisidir. Panorama, Quest, Panorama, Quest, Münferit seçilebilir. |
| Distribütör Kriteri | Distribütör Kriter Tipi ne göre değişkenlik gösteren bir alandır. Örneğin Distribütör Kriter Tipi “Kod” seçilirse sisteme kayıtlı tüm Distribütörler gelir ve tek bir Distribütör seçilebilir fakat Distribütör Kriter Tipi “Grup” seçilir ise Distribütör grupları seçilir. |
| ST Grup | Satış Temsilcisi Grup tanımı için program tarafından verilen artışlı numaradır. |
| Şablon | Ürün şablonu listesinden şablon seçimi yapılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Bilgileri kaydetmeye yarar. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
