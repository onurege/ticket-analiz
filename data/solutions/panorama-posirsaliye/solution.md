# Pos İrsaliye

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Pos İşlemleri → Tüketici Hızlı Satış (POS)`

## Ekran Tanımı

Hızlı satışın yapılabildiği ekrandır. Müşteriye hem satışın hem tahsilatın aynı anda bu ekrandan yapılabilmektedir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Belge Kodu | Program tarafından verilen artışlı numaradır. |
| Matbu No | Fiziki evrağın resmi numarasıdır. |
| Müşteri Kodu | İşlem yapılacak müşterinin seçildiği yerdir. |
| Satış Temsilcisi | Satışı gerçekleştiren satış elemanının seçildiği yerdir. |
| Depo Kodu | Satışın hangi depodan yapılacağının seçildiği yerdir. |
| Mağaza Kodu | Temsilcinin kartı üzerindeki araç depo bilgisidir. |
| Satış Tip | Toptan-Parakende seçiminin yapıdığı yerdir. |
| Toplama Durumu | Mobilden girilen Fatura depocu terminaline düşüyor ve depocu tarafından yapılan işlemin sonucu ile ilgili bilgi bu alandan görülür. |
| İşlem Saati | İşlem yapıldığı saat bilgisidir. |
| İşlem Tarihi | İşlemin yapıldığı sistem saatidir. |
| Vade Günü | Belgenin vade günün girildiği alandır. |
| Vade Tarihi | Belgenin vade tarihinin olduğu alandır. |
| Sevk Tarihi | Belgenin vade tarihinin olduğu alandır. |
| Servis Tarihi | PanoramaCalldesk kullanılıyor ise servisin müşteriye gitme tarihinin girildiği alandır. |
| Müşteri Kodu | İşlem yapılacak müşterinin seçileceği alandır. |
| Ünvan | Seçilen müşterinin ünvan bilgisidir. |
| Adres | Seçilen müşterinin adres bilgisidir. |
| İl | Seçilen müşterinin il bilgisidir. |
| İlçe | Seçilen müşterinin ilçe bilgisidir. |
| Vergi Dairesi | Seçilen müşterinin vergi dairesi bilgisidir. |
| Sevk Adresi | Seçilen müşteri üzerinde sevk adresi var ise sevk adresinin seçildiği alandır. |
| TC Kimlik No | Seçilen müşteri gerçek kişi ise TC Kimlik No bilgisidir. |
| Vergi No | Seçilen müşteri tüzel kişi ise vergi no bilgisidir. |
| Vergi Tipi | Seçilen müşterinin vergi bilgisidir. |
| Telefon | Seçilen müşterinin telefon bilgisidir. |
| Cep Telefonu | Seçilen müşterinin cep telefonu bilgisidir. |
| Mail Adresi | Seçilen müşterinin mail adresidir. |
| Cinsiyet | EnrouteCalldesk kullanılıyor ise müşterinin cinsiyet bilgisidir. |
| Barkod | Ürünün barkod bilgisine göre girebildiğimiz alandır. |
| Ürün | Ürünü ürün rehberinden seçebildiğimiz alandır. |
| Ürün Adı | Ürünün adına göre girebildiğimiz alandır. |
| Seri No | Üründe seri takibi var ise seri bilgisinin girildiği yerdir. |
| Miktar | Girilen ürünün miktarının girildiği yerdir. |
| Birim | Ürünün hangi birimden satışı yapıldığının bilgisinin bulunduğu alandır. |
| Depo Miktarı | Seçilen üründe seçilen depodaki miktarıdır. |
| Fiyat | Seçilen ürünün kdvsiz birim fiyatıdır |
| KDV’li Br.F. | Seçilen ürünün kdv dahil birim fiyatıdır. |
| Brüt Tutar | Ürünün Fiyat*Miktar bilgisidir. |
| İndirim % | Temsilcinin serbest iskonto yetkisi var ise satır iskonto yüzde olarak girilebilen alandır. |
| İndirim TL | Temsilcinin serbest iskonto yetkisi var ise satır iskonto tutar olarak girilebilen alandır. |
| Tip | Müşteriden alınan tahsilatın tip seçiminin yapıldığı alandır. |
| Makbuz No | Alınan Tahsilatın fiziki numarasının girildiği yerdir. |
| Tutar | Alınan tahsilatın tutarı girilir. |
| Kom. Tutarı | Tip kredi kartı seçilir ise ve komison tutarı tanımlı ise komisyon tutar bilgisinin bulunduğu alandır. |
| Vade Tarihi | Girilen tahsilatın vade tarihi bilgisidir. |
| İşlem Tipi | Girilen tahsilatın hareket tipidir. |
| Banka | Tahsilat tipi banka bilgisi gerektirir ise seçim yapılan alandır. |
| Şube | Bankanın şube bilgisinin girildiği yerdir. |
| Çek No | Tahsilat tipi çek seçilir ise çek numarasının girildiği alandır. |
| Kom.Oranı | Komisyon oranının girildiği yerdir. |
| Taksit | Taksit bilgisinin girildiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yazdır | Girilen belgenin yazdırma işlemini yapar. |
| Kaydet | Girilen belge ve tahsilatın kaydetme işlemini yapar |
| Kaydet ve Yeni | Girilen belgeleri kayıt ettikten sonra yeni bir ekran açar |
| Belge Toplama | Belgedeki ürünlerin Depocuya gönderilip, depocu tarafından toplanılması gerekiyor ise belge toplama butonundan işlem yapılır. |
| Ürün Sorgulama | Ürünlerin ürün kodu, grup, ek grup(Marka /Mevsim ) , Marka/ Grup ve stok durumuna göre ürünlerin hızlı bir şekilde sorgulandığı ekrandır.(Bu ekranın ile ilgili bilgiye ürün Sorgulama Ekranındaki Yardım butonundan ulaşılabilir.) |
| Rezerv Depo Transfer | Rezerve depoya transfer yapılması sağlanır |
| Kapat | Açılan ekran kapanır |
