# Bayi Tanım

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım Kanalı → Tanımlamalar → Bayi Tanım`

## Ekran Tanımı

Bayi tanımlama ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Takip kodu bilgisidir. |
| Durum | Durum bilgisidir. “Aktif”, “Pasif” değerler alabilir. |
| Kod | Kod bilgisidir. |
| Ünvan | Bayi Ünvanıdır. |
| İlgili Kişi | İlgili kişi bilgisidir. |
| Kısa Adı | Kısa ad bilgisidir. |
| Telefon | Telefon bilgisidir. |
| Cep Tel No | Cep telefonu bilgisidir. |
| Mail Adresi | Mail adresi bilgisidir. |
| İl | İl plaka kodu bilgisidir. |
| İlçe Kod | İlçe bilgisidir. |
| Bölge Kodu | Bölge bilgisidir. |
| Müşteri | Müşteri bilgisidir. |
| Adres | Adres bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Kapat | Değişiklikleri kaydeder ve ekranı kapartır. |
| Kapat | Yapılan değişiklikleri kaydetmenden ekranı kapamaya çalışır. |
