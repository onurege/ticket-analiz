# Geç Sevk İptal Nedenleri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Tanımlamalar → Geç Sevk İptal Nedenleri`

## Ekran Tanımı

Geç sevk edilen ya da iptal edilen sipariş kalemleri için girilecek nedenlerin tanımlandığı ekrandır. Bu nedenler Sipariş Geç Sevk İptal Neden giriş ekranında kullanılır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | İlgili tanıma program tarafından verilen artışlı numaradır. |
| Distribütör Kodu | Tanımın hangi distribütöre ait olduğunu belirten distribütör kod bilgisidir. |
| Uygulama Yeri | Uygulama yeri bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemin kaydedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
