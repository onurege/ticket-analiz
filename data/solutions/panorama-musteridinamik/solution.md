# Müşteri Dinamik Grup

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteri → Tanımlamalar → Müşteri Dinamik Grup`

## Ekran Tanımı

Müşteri dinamik grup bilgilerinin tanımlandığı ekrandır. Öncelikle ekrana, sisteme girilmiş olan müşteri dinamik grubu listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Müşteri dinamik grubuna sistem tarafından verilen artışlı numaradır. |
| Distribütör Kodu | Müşteri dinamik grubu distribütör kodudur. Müşteri dinamik grubunun hangi distribütöre ait olduğu bu alan üzerinden izlenir. |
| Uygulama Yeri | Müşteri ek grubunun kullanılacağı modül bilgisinin tutulduğu alandır. Panorama / Quest / Panorama ve Quest / Münferit / Hepsi / Stokbar değerlerini alabilir. |
| Kriter Tipi | Dinamik grup, müşteri hangi kriteri için yapılacağı belirtilir. Kod / Grup / Ek Grup gibi değerler alabilir. |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Detayda yeni tanım yapmak için kullanılır. |
| İzle | Detaydaki kaydı görüntülemek için kullanılır. |
| Sil | Detaydaki kaydın silinmesi için kullanılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| T.Kod | Başlık bilgilerinden seçilen Kriter Tipi alanının içeriğine göre müşteri dinamik grubu detayında olacak müşteri, müşteri grupları veya müşteri ek gruplarının seçiminin yapılabileceği alandır. |
| Kaydet | Yapılan işlemin kaydedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
