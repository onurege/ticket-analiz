# St Depo Hareket İzin

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Tanımlamalar → ST Depo Hareket İzin`

## Ekran Tanımı

Araç tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Aracın hangi distribütöre ait olduğunu belirten, distribütör kod bilgisidir. |
| Satış Temsilcisi | Satış Temsilcisi kodudur. |
| ST Kodu | Listelenen Satış temsilcisinin kod bilgisidir. |
| ST/SE Adı | Listelenen ST/SE Adı bilgisidir. |
| Depo Kodu | Depo kodu bilgisidir. |
| Depo Adı | Depo adı bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Ekranda bulunan kayıtların yenilenmesini sağlayan butondur. |
| İşlemi Başlat | Gerekli bütün alanlar doldurulduktan sonra “İşlemi Başlat” butonuna tıklanır ve işlem gerçekleştirilir. |
| Listele | Kayıtları listelemeye yarar. |
| Temizle | Yapılan seçimleri temizlemeye yarar. |
