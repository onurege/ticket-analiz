# Hesap-Finans

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tanımlamalar → Hesap - Finans`

## Ekran Tanımı

Finans modülünde banka hesap bilgilerinin tanımlandığı ekrandır. Öncelikle ekrana sisteme girilmiş olanlar finans hesapları listelenir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Genel - Finans tanımına ait program tarafından verilen artışlı numaradır. |
| Durum | Tanımlanan hesaba ait durum bilgisidir. Aktif ve Pasif değerlerini alır. |
| Hesap Adı | Tanımlanan hesabın adının girildiği metin sahasıdır. |
| Distribütör Kodu | Tanımlanan hesabın bağlı olduğu distribütör kodudur |
| Banka Kodu | Hesabın bağlı olduğu banka bilgisidir ve rehber alandır. |
| Şube | Hesabın bağlı olduğu şube bilgisidir ve rehber alandır. |
| Hesap No | Hesap numarası alanıdır. |
| IBAN No | Hesaba ait IBAN numarası bilgisidir. |
| Banka Cari | Banka cari bilgisinin Borç, Alacak ve Bakiye rakamlarının gösterildiği alandır. Edit edilemez. |
| Takas Çek | Takas Çek bilgisinin Borç, Alacak ve Bakiye rakamlarının gösterildiği alandır. Edit edilemez. |
| Kesilen Çek | Kesilen Çek bilgisinin Borç, Alacak ve Bakiye rakamlarının gösterildiği alandır. Edit edilemez. |
| Takas Senet | Takas Senet bilgisinin Borç, Alacak ve Bakiye rakamlarının gösterildiği alandır. Edit edilemez. |
| Toplam | Tüm Borç, Alacak ve Bakiye alanlarının dip toplamlarının gösterildiği alandır. Edit edilemez. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Yeni | Yeni kayıt girmek için kullanılan butondur. |
| Düzenle | Mevcut kayıtlarda düzenleme amacıyla kullanılan butondur. |
| İzle | Mevcut kayıtlarda izleme yapabilmek için kullanılan butondur. |
| Kopyala | Listede seçili olan kaydı kopyalamak için kullanılan butondur. |
| Sil | İptali istenen kaydı silmek için kullanılan butondur. |
