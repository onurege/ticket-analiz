# Alternatif Ödeme Tipleri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriteri → Satış Kriterleri → Alternatif Ödeme Tipleri`

## Ekran Tanımı

Alternatif Ödeme Tipleri ekranırdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Ödeme Tipi | Ödeme Tipi bilgisidir. “Peşin”, “Çek”, “Senet”, “Kredi Kartı”, “Vadeli”, “Ticari Kart”, “DBS”, “Havale/EFT”, “Sanal Pos” değerleri alabilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemin kaydedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
| Alternatif Ödeme Tipi | Sadece el terminalinde ödeme seçeneği olması açısından belirlenen müşteriye belirlenen diğer ödeme tipleri seçeneklerine alternatif ödeme tipi denir. |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Detayda yeni tanım yapmak için kullanılır. |
| İzle | Detaydaki kaydı görüntülemek için kullanılır. |
| Sil | Detaydaki kaydın silinmesi için kullanılır. |
