# Raf Tanım

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Tanımlamalar → Raf Tanım`

## Ekran Tanımı

Raf tanımlamasının yapıldığı ekrandır. Öncelikle ekrana, sisteme girilmiş olan raf tanım listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Sistem tarafından takip edilen artışlı numara bilgisidir. |
| Durum | Raf tanımının durumunu belirtir. Aktif ve Pasif değerler alabilmektedir. |
| Distribütör Kodu | Raf tanımının hangi distribütör için yapıldığının bilgisidir. |
| Depo Kodu | Raf tanımının hangi depo için yapıldığı bilgisidir. |
| Raf Kodu | Rafa verilen kod bilgisinin girildiği alandır. |
| Koridor | Rafın bulunduğu koridor bilgisinin girildiği alandır. |
| Palet | Rafın palet bilgisinin girildiği alandır. |
| Raf No | Raf numarasının tanımlandığı alandır. |
| Kapasite | Raf kapasite bilgisinin girildiği alandır. |
| En | Rafın en bilgisinin girildiği alandır. |
| Boy | Rafın boy bilgisinin girildiği alandır. |
| Derinlik | Rafın derinlik bilgisinin girildiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Yapılan tanımı kaydeder ve Yeni tanım için ekranı açık bırakır. |
| Kaydet ve Kapat | Yapılan tanımı kaydeder ve ekranı kapatır. |
| Kapat | Ekranı kapatır. |
