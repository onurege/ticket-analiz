# Depo Giriş

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri → Stok Hareketleri → Depo Giriş`

## Ekran Tanımı

Depo’ya stok girişi yapılmasına yarar. Menüde ekrana tıklandığına ilk önce depo giriş kayıtlarının bulunduğu liste gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Belge Kodu | Program tarafından verilen artışlı numaradır. |
| İşlem Tarih | Depo giriş kaydının girileceği tarihtir. Varsayılan olarak günün tarihi gelir. Zorunlu alandır. |
| Distribütör Kodu | Depo giriş kaydının hangi distribütöre ait olduğu bu alan üzerinden izlenir.Zorunlu alandır. |
| Hareket Tip | Depo giriş kayıtları için tanımlanmış hareket tiplerinin seçildiği zorunlu bir alandır. |
| Giriş Depo | Ekran detayında girilen stok miktarlarının hangi deponun stok bakiyesine ekleneceğinin belirlendiği zorunlu bir alandır. |
| Depo Hareket Nedeni | Tanımlanmış olan depo hareket nedenlerinin seçildiği alandır. |
| Özel Kod | Kullanıcının özel olarak takip etmek istediği bir kod bilgisi girilebilir. |
| Matbu No | Depo giriş kaydı yazdırıldığında matbu numaranın otomatik yazıldığı bir alandır. İsteğe bağlı olarak manuel olarak ta değer girilebilir. |
| Ürün | Detayda eklenmiş olan ürüne ait kod bilgisidir. |
| Distribütör Kodu | Depo giriş kaydının hangi distribütöre ait olduğu bu alan üzerinden izlenir. Zorunlu alandır. |
| Depo Miktarı | Ürünün belge başlığında seçilen depodaki mevcut stok miktarıdır. Stok miktarı ana birim bazında görüntülenir ancak ekrandaki birim alanından 1.birim yerine 2.birim / 3.birim / 4.birim / 5.birim gibi birimler yapıldığında depo miktarı bu birime göre gösterilecektir. |
| Birim | Belge detayına eklenen ürünün hangi birimden eklenmiş olduğunu gösterir. |
| Miktar | Belge detayına eklenen ürünün miktar değerini gösterir. |
| Birim Fiyat | Belge detayına eklenen ürünün seçilen birim bazında fiyat bilgisini gösterir. |
| Net Tutar | Belge detayına eklenmiş ürünün birim fiyat ve miktarının çarpım değeridir. |
| Özel Kod | Kullanıcının özel olarak takip etmek istediği bir kod bilgisi girilebilir. |
| Tip | Belge detayına eklenen ürün satırının tipini belirtir. Depo giriş belgesinde yalnızca ürün “ U ” değerini alır. |
| Durum | Depo giriş kaydının durum bilgisini gösterir. Aktif ve Silindi değerlerini alır. |
| Aktarım Durumu | Depo giriş kaydının ticari sistemlere (Netsis, Logo, Eta vs..) aktarım durumunu gösterir. |
| Net Tutar | Depo giriş kaydının toplam tutarını gösterir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Belgeyi kaydetmek için kullanılır. |
| Kapat | Ekranı Kapatma işlemi yapar. |
| Yenile | Belge detay listesini yenilemek için kullanılır. |
| Yeni | Belge detaya yeni ürün eklemek için kullanılır. |
| Düzenle | Belge detay satırında düzenleme yapmak için kullanılır. |
| İzle | Detaya eklenen satırı izle modunda açar. |
| Sil | Detaya eklenen satırı silmek için kullanılır. |
| Hızlı Giriş | Belge detayına eklenecek ürünlerin tek bir ekrandan seçilip miktar ve birim değerlerinin hızlı bir şekilde girilmesi için Hızlı Giriş kullanılabilir. |
| Toplu Kod Girişi | Ürün barkod kodu, ürün kodu veya ağırlıklı ürün barkod girişi ile belge detayına ürün eklenebilir. |
