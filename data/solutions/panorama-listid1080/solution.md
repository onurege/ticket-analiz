# El Terminal

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi→ El Terminal İşlemleri → El Terminal`

## Ekran Tanımı

El terminallerinin Panorama üzerinden bilgi alis-verisi yapabilmesi için model, seri, device id, distribütör ve açiklama bilgilerinin tanimlandigi ekrandir. Öncelikle ekrana, sisteme

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Seri No | El terminalinin Seri No bilgisinin girilecegi alandir. Girilmesi zorunlu sahadır. |
| Distribütör Kodu | El terminalinin hangi distribütöre ait oldugu bilgisinin girilecegi alandir. |
| Model | El terminalinin Model bilgisidir. |
| Device ID | El terminalinin device ID bilgisinin girilecegi alandir. El terminali üzerine program kurulumu yapilarak program üzerinde Ek Islemler menüsünde yer alan Device ID bölümüne girilerek bu numara görülebilir. |
| Alım Tarihi | El terminalinin tanimlandigi tarihin tutuldugu alandir. |
| Kullanımda | El terminalinin kullanilip kullanilmadigi bilgisinin tutuldugu alandir. Kullanilmayan el terminalleri için bu alan isaretli olmamalidir. |
| IMEI No. | El terminalinin IMEI No bilgisinin girilecegi alandir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemlerin kayıt edilmesini sağlar. |
| İptal | Yapılan işlemlerin kayıt edilmeden, ekranın kapanmasını sağlar. |
