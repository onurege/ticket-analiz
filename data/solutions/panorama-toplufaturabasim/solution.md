# Toplu Fatura Basım

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Belge Basım İşlemleri →Toplu Fatura Basım`

## Ekran Tanımı

Toplu Fatura Basım Ekranından Tipi Fatura olan belgelerin toplu olarak yazdırma işlemi yapılıyor.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Matbu No | Belge basımı yapılacak faturanın matbu no değeri girilir ve detayda bu matbu numarasına ait fatura listelenir. |
| Distribütör Kodu | Basımı yapılacak belgelerin bağlı olduğu distribütör seçimi bu alandan yapılır. |
| İlk Belge No | Belge numarasına bağlı olarak Faturalar listelenecek ise, yani yazdırılacak faturaların belge numaraları sıralıysa ilk Faturanın belge no değeri bu alana yazılır |
| Son Belge No | Belge numarasına bağlı olarak Faturalar listelenecek ise, yani yazdırılacak faturaların belge numaraları sıralıysa son Faturanın belge no değeri bu alana yazılır. |
| Başlangıç Tarihi | Yazdırılacak belgelerin işlem tarihlerini kapsayan tarih aralığı için Başlangıç Tarih değeri bu alana yazılır. |
| Bitiş Tarihi | Yazdırılacak belgelerin işlem tarihlerini kapsayan tarih aralığı için Bitiş Tarih değeri bu alana yazılır. |
| Müşteri Kriter Tipi | Müşteri kriteri vererek Faturaların detayda listelenmesi ve buna bağlı olarak yazdırılma işlemi yapılacak ise Müşteri Kriter Tip alanından Kod, Grup Ek grup değerlerinden biri seçilerek Müşteri Kriteri rehberinde listelenecek değerler belirlenir. |
| Müşteri Kriter | Müşteri kriteri vererek Faturaların detayda listelenmesini istiyorsak ve buna bağlı olarak yazdırma işlemi yapılacak ise, Müşteri Kriter Tip alanında seçmiş olduğumuz değere göre Müşteri Kriter seçimi yapılır ve seçilen müşterilere ait faturalar detayda listelenir. |
| Satış Temsilcisi | Satış Temsilcisine bağlı Faturalar yazdırılacak ise; bu alandan Satış temsilcisi seçimi yapılarak detayda sadece seçilen satış temsilcisine ait faturaların listelendiğini görürüz. |
| Bölge | Belli bölgelerdeki müşterilerin Faturaları yazdırılacak ise, bu alandan Bölge Kodu seçilerek detayda seçilen bölgeye ait müşterilerin faturalarının listelendiğini görürüz. |
| Depo | Depo Kodu kriterine bağlı olarak Faturalar yazdırılacak ise, bu alandan Depo Kodu seçilerek detayda seçilen depolara ait faturaların listelendiğini görürüz. |
| Yükleme Kodu | Belli bir yükleme koduna ait Faturalar yazdırılacak ise; bu alandan yükleme kodu seçilerek detayda ilgili yüklemedeki faturalarının listelendiğini görürüz. |
| Araçlar | Araç Kodu kriterine bağlı olarak Faturalar yazdırılacak ise; bu alandan Araç Kodu seçilerek detayda seçilen araçlara ait faturaların listelendiğini görürüz. |
| Hareket Tipi | Hareket Tipi kriterine bağlı olarak Faturalar yazdırılacak ise; bu alandan Fatura hareket tipi seçilerek detayda seçilen hareket tiplerine ait faturaların listelendiğini görürüz. |
| Belge Tipi | Bu alanda Satış, Alış, Depozito İade, Satıcıya İade, Verilen Hizmet, Müşteriden İade Faturası(Bozuk) ve Müşteriden İade Faturası(Sağlam) belge tipleri listelenir ve yazdırılacak faturaların belge tipi bu alandan seçilir. |
| Satış Tip | Satış Tipi alanında Hepsi, Açık, Kapalı değerleri listelenir ve bu alandan seçmiş olduğumuz satış tipindeki faturalar detayda listelenir ve yazdırma işlemi yapılabilir. |
| Giriş Tip | Giriş Tipi alanında Hepsi, Manuel, El Terminali, Aktarım değerleri listelenir ve bu alandan seçmiş olduğumuz giriş tipindeki faturalar detayda listelenir ve yazdırma işlemi yapılabilir. |
| Ödeme Tipi | Ödeme Tipi alanında Hepsi, Peşin, Çek, Senet, Kredi Kartı, Açık Hesap, Ticari Kart, DBS, Havale/ EFT, Mail order değerleri listelenir ve bu alandan seçmiş olduğumuz ödeme tipindeki faturalar detayda listelenir ve yazdırma işlemi yapılabilir. |
| E-Belge Basım Tipi | Bu alanda yazdırılacak ve yazdırılmayacak değerleri listelenir. E- Faturaların yazdırılması isteniyor ise bu alandan Yazdırılacak değerinin seçilmesi gerekiyor ve buna bağlı olarak e- faturalar detayda listelenir ve yazdırılabilir. |
| Belge Basım Durumu | Bu alanda Yazdırılmadı, Yazdırıldı ve Hepsi seçenekleri listelenir. Daha önceden yazdırılmış faturaların tekrar toplu belge basımından yazdırılması isteniyor ise Yazdırıldı değeri seçilerek detayda listelenmesi sağlanır, Hepsi değeri seçilir ise daha önceden yazdırılan ve yazdırılmayan faturaların tümü diğer kriterlere bağlı olarak listelenir. Yazdırılmadı değerinin seçilmesi durumunda daha önceden yazdırılmayan faturalar yine diğer kriterlere bağlı olarak detayda listelenir. |
| Belge Kodu | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen faturaların Belge Kodu değerini gösterir. |
| Müşteri Kodu | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen faturaların Müşteri Kodu değerini gösterir. |
| Müşteri Adı | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen faturaların Müşteri Adı değerini gösterir. |
| ST Kodu | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen faturaların Satış Temsilcisi Kodunu gösterir. |
| Satış Temsilcisi | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen faturaların Satış Temsilcisi Adını gösterir. |
| Araç Kodu | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen faturaların Araç Kodunu gösterir. |
| Dağıtım Sıra | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen faturaların Dağıtım sırasıdır. |
| Plaka | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen faturaların Araç Plaka Kodunu gösterir. |
| İşlem Tarihi | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen faturaların İşlem Tarihini gösterir. |
| Net Tutar | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen faturaların Net Tutar değerini gösterir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Listele butonuna bastığımızda kriter alanına girilen kriterlere bağlı olarak faturalar detay alanında listelenir. |
| Temizle | Temizle butonuna bastığımızda kriter alanına girilen kriterler ve bunlara bağlı olarak detay alanında listelenen faturalar temizlenir. |
| Kapat | Fatura Toplu Basım ekranını kapatır. |
| Yenile | Detay listesinin yenilenmesini sağlar |
| Yazdır | Detay alanında listelenen ve daha sonra seçimi yapılan Faturaların Yazdırılması sağlanır. Yazdır butonuna bastığımızda Basım Ekranı açılır. |
| İzle | Detay alanında listelenen ve seçimi yapılan Fatura izle modunda açar. |
