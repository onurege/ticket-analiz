# Tercihli Ürün Satamama Nedeni

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri → Satış Kriterleri → Satamama Nedeni`

## Ekran Tanımı

El terminallerinde satış yapılamadığı durumlarda neden satış yapılamadığını göstermek için girilecek olan nedenin tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Tercihli Ürün Satamama Nedeninin kodunun girildiği mecburi alandır. |
| Ad | Tercihli Ürün Satamama Nedeninin Adının girildiği mecburi alandır |
| Distribütör Kodu | İşlem yapılacak Distribütörün seçildiği rehber alandır. |
| Uygulama Yeri | Tercihli Ürün Satamama nedeninin kullanılacağı yerin seçildiği kombo box alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Ekranda bilgileri girilmiş kullanıcıyı yarar. |
| İptal | Ekranı kaydetmeden Kapatma işlemi yapar. |
