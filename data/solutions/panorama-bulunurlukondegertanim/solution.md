# Bulunurluk Ön Değer Tanımlama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri → Anket ve Bulunurluk → Bulunurluk Ön Değer Tanımlama`

## Ekran Tanımı

Bulunurluk tanımları için ön değer tanımı yapılır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Durum | Durum bilgisidir. Aktif olan kayıtlar sahada uygulanır. |
| Distribütör Kodu | İlgili tanımın geçerli olacağı distribütör bilgisi seçilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Ürün Detayı | Ürün detay ekranı açılır. Ön değer için rehberden ürün seçimi yapılır. |
| Kaydet ve Yeni | Tanım kaydedilir yeni tanım için form açılır. |
| Kaydet ve Kapat | Tanım kaydedilir form kapanır. |
| Kapat | Form kaydedilmeden kapanır. |
| Seç ve Kapat | Ürün seçimi yapıldıktan sonra formun kapatılmasını sağlar. |
| Sadece Seçilenler | Sadece seçilen ürünler listlenir. |
| Hepsi | Bütün ürünler listelenir. |
| Kapat | Form kapanır. |
| Yenile | Sayfayı yeniler. |
| İzle | Seçilen ürün kaydı hakkında detaylı bilgi ekranı açılır. |
