# Satamama Nedeni

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri→ Satış Kriterleri→Satamama Nedeni`

## Ekran Tanımı

El terminallerinde satış yapılamadığı durumlarda neden satış yapılamadığını göstermek için girilecek olan nedenin tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Satamama Nedeninin kodunun girildiği mecburi alandır. |
| Ad | Satamama Nedeninin Adının girildiği mecburi alandır |
| Distribütör Kodu | İşlem yapılacak Distribütörün seçildiği rehber alandır. |
| Uygulama Yeri | Satamama nedeninin kullanılacağı yerin seçildiği kombo box alandır. |
| Bulunurluk Toplancak mı? | Eğer el termianlinde bu satamama nedeni seçilirse hemen arkasından bulunurluk toplama ekranının açılıp açılmayacağını belirten kombo box alandır. |
| Çağrı mrk. Satamama Nedeni | Çağrı merkesi satış ekranında kullanılacak ise seçilecek olan kutucuktur. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Ekranda bilgileri girilmiş kullanıcıyı yarar. |
| İptal | Ekranı kaydetmeden Kapatma işlemi yapar. |
