# Araç Cinsi

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Tanımlamalar → Araç Cinsi`

## Ekran Tanımı

Araç Cinsinin tanımlandığı alandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Araç Cins Kod | Araç Cinsi kodu bilgisidir. |
| Araç Cinsi | Araç Cinsi bilgisidir. |
| Araç Ağırlığı | Araç Ağırlığı bilgisidir. |
| Araç Hacmi | Araç Hacim bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Kapat | Yapılan işlemin kaydedilmesini ve kapatması sağanır. |
| Kapat | Ekranı kapatır. |
