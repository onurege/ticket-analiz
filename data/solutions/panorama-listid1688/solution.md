# Firma

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri → Kontrar İşlemleri → Ticari Pazarlama Firma`

## Ekran Tanımı

Ticari Pazarlama Firma tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Firma kayıtı kod bilgisidir. |
| Ad | Firma isim bilgisidir. |
| Kısa Adı | Firma kısa ad bilgisidir. |
| Durum | Durum bilgisidir. |
| Tip | Tip bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Ekranda bilgileri girilmiş kullanıcıyı yarar. |
| İptal | Ekranı kaydetmeden Kapatma işlemi yapar. |
