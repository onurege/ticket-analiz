# Anket Cevaplarını Silme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → İşlemler → Anket Cevaplarını Değiştirme`

## Ekran Tanımı

Anket cevaplarının değiştirilmesini sağlayan ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İlk İşlem Tarihi | Silinecek anketleri filtre etmek için ilk işlem tarihinin belirtildiği alandır. |
| Son İşlem Tarihi | Silinecek anketleri filtre etmek için son işlem tarihinin belirtildiği alandır. |
| Anket Tanım Kod | Silinecek ankete ait tanım kodunun rehberden seçildiği alandır. |
| Distribütör Kriter Tipi | Silicenek anketleri filtrelemek için distribütör kriter tipinin girildiği alandır. Kod / Grup değerlerini alabilir. |
| Distribütör Kriteri | Verilen kriter tipine göre distribütörlerin listeden seçildiği alandır. |
| Müşteri Kriter Tipi | Silinecek anketleri filtrelemek için müşteri kriter tipinin girildiği alandır. Kod / Grup / Ek Grup / Dinamik Grup değerlerini alabilir. |
| Müşteri Kriteri | Verilen Müşteri Kriter Tipine göre müşteri/müşterilerin listeden seçildiği alandır. |
| Toplu Ekle | Toplu olarak müşteri seçimi yapılacağında, müşteri kodu veya müşteri takip kodlarının toplu olarak girilerek işlem yapılabilen alandır. |
| Eklenemeyen | Toplu Ekle alanına girilen müşteri kodu yada müşteri takip kodu sistemde bulunamaz ise bu bulunamayan kodun gösterildiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Belirtilen kriterlere göre müşterilere ait anketlerin listelenmesi için kullanılır. |
| Temizle | Belirtilen kriterlerin ekrandan temizlenmesi için kullanılır. |
| Tarihçe | Daha önceki değişiklerin gösterilmesi için kullanılır. |
| Kapat | Ekranı kapatır. |
| Yenile | Ekranı yeniler. |
| \|< | Listede en başa geçmek için kullanılır. |
| >\| | Listede en son sayfaya geçmek için kullanılır. |
| Sil | Detayda seçilen anketin silinmesi için kullanılır. |
| Kapat | Ekranı kapatır. |
| Yenile | Listeyi yeniler. |
| \|< | Listede en başa geçmek için kullanılır. |
| >\| | Listede en sona geçmek için kullanılır. |
