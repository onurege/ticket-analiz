# Dağıtıcı

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Tanımlamalar → Dağıtıcı`

## Ekran Tanımı

Dağıtıcı tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Dağıtıcıya program tarafından verilen artışlı numaradır. |
| Kod | Tanımlanan dağıtıcı için kullanıcının verdiği koddur. |
| Distribütör Kodu | Dağıtıcının hangi distribütöre ait olduğunu belirten, distribütör kod bilgisidir. |
| Ad Soyad | Dağıtıcının isim soyadı bilgisidir. |
| Satış Temsilcisi | Dağıtıcıya tanımlanmış olan satış temsilcisi kodudur. |
| Müşteri Adı | Dağıtıcıya satış temsilcisi tanımlamaya ek olarak müşteri adı da tanımlanabilir. Müşteri Adı kriteri özel olarak istenilen müşteriye satış temsilcisi atanması bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemleri kaydetmek için kullanılan butondur. |
| İptal | Ekranı kaydetmeden kapatmaya yarayan butondur. |
