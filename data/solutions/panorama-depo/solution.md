# Depo

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri → Tanımlamalar → Depo`

## Ekran Tanımı

Depo bilgilerinin tanımlandığı ekrandır. Öncelikle ekrana, sisteme girmiş olan depo listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Program tarafından verilen artışlı numaradır. |
| Durum | Deponun durum bilgisidir. Aktif/Pasif değerlerini alır. |
| Ad | Depoya verilen isimdir. Girilmesi zorunlu sahadır. |
| Varsayılan | Tanımlanan deponun hareket kayıtlarında varsayılan değer olarak gelmesi isteniyor ise bu saha seçilmelidir. |
| Tür | Deponun, merkez, araç, bozuk iade, bölge, rezerve, emanet, lastik otel depo olduğu bilgisidir. |
| Uygulama Yeri | Uygulama yeri bilgisidir. Panorama/Quest/Panorama&Quest/Münferit/Hepsi/Stokbar değerlerini alabilir. |
| Distribütör Kodu | Depo distribütör kodudur. Deponun hangi distribütöre ait olduğu bu alan üzerinden izlenir. |
| Dist. Depo Kod | Ticari paket aktarımlarında tanımlanan depo kodunun ticari paket karşılığının yazılacağı alandır. Girilmesi zorunlu sahadır. |
| Adres | Deponun bulunduğu adres bilgisi girilir. |
| Adres 2 | Deponun bulunduğu adres bilgisinin devamı girilir. |
| Posta Kodu | Deponun bulunduğu adrese ait posta kodu bilgisi girilir. |
| İl Kod | Deponun bulunduğu adrese ait il kodu bilgisi seçilir. |
| İlçe Kod | Deponun bulunduğu adrese ait ilçe kodu bilgisi seçilir. |
| Depo Grup | Depo grup bilgisi seçilir. |
