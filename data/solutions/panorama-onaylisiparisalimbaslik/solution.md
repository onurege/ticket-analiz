# Onaylı Sipariş Alım

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Satış İşlemleri →Onaylı Sipariş Alım`

## Ekran Tanımı

Onaylı Sipariş Alım ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Onaylı Sipariş Alım kod bilgisidir. |
| Tarih | Sipariş tarih bilgisidir. |
| Dist. Kodu | Distribütör kodu bilgisidir. |
| Müşteri Kodu | Müşteri kodu bilgisidir. |
| Satış Temsilcisi | Satış temsilcisi bilgisidir. |
| İşlem Tipi | İşlem tipi bilgisidir. |
| Depo | Depo bilgisidir. |
| Fiyat Farkı İptal Neden | Fiyat farkı iptal neden bilgisi seçilir. |
| Stok Seviye İptal Neden | Stok seviye iptal nedenidir. |
| Son Fatura Sayısı | Son fatura sayısı nedenidir. |
| Sipariş Miktarı Katsayısı | Sipariş miktarı katsayısı bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İşlemi Başlat | İşlemi başlatmak için kullanılır. |
| Excel Import | Excel dosyasını belgeye yerleştirir. |
| Kapat | Onaylı sipariş ekranını kapatır. |
| Düzenle | Detay alanında listelenen ve seçimi yapılan onaylı siparişi düzenleme modunda açar. |
