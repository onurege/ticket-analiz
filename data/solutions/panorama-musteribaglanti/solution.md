# Müşteri Bağlantı

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → Parametreler → Müşteri Bağlantı`

## Ekran Tanımı

Müşteri ek saha bağlantısı için kullanılan ekrandır. İlk olarak sistemde tanımlı bağlantı tanımları listelenir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Ek saha takip kodu bilgisinin rehberden seçildiği alandır. |
| Seçenek | Ek saha seçeneğinin rehberden seçildiği alandır. |
| Bağlı Ek Saha Kodu | EK sahanın bağlı olacağı diğer ek sahanın belirtildiği alandır. |
| Bağlı Ek Saha Seçenek | Ek saha seçeneğinin bağlı olucağı diğer ek saha seçeneğinin belirtildiği alandır. |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Detayda yeni tanım yapılmasını sağlar. |
| Düzenle | Detaydaki tanımı düzenlemek için kullanılır. |
| İzle | Detaydaki tanımın detayını görüntülemek için kullanılır. |
| Sil | Detaydaki tanımı siler. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Tanımı kaydeder. |
| Kapat | Tanım ekranını kapatır. |
