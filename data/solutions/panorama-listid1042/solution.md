# Matbu No

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Tanımlamalar`

## Ekran Tanımı

Belgeler için matbu numaraların tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Matbu No tanımına sistem tarafından otomatik olarak verilen numaradır. |
| Matbu Tipi | Matbu No tanımının hangi belge türü için yapılacağının seçildiği alandır. |
| Distribütör Kodu | Matbu No tanımının hangi distribütör için yapılacağının seçildiği alandır. |
| Dizayn Kodu | Matbu No tanımının hangi dizayna bağlanacağının seçildiği alandır. |
| Fiyatsız Dizayn | Matbu No tanımının hangi fiyatsız dizayna bağlanacağının seçildiği alandır. Fiyatsız dizayn alanı sadece Matbu Tipi “İrsaliye” seçildiği durumlarda kullanılabilir. |
| Seri No | Matbu belgenin seri numarasının yazıldığı alandır. |
| Son No | Matbu belgenin son seri numarasının yazıldığı alandır. |
| Min Uzunluk | Matbu numaranın en az kaç karakter olması gerektiğinin belirlendiği alandır. |
| Max Uzunluk | Matbu numaranın en fazla kaç karakter olması gerektiğinin belirlendiği alandır. |
| Kopya Sayısı | Matbu belgenin yazdırılma sırasında kaç kopya yazdırılacağının belirlendiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Yeni | Yeni matbu numarası tanımlamak için kullanılan butondur. |
| Düzenle | Önceden tanımlanmış matbu numaralarını düzenlemek için kullanılan butondur. |
| İzle | Önceden tanımlanmış matbu numaralarını izlemek için kullanılan butondur. |
| Kopyala | Önceden tanımlanmış matbu numaralarını kopyalamak için kullanılan butondur. |
| Sil | Önceden tanımlanmış matbu numaralarını silmek için kullanılan butondur. |
