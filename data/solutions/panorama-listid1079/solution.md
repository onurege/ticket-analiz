# Model

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → El Terminali → Model`

## Ekran Tanımı

El terminali Model ve device id bilgilerinin tanımlandığı alandır. El bilgisayarının güncel rut, ürün, fiyat bilgilerini alabilmesi için yapılan tanımlardır. Öncelikle ekrana, sisteme tanımlı olan El terminali listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Model kod bilgisidir. |
| Ad | Model isim bilgisidir. |
| Kamera Uyg. | Kamera Uygulaması bilgisidir. |
| Tip | Cihazın işletim sistemi bilgisidir. |
| Platform Kodu | Platform kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemleri kaydetmek için kullanılan butondur. |
| İptal | Ekranı kaydetmeden kapatmaya yarayan butondur. |
