# Sipariş Başlık Düzeltme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Sistem Yönetimi 🡪 Ek İşlemler 🡪 Sipariş Başlık Düzeltme`

## Ekran Tanımı

Siparişlerin Başlık Bilgilerinin düzenlendiği ekrandır. Öncelikle ekrana, sisteme girilmiş olan Belge Başlık düzeltme sipariş gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Distribütör kod bilgisinin görüldüğü alan |
| Müşteri Kodu | Müşteri kodunun görüldüğü alan |
| İşlem Tarihi | Belge İşlem tarihinin görüldüğü alan |
| Belge Kodu | Belgenin kod bilgisinin görüldüğü alan |
| Vade Günü | Belge üzerinde bulunan vade gününün görüldüğü alan |
| Matbu No | Belge matbu Numarası değiştirmek isteniyor ise bu alana ilgili belge numarasının yazılması gerekmektedir. |
| Özel Kod | Belge Özel Kodu değiştirilmek isteniyor ise bu alana belge özel kodunun yazılması gerekmektedir. |
| Sevk Tarihi | Belgenin Sevk tarihi değiştirilmek isteniyor ise tarih girişi yapılması gereken alandır. |
| Ödeme Tipi | Ödeme tipi değiştirilmek isteniyor ise, listeden seçim yapılarak değiştirme yapılabilir. |
| Sipariş Mail Durumu | Sipariş mail durumu gönderildi veya gönderilmedi olarak seçim yapılarak değiştirilebilir. |
