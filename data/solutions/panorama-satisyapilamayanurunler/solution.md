# Satış Yapılamayan Ürünler

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Ürün Yönetimi → Satış Yapılamayan Ürünler`

## Ekran Tanımı

Satışı yapılamayan ürünlerin tanımlandığı ekrandır. Öncelikle ekrana, sisteme girilmiş olan satışı yapılamayanların listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Durum | Tanımın durum bilgisidir. Aktif/Pasif değerlerini alır. |
| Başlangıç Tarihi | Tanımın hangi tarihten itibaren geçerli olucağının bilgisinin girildiği alandır. |
| Bitiş Tarihi | Tanımın hangi tarihte son bulacağı bilgisinin girildiği alandır. |
| Distribütör Kodu | Tanımın hangi distribütör için geçerli olucağının belirtildiği alandır. |
| Müşteri Kriter Tipi | Tanımın hangi müşteri için geçerli olucağını belirlemek için kullanılır. Kod / Grup / Ek Grup / Ek Saha / Dinamik Grup değerlerini alabilir. |
| Müşteri Kriter | Kriter tipinde yapılan seçime göre değerlerin seçildiği alandır. |
| Rut Kriter | Rut kriterinin belirtildiği alandır. Rut İçi / Rut Dışı / Hepsi değerlerini alabilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Ürün Detayı | Satışı yapılamayan ürün yada ürünlerin seçilmesi için ürün ekranını açar. |
| Kaydet ve Yeni | Tanımı kaydeder ve yeni tanım için boş form açar. |
| Kaydet ve Kapat | Tanımı kaydeder ve ekranı kapatır. |
| Kapat | Ekranı kapatır. |
| Seç ve Kapat | Seçilen kayıtları kaydeder ve ekranı kapatır. |
| Sadece Seçilenler | Listede sadece seçilenleri gösterir. |
| Hepsi | Listede bütün kayıtları gösterir. |
| Kapat | Ekranı kapatır. |
| Yenile | Listeyi yeniler. |
| İzle | Ürün detaylarını görüntüler. |
| \|< | Listede en baş sayfaya gidilmesini sağlar. |
| \|> | Listede en son sayfaya gidilmesini sağlar. |
