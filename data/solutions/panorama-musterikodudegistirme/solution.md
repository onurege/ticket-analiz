# Müşteri Kodu Değiştirme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → İşlemler → Müşteri Kodu Değiştirme`

## Ekran Tanımı

Müşteri kodu değiştirme ekrandır.

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Verilen kriter tipine göre distribütörlerin listeden seçildiği alandır. |
| İlk Müşteri | İlk Müşteri Kodunun seçildiği ekrandır. |
| Son Müşteri | Son Müşteri Kodunun seçildiği ekrandır. Tek bir müşteri seçilmek isteniyor ise İlk Müşteri ve Son Müşteriye aynı kod girilmelidir. |
| Grup Kodu | Müşteri Grubunun seçildiği ekrandır. |
| Ek Grup | Müşteri Ek Grubunun seçildiği ekrandır. |
| Dinamik Grup | Müşteri Dinamik Grubunun seçildiği ekrandır. |
| İptal/Pasif Müşteriler Listelensin | İptal/Pasif Müşteriler listelensin istenirse seçilebilir. |
| İşlem Başlat | İşlem başlatmak için kullanılır |
| Listele | Girilen değerlere göre listeleme işlemi gerçekleştirir. |
| Kapat | Ekranı kapatır. |
| Yenile | Ekranı yeniler. |
| Düzenle | Seçilen belgeyi düzenlemek için kullanılır. |
