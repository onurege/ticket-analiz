# İptal/ Pasif Nedeni Ek Saha Bağlantı

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım Kanalı → Parametreler → İptal/Pasif Nedeni Ek Saha Bağlantı`

## Ekran Tanımı

İptal/Pasif nedeni ek saha bağlantı tanımının yapıldığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Kod bilgisidir. |
| Distribütör Kriter Tipi | Burada seçilen değere göre distribütör kriterde distribütör listesi gelir. Kod / Grup değerlerini alabilir. |
| Distribütör Kriteri | Distribütör kriter tipi olarak kod seçilmiş ise bu alanda distribütör listesinden distribütör seçimi yapılır. Eğer ki distribütör kriter tipi Grup seçilmiş ise distribütör grup listesinden grup seçimi yapılır. |
| Pasif Neden Kod | Pasif neden kod bilgisidir. |
| Ek Saha Kriter | Ek saha kriter bilgisidir. |
| Ek Saha Kriteri | Ek saha kriteri bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Yapılan tanımı kaydeder ve yeni tanım ekranını getirir. |
| Kaydet ve Kapat | Yapılan tanımı kaydeder ve ekranı kapatır. |
| Kapat | Ekranı kapatır. |
