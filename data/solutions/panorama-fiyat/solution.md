# Fiyat Tanımlama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri → Fiyat İşlemleri → Fiyat Tanımlama`

## Ekran Tanımı

Ürün fiyatlarının tanımlanabileceği ekrandır. Menüde ekrana tıklandığına ilk önce fiyat kayıtlarının bulunduğu liste gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Durum | Fiyat tanımının durum bilgisini gösterir. Aktif ve Pasif değerlerini alır. |
| Başlangıç Tarihi | Fiyat tanımın geçerli olmaya başlayacağı tarihin girildiği zorunlu bir alandır. |
| Bitiş Tarihi | Fiyat tanımın geçerlilik süresinin dolacağı tarihin girildiği zorunlu bir alandır. |
| Uygulama Yeri | Ürünün uygulama yerinin belirtildiği alandır. Panorama / Quest / Panorama ve Quest /Münferit / Hepsi / Stokbar değerlerini alabilir. |
| Distribütör Kriter Tipi | Tanımın Hangi Distribütör için geçerli olacağını belirlemek için kullanılır. Kriter tipi Kod, Grup, Ek Grup, Dinamik Grup değerleri alır. |
| Distribütör Kriteri | Kriter tipinde yapılan seçime göre değerlerin seçildiği alandır. Örneğin Kriter tipi kod seçildi ise Distribütör kodu listesi üzerinden distribütör seçimi yapılır. Boş bırakıldığı durumda tüm distribütörler için geçerli olacaktır. Fiyat sadece seçili distribütörler için uygulanır. |
| Müşteri Kriter Tipi | Tanımın Hangi Müşteri için geçerli olacağını belirlemek için kullanılır. Kriter tipi Kod, Grup, Ek Grup, Dinamik Grup, Bölge, Satış Temsilcisi, Ek Saha değerlerini alır. |
| Müşteri Kriteri | Müşteri Kriter Tipi ne göre değişkenlik gösteren bir alandır. Örneğin Müşteri Kriter Tipi “Kod” seçilirse sisteme kayıtlı tüm Müşteriler gelir ve tek bir Müşteri seçilebilir fakat Müşteri Kriter Tipi “Grup” seçilir ise Müşteri grupları seçilir ve o gruba dahil olan tüm müşterilere fiyat tanımlanmış olur. |
| Ek Saha Kriteri | Müşteri kriter tipinde Ek Saha seçimi yapıldığında devreye giren alandır. Ek saha kriterinin seçimi için kullanılır. |
| Ürün | Fiyat Tanımının hangi ürün için yapılacağı seçilir. Rehbere dolan ürün listesinden bir ürün seçilir. |
| Para Birimi | Fiyat tanımın hangi para cinsinden yapılacağı seçilir. Rehberde yer alan para birimlerinden biri seçilir. |
| Yeni Fiyat | Satış belgelerinde geçerli olacak fiyat bilgisinin girildiği zorunlu bir alandır. |
| Eski Fiyat | Eski fiyatın girilebileceği alandır. |
| Alış Fiyatı | Alış belgelerinde geçerli olacak fiyat bilgisinin girildiği alandır. |
| Bozuk İade | Bozuk iade belgelerinde geçerli olacak fiyat bilgisinin girildiği alandır. |
| Sağlam İade | Sağlam iade belgelerind geçerli olacak fiyat bilgisinin girildiği alandır. |
| Tavsiye Fiyat | Tavsiye fiyat bilgisinin girildiği alandır. |
| ÖTV/ÖİV Tip | Fiyat tanımına eklenecek Ötv veya Öiv’nin tipinin belirlendiği alandır. Tutar ve Yüzde değerleri seçilebilir. |
| ÖTV/ÖİV Değeri | ÖTV/ÖİV Tipi alanındaki değere göre fiyata eklenecek ÖTV/ÖİV yüzdesini veya tutarının girildiği alandır. |
| Ödeme Planı | Tanımlanan fiyatın hangi ödeme planı için geçerli olacağının seçildiği alandır. Rehberde ödeme planlarından biri seçilebilir. |
| Satış Kdv Oranı | Satış fiyatının üstüne eklenecek kdv’nin oran bilgisinin girildiği alandır. Fiyat tanımı üzerinden girilmezse ürün tanımı üzerindeki kdv oranı geçerlidir. |
| Alış Kdv Oranı | Satış fiyatının üstüne eklenecek kdv’nin oran bilgisinin girildiği alandır. Fiyat tanımı üzerinden girilmezse ürün tanımı üzerindeki kdv oranı geçerlidir. |
| KDV/ÖİV Dahil | Tanımlanan fiyat bilgilerinin içerisinde Kdv ve Öiv tutarlarının da yer aldığını belirtmek için kullanılır. |
| Geri Kazanım Değeri | Ürünün geri dönüştürükme maliyetinin girildiği alandır. |
| Satış | Girilen ÖTV/ÖİV değerlerinin satış belgelerinde geçerli olacağını belirtir. |
| Alış | Girilen ÖTV/ÖİV değerlerinin alış belgelerinde geçerli olacağını belirtir. |
| Satıcıya İade | Girilen ÖTV/ÖİV değerlerinin satıcıya iade belgelerinde geçerli olacağını belirtir. |
| İade | Girilen ÖTV/ÖİV değerlerinin müşteriden iade belgelerinde geçerli olacağını belirtir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Fiyatı kaydetmek ve boş bir fiyat ekranı açmak için kullanılır. |
| Kaydet ve Kapat | Fiyatı kaydetme ve ekranı kapatma işlemi yapar. |
| Kapat | Ekranı kapatma işlemi yapar. |
