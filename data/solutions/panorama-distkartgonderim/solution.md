# Dist. Kart Gönderim

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri → Kart Takip → Dist`

## Ekran Tanımı

Kart gönderim ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Kart talebi belge kod bilgisidir. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| Gönderim Durum | Gönderim durum bilgisidir. |
| Talep Kodu | Talep kodu bilgisidir. |
| Talep Tip | Talep tipi bilgisidir. |
| Talep Miktarı | Talep miktarı bilgisidir. |
| Gönderim Miktarı | Gönderim miktarı bilgisidir. |
| Gönderim Tarihi | Gönderim tarihi bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Ekranda bilgileri girilmiş kullanıcıyı yarar. |
| İptal | Ekranı kaydetmeden Kapatma işlemi yapar. |
| Yenile | Detay alanı bilgilerini yenilemek için kullanılır. |
| Yeni | Yeni kayıt eklemek için kullanılır. |
| İzle | Detay ekranında bulunan kayıtları incelemek için kullanılır. |
| Sil | Detay ekranında seçili kayıtı silmek için kullanılır. |
