# Ürün Dinamik Grup

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Tanımlamalar → Ürün Dinamik Grup`

## Ekran Tanımı

Ürün dinamik grup bilgilerinin tanımlandığı ekrandır. Öncelikle ekrana, sisteme girilmiş olan ürün dinamik grubu listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Ürün dinamik grubuna sistem tarafından verilen artışlı numaradır. |
| Distribütör Kodu | Ürün dinamik grubu distribütör kodudur. Ürün dinamik grubunun hangi distribütöre ait olduğu bu alan üzerinden izlenir. |
| Uygulama Yeri | Ürün ek grubunun kullanılacağı modül bilgisinin tutulduğu alandır. Panorama / Quest / Panorama ve Quest / Münferit / Hepsi / Stokbar değerlerini alabilir. |
| Kriter Tipi | Dinamik grup, ürünün hangi kriteri için yapılacağı belirtilir. Kod / Grup / Ek Grup gibi değerler alabilir. |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Detayda yeni tanım yapmak için kullanılır. |
| İzle | Detaydaki kaydı görüntülemek için kullanılır. |
| Sil | Detaydaki kaydın silinmesi için kullanılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Başlık bilgilerinden seçilen Kriter Tipi alanının içeriğine göre ürün dinamik grubu detayında olacak ürün, ürün grupları veya ürün ek gruplarının seçiminin yapılabileceği alandır. |
| Kaydet | Yapılan işlemin kaydedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
