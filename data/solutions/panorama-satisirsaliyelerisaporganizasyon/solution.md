# Satış Irsaliyeleri SAP Organizasyonu

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → İşlemler → Satış İrsaliyeleri SAP Entegrasyonu`

## Ekran Tanımı

Satış irsaliyesi merkez entegrasyon ile SAP’ye aktarım arayüzüdür.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Başlangıç Tarihi | Aktarım yapılacak başlangıç tarihi seçimi yapılır. |
| Bitiş Tarihi | Aktarım yapılacak bitiş tarihi seçimi yapılır. |
| Distribütör Kodu | Aktarım yapılacak distribütör seçimi yapılır. |
| Key Account Grup Kod | Aktarım yapılacak grup kodu seçimi yapılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Aktar | Seçilen kriterlere göre kayıtların aktarılmasını sağlar. |
| Kapat | Formun kapatılmasını sağlar. |
