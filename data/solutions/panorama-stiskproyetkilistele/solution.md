# ST IskPro Yetki Listele

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Parametreler → St IskPro Yetki Liste`

## Ekran Tanımı

Satış temsilcisi iskonto promosyon yetki listeleme ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yetki Tipi | Panorama ’da girilmiş olan yetki tipinin seçildiği alandır. |
| Başlangıç Tarihi | Listelenecek Entegrasyon sonuçlarının başlangıç tarihinin girildiği alandır. |
| Bitiş Tarihi | Listelenecek Entegrasyon sonuçlarının bitiş tarihinin girildiği alandır. |
| Distribütör Kriter Tipi | Kayıları Listelenecek Distribütörün kriterini seçildiği alandır. |
| Distribütör Kriteri | Kayıtları listelenecek Distribütörün belirtildiği alandır. |
| Satış Temsilcisi Adı | Hangi Satış Temsilcisine ait bilgilerin listeleneceği bilgisidir. |
| Müşteri Kriter Tipi | Müşteri kriterinin hangi tipte olduğu bilgisidir. Kod, Grup, Ek Grup Dinamik Grup seçilebilir. |
| Müşteri | Müşteri bilgisidir. |
| Ürün Kriter Tipi | Ürünlerin seçileceği kriterlerin girildiği alandır değer olarak (Kod, Grup, Ek Grup, Dinamik Gurup) değerlerini almaktadır. |
| Ürün | Ürün kod bilgisidir. |
| Sıra | St IskPro yetki sırası bilgisidir. |
| Yetki Kodu | Yetki kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Seçilen kriterlere uyumlu olan bilgileri listelemeye yarar. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
