# Promosyon Bütçe Tanımı

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ve Pazarlama Kriterleri → İskonto Promosyon → Promosyon Bütçe Tanımı`

## Ekran Tanımı

Serbest promosyon bütçe tanımı yapıldığı ekrandır. Bu ekrandan girilen verilere göre belgede ilgili müşteri için serbest promosyon limiti uygulanır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Durum | Tanımın durum bilgisidir. Aktif olan kayıtlar geçerlidir. |
| Başlangıç Tarihi | Bütçe tanımının geçerli olduğu başlangıç tarihidir. |
| Bitiş Tarihi | Bütçe tanımının sonlandığı tarih bilgisidir. |
| Distribütör Kodu | Bütçe tanımının yapılacağı Distribütör seçilir. |
| Müşteri Kriter Tipi | Bütçe tanımının yapılacağı müşteri kriter tipi seçilir. Kod,Grup,Ek Grup,Dinamik Grup,Satış Temsilcisi değerlerini alabilir. |
| Müşteri Kriteri | Seçilen müşteri kriter tipine göre müşteri kriteri seçimi yapılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Kapat | Girilen değerleri kaydeder. |
| Listele | Seçilen müşteri kriterine göre listeleme yapar, bu şekilde bütçe değerleri girilebilir. |
| Temizle | Ekranda seçilen kriterleri temizler. |
| Kapat | Ekranı kaydetmeden kapatır. |
| Düzenle | Listelenen müşteri kaydı seçilerek Düzenle butonu ile bütçe değeri girilir. |
| İzle | Listelenen müşteri kaydını izle modunda açar. |
| İlk Sayfa | Listeleme yapılan sayfada ilk sayfaya gider. |
| Önceki Sayfa | Listeleme yapılan sayfada bir önceki sayfaya gider. |
| Sonraki Sayfa | Listeleme yapılan sayfada bir sonraki sayfaya gider. |
| Son Sayfa | Listeleme yapılan sayfada son sayfaya gider. |
