# Satılabilir Ürün

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Ürün Yönetimi → Satılabilir Ürün`

## Ekran Tanımı

Belirtilen kriterlere ait belgedeki ürünlerin satılabilir tanımlarının yapıldığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Satılabilir ürün tanımına sistem tarafından verilen artışlı numaradır. |
| Durum | Satılabilir ürün tanımının durum bilgisidir. Aktif / Pasif değerlerini alabilirler. |
| Uygulama Yeri | Hangi modül için kullanılacağı bilgisidir. Panorama / Quest / Panorama ve Quest / Münferit / Hepsi / Stokbar / VerSiparişi |
| Başlangıç Tarihi | Tanımlanan satılabilir ürün ayarlarının hangi tarihten itibaren geçerli olucağı bilgisinin girildiği alandır. |
| Bitiş Tarihi | Tanımlanan satılabilir ürün ayarlarının hangi tarihte biteceği bilgisinin girildiği alandır. |
| Tür | Satılabilir ürün tanımının hangi tür belgelerde kullanılacağının tanımlandığı alandır. Satış / Alış / Müşteriden İade Sağlam / Satıcıya İade / Müşteriden İade Bozuk / Konsinye Satış / Konsinye Alış / Manuel Araç Yükleme / Araç Yükleme Talep / Araç Yükleme Talep (Tüccar) / Depo Giriş / Depo Çıkış / Depolar Arası Transfer / Hepsi |
| Ürün Kriter Tipi | Ürün kriterinin belirtildiği alandır. Kod / Grup / Hiyerarşi 1 değerlerini alabilir. |
| Hareket Tip | Satılabilir ürünün önceden tanımlanmış olan hangi hareket tipi için uygulanacağı seçilir. |
| Distribütör Kriter Tipi | Tanımlanan mesaj ayarları sadece seçilen distibütör kriter tipine uygulanır. Bu değere göre distribütör kriterde distribütör listesi gelir. Kod / Grup değerlerini alabilir. |
| Distribütör Kriteri | Distribütör kriter tipi olarak kod seçilmiş ise bu alanda distribütör listesinden distribütör seçimi yapılır. Eğer distribütör kriter tipi grup seçildiyse distribütör grup listesinden distribütör seçimi yapılır. |
| ST Kriter Tip | Bu alanda seçilen değere göre Satış Temsilcisinde satış temsilcisi listelenir. Kod / Grup değerlerini alabilir. |
| Satış Temsilcisi | Satış Temsilcisi tipi olarak kod seçilmiş ise bu alanda satış temsilcisi listesinden satış temsilcisi seçimi yapılır. Eğer satış temsilcisi kriter tipi grup seçildiyse satış temsilcisi grup listesinden satış temsilcisi seçimi yapılır. |
| Müşteri Kriter Tipi | Bu alanda seçilen değere göre Müşteri Kriterde müşteriler listelenir. Kod / Grup / Ek Grup / Ek Saha / Dinamik Grup değerlerini alabilir. |
| Müşteri Kriter | Müşteri Kriter Tipinde seçilen değere göre Müşteriler listelendiği ve tanım yapıldığı alandır. |
| Ek Saha Değer | Müşteri Kriter Tipi Ek Saha olduğunda aktif olmaktadır. Müşteri Kriter Tipi Ek Saha ve Müşteri Kriteri de buna bağlı tanımlardan birisi seçildiğinde ek saha seçiminin yapıldığı alandır. |
| Uygulama Kriterleri Kullanılsın | Uygulama kriterlerini aktif hale getirmek için kullanılır. Bu alan aktif edildiğinde diğer kriterler pasif hale geçicektir. |
| Uygulama Kriterleri | Uygulama kriterleri aktif hale getirildiğinde Satılabilir Ürün Uygulama Kriter tanımlarından seçim yapılabildiği alandır. |
| Yönetim HY-1 | Yönetim Hiyerarşisi 1 listesinden seçim yapılır. |
| Yönetim HY-2 | Yönetim Hiyerarşisi 2 listesinden seçim yapılır. |
| Ekle | Satılabilir ürün tanımının yasaklanacağı yerlerin listelenmesi sağlanır. |
| Sil | Satılabilir ürün tanımının yasaklanacağı yerlerin listeden silinmesini sağlanır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Ürün Detayı | Hangi ürünlerin satılabilir ürün olucağının seçiminin yapıldığı ekranı açar. |
| Kaydet ve Yeni | Tanımı kaydeder ve yeni tanım için ekrana boş form getirir. |
| Kaydet ve Kapat | Tanımı kaydeder ve ekranı kapatır. |
| Kapat | Ekranı kapatır. |
| Seç ve Kapat | Seçilen kayıtları kaydeder ve ekranı kapatır. |
| Sadece Seçilenler | Listede sadece seçilenleri gösterir. |
| Hepsi | Listede bütün kayıtları gösterir. |
| Kapat | Ekranı kapatır. |
| Yenile | Listeyi yeniler. |
| İzle | Ürün detaylarını görüntüler. |
| \|< | Listede en baş sayfaya gidilmesini sağlar. |
| \|> | Listede en son sayfaya gidilmesini sağlar. |
