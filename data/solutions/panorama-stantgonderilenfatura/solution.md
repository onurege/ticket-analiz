# Stant Fatura Gönderme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri 🡪 Fon ve Bütçe Yönetimi 🡪 İşlemler 🡪 Stant Fatura Gönderme`

## Ekran Tanımı

Stant faturası gönderme ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Sistem tarafından takip edilen artışlı numara bilgisidir. |
| Durum | Durum bilgisidir. |
| Gönderilme Statü | Gönderilme statü bilgisidir. “Yeni” veya “Gönderildi” değerlerini almaktadır. |
| Matbu No | Matbu no bilgisidir. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| Tutar | Tutar bilgisidir. |
| Fatura Tarihi | Fatura tarihi bilgisidir. |
| Gönderilme Tarihi | Gönderilme tarihi bilgisidir. |
| Not | Not bilgisidir. |
| Müşteri Kodu | Müşteri kodu bilgisidir. |
| Tarih Aralığı | Tarih aralığı bilgisidir. |
| Fatura Seçimi | Fatura seçimi bilgisidir. |
| Onay | Onay durumu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Listeyi yenilemeye yarar. |
| İzle | Listedeki değerleri izlemek için kullanılır. |
| Sil | Listede seçili belgeyi silmek için kullanılır. |
| Toplu Ekleme | Toplu detay eklemek için kullanılır. |
| Kaydet ve Yeni | Belgeyi kaydeder ve yeni belge oluşturur. |
| Kaydet ve Kapat | Belgeyi kaydeder ve kapatır. |
| Kapat | Ekranı kaydetmeden Kapatma işlemi yapar. |
| Fatura Bas | Fatura çıktısı almak için kullanılır. |
