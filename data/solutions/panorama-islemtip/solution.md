# Hareket Tip

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Sistem Yönetimi 🡪 Genel Tanımlamalar 🡪 Hakaret Tip`

## Ekran Tanımı

Hareket Tip bilgilerinin tanımlandığı ekrandır. Temel tanımlama formudur. Farklı belge tipleri için Hareket Tipi tanımlayacağımız ekrandır. İşlem Grup Kodu ve Açıklama bilgileri girilmeden kayıt yapılamaz.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Hareket Tip için tanımlanan numaradır. |
| Tip | Hangi belge için kullanılmak isteniyorsa, belge tipinin seçildiği alandır. |
| Ödeme Tipi | Ödeme Tipinin seçildiği alandır. |
| Kriter Tipi | Kod veya Grup seçiminin yapıldığı alandır. |
| Maksimum Ara Toplam |  |
| Vade | Vade günün yazıldığı alandır. Parametreye bağlı olarak istenirse belge üzerinde bulunan vade günü buradan set edilebilir. |
| Özel Ürün Kod | Özel ürün için tanımlanan hareket tiplerinde, ürünün seçildiği alandır. |
| Sipariş Mail Adresi | Mail adresinin yasıldığı alandır. |
| Sıralama Değeri |  |
| El Terminaline gönderilmesin | El terminalinde kullanılmak istenmiyorsa işaretlenir. |
| Sıralama Değeri Değiştirilemesin | Hareket tipte yazılmış olan Vade, Belge üzerinde değiştirilmesi istenmiyorsa bu kutucuk işaretlenir. |
| Kriter Kodu | Tanımlanan Hareket Tip in hangi Distribütör için kullanılacağının seçimi yapıldığı alan. |
