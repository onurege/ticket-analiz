# Ürün Grup

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Tanımlamalar → Ürün Grup`

## Ekran Tanımı

Ürün grup bilgilerinin tanımlandığı ekrandır. Öncelikle ekrana, sisteme girilmiş olan ürün grubu listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Ürün grubuna verilen artışlı numaradır. |
| Ad | Ürün grubuna verilen isimdir. |
| Kısa Adı | Ürün grubuna verilen kısa isimdir. |
| Distribütör Kodu | Ürün grubu distribütör kodudur. Ürün grubunun hangi distribütöre ait olduğu bu alan üzerinden izlenir. |
| Uygulama Yeri | Ürün grubunun kullanılacağı modül bilgisinin tutulduğu alandır. Panorama / Quest / Panorama ve Quest / Münferit / Hepsi / Stokbar değerlerini alabilir. |
| Katsayı | Sözleşme tanımında kullanılan katsayıdır. |
