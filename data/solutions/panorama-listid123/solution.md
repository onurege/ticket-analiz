# Müşteri Ek Grubu

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → Tanımlamalar → Müşteri Ek Grup`

## Ekran Tanımı

Müşteri ek grup bilgilerinin tanımlandığı ekrandır. Öncelikle ekrana, sisteme girilmiş olan müşteri ek grubu listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Müşteri ek grubuna verilen artışlı numaradır. |
| Ad | Müşteri ek grubuna verilen isimdir. |
| Distribütör Kodu | Müşteri ek grubu distribütör kodudur. Müşteri ek grubunun hangi distribütöre ait olduğu bu alan üzerinden izlenir. |
| Uygulama Yeri | Müşteri ek grubunun kullanılacağı modül bilgisinin tutulduğu alandır. Panorama / Quest / Panorama ve Quest / Münferit / Hepsi / Stokbar değerlerini alabilir. |
| Renk | Müşteri grubuna ait rengi belirtir. Diamond / Gold / Silver / Bronz / Red değerlerini alabilir. |
| Müşteri Grup | Müşteri ek grubunun bağlı olucağı grup seçiminin yapıldığı alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan tanımı kaydeder. |
| İptal | Ekranı kapatır. |
