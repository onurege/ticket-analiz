# Satış Temsilcisi Parametreler

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Parametreler → Satış Temsilcisi Parametreler`

## Ekran Tanımı

Satış temsilcilerinin yetkilerinin ve genel özelliklerinin verildiği ekrandır. Daha önceden tanımlanmış satış temsilcileri tanımlarına düzenle ile girilerek parametreler tanımlanır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Takip kodu bilgisidir. |
| Kod | Önceden tanımlanmış müşteri listesinden seçilen müşterilerin kod bilgilerinin otomatik olarak geldiği ekrandır. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| Menü Grubu | Menü grubu bilgisidir. |
| Durum | Aktif/Pasif durumunu belirleyen seçenektir |
| ST/SE Adı | Satış temsilcisi/Satış elemanı adıdır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Kapat | Rutu kaydet ve ekranı kapatmaya yarar. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
| Yenile | Listeyi yenileme işlemine yarayan butondur. |
| Düzenle | Satış Temsilcisi Parametreleri düzenlemeye yarar. |
| İzle | Satış Temsilcisi Parametreleri izlemeye yarar. |
| Parametre Detay | Parametre detay ekranını açmaya yarar. |
