# Tercihli Ürün

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Ürün Yönetimi → Tercihli Ürün`

## Ekran Tanımı

Ürünlerin sıra olarak en başta gözükmesi ve çeşitli renk değerleri ile renklerinin değiştirilerek tercihli ürün olarak belirtilmesini sağlayan ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Tercihli ürün tanımına sistem tarafından verilen artışlı numaradır. |
| Durum | Tercihli ürün tanımının durum bilgisidir. Aktif / Pasif değerlerini alabilir. |
| Başlangıç Tarihi | Tanımlanan tercihli ürün ayarlarının hangi tarihten itibaren geçerli olucağı bilgisi girilir. |
| Bitiş Tarihi | Tanımlanan tercihli ürün ayarlarının hangi tarihten itibaren geçersiz olucağı bilgisi girilir. |
| Distribütör Kriter Tipi | Tanımlanacak tercihli ürünün hangi distribütör kriter tipine uygulanacağının belirlendiği alandır. Kod / Grup / Ek Grup / Dinamik Grup değerlerini alabilir. |
| Tip | Ürün tip seçiminin yapıldığı alandır. Tercihli / Liste Dışı değerlerini alabilir. |
| Distribütör Kriteri | Distribütör Kriter Tipi seçimine bağlı olarak distribütör kriter seçiminin yapıldığı alandır. |
| Müşteri Kriter Tipi | Tanımlanacak tercihli ürünün hangi müşteri kriter tipine uygulanacağının belirlendiği alandır. Kod / Grup / Ek Grup / Dinamik Grup / Bölge / Satış Temsilcisi değerlerini alabilir. |
| Müşteri Kriter | Müşteri Kriter Tipi seçimine bağlı olarak müşteri kriter seçiminin yapıldığı alandır. |
| Müşteri Kriter Tipi 2 | Tanımlanacak tercihli ürünün hangi müşteri kriter tipine uygulanacağının belirlendiği alandır. Kod / Grup / Ek Grup / Dinamik Grup / Bölge / Satış Temsilcisi değerlerini alabilir. |
| Müşteri Kriter | Müşteri Kriter Tipi seçimine bağlı olarak müşteri kriter seçiminin yapıldığı alandır. |
| Ürün Kriter Tipi | Ürün Kriter Tipi seçimine bağlı olarak ürün kriter seçiminin yapıldığı alandır. |
| Zorunluluk | Zorunluluk kriteri için tercihli ürünün tercihe yönelik olması isteniliyorsa tercihli, direk tercihli ürünün zorunlu olması isteniliyorsa zorunlu seçeneği seçilir. |
| Belge girişinde yeni ürün mesajı verilsin | Bu seçenek işaretli olduğunda Müşteriye girildiğinde yeni ürün mesajı ve androidde de kırmızı bir işaret şeklinde bir uyarı belirmektedir. |
| Toplu Ekle / Çıkar | Ürün Kod yada Ürün Text Kod seçilerek seçilerek bu seçime uygun değerler girilerek toplucak ürünlerin eklenmesi yada çıkarılması yapılır. |
| Eklenemeyen | Bir ürün eklenemediği zaman bu kısıma atılır. |
| Yenile | Listeyi yeniler. |
| Yeni | Tercihli ürünün ve ürüne ait detayların girilmesi için kullanılır. |
| Detay Düzenle | Detaydaki kaydı düzenlemek için kullanılır. |
| İzle | Detaydaki kaydın detaylarını görüntülemek için kullanılır. |
| Sil | Detaydaki kaydı silmek için kullanılır. |
| Yukarı | Detaydaki kayıdın sıralamada yukarı taşınması için kullanılır. |
| Aşağı | Detaydaki kayıdın sıralamada aşağı taşınması için kullanılır. |
| Ekle | Toplu Ekle/Çıkar alanına girilen değerlerin detaya eklenmesi için kullanılır. |
| Çıkar | Toplu Ekle/Çıkar alanına girilen değerlerin detaydan çıkarılması için kullanılır. |
| \|< | Listede en baş sayfaya gidilmesini sağlar. |
| >\| | Listede en son sayfaya gidilmesini sağlar. |
| Ürün Kriteri | Tercihli ürün olarak tanımlanıcak ürünün seçildiği alandır. |
| Sıra | Sıra numarasının verildiği alandır. |
| Takip Gün Sayısı |  |
| Renk Kodu | Hangi renkte gözüküceğinin belirtildiği alandır. |

## Buton Açıklamaları
