# İskonto Promosyon Onaylama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ve Pazarlama Kriterleri → İskonto Promosyon → İskonto Promosyon Onaylama`

## Ekran Tanımı

Iskonto/Promosyon tanımlanırının onaylandığı ekrandır. Bu özelliğin çalışması için ‘Panorama Çalışma Parametreleri’ nde bulunan ‘İskonto Onaylayarak Geçerli Olsun’ parametresinin seçili olması gerekmektedir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Onaylanacak iskonto tanımlarının ait olduğu distribütör seçimi yapılır. |
| Geçerlilik Tarihi | Onaylama işlemi için, listelenecek iskonto tanımlarının geçerli olduğu tarih girişi yapılır. |
| Belge Tür | Listelenecek iskonto tanımlarının onay durumu seçilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Giriş yapılan kriterlere göre iskonto tanımları listelenir. |
| Kapat | Ekran kapatılır. |
| Yenile | Listelenen ekranın yenilenmesini sağlar. |
| İlk Sayfa | Listeleme yapılan sayfada ilk sayfaya gider. |
| Önceki Sayfa | Listeleme yapılan sayfada bir önceki sayfaya gider. |
| Sonraki Sayfa | Listeleme yapılan sayfada bir sonraki sayfaya gider. |
| Son Sayfa | Listeleme yapılan sayfada son sayfaya gider. |
| Onay | Listelenen iskonto kayıtlarından seçilenlerin onaylanmasını sağlar. |
| İzle | Kaydın izle modunda açılmasını sağlar. |
| İptal | Seçim sonrası işlemi iptal eder. |
