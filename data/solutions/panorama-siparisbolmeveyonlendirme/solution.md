# Sipariş Bölme ve Yönlendirme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Satış İşlemleri → Sipariş Bölme ve Yönlendirme`

## Ekran Tanımı

Sipariş bölme ve yönlendirme ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Başlangıç Tarihi | Başlangıç tarihi bilgisidir. |
| Bitiş Tarihi | Bitiş tarihi bilgisidir. |
| Sevk Tarihi | Sevk tarihi bilgisidir. |
| Yönlendirme Durum | Yönlendirme durumu bilgisidir. |
| Distribütör Onay | Distribütör onayı bilgisidir. |
| Sevk Durum | Sevk durum bilgisidir. |
| Fatura Durum | Fatura durum bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Belgeleri detay ekranında listelemek için kullanılır. |
| İzle | Seçilen belgeyi gözlemlemek için kullanılır. |
| Böl ve Yönlendir | Seçilen siparişi bölüp yönlendirmek için kullanılır. |
| Yönlendirme | Distribütöre yönlendirmek için kullanılır. |
| Yönlendirme İptal | Distribütöre yönlendirmeyi iptal etmek için kullanılır. |
| Distribütör Onayı | Distribütör onaylamak için kullanılır. |
| Distribütör Ret/Onayı İptal | Distribütör onay veya red işlemini iptal etmek için kullanılır. |
| SAP Aktar | SAP Aktarmak için kullanılır. |
| Kapat | Ekranı Kapatma işlemi yapar. |
