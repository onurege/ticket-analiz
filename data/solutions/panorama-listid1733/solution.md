# Kart Talep

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri → Kart Takip → Kart Talep`

## Ekran Tanımı

Kart Talep ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Kart talebi belge kod bilgisidir. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| Durum | Belge durum bilgisidir. |
| Talep Nedeni | Talep nedeni bilgisidir. |
| Talep Tip | Talep tipi bilgisidir. |
| Talep Miktarı | Talep miktarı bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Ekranda bilgileri girilmiş kullanıcıyı yarar. |
| İptal | Ekranı kaydetmeden Kapatma işlemi yapar. |
