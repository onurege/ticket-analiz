# Duyuru / Katılım Onay

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ve Pazarlama Kriterleri → İskonto Promosyon → Duyuru/Katılım Onay`

## Ekran Tanımı

Duyuru/Katılım onay ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Bütçe tanımının yapılacağı Distribütör seçilir. |
| Onay Durumu | Duyuru/Katılım onay durumu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Seçilen müşteri kriterine göre listeleme yapar, bu şekilde bütçe değerleri girilebilir. |
| Kapat | Ekranı kaydetmeden kapatır. |
| Onay | Detay ekranında seçilen duyuru/katılım belgelesini onaylamak içni kullanılır. |
| Ret | Detay ekranında seçilen duyuru/katılım belgelesini reddetmek içni kullanılır. |
| Not Girişi | Detay ekranında seçilen duyuru/katılım belgelesini not girişi yapılması için yapılır. |
