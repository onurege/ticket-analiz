# Kredi Kartı

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tahsilat İşlemleri → Kredi Kartı`

## Ekran Tanımı

Kredi Kartı tahsilat işleminin yapıldığı adımdır. Ekrana daha önceden girilmiş olan Kredi Kartı tahsilatları gelir. Önceki girilen Kredi Kartı tahsilatları üzerinde iptal, düzenleme işlemi yapılabilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Kredi Kartı tahsilata program tarafından verilen artışlı numaradır. |
| Belge No | Kredi Kartı tahsila ta verilecek olan belge kodudur. |
| Makbuz No | Tahsilat makbuzu üzerindeki girilecek olan makbuz numarasıdır. |
| İşlem Tarihi | Kredi Kartı tahsilatının yapıldığı tarih bilgisinin girildiği alandır. |
| Distribütör Adı | Distribütör bilgisinin girildiği alandır. |
| Müşteri | Kredi Kartı tahsilatının yapıldığı müşteri kodudur. |
| Satış Temsilcisi | Müşterini rutunda olduğu satış temsilcisi bilgisidir. |
| Tahsilat ST Kod | Tahsilatı yapan satış temsilcisi bilgisidir. |
| İşlem Tipi | Tahsilat işlem tipinin girildiği alandır. |
| Döviz Tipi | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsidir. |
| Dövizli Tutar | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsinden tutarıdır. |
| Kur | Döviz takibi yapılıyor ise işlem yapılan tahsilatın o günkü kur bilgisidir. |
| Tutar | Yapılan Kredi Kartı tahsilatın tutarının girildiği alandır. |
| Ödeme Planı | Tanımlı ödeme planı var ise seçildiği alandır. |
| Vade Tarihi | Alınan Kredi Kartı tahsilatının vade günün girildiği alandır. |
| Banka | Kredi kartı kaydı yapılan banka bilgisidir. |
| Kredi Kart No | Kredi Kartının numara bilgisidir. |
| Bakiye | Kredi Kartı Tahsilat kaydı girilen müşterinin bakiye bilgisidir. |
| Basım | Kredi Kartı Tahsilatı kaydının basım durum bilgisidir. |
| Giriş Tip | Kredi Kartı Tahsilatı kaydının giriş tipi bilgisidir. |
| Durum | Kredi Kartı Tahsilat kaydının belge durum bilgisidir. Aktif, silindi değerlerini alır. |
| Fatura Kodu | Kredi Kartı kaydının fatura kodu bilgisidir. Girilen faturaya karşılık oluşan takip kodu bilgisidir. |
| Sipariş Kodu | Kredi Kartı kaydının sipariş kodu bilgisidir. Girilen siparişe karşılık oluşan takip kodu bilgisidir. |
| Aktarım Durumu | Kredi Kartı Tahsilat kaydının ticari paket aktarım durum bilgisidir. Aktarıldı, Aktarılmadı değerlerini alır. |
| Tahsilat İptal Nedeni | Tahsilat iptal nedeninin girilebildiği alandır. |
| Proforma Fatura Kodu | Kredi Kartı kaydının proforma fatura kodu bilgisidir. Girilen proforma faturaya karşılık oluşan takip kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Yeni | Yeni kayıt girmek için kullanılan butondur. |
| Düzenle | Mevcut kayıtlarda düzenleme amacıyla kullanılan butondur. |
| İzle | Mevcut kayıtlarda izleme yapabilmek için kullanılan butondur. |
| Kopyala | Listede seçili olan tahsilatı kopyalamak için kullanılan butondur. |
| Sil | İptali istenen kaydı silmek için kullanılan butondur. |
| Yazdır | Tanımlı dizayna göre çıktı alınmak istendiğinde kullanılan butondur. |
| Fatura Kapat | Fatura kapatma ekranını açmaktadır. |
