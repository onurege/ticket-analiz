# Tatil Tanım

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Tanımlamalar`

## Ekran Tanımı

Tatil günlerinin sisteme tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Tatil tanımına sistem tarafından otomatik olarak verilen numaradır. |
| Distribütör Kodu | Tatil tanımının hangi distribütör için yapılacağının seçildiği alandır. |
| Başlangıç Tarihi | Tatil tanımı başlangıç tarihinin belirlendiği alandır. |
| Bitiş Tarihi | Tatil tanımı bitiş tarihinin belirlendiği alandır. |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Yeni | Yeni tatil tanımlamak için kullanılan butondur. |
| Düzenle | Önceden tanımlanmış tatili düzenlemek için kullanılan butondur. |
| İzle | Önceden tanımlanmış tatili izlemek için kullanılan butondur. |
| Kopyala | Önceden tanımlanmış tatili kopyalamak için kullanılan butondur. |
| Sil | Önceden tanımlanmış tatili silmek için kullanılan butondur. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Değişiklikleri kaydet ve yeni belge tanımlama butonudur. |
| Kaydet ve Kapat | Değişiklikleri kaydet ve ekranı kapatmaya yarar. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
