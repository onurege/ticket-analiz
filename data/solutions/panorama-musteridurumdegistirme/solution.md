# Müşteri Durum Değiştirme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → İşlemler → Müşteri Durum Değiştirme`

## Ekran Tanımı

Müşteri/müşterilere ait durum bilgisinin değiştirildiği ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kriter Tipi | Durum değiştirilecek müşteri/müşterileri filtrelemek için kriter tipinin girildiği alandır. Kod / Grup değerlerini alabilir. |
| Distribütör Kodu | Verilen kriter tipine göre distribütörlerin listeden seçildiği alandır. |
| Müşteri Kriter Tipi | Durum değiştirilecek müşteri/müşterileri filtrelemek için müşteri kriter tipinin girildiği alandır. Kod / Grup / Ek Grup / Dinamik Grup değerlerini alabilir. |
| Müşteri Kodu | Verilen Müşteri Kriter Tipine göre müşteri/müşterilerin listeden seçildiği alandır. |
| Güncellenicek Durum | Müşterinin güncellenicek durumunun belirtildiği alandır. Aktif / Pasif / İptal değerlerini alabilir. |
| Toplu Ekle | Toplu olarak müşteri seçimi yapılacağında, müşteri kodu veya müşteri takip kodlarının toplu olarak girilerek işlem yapılabilen alandır. |
| Müşteri Pasif/İptal Neden | Müşterinin pasife yada iptale çekileceğinde neden açıklamasının belirtildiği alandır. |
| Eklenemeyen | Toplu Ekle alanına girilen müşteri kodu yada müşteri takip kodu sistemde bulunamaz ise bu bulunamayan kodun gösterildiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Belirtilen kriterlere göre müşterilerin listelenmesi için kullanılır. |
| Temizle | Belirtilen kriterlerin ekrandan temizlenmesi için kullanılır. |
| Tarihçe | Daha önceki değişiklerin gösterilmesi için kullanılır. |
| Kapat | Ekranı kapatır. |
| Yenile | Ekranı yeniler. |
| Durum Güncelle | Durum güncelleme işlemini başlatır. |
| \|< | Listede en başa geçmek için kullanılır. |
| >\| | Listede en sona geçmek için kullanılır. |
| Kapat | Ekranı kapatır. |
| Yenile | Listeyi yeniler. |
| \|< | Listede en başa geçmek için kullanılır. |
| >\| | Listede en sona geçmek için kullanılır. |
