# Ürün Seviyeli Grup 1

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Parametreler → Ürün Seviyeli Grup 1`

## Ekran Tanımı

Ürün Seviyeli Grup 1 tanımlarının yapıldığı ekrandır. Ürünlere ait grup seviyeleri bu ekranda tanımlanmaktadır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Seviye Sayısı | Ürün gruplarına ait kaç seviye olucağının tanımlandığı alandır. |
| Seviye Ayracı | Ürün grup seviyeleri ve alt seviyeleri birbirleriyle ayraç aracılığıyla ilişkilendirilir. Bu ayraç tanımının yapıldığı alandır. |
| Sıra | Seviyeli Grup sıra tanımının yapıldığı alandır. |
| Karakter Sayısı | Seviyeye ait numaranın kaç karakterli olduğunun tanımının yapıldığı alandır. |
| Seviye Adı | Seviyeye ait isim tanımının yapıldığı alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Detayda yeni tanım yapılmasını sağlar. |
| Düzenle | Detaydaki tanımın düzenlenmesini sağlar. |
| İzle | Detaydaki tanıma ait detayların görüntülenmesini sağlar. |
| İşlemi Başlat | Ürün Seviyeli Grup 1 tanımını oluşturur. |
| Kapat | Ekranı kapatır. |
