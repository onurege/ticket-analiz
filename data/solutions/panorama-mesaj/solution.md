# Mesaj

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → İşlemler → Mesaj`

## Ekran Tanımı

Satış ekibinin mesajlarını özelleştirebildiği ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kodu | Mesaj değer tanımına program tarafından verilen artışlı numaradır. |
| Durum | Durum bilgisidir, aktif/pasif değerlerini alabilir. |
| Başlangıç Tarihi | Tanımlanan mesaj ayarlarının hangi tarihten itibaren geçerli olacağı bilgisi girilir. |
| Bitiş Tarihi | Tanımlanan mesaj ayarlarının hangi tarihten itibaren geçersiz olacağı bilgisi girilir. |
| Hareket Tip | Kriter olarak Hareket Tipinin seçileceği alandır. |
| Merkez ST Bilgiler Hazırlansın | Merkez ST bilgilerinin hazırlanması istenilirse kutucuk işaretlenir. Kutucuk işaretlendiği zaman distribütör kriter tipi ve distribütör kriteri otomatik olarak devre dışı olur. |
| İletilecek Yer | İletilecek Yer bilgisidir. “Satış Temsilcisi”, “Müşteri”, “Ürün” değerleri alabilir. |
| Distribütör Kriter Tipi | Tanımlanan mesaj ayarları sadece seçilen distribütör kriter tipine uygulanır.  Kod seçilirse Distribütör Kod listesi gelir.  Grup seçilirse Distribütör Grup listesi gelir.  Ek grup seçilirse Distribütör Ek Grup listesi gelir.  Dinamik grup seçilirse Distribütör Dinamik grup listesi gelir. |
| Distribütör Kriteri | Distribütör kriterinin belirlendiği sahadır |
| ST Kriter Tipi | Mesajın iletileceği yer olarak Satış Temsilcisi seçildiğinde Müşteri Kriter tipi, müşteri kriteri, ürün kriter tipi ve ürün kriteri alanları pasif olur. ST kriter tipi kod, grup yöneticisi ya da grup seçilir. |
| ST Kritet | ST Kriteri satış temsilcisi listesinden seçilir. |
| Müşteri Kriter Tipi | İletilecek Yer müşteri seçildiğinde aktif olan alandır. Müşteri kriter tipi kod, grup, ek grup, dinamik grup ya da bölge olarak seçilir. |
| Müşteri Kriter | İletilecek Yer müşteri seçildiğinde aktif olan alandır. Müşteri Kriteri müşteri listesinden seçilir. |
| Ek Saha Kriteri | Ek saha kriteri bilgisidir. |
| Ürün Kriter Tipi | İletilecek Yer ürün seçildiğinde aktif olan alandır. Ürün kriter tipi kod, grup, ek grup ya da dinamik grup seçilir. |
| Ürün Kriteri | İletilecek Yer ürün seçildiğinde aktif olan alandır. Ürün kriteri ürün listesinden seçilir. |
| Gösterim Tip | Bilgi ekranında görüntüle ya da listede otomatik görüntüle seçeneklerinden biri seçilir. |
| İskonto Kodu | İskonto kodu bilgisidir. |
| Mesaj | Mesaj alanına ayarları tanımlanan mesajın içeriği yazılır. Zorunlu alandır. |
| Biçim | Yazı biçimini seçtiğimiz alandır. |
| Süre | Bildirim süresi bilgisidir. |
| Tekrar | Bildirimin tekrar sayısını belirtir |
| Açılış | Mesajın gösterileceği ekranın seçimlerinin yapıldığı alandır. |
| Sipariş | Mesajın gösterileceği ekranın seçimlerinin yapıldığı alandır. |
| Teslim Fişi | Mesajın gösterileceği ekranın seçimlerinin yapıldığı alandır. |
| Fatura | Mesajın gösterileceği ekranın seçimlerinin yapıldığı alandır. |
| Panorama Çağrı Merkezi Satış | Mesajın gösterileceği ekranın seçimlerinin yapıldığı alandır. |
| Kapanış | Mesajın gösterileceği ekranın seçimlerinin yapıldığı alandır. |
| Müşteri | Mesajın gösterileceği ekranın seçimlerinin yapıldığı alandır. |
| İrsaliye | Mesajın gösterileceği ekranın seçimlerinin yapıldığı alandır. |
| Tahsilat | Mesajın gösterileceği ekranın seçimlerinin yapıldığı alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Değişiklikleri kaydet ve yeni belge tanımlama butonudur. |
| Kaydet ve Kapat | Değişiklikleri kaydet ve ekranı kapatmaya yarar. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
