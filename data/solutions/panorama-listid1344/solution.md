# Satış Temsilcisi İade Limit

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Tanımlamalar → Satış Temsilcisi İade Limit`

## Ekran Tanımı

Satış temsilcisi iade limit ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artan matrisli numaradır . |
| Durum | Durum bilgisidir. |
| Başlangıç Tarihi | Başlangıç tarihi bilgisidir. |
| Bitiş Tarihi | Bitiş tarihi bilgisidir. |
| Distribütör Kodu | Distribütör kodunun seçildiği sahadır. |
| Tutar | Tutar bilgisidir. |
| Satış Temsilcisi | Satış Temsilcisi bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemleri kaydetmek için kullanılan butondur. |
| İptal | Ekranı kaydetmeden kapatmaya yarayan butondur. |
