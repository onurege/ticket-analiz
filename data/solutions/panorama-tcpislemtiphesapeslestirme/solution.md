# İşlem Tip - Hesap

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → Ticari Paket → Eşleştirme → İşlem Tipi Hesap`

## Ekran Tanımı

Panoramada kullanılan işlem tipine göre ticari pakette kullanılan hesap kodu bilgisi girililir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Adı | İşlem yapılacak distribütör seçimi yapılır. |
| TP HESAP KOD | Ticari paket hesap kod değeri girilir. |
| MASRAF KOD | Ticari paket masraf kodu değeri girilir. |
| KOD | Panorama hareket tipi kodu gösterilir. |
| AÇIKLAMA | Panorama hareket tipi açıklaması gösterilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kapat | Form kapatılır. |
| Yenile | Sayfanın yenilenmesini sağlar. |
| Düzenle | Seçilen alanda değer girişi yapılmasını sağlar. |
| Yön Tuşları | Listelenen hareket tipi kayıtlarında ileri-geri sayfa geçişi yapılmasını sağlar. |
