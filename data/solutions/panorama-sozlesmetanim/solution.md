# Sözleşme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriteri → Satış Kriterleri → Sözleşme`

## Ekran Tanımı

Sözleşme bilgilerinin tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Sözleşme İzlemeye sistem tarafından otomatik olarak verilen artışlı numaradır. |
| Durum | Sözleşme izleme durum bilgisidir, aktif/pasif değerlerini alabilir. |
| Başlangıç Tarihi | Tanımlanan raf ömrü izleme ayarlarının hangi tarihte geçerli olacağı bilgisi girilir. |
| Bitiş Tarihi | Tanımlanan raf ömrü izleme ayarlarının hangi tarihten itibaren geçersiz olacağı bilgisi girilir. |
| Sözleşme Tipi | Sözleşme tipi listesinden sözleşme tipi seçilir. |
| Merkez Sözleşme Kodu | Merkez sözleşme kodu bilgisidir. |
| Distribütör Kodu | Distribütör bilgisinin seçildiği alandır. |
| Müşteri Kodu | Müşteri listesinden seçim yapılır. |
| Sözleşme Nedeni | Sözleşme nedenleri listesinden sözleşme nedeni seçilir. |
| Sözleşme Grubu | Sözleşme grubu listesinden sözleşme grubu seçimi yapılır. |
| Nokta Özelliği | Nokta özelliği listesinden nokta özelliği seçimi yapılır. |
| Sözleşme Kriteri | Sözleşme kriteri bilgisidir. |
| Sözleşme Notu | Sözleşme Notu bilgisidir. |
| Hedef Ürün Kriter Tip | Hibe tanımında, Hedef Ürün Kriter Tipi bilgisidir. “Kod”, “Grup”, “Ek Grup”, “Dinamik Grup” değerleri alabilir. |
| Ürün | Ürün bilgisidir. |
| Tip | Ürün hibe tipi bilgisidir. “Miktar”, “TL”, “Ağırlık”, “Hacim”, “Litre” değerleri alabilir. |
| Hedef Değeri | Hedef değer bilgisidir. |
| Hedef Kademe | Hedef kademe bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İskonto | İskonto oluşturma butonudur. |
| Kaydet ve Yeni | Yapılan işlemin kaydedilmesini sağlar ve yeni belge açar. |
| Kaydet ve Kapat | Yapılan işlemin kaydedilmesini sağlar ve ekranı kapatır. |
| Kapat | Ekranı kapatır. |
