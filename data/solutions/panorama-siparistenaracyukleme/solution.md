# Siparişten Araç Yükleme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri → Dağıtım İşlemleri → Siparişten Araç Yükleme`

## Ekran Tanımı

Müşterinin talep ettiği Sipariş belgelerinin Sevkiyat planının yapıldığı ve Fatura yada İrsaliye belgelerinin oluşturulduğu ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Yapılan işleme ait program tarafından otomatik verilen kayıt numarasıdır. |
| Distribütör Kodu | Sipariş kayıtlarının bağlı olduğu Distribütör kodunun seçildiği alandır. |
| Uygulanan Iskonto Promosyonları İptal Et ve Yeniden Hesapla | İşaretli olduğunda; Sipariş kayıtlarına uygulanmış olan Iskonto/Promosyonların yeniden hesaplanması sağlamaktadır. |
| İş Bildirim | İşaretli olduğunda; Sipariş kayıtlarının Fatura/İrsaliye dönüştürmeden önce İş Bildirim belgesi oluşturmaktadır. Bu işlemde Stok ve Bakiye değerlerini değiştirmez. İş Bildirim işlemi tamamlandıktan sonra Stok ve Bakiye hareketleri oluşur. |
| Döviz Kur Bilgileri Güncellensin | İşaretli olduğunda; Belgelerdeki döviz kur bilgilerinin güncellenmesini sağlar. |
| Başlangıç Tarihi | Alınan Sipariş belgelerinin hangi tarihten başlayacağı bilgisidir. |
| Bitiş Tarihi | Alınan Sipariş belgelerinin hangi tarihte biteceği bilgisidir. |
| Başlangıç Sevk Tarihi | Alınan Sipariş belgelerinin hangi Sevk tarihinden başlayacağı bilgisidir. |
| Bitiş Sevk Tarihi | Alınan Sipariş belgelerinin hangi Sevk tarihinde biteceği bilgisidir. |
| Araçlar | Alınan siparişlerin yükleneceği araç ya da araçların seçiminin yapıldığı alandır. |
| Hareket Tip | Alınan siparişlerin hangi hareket tipinde geleceğinin seçiminin yapıldığı alandır. |
| Satış Temsilcileri | Alınan siparişlerinin hangi Satış Temsilcileri üzerinden listelendiği seçiminin yapıldığı alandır. |
| Bölge Kodları | Alınan Siparişlerin hangi müşteri Bölge üzerinden listeleneceği seçiminin yapıldığı bilgisidir. |
| Detay Kriterler | Daha detaylı kriter girilmek isteniyorsa “Detay Kriterler” butonuna basılarak yeni bir ekranda, müşteri kodu (Rehber), Müşteri Grup (Çoklu seçim), Müşteri Ek Grup (Çoklu seçim), Depo Kodları (Çoklu Seçim), Özel Kod (Text Alan), Siparişin Tipi (Açık, Kapalı), Ödeme tipi (Peşin, Çek, Senet, Kredi kartı, Açık Hesap, Ticari kart, DBS, Havale/EFT) |
| Seçilen Araç | Siparişlerin yükleme yapılacağı araç seçilir. |
| Araç Nokta Sayısı | Araç Tanımı üzerindeki Nokta Sayısı değerini göstermektedir. |
| Nokta Sayısı | Seçilen araç bilgisi içerisindeki müşteri sayısını göstermektedir. |
| Belge Sayısı | Seçilen araç bilgisi içerisindeki belge sayısını göstermektedir. |
| Durum | Yükleme işleminin Hesap kapatma işlemindeki durum bilgisini göstermektedir. |
| Dağıtıcı | “Dağıtıcı Kodu” ve “Dağıtıcı Adı” seçilen araç için seçilen dağıtıcının kodunu ve adını göstermektedir. |
| Yüklenen Ağırlık % | Parametrik olarak Seçilen Araç içerisindeki yüzdesel olarak ağırlık bilgisini göstermektedir. |
| Yüklenen Hacim % | Parametrik olarak Seçilen Araç içerisindeki yüzdesel olarak hacim bilgisini göstermektedir. |
| Araç Hacim | Seçilen aracın maksimum alabileceği Hacim miktar bilgisidir. |
| Toplam Hacim | Seçilen araca yüklenmiş belgelerin depozito iade kalemleri hariç diğer ürünlerinin toplam hacim miktarını belirtir. |
| Hacim Oran (%) | Araç Hacim ile Toplam Hacim arasındaki yüzdesel oranı gösterir. |
| İade Hacim | “İade Hacim” seçilen araca yüklenmiş belgelerin tüm depozito iade kalemlerinin toplam hacim miktarını gösterir. |
| İade Hacim Oran (%) | İade Hacim alanının yüzdesel Oranını gösterir. |
| Araç Ağırlık | Seçilen aracın maksimum alabileceği Ağırlık miktar bilgisidir. |
| Toplam Ağırlık | Seçilen araca yüklenmiş belgelerin depozito iade kalemleri hariç diğer ürünlerinin toplam ağırlık miktarını gösterir. |
| Ağırlık Oran (%) | Araç Ağırlık ile Toplam Ağırlık arasındaki yüzdesel oranı gösterir. |
| İade Ağırlık | “İade Ağırlık” seçilen araca yüklenmiş belgelerin tüm depozito iade kalemlerinin toplam ağırlık miktarını gösterir. |
| İade Ağırlık Oran (%) | İade Ağırlık alanının yüzdesel Oranını gösterir. |
| İrsaliye Tarihi | Oluşacak İrsaliye belgelerinin tarihlerinin belirlendiği alandır. |
| Fatura Tarihi | Oluşacak Fatura belgelerinin tarihlerinin belirlendiği alandır. |
| Depo Kodu | Oluşacak belgelerin hangi depo üzerinden yapılacağının bilgisidir. |
| İrsaliye Hareket Tipi | Oluşacak İrsaliye belgelerinin Hareket Tiplerinin belirlendiği alandır. |
| Fatura Hareket Tipi | Oluşacak Fatura belgelerinin Hareket Tiplerinin belirlendiği alandır. |
| Depo Kodu | Seçilen ürünün depo kodunu gösteren alandır. |
| Depo Miktarı | Seçilen ürünün depo stok miktarının gösterildiği alandır. |
| Ürün Sayısı | Toplam ürün kalem sayısının gösterildiği alandır. |
| P.Miktar | Promosyon kalem miktarının gösterildiği alandır. |
| Miktar | Seçilen ürün miktarını göstermektedir. |
| K.Ürün Sayısı | Kritik ürün Sayısını göstermektedir. (Stok problemi olan) |
| Tümünü Sıfırla | Ürün miktarlarının toplu halde sıfırlanmasını sağlamaktadır. |
| Sevk Miktarlarını Eşitle | Ürün miktarları ile sevk miktarlarını eşitlemeyi sağlamaktadır. |
| Yenile | Ekradaki verilerin yenilenmesini sağlamaktadır. |
| Düzenle | Seçilen ürünün müşteriye ait sevk miktarını düzenlemeyi sağlamaktadır. |
| İzle | Seçilen ürünün müşteriye ait sevk miktarını izlemeyi sağlamaktadır. |
| Sıfrla | Seçilen ürünün müşteriye ait sevk miktarını sıfırlamaktadır. |
| Ağırlıklı Dağıtım | Miktar oranına göre sevk miktarını otomatik belirlemeyi sağlamaktadır. |
| Sıralı Dağıtım | Belge sayısına göre sevk miktarlarını eşit olarak otomatik belirlemeyi sağlamaktadır. |
| Kapat | Ekranı kapatma işlemi yapar. |
| Araçlar | Kriter alanında seçilen araç/araçların bilgisi gösterilmektedir. |
| Seçilen Araçlar (Max 6) | Kriter alanında belirlenmiş araçlardan çoklu seçim yapılabilen alandır. En fazla 6 araç seçimi yapılabilmektedir. |
| Toplam KM | Seçilen araçlara ait toplam KM değerini göstermektedir. |
| Seçilen Rut KM | Seçilen noktalara ait yapılacak Rut KM göstermektedir. |
| Müşteri Rut Sıra | Rut sıra değeri girilerek müşterinin belirginleştirilmesini sağlamaktadır. |
| Kapat | Ekranı kapatma işlemi yapar. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Temizle | Sahaya girilen bilgilerin silmesini sağlar. |
| Sevk Tarihleri | Sevk Tarihleri ekranının açılmasını sağlamaktadır. |
| Listele | Girilen kriterlere göre ilgili kayıtların listelenmesini sağlamaktadır. |
| Stok Seviye Kontrol | Stoğu yetersiz olan ürünlerinin kontrolü yapmaktadır. Stok yetersiz olduğu durumda Kısmi Sevk Ekranını açmaktadır. |
| Yükleme Rp. | Yükleme raporunun görüntülenmesini sağlamaktadır. |
| Sevkiyat Rp. | Sevkiyat raporunun görüntülenmesini sağlamaktadır. |
| Yükleme Geri Al | Yapılan yükleme işleminin geri alınmasını sağlamaktadır. |
| Rut Görüntüle | Harita ekranının açılmasını sağlamaktadır. |
| Yükleme Özet | Yükleme özet ekranının açılmasını sağlamaktadır. |
| Depo İşlemleri | RF Check parametresine bağlı olarak depo işlemlerinin yapılabilmesini sağlamaktadır. |
| Sadakat | Sadakat İskontosu Seçimi ekranını açmaktadır. |
| Belgeleştir | Bu butona basıldığında “Yüklenemeyenler” hariç diğer tüm araçlardaki siparişler araç sırası ve araç içerisindeki sipariş sırasına göre irsaliyeleri ve faturaları oluşturulma işlemine girer. ( İrsaliye siparişin detayları ile aynı şekilde oluşturulur. ) |
| Kaydet | Bu buton yardımı ile yükleme kaydedilebilir. Kaydet butonuna basıldığında “Kriterler” gruplamasındaki ve yüklenemeyenler aracı veritabanına kaydedilmez. |
| Kapat | Ekranı Kapatma işlemi yapar. |
| Yenile | Liste kayıtlarının güncellenmesini sağlamaktadır. |
| Düzenle | Belgenin kayıt tipi ya F(Fatura) ya da I(İrsaliye) olarak düzenlenir |
| Belge İzle | Alınan siparişin ürün detay bilgilerinin izlenmesini sağlar. |
| Transfer | Yüklenmemiş olan siparişlerden seçilenlerin araca transferi sağlanır. |
| Fatura/İrsaliye | Listelenen bütün siparişlerin fatura ve irsaliye olarak otomatik çevrilmesini sağlar. |
| Kısmi Sevk | İstenilen belgelerin parçalı bir şekilde sevk edilebilmesini sağlar |
| İsk./Pro. | Seçilen siparişlerde iskonto/promosyon olup olmadığı gösterilir |
| Seçili Ağ./Hac. | Seçili kayıtlar için Ağırlık ve Hacim toplamlarını göstermektedir. |
| Bekleyen Siparişler | Seçilen kayıt müşterisinin sevk edilmemiş siparişlerini listelemektedir. |
| Yukarı | Belgele sıralamasını aşağı yönde yer değiştirmesini sağlar |
| Aşağı | Belgele sıralamasını yukarı yönde yer değiştirmesini sağlar |
| Yüklenemeyen Kontrol | Seçilen kayıt müşterisinin Yüklenemeyenler bölümündeki kayıtlarını belirgin hale getirmektedir. |
