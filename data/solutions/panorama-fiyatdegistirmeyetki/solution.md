# Fiyat Değiştirme Yetki Tanımlama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Parametreler → Fiyat Değiştirme Yetki Tanımlama`

## Ekran Tanımı

Fiyat değiştirme yetkisinin tanımlandığı ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Berilen artışlı numaradır. |
| Durum | Durum bilgisidir, aktif/pasif değerlerini alabilir. |
| Başlangıç Tarihi | Listelenecek Entegrasyon sonuçlarının başlangıç tarihinin girildiği alandır. |
| Bitiş Tarihi | Listelenecek Entegrasyon sonuçlarının bitiş tarihinin girildiği alandır. |
| Hareket Tip | Kriter olarak Hareket Tipinin seçileceği alandır. |
| Distribütör Kriter Tipi | Kayıları Listelenecek Distribütörün kriterini seçildiği alandır. |
| Distribütör Kriteri | Kayıtları listelenecek Distribütörün belirtildiği alandır. |
| ST Kriter Tipi | Satış temsilcisi kriter tipi bilgisidir. |
| Satış Temsilcisi | Hangi Satış Temsilcisine ait bilgilerin listeleneceği bilgisidir. |
| Ürün Kriter Tipi | Ürünlerin seçileceği kriterlerin girildiği alandır değer olarak (Kod, Grup, Ek Grup, Dinamik Gurup) değerlerini almaktadır. |
| Ürün Kriteri | Ürün kriteri bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Değişiklikleri kaydet ve yeni belge tanımlama butonudur. |
| Kaydet ve Kapat | Değişiklikleri kaydet ve ekranı kapatmaya yarar. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
