# Toplu Müşteri Değerleri Değiştirme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → İşlemler → Toplu Müşteri Değerleri Değiştirme`

## Ekran Tanımı

Müşteri kartı üzerinde bulunan Kredi Çarpanı 1, Kredi Çarpanı 2, İskonto Oranı, Vade Günü gibi değerlerin toplu olarak değiştirilmesini sağlayan ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Grup Kodu | Müşterilerin Grup Kodu bilgisine göre filtrelenmesi için Grup bilgisinin girildiği alandır. |
| Dinamik Grup | Müşterilerin Dinamik Grup bilgisine göre filtrelenmesi için Dinamik Grup bilgisinin girildiği alandır. |
| Satış Temsilcisi | Müşterilerin Satış Temsilcisi bilgisine göre filtrelenmesi için Satış Temsilcisi bilgisinin girildiği alandır. |
| Bölge | Müşterilerin filtrelenmesi için Bölge bilgisinin girildiği alandır. |
| Değer Tipi | Değer Tipinin belirtildiği alandır. Yüzde / Değer değerlerini alabilir. |
| Ek Grup Kodu | Müşterilerin Ek Grup Kodu bilgisine göre filtrelenmesi için Ek Grup Kodu bilgisinin girildiği alandır. |
| Değiştirilecek Saha | Değiştirilecek sahanın hangisi olduğunun belirtildiği alandır. Kredi Çarpanı 1, Kredi Çarpanı 2, İskonto Oranı, Vade Günü değerlerini alabilir. |
| Alt Limit | Müşteri Alt Limit değerinin girildiği alandır. |
| Üst Limit | Müşteri Üst Limit değerinin girildiği alandır. |
| Değer | Değer bilgisinin girildiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Tanımı kaydeder. |
| Listele | Girilen kriter bilgilerine göre detayda müşterilerin listelenmesini sağlar. |
| Temizle | Girilen kriter bilgilerini ekrandan kaldırır. |
| Kapat | Ekranı kapatır. |
| Ekle | Belirtilen kriterlere göre müşteri özelliği değiştirlecek müşterinin listeden seçilerek bir aaşağıdaki değiştirilecek listesine kayıdın atılmasını sağlar. |
