# Fatura Gecikme Günü Hesaplama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → İşlemler → Fatura Gecikme Günü Hesaplama`

## Ekran Tanımı

Tanmlanmış geciken faturalarıın hesaplamalarının yapıldığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Başlangıç Tarihi | Tanımlanan Fatura Gecikme Günü Hesaplama ayarlarının hangi tarihte geçerli olacağı bilgisi girilir. |
| Bitiş Tarihi | Tanımlanan Fatura Gecikme Günü Hesaplama ayarlarının hangi tarihten itibaren geçersiz olacağı bilgisi girilir. |
| Distribütör Kodu | Tanımın hangi distribütöre ait olduğunu belirten distribütör kod bilgisidir. Zorunlu alandır. |
| Satış Temsilcisi | Satış Temsilcisi bilgisinin seçildiği ekrandır. Zorunlu alandır. |
| Esas Alınacak Tarih | Fatura gecikme günü için esas alınacak tarih bilgisi girilir. |
| İşlem Tarihi | Fatura gecikme gününden itibaren hangi gün işlem yapılacağı girilir. |
| Kullanılacak Tarih | Kullanılacak tarih seçiminin yapıldığı alandır. Vade ya da işlem seçimi yapılır. |
| Max. Tolerans | Fatura gecikme gününden sonra maksimum gösterilecek tutar girilir. |
| Sadece Gecikmişler | Sadece faturası gecikmişlerin gösterilmesi istenilirse kutucuk işaretlenir. |
| Kod | Tanımlanmış gecikmiş faturanın takip kodu bilgisidir. |
| Kriter | Kriter bilgisidir. |
| Durum | Durum bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Kaydet | Bütün alanlar doldurulduktan sonra belirlenen kriterlere göre listelemenin başlatılması için İşlemi Başlat butonuna tıklanılır. |
| Kapat | Ekranı kaydetmeden kapatır. |
| Yenile | Listeyi yenilemek amaçlı kullanılır. |
| İzle | Kayıtları incelemek için kullanılır. |
