# Depo Grup

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri → Tanımlamalar → Depo Grup`

## Ekran Tanımı

Depo Grup ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Tanımlanması yapılan depo grup kod tanımının kod bilgisidir. |
| Distribütör Kodu | Tanımlanması yapılan depo grup kodunun hangisi distribütöre ait olduğunun bilgisidir. |
| Depo Kodu | İlgili deponun seçileceği sahadır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Belge detay listesini yenilemek için kullanılır. |
| Yeni | Belge detaya yeni ürün eklemek için kullanılır. |
| İzle | Detaya eklenen satırı izle modunda açar. |
| Sil | Detaya eklenen satırı silmek için kullanılır. |
