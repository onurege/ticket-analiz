# Özel Ürün Kodları

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Tanımlamalar → Özel Ürün Kodları`

## Ekran Tanımı

Özel ürün kodu bilgilerinin tanımlandığı ekrandır. Herhangi bir neden ile ürünler için tanımlanmış olan ürün kodları dışında farklı kodlar tanımlanması gerektiğinde kullanılan ekrandır. Örneğin, bazı zincir mağazalar kendilerine kesilen faturalarda kendi ürün kodlarını görmek isterler. Bu amaçla ürün kodlarının karşılarına zincir mağazanın kendi ürün kodları tanımlanır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Tanımlanması yapılan özel ürün kod tanımının kod bilgisidir. Girilmesi zorunlu sahadır. |
| Ad | Tanımlanması yapılan özel ürün kodunun açıklama bilgisidir. Girilmesi zorunlu sahadır. Verilen zincir mağaza ismi girilebilir. |
| Distribütör Kodu | Tanımlanması yapılan özel ürün kodunun hangisi distribütöre ait olduğunun bilgisidir. |
| Uygulama Yeri | Özel ürün kodlarının nerede işleme tutulacağı seçilir. Zorunlu alandır. Panorama / Quest / Panorama ve Quest / Münferit / Hepsi / Stokbar değerlerini alabilir. |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Detayda yeni tanım yapılmasını sağlar. |
| İzle | Detayda yapılan tanımın görüntülenmesini sağlar. |
| Düzenle | Detayda yapılan tanımın düzenlenmesini sağlar. |
| Sil | Detayda yapılan tanımın silinmesini sağlar. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Ürün | Eşleştirme yapılacak ürünün rehberden seçildiği alandır. |
| Özel Ürün Kod 1 | Müşterinin istemiş olduğu kendi ürün kodunun girildiği alandır. |
| Özel Ürün Kod 2 | Müşterinin istemiş olduğu kendi ürün kodunun girildiği ikincil alandır. |
| Kaydet | Detay tanımını kaydeder. |
| İptal | Detay tanımını iptal eder. |
| Kaydet | Yapılan tanımın kadedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
