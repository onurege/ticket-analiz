# Hafta

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Sistem Yönetimi→ Genel Parametreler → Hafta`

## Ekran Tanımı

Hafta tanımlarının yapıldığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Sistemin otomatik olarak artışlı verdiği koddur. |
| Başlangıç Tarihi | Tanımı yapılacak haftanın başlangıç tarihinin girildiği alandır. |
| Bitiş Tarihi | Tanımı yapılacak haftanın bitiş tarihinin girildiği alandır. |
| Hafta Kod | Tanımı yapılan haftanın kodunun seçildiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemlerin kayıt edilmesini sağlar. |
| İptal | Yapılan işlemlerin kayıt edilmeden, ekranın kapanmasını sağlar. |
