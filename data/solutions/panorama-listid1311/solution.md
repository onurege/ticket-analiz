# Lojistik Onay İptal Neden

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri → Tanımlamalar → Lojistik Onay İptal Neden`

## Ekran Tanımı

Lojistik Onay Nedenlerinin tanımlandığı alandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Ad | Lojistik Onay İptal Nedenlerine verilen isimdir. Girilmesi zorunlu sahadır. |
| Distribütör Kodu | İlgili Distribütörün seçileceği sahadır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemin kaydedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
