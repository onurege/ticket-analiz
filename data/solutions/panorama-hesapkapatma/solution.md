# Hesap Kapatma

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Satış İşlemleri → Hesap Kapatma`

## Ekran Tanımı

Panorama Mobil Satış Dağıtım modülü içerisinde “Soğuk Satış” uygulaması olan müşterilerde kullanılmak üzere dağıtıcıların dönüş işlemlerinde “Hesap Kapatma İşlemleri” kullanılır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Faturanın kayıtlı olduğu distribütörün kodu seçilir |
| Dağıtıcı Kodu | Ürünleri dağıtan kişinin kodu seçilir |
| Hareket Tipi | Distribütör Kodu seçildiğinde otomatik olarak gelir ( Bakiye Dengeleme gibi) |
| Tahsilat ST Kod | Tahsilatı yapan satış temsilcisi bilgisidir |
| İşlem Tarihi | İşlemin yapıldığı tarih seçilir. |
| Hesap Kapatma Tarihi | Hesap kapatma tarihi bilgisidir. |
| Yükleme Detay Kod | Siparişten Araç Yükleme ekranından seçilir ve oradaki yükleme kodu otomatik olarak gelir |
| Otomatik Tahsilatlar Yaratılmasın | Otomatik Tahsilatlar Yaratılmasın istenirse seçilebilir. |
| Kod | Belge kod bilgisidir. |
| Alış Faturası | Alış faturası bilgisidir. |
| Satış Faturası | Satış faturası bilgisidir. |
| Durum | Fatura iptal ise iptal, geçerliliğini koruyorsa aktif değerlerini alır. |

| Nakit | Nakit tutar bilgileridir. “Fatura(Satış)”, “Tahsilat”, “Fatura(Alış/İade)”, “Gelen”, “Fark” başlığı altında bilgiler görülebilir. |
