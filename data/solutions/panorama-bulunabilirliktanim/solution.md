# Bulunurluk

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri → Anket ve Bulunurluk → Bulunurluk`

## Ekran Tanımı

Sahada uygulanmak üzere Bulunurluk tanımlarının yapıldığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Bulunurluk tanıma verilen otomatik artışlı numaradır. |
| Durum | Bulunurluk durum bilgisidir, aktif/pasif değerlerini alabilir. |
| Başlangıç Tarihi | Tanımlanan Bulunurluk ayarlarının hangi tarihte geçerli olacağı bilgisi girilir. |
| Bitiş Tarihi | Tanımlanan Bulunurluk ayarlarının hangi tarihten itibaren geçersiz olacağı bilgisi girilir. |
| Uygulama Tipi | Bulunurluğun uygulanacağı tip Var/Yok-Miktar veya Fiyat cinsinden seçilebilir. |
| Müşteri Sayısı | Bulunurluk tanımının uygulancağı müşteri sayısı girilir. |
| Dsitribütör Kriter Tipi | Tanımlanan tercihli ürün ayarları sadece seçilen distribütör kriter tipine uygulanır. Kod seçilirse Distribütör Kod listesi gelir. Grup seçilirse Distribütör Grup listesi gelir. Ek grup seçilirse Distribütör Ek Grup listesi gelir. Dinamik grup seçilirse Distribütör Dinamik grup listesi gelir. |
| Distribütör Kriteri | Distribütör kriterinin seçildiği alandır. |
| Müşteri Kriter Tipi | Müşteri Kriter Tipi Kod/Grup/Ek Grup/Dinamik Grup/Bölge/Satış Temsilcisi/Ek Saha değerlerini alabilir. |
| Müşteri Kiter | Müşteri Kriterinin seçildiği alandır. |
| Ek Saha | Müşteri Kriter seçeneği ‘Ek Saha’ seçildi ise ilgili alandan bulunurluk tanımının çıkacağı ‘Ek Saha’ bilgisi seçilir. |
| Ürün Kriter Tipi | Ürün Kriter Tipi Kod/Grup/Ek Grup/Dinamik Grup değerlerini alabilir. |
| Giriş Tip | Giriş Tip Detaylı/Gruplu değerlerini alabilir. |
| Frekans | Frekans Sürekli/Bir kez/Tekrarlı değerlerini alabilir. |
| Gün Sayısı | Frekans değeri olarak ‘Tekrarlı’ seçildiğinde aktif olur. Gün sayısı girilerek o değerde tekrar sağlanır. |
| Zorunlu | Seçilir ise ilgili bulunurluk tanıma giriş yapılması zorunlu hale gelir. |
| Ürün Kısa Adları Kullanılsın | Seçilir ise ilgili bulunurluk tanımında ürün kısa ad’ları listlenir. |
| Stokta Olmayan Ürünleride Hazırla | Seçilir ise ilgili bulunurluk tanımı için stokta olmayan ürünlerde hazırlanır. |
| Ürün Kriteri | Bulunurluk tanımının yapılacağı ürünler seçilir. Üst kısımda seçilen ‘Ürün Kriter Tip’ değerine göre rehber açılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfanın yenilenmesini sağlar. |
| Yeni | Yeni bulunurluk detayı tanımlanır. |
| İzle | Kaydedilen bulunurluk detay kaydını izle modunda açar. |
| Sil | Kaydedilen bulunurluk detay kaydını siler. |
