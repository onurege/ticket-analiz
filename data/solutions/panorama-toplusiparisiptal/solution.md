# Toplu Sipariş İptali

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Satış İşlemleri →Toplu Sipariş İptali`

## Ekran Tanımı

Yüklemesi yapılmamış siparişler bu ekrandan toplu olarak iptal edilebilir. Kriter alanından girilen değerlere bağlı olarak ilgili kriterlere ait siparişler detay alanında listelenir ve daha sonra çoklu seçim yapılarak seçilmiş olan siparişler iptal edilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Başlangıç Tarihi | Hangi tarihler arasındaki siparişlerin detayda listelenmesini istiyorsak başlangıç tarihi değeri bu alana girilir. |
| Bitiş Tarihi | Hangi tarihler arasındaki siparişlerin detayda listelenmesini istiyorsak bitiş tarihi değeri bu alana girilir. |
| Dist. Kodu | İptal edilecek Siparişlerin bağlı olduğu distribütör seçimi bu alandan yapılır. |
| Müşteri Kriter Tipi | Belli bir müşterinin, belli grup veya ek gruba ait müşterilerin siparişlerinin detayda listelenmesini istiyorsak bu alandan kod, grup veya ek grup seçimi yapılır. |
| Müşteri Kriter | Müşteri Kriter Tip alanında seçmiş olduğumuz değere göre müşteri kriter alanındaki rehberde listelenen değerler değişir. Bu alanda çoklu seçim yapılabilir. |
| Bölge | Bir bölgeye ait müşterilerin siparişlerinin iptali olacak ise bölge alanından ilgili bölge değeri seçilir. Tekli seçim yapılır. |
| St Kodu | Belli satış temsilcilerine ait siparişlerin iptal edilmesi isteniyor ise ilgili satış temsilcileri bu alandan seçilir. |
| İlk Sipariş No | Ardışık sıraya ait siparişler iptal edilmek isteniyor ise ilk sipariş kodu bu alandan seçilir. |
| Son Sipariş No | Ardışık sıraya ait siparişler iptal edilmek isteniyor ise son sipariş kodu bu alandan seçilir. |
| Sipariş Durum | Sipariş durumuna göre listeleme yapılabilir, sipariş durumu alanında Hepsi veya Kısmı sevk seçimi yapılarak siparişlerin listelenmesi sağlanabilir. |
| Belge Tipi | Listelenecek siparişlerin Açık Kapalı olma durumuna göre listelenmesi sağlanır. Açık siparişler listelenmek isteniyorsa Açık değeri seçilir. Kapalı siparişler listelenmek isteniyorsa Kapalı seçimi yapılır. Açık ve kapalı siparişlerin listelenmesini istiyorsak Hepsi değeri seçilir. |
| Ödeme Tipi | Ödeme tipine göre siparişler listelenmek isteniyor ise bu alandan ilgili ödeme tipi değeri seçilir. |
| Özel Kod | Özel koda ait siparişlerin listelenmesi isteniyor ise ilgili özel kod değeri bu alana girilir. |
| Hareket Tipi | Belli bir hareket tipine ait siparişlerin listelenmesini istiyorsak, ilgili hareket tipi bu alandan seçilir. |
| Kod | İptal edilmek istenen siparişler herhangi bir kritere bağlı kalmadan bu alandan seçilebilir. |
| Toplu Ekleme | Sipariş kodlarının elle bu alana yazılması sonucunda listele butonuna bastığımızda seçilen siparişler detaya eklenir. Toplu ekle alanına birden fazla sipariş kodu yazılması durumunda sipariş kodları virgül ile ayrılır. |
| Eklenemeyen | Toplu ekleme alanına girilen sipariş kodlarından herhangi birinin eklenmemesi durumunda eklenemeyen siparişin kodunu Eklenemeyen alanında görürüz. |
| Belge Kodu | Seçilen kriterlere göre listelenen siparişlerin Belge kodu değerlerini gösterir. |
| Müşteri Kodu | Seçilen kriterlere göre listelenen siparişlerin Müşteri kodu değerlerini gösterir. |
| Müşteri Adı | Seçilen kriterlere göre listelenen siparişlerin Müşteri adı değerlerini gösterir. |
| İşlem Tarihi | Seçilen kriterlere göre listelenen siparişlerin İşlem Tarihi değerlerini gösterir. |
| Satış Temsilcisi | Seçilen kriterlere göre listelenen siparişlerin Satış temsilcisi kodunu ve adını gösterir. |
| Net Tutar | Seçilen kriterlere göre listelenen siparişlerin Net tutar değerlerini gösterir. |
| Dist | Seçilen kriterlere göre listelenen siparişlerin Distribütör kodu değerlerini gösterir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Listele butonuna bastığımızda kriter alanına girilen kriterlere bağlı olarak siparişler detay alanında listelenir. |
| Temizle | Temizle butonuna bastığımızda kriter alanına girilen kriterler ve bunlara bağlı olarak detay alanında listelenen siparişler temizlenir. |
| Kapat | Toplu Sipariş İptali ekranını kapatır. |
| Yenile | Detay listesinin yenilenmesini sağlar |
| İptal | Detay alanında listelenen ve daha sonra seçimi yapılan siparişlerin İptal edilmesini sağlar. |
| İzle | Detay alanında listelenen ve seçimi yapılan siparişi izle modunda açar. |
