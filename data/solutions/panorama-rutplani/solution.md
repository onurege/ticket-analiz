# Rut Planı

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `İş Akış → Onay İşlemleri → Rut İşlemleri`

## Ekran Tanımı

Onaylanacak/Reddedilen/Onaylanan alış siparişlerinin listelendiği ekrandır. Onaylama ve red işlemi bu ekrandan yapılmaktadır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Kullanıcı Grup | Listeleme yapılacak kullanıcı grubu seçilir. |
| Satış Temsilcisi | Dağıtıcıya tanımlanmış olan satış temsilcisi kodudur. |
| Onay Bekleyenler | Sadece onay bekleyenlerin listelenmesi sağlanılır. |
| Onayladıklarım | Sadece onaylananların listelenmesi sağlanır. |
| Reddettiklerim | Sadece reddedilenlerin listelenmesi sağlanılır. |
| İşlem | İşlem tipi “Bekliyor” veya “Onay” seçilerek kriter belirlenmesi yapılıp “Kaydet” butonuna tıklanılarak işlem tipinin değiştirilmesi sağlanılır. |
| Ret Neden | Ret nedenini belirtildiği açıklama alanıdır. |
| Takip Kodu | Dağıtıcıya program tarafından verilen artışlı numaradır. |
| Distribütör Kodu | Dağıtıcının hangi distribütöre ait olduğunu belirten, distribütör kod bilgisidir. |
| Kod | Önceden tanımlanmış müşteri listesinden seçilen müşterilerin kod bilgilerinin otomatik olarak geldiği ekrandır. |
| Toplu Ekle/Çıkar | Rut tanımına eklenmek istenilen müşterilerin kodlarının toplu olarak eklenmesinin sağlandığı ekrandır. |
| Ziyaret Başlangıç | Tanımlanmak istenilen rut güzergâhı için ziyaret başlangıcı belirlenir. |
| Müşteri Kodu | Müşteri kod bilgisidir. |
| Müşteri Takip Kodu | Müşteri Takip kodu bilgisidir. |
| Müşteri Merkez Kod | Müşteri Merkez kodu bilgisidir. |
| Tip | Araç tip bilgisidir. Dağıtıcı ya da Satış Temsilcisi seçilebilir. |
| Onay | Onay durum bilgisidir. |
| Eklenemeyen | Toplu ekleme ekranından eklenmek istenilen müşteri kodlarından eklenemeyenlerin listelendiği sahadır. |
| Ziyaret Artış | Tanımlanmak istenilen rut güzergâhı için ziyaretin artış sayısı belirlenir. |
| Harita Sıralama Kriteri | Sıra veya Ziyaret Sırası değeri alır. Harita Sıralama Kriterini belirler. |
| Toplam KM | Toplam Km bilgisidir. |
