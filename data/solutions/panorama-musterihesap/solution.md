# Müşteri-Hesap

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tanımlamalar → Müşteri - Hesap`

## Ekran Tanımı

Tanımlanan banka hesap bilgilerinin müşterilere atandığı ekrandır. Gelen listede ilgili müşteri seçilip düzenle ile giriş yapılarak bu müşterinin çalıştığı banka, hesap, şube bilgileri tanımlanır. Ekrana müşteri listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Müşterinin takip kod bilgisidir. Edit edilemez. |
| Uygulama Yeri | Müşteri kartından gelen uygulama yeri bilgisidir. Panorama, Quest, Panorama ve Quest, Münferit,Hepsi,Stokbar,CallDesk değerleri gelir. Edit edilemez. |
| Müşteri Kodu | Atanacak hesabın müşteri kod bilgisidir. |
| Durum | Hesabın atanacağı müşterinin durumunu gösteren sahadır. Sadece Aktif müşteriler bu listede gözükür. Edit edilemez. |
| Ünvan | Müşterinin ünvan bilgisinin gösterildiği alandır. Edit edilemez. |
| Distribütör Kodu | Müşterinin bağlı bulunduğu distribütörün gösterildiği ekrandır. Edit edilemez. |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Yeni | Müşteriye banka hesabı eklemek için kullanılan butondur. |
| Düzenle | Mevcut banka hesabını değiştirmek için kullanılan butondur. |
| İzle | Banka hesabını izlemek için kullanılan butondur. |
| Sil | Müşteri ile banka hesabı arasındaki bağı kopartabilmek için kullanılan butondur. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Düzenle | Mevcut kayıtlarda düzenleme amacıyla kullanılan butondur. |
| İzle | Mevcut kayıtlarda izleme yapabilmek için kullanılan butondur. |
