# Rut Optimizasyon Tanım

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri → Dağıtım İşlemleri → Rut Optimizasyonu → Rut Optimizasyon Tanım`

## Ekran Tanımı

Rut optimizasyonu tanım ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yükleme Kodu | Yükleme kodu bilgisidir. |
| Distribütör Kodu | Kaydın hangi distribütöre ait olduğunu belirten, distribütör kod bilgisidir. |
| İş Bildirim | İş bildirim yapılsın istenirse seçilir. |
| Başlangıç Tarihi | Başlangıç tarihi bilgisidir. |
| Bitiş Tarihi | Bitiş tarihi bilgisidir. |
| Başlangıç Sevk Tarihi | Başlangıç sevk tarihi bilgisidir. |
| Bitiş Sevk Tarihi | Bitiş sevk tarihi bilgisidir. |
| Araçlar | Araç seçiminin yapıldığı alandır. |
| Hareket Tip | Hareket tipi bilgisidir. |
| Satış Temsilcisi | Satış temsilcisi bilgisidir. |
| Bölge Kodları | Bölge kodları bilgisidir. |
| ST Grup | Satış temsilcisi grup bilgisidir. |
| Rut Opt. Parametre | Rut Optimizasyonu parametre bilgisidir. |
| Belge Sayısı | Belge sayısı bilgisidir. |
| Nokta Sayısı | Nokta sayısı bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Temizle | Temizle butonuna bastığımızda kriter alanına girilen kriterler ve bunlara bağlı olarak detay alanında listelenenler temizlenir. |
| Sevk Tarihleri | Sevk tarihi bilgilerinin görüldüğü ekranı açmaktadır. |
| Listele | Listele butonuna bastığımızda kriter alanına girilen kriterlere bağlı olarak İrsaliyeler detay alanında listelenir. |
| Stok Seviye Kontrol | Stok seviye kontrolünün yapıldığı ekranı açmaktadır. |
| Yükleme Rp. | Yükleme Raporunun gözlemlendiği ekrandır. |
| Sevkiyat Rp. | Sevkiyat Raporunun gözlemlendiği ekrandır. |
| Rut Optimizasyonu | Rut optimizasyonu ekranını açmaktadır. |
| Sonuç | Sonuçlandırmak için kullanılan butondur. |
| Belgeleştir | Belgeleştirilmesinin istenirse seçilir. |
| Kaydet | Belgeyi Kaydetmek için kullanılır |
| Kapat | Sipariş ekranını kapatır. |
