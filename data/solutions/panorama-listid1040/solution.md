# Komisyon

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tanımlamalar → Komisyon`

## Ekran Tanımı

Banka komisyon tanımlarının yapıldığı ekrandır. Öncelikle ekranan sisteme girilmiş olan komisyon tanımları listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen otomatik artışlı numaradır. |
| Durum | Tanımlanan komisyonun durum bilgisidir. Aktif ve Pasif değerlerini alır. |
| Başlangıç Tarihi | Komisyon tanımın geçerli olduğu başlangıç tarihidir. |
| Bitiş Tarihi | Komisyon tanımının geçerli olduğu bitiş tarihidir. |
| Banka Kodu | Anlaşmalı banka bilgisidir ve rehber ekranıdır. |
| Komisyon Oranı | Komisyon oranın girildiği alandır. |
| Kredi Kartı Banka Kodu | Kredi kartının bağlı bulunduğu bankanın seçildiği rehber alanıdır. |
| Pos Vade Günü | Pos işlemlerinde paranın ne kadar sürede hesaba geçtiğinin bilgisidir. |
| Distribütör Kodu | Komisyon tanımın uygulanacağı distribütör bilgisidir. |
| Varsayılan POS Bankası | POS Bankasının varsayılan olarak ayarlandığı check box’tır. Sadece tek komisyon varsayılan olabilir. |
