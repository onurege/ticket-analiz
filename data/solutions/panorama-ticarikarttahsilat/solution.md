# Ticari Kart

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tahsilat İşlemleri → Ticari Kart`

## Ekran Tanımı

Ticari Kart tahsilat işleminin yapıldığı adımdır. Ekrana daha önceden girilmiş olan Ticari Kart tahsilatları gelir. Önceki girilen Ticari Kart tahsilatları üzerinde iptal, düzenleme işlemi yapılabilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Ticari Kart tahsilatına program tarafından verilen artışlı numaradır. |
| Belge No | Ticari Kart tahsila tına verilecek olan belge kodudur. |
| Makbuz No | Tahsilat makbuzu üzerindeki girilecek olan makbuz numarasıdır. |
| İşlem Tarihi | Ticari Kart tahsilatının yapıldığı tarih bilgisinin girildiği alandır. |
| Distribütör Adı | Distribütör bilgisinin girildiği alandır. |
| Müşteri | Ticari Kart tahsilatının yapıldığı müşteri kodudur. |
| Satış Temsilcisi | Müşterini rutunda olduğu satış temsilcisi bilgisidir. |
| Tahsilat ST Kod | Tahsilatı yapan satış temsilcisi bilgisidir. |
| İşlem Tipi | Tahsilat işlem tipinin girildiği alandır. |
| Döviz Tipi | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsidir. |
| Dövizli Tutar | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsinden tutarıdır. |
| Kur | Döviz takibi yapılıyor ise işlem yapılan tahsilatın o günkü kur bilgisidir. |
| Tutar | Yapılan Ticari Kart tahsilatın tutarının girildiği alandır. |
| Ödeme Planı | Tanımlı ödeme planı var ise seçildiği alandır. |
| Vade Tarihi | Alınan Ticari Kart tahsilatının vade günün girildiği alandır. |
| Banka | Ticari Kart kaydı yapılan banka bilgisidir. |
| Kredi Kart No | Ticari Kartının numara bilgisidir. |
| Bakiye | Ticari Kart Tahsilat kaydı girilen müşterinin bakiye bilgisidir. |
| Basım | Ticari Kart Tahsilatı kaydının basım durum bilgisidir. |
| Giriş Tip | Ticari Kart Tahsilatı kaydının giriş tipi bilgisidir. |
| Durum | Ticari Kart Tahsilat kaydının belge durum bilgisidir. Aktif, silindi değerlerini alır. |
| Fatura Kodu | Ticari Kart kaydının fatura kodu bilgisidir. Girilen faturaya karşılık oluşan takip kodu bilgisidir. |
| Sipariş Kodu | Ticari Kart kaydının sipariş kodu bilgisidir. Girilen siparişe karşılık oluşan takip kodu bilgisidir. |
| Aktarım Durumu | Ticari Kart Tahsilat kaydının ticari paket aktarım durum bilgisidir. Aktarıldı, Aktarılmadı değerlerini alır. |
| Tahsilat İptal Nedeni | Tahsilat iptal nedeninin girilebildiği alandır. |
| Proforma Fatura Kodu | Ticari Kart kaydının proforma fatura kodu bilgisidir. Girilen proforma faturaya karşılık oluşan takip kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Yeni | Yeni kayıt girmek için kullanılan butondur. |
| Düzenle | Mevcut kayıtlarda düzenleme amacıyla kullanılan butondur. |
| İzle | Mevcut kayıtlarda izleme yapabilmek için kullanılan butondur. |
| Kopyala | Listede seçili olan tahsilatı kopyalamak için kullanılan butondur. |
| Sil | İptali istenen kaydı silmek için kullanılan butondur. |
| Yazdır | Tanımlı dizayna göre çıktı alınmak istendiğinde kullanılan butondur. |
