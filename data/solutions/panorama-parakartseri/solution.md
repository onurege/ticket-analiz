# Promosyon Para Kart Giriş

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri 🡪 Stok Hareketleri 🡪 Promosyon Para kart Girişi`

## Ekran Tanımı

Promosyon Para Kart ürünlerinin seri bazında girişi için kullanılan ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Ürün Kodu | Para Kart Ürünlerinin listelendiği Tek seçimli zorunlu Rehber alandır. |
| Tek seri Girişi | Tek bir ürün için seri girişi yapılacaksa, tıklanan seçim alanıdır. |
| Seri Aralık Girişi | Birden çok ürün için seri girişi yapılacaksa, tıklanan seçim alanıdır. |
| Seri Giriş | Tek Seri girişi veya çoklu seri girişi için başlangıç serisinin girildiği alandır. |
| Seri Giriş | Çoklu seri girişi için başlangıç serisinin girildiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Ekranı kaydedip yeni bir ekran açmak için kullanılacak olan butondur. |
| Kaydet ve Kapat | Ekranı kaydedip kapatmak için kullanılacak olan butondur. |
| Kapat | Ekranı Kapatma işlemi yapar. |
