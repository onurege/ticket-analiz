# Otomatik Stok Seviye

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri → Tanımlamalar → Otomatik Stok Seviye`

## Ekran Tanımı

Otomatik Stok Seviye ekranı ile seçilen distribütörün seçilen deposuna otomatik stok seviye artırımı yapılabilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından otomatik verilen kod. |
| Durum | Tanımlanan ya da tanımlanacak olan otomatik stok seviye tanımının aktif veya pasif olduğunun seçimi yapılır. |
| Distribütör Kodu | Depo distribütör kodudur. Deponun hangi distribütöre ait olduğu bu alan üzerinden izlenir. |
| Depo Kodu | Seçilen distribütöre ait hangi depoya otomatik stok seviye yapılacağı seçilir. |
| Rut Kodu | Rut Tanım Başlık sahasından tanımlanmış rut kodu seçilir. |
| Ürün | Ürün listesinden hangi ürüne ait otomatik stok seviye artırımı yapılacağı seçilir. |
| Birim Sıra | Otomatik stok seviye yapılacak ürün biriminin adet mi, koli mi ya da palet mi olacağı seçilir. |
| Min Miktar | Otomatik stok seviye yapılacak ürün için minimum miktar sınırı belirtilmek zorundadır. |
| Max Miktar | Otomatik stok seviye yapılacak ürün için maksimum miktar sınırı belirtilmek zorundadır. |
| Yükleme | Otomatik stok seviye yapılacak net ürün miktarı belirtilmek zorundadır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Belge detay listesini yenilemek için kullanılır. |
| Yeni | Belge detaya yeni ürün eklemek için kullanılır. |
| Düzenle | Belge detay satırında düzenleme yapmak için kullanılır. |
| İzle | Detaya eklenen satırı izle modunda açar. |
| Sil | Detaya eklenen satırı silmek için kullanılır. |
