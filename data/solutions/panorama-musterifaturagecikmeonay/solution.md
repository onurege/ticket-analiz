## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Durum | İşlem durum bilgisidir. Aktif/Pasif değerlerini alır. |
| Distribütör Kodu | Tanımın hangi distribütöre ait olduğunu belirten distribütör kod bilgisidir. |
| Müşteri Kodu | Müşteri kodunun seçildiği alandır. |
| Başlangıç Tarihi | Tanımlanan Fatura Gecikme Günü Onay hangi tarihte geçerli olacağı bilgisi girilir. |
| Bitiş Tarihi | Tanımlanan Fatura Gecikme Günü Onay hangi tarihte biticeği bilgisi girilir. |

## Buton Açıklamaları

| Alan | Açıklama |
