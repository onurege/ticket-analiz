# Araç

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Tanımlamalar → Araç`

## Ekran Tanımı

Araç tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Dağıtıcıya program tarafından verilen artışlı numaradır. |
| Kod | Tanımlanan araç için kullanıcının verdiği koddur. |
| Tip | Araç tip bilgisidir. Dağıtıcı ya da Satış Temsilcisi seçilebilir. |
| Distribütör Kodu | Aracın hangi distribütöre ait olduğunu belirten, distribütör kod bilgisidir. |
| Dağıtıcı Kodu | İlgili araçta hangi dağıtıcının kullanılacağı bilgisidir. |
| Durum | Aktif/Pasif durum bilgisidir. |
| Soför Adı | Araç kullanan Şoför bilgisidir. |
| Ağırlık | Ağırlık biriminden araç kapasite bilgisidir. |
| Marka | Araç marka bilgisidir. |
| Plaka 1 | Araç plaka bilgisidir. |
| Plaka 2 | Araç plaka bilgisidir. |
| Muavin Kodu | Varsa araç muavin kodu bilgisidir. |
| Muavin | Varsa araç muavin bilgisidir. |
| Hacim | Hacim biriminden araç kapasite bilgisidir. |
| Model | Araç model bilgisidir. |
| Ruhsat No | Araç ruhsat no bilgisidir. |
| Depo Kodu | Depo kodu bilgisidir. |
| Yükleme Kriter Tip | Yükleme Kriter Tipi. Kod, Grup, Ek Grup, Dinamik Grup, Bölge, Satış Temsilcisi, İşlem Tipi ve Rut seçilebilir. |
| Yükleme Kriter | Seçilen yükleme kriter tipine göre, yükleme ekranında siparişlerin hangi kritere göre araca otomatik yükleneceği bilgisidir. |
| Yükleme Kriter Tip2 | Yükleme Kriter Tipi. Kod, Grup, Ek Grup, Dinamik Grup, Bölge, Satış Temsilcisi, İşlem Tipi ve Rut seçilebilir. |
| Yükleme Kriter2 | Seçilen yükleme kriter tipine göre, yükleme ekranında siparişlerin hangi kritere göre araca otomatik yükleneceği bilgisidir. |
| Araç Cinsi | Araç cins bilgisidir.(Araç ağırlığı ve Araç hacmi bilgilerine göre araç cins seçimi yapılmaktadır.) |
| Nokta Sayısı | Nokta sayısı bilgisidir. |
| Yük/Servis/Hizmet Tipi | Yük/Servis/Hizmet Tipi bilgisidir. |
| Kapasite | Kapasite bilgisidir. |
| Ortalama Hız | Ortalama Hız bilgisidir. |
| Km Başına Maliyet | Km başına yakıt maliyeti bilgisidir. |
| Kalkış Tarihi | Kalkış tarihi bilgisidir. |
| Kalkış Zamanı | Kalkış tarihi bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemleri kaydetmek için kullanılan butondur. |
| İptal | Ekranı kaydetmeden kapatmaya yarayan butondur. |
