# SAP'den Verilen Siparişlerin Alınması

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → İşlemler → SAP'den Verilen Siparişlerin Alınması`

## Ekran Tanımı

Merkez entegrasyon ile SAP’den verilen sipariş kayıtlarının aktarım arayüzüdür.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Tarih | Aktarım yapılacak tarih seçimi yapılır. |
| Distribütör Kodu | Aktarım yapılacak distribütör seçimi yapılır. |
| Depo Kodu | Aktarım yapılacak depo seçimi yapılır. |
| Satış Temsilcisi | Aktarım yapılacak satış temsilcisi seçimi yapılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Aktar | Girilen kriterlere göre aktarım yapılmasını sağlar. |
| Sonuç İzleme | Aktarım sonuçlarının kontrol edilmesini sağlar. |
| Kapat | Ekranı kapatır. |
