# Alınan Sipariş

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `iş Akış → Onay İşlemleri → Alınan Sipariş`

## Ekran Tanımı

Onaylanacak/Reddedilen/Onaylanan satış siparişlerinin listelendiği ekrandır. Onaylama ve red işlemi bu ekrandan yapılmaktadır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kullanıcı Grup | Listeleme yapılacak kullanıcı grubu seçilir. |
| Onay Bekleyenler | Sadece onay bekleyenlerin listelenmesi sağlanılır. |
| Onayladıklarım | Sadece onaylananların listelenmesi sağlanır. |
| Reddettiklerim | Sadece reddedilenlerin listelenmesi sağlanılır. |
| Ödeme/Hareket/Giriş Tipi | Ayrıca tanımlanması yapılmış özel hareket, ödeme tipi veya giriş tipi varsa ve ona göre filtreleme yapılmak istenilirse o tip seçilir. Bütün listenin görüntülenmesi istenilirse “Tümü” kriteri seçilir. |
| İşlem | İşlem tipi “Bekliyor” veya “Onay” seçilerek kriter belirlenmesi yapılıp “Kaydet” butonuna tıklanılarak işlem tipinin değiştirilmesi sağlanılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan değişikliklerin kaydedilmesini sağlar. |
| İzle | Tanımın izle ile açılmasını sağlar. |
| Tarihçe | Onay geçmişini gösterir. |
| Düzenle | İzin verilen yetki dahilinde belgenin düzenleme butonudur. |
| Başlık Düzenle | İzin verilen yetki dahilinde belgenin başlık bilgisi düzenleme butonudur. |
| Detay Düzenle | İzin verilen yetki dahilinde belgenin detay bilgisi düzenleme butonudur. |
| Kapat | Ekranı kaydetmeden kapatır. |
| Ret Neden | Ret Nedenin kod bilgisidir. |
| Geri Gelme Nedeni | Önceki adıma gönder adımından gelen açıklama bilgisidir. |
| Kod | Belgenin takip kod bilgisidir. |
| Erp Kod | Müşterinin takip kod bilgisidir. |
| Ünvan | Müşterinin ünvan bilgisidir. |
| Tutar | Belgenin tutar bilgisidir. |
| İşlem Tarihi | Belgede yapılan son değişiklik tarihidir. |
| Distribütör Kodu | Distribütörün takip kod bilgisidir. |
| Dist | Belgenin tutar bilgisidir. |
| Yıl | Çalışma yılının gösterildiği alandır. |
| Ağırlık | Distribütörün kod bilgisidir. |
| Oran | Belgenin oran bilgisidir. |
| İskonto Tutarı | Belgenin iskonto tutar bilgisidir. |
| Özel Kod | Varsa belgenin özel kod bilgisidir. |
