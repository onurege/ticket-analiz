# Pm Menü Sıralama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Parametreler → PM Menu Sırlama`

## Ekran Tanımı

PM menü sıralaması işlemine yarar

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Şablon kod bilgisidir. |
| Durum | Ürün durum bilgisidir. Aktif, Pasif değerlerini alır. |
| Pm İşlemleri | Pm işlem listesi alanıdır. |
| Kod | Menü kodu bilgisidir. |
| Sıra | Menü sırası bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Kapat | Değişiklikleri kaydet ve ekranı kapatmaya yarar. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
| Yukarı | Listede seçili alanın sırasını değiştirmek için kullanılır. |
| Aşağı | Listede seçili alanın sırasını değiştirmek için kullanılır. |
| Ekle | Seçilen kriterlere uygun olan bilgileri listelemeye yarar. |
| Temizle | Listeyi temizlemek için kullanılır. |
