# Fatura Tahsilat

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → DBS İşlemleri → Fatura Tahsilat`

## Ekran Tanımı

Ekran Faturaların web servis kullanılarak banka tarafındaki tahsilat bilgilerinin görüntülendiği ekrandır. Banka seçimi sadece Panorama sistem yöneticisi ekranında yer alan DBS Entegrasyon ekranında kullanılan bankalar üzerinden listelenmekte ve işlem yapılmaktadır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Banka | Banka bilgisinin seçildiği alandır. |
| Distribütör Bölge | Merkez kullanıcı için distribütör bölge kriterinin verilebildiği alandır. |
| Distribütör Kodu | Distribütör kriterinin verildiği alandır. |
| Vade Başlangıç Tarihi | Listenecek kayıtların vade tarihinin başlangıcıdır |
| Bitiş Tarihi | Bitiş tarihinin girildiği alandır. |
| Distribütör Bölge | Listelenen kayıtların hangi bölgede olduğu bilgisinin olduğu alandır. |
| Distribütör Adı | Listelenecek belgelerde geçerli yada iptal belgelerin listelenmesi için kullanılan kriterdir. |
| Banka Adı | Listelenen kayıt bankada ise banka bilgisinin bulunduğu alandır. |
| Tutar | Listelenen belgenin tutar bilgisidir. |
| Fatura No | Listelenen kayıdının sistem takip numarasıdır. |
| Ödeme Durumu | Ödendi bilgisinin olduğu alandır. |
| Ödeme Tarihi | Ödeme tarihinin tutulduğu alandır. |
| Tahsilat Tarihi | Tahsilat tarihinin tutulduğu alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Girilen kriterlere göre verilen listelenmesini sağlar. |
| Excel’e Aktar | Ekranda listelenen verilerin excel formatında kayıt işleminin yapılmasını sağlar |
| Pdf’e Aktar | Ekranda listelenen verilerin pdf formatında kayıt işleminin yapılmasını sağlar |
| Kapat | Açılan ekran kapanır. |
