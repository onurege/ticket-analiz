# Kasa

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tanımlamalar → Kasa`

## Ekran Tanımı

Kasa bilgilerinin tanımlandığı ekrandır. Öncelikle ekrana sisteme girilmiş kasa listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Kasa tanımına program tarafından verilen artışlı numaradır. |
| Durum | Kasanın durum bilgisidir. Aktif ve Pasif değerlerini alır. |
| Kasa Kod | Kasa kodunun belirlendiği sayısal sahadır ve zorunludur. |
| Dist | Distribütör kodunun girildiği rehber alanıdır ve zorunludur. |
| Satış Temsilcisi | Kasaya Satış Temsilcilerinin atanabileceği çoklu seçim rehber alanıdır. |
| Aylar(Ocak..Aralık) | Ay bazında kasa giriş çıkış ve buna bağlı o ayki bakiye kısmının gösterildiği alandır. |
| Toplam | Tüm ay giriş ve çıkışları ile kasanın mevcut bakiyesinin gösterildiği alandır. |
