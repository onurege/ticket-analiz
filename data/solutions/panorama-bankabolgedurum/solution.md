# Banka Bölge Durum

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tahsilat İşlemleri → Banka Bölge Durumu`

## Ekran Tanımı

Banka bölge durumu ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Banka | Banka bilgisidir. |
| Distribütör Bölge | Distribütör bölge bilgisidir. |
| Başlangıç Tarihi | Başlangıç tarihi bilgisidir. |
| Bitiş Tarihi | Bitiş tarihi bilgisidir. |
| Tarih Tipi | Tarih tipi bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Girilen kriterlere göre listeleme işlemini gerçekleştirir. |
| Excel’e Aktar | Listeyi excel’e aktarır. |
| Pdf’e Aktar | Listeyi pdf dosyasına aktarır. |
| Kapat | Belgeyi kapatmak için kullanılır. |
| Banka Bölge Durum | Detay alanında banka bölge durumunun gösterir. |
| Ay | Detay alanının ay bazında görüntülenmektedir. |
