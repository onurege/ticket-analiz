# Tüccar Plasiyer Onay Ekranı

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri → Stok Hareketleri → Tüccar Plasiyer Onay`

## Ekran Tanımı

Tüccar plasiyer onay ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Başlangıç Tarihi | Başlangıç tarihi ekranıdır. |
| Bitiş Tarihi | Bitiş tarihi ekranıdır. |
| Distribütör Kodu | Depo çıkış kaydının hangi distribütöre ait olduğu bu alan üzerinden izlenir. |
| Satış Temsilcisi Kod | Satış temsilcisi kod bilgisidir. |
| Lojistik Onay | Lojistik onayı bilgisidir. |
| Distribütör Onayı | Distribütör onayı bilgisidir. |
| ST Onaylı | Satış temsilcisi onaylı bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Belge detay listesini yenilemek için kullanılır. |
| Düzenle | Belge detay satırında düzenleme yapmak için kullanılır. |
| İzle | Detaya eklenen satırı izle modunda açar. |
| Belge İptali | Belge İptalinin yapılması sağlanır. |
| Lojistik Onayı | Lojistik onayı verilmesi istenirse kullanılır. |
| Lojistik Onayı İptali | Lojistik onayı iptali işlemi için kullanılır. |
| Distribütör Onayı | Distribütör onayı verilmesi istenirse kullanılır. |
| Distribütör Onayı İptal | Distribütör onayı iptali işlemi için kullanılır. |
| Kapat | Ekranı kapatmak için kullanılır. |
