# Yatırım

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriteri → Satış Kriterleri → Yatırım`

## Ekran Tanımı

Yatırım bilgilerinin tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Yatırım detayına verilen artışlı numaradır. |
| Yatırım Tipi | Yatırım tipi bilgisidir. “Bedelsiz”, “Duran Varlık”, “Ek İskontolar”, “Peşin Ciro” Değerlerini alabilir. |
| Giriş Tarihi | Giriş Tarihi bilgisidir. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| Merkez Müşteri Kodu | Merkez müşteri kodu bilgisidir. |
| Müşteri Kodu | Müşterinin kod bilgisidir. |
| Üretici | Üretici kodu bilgisidir. |
| Sözleşme Başlangıç Tarihi | Sözleşmenin başladığı tarihi belirten alandır. |
| Sözleşme Bitiş Tarihi | Sözleşmenin bitiş tarihini belirten alandır. |
| Ay Sayısı | Toplam ay sayısı bilgisidir. |
| Yatırım Faturası geldi mi? | Yatırım faturası geldiyse seçilir. |
| Teminat Vadesi | Teminat Vadesi bilgisidir. |
| Teminat Türü | Teminat türü bilgisidir. |
| Taahhüt Birimi | Taahhüt birini bilgisidir. |
| Sözleşme Peşin Ciro Tutarı | Sözleşme Peşin Ciro Tutar bilgisidir. |
| Taahhüt Tutarı | Taahhüt tutar bilgisidir . |
| Teminat Tutarı | Teminat tutar bilgisidir. |
| Sözleşme Tanımlama Tarihi | Sözleşme tanımlama tarihi bilgisidir. |
| Fiili Peşin Ciro Tutarı | Fiili peşin ciro tutar bilgisidir. |
| Giderleşen Ciro Tutarı | Giderleşen Ciro Tutar bilgisidir. |
| Cari Yıl Giderleşen | Cari Yıl Giderleşen bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemin kaydedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
| Ay | Yatırımın gerçekleştiriği ay bilgisidir. |
| Yıl | Yatırımın gerçekleştiriği yıl bilgisidir. |
| Taahhüt Gerçekleşen | Taahhüt Gerçekleşen bilgisidir. |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Detayda yeni tanım yapmak için kullanılır. |
| Düzenle | Detaydaki kaydı düzenlemek için kullanılır. |
| İzle | Detaydaki kaydı görüntülemek için kullanılır. |
| Sil | Detaydaki kaydın silinmesi için kullanılır. |
