# Kullanıcı

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Sistem Yönetimi → Yetki İşlemleri → Kullanıcı`

## Ekran Tanımı

Sistemi kullanacak kullanıcıların tanımlandığı ekrandır. Öncelikle ekrana, sisteme tanımlı olan kullanıcıların listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen otomatik artışlı kullanıcı kodudur. |
| Tip | Sistemi kullanacak kullanıcının tipini belirler. Merkez, Distribütör/Bayi, Müşteri, Rapor seçeneklerinden biri seçilir. |
| Dil Kodu | Kullanıcının sistemde kullanacağı varsayılan dili seçmeye yarar. Türkçe ve İngilizce değerlerini alır. |
| Onay | Tanımlanan kullanıcının onay bilgisini görüntüler. Elle değiştirilemez bir alandır. |
| Kullanıcı İsmi | Sistemi kullanacak olan kullanıcı için belirlenecek kullanıcı adıdır. Sistemde aynı isimde sadece bir kullanıcının olmasına izin verilir. |
| Durum | Kullanıcının durumunu belirler. Aktif ve pasif değerlerini alır. Sistemi sadece aktif olan kullanıcılar kullanabilir. |
| Yetki Tip | Kullanıcının hangi yetki tipine dahil olduğunu belirler. Tanımlı yetki tiplerinden biri seçilir. Zorunlu alan değildir. |
| Gruplar | Kullanıcının dahil olacağı kullanıcı grupları seçilir. Zorunlu alan değildir. |
| Ad Soyad | Kullanıcının ad ve soyad bilgisi girilir. Zorunlu değildir. |
| Mail Adresi | Sistemi kullanacak kullanıcının mail adresi girilir. Zorunlu alan değildir. |
| Şifre | Sistemi kullanacak kullanıcının şifresi girilir. Zorunlu alandır. |
| Şifre(Tekrar) | Şifre alanına girilmiş olan değer ile aynı değer girilmelidir. Zorunlu alandır. |
| Kullanıcı bir sonraki oturumda şifre değiştirmeli | İşaretlendiğinde kullanıcı tanımında belirlenen şifre ile sisteme girildiğinde kullanıcıya şifre değişikliği ekranı çıkartılmasına yarar. |
| Kullanıcı şifreyi değiştiremez | İşaretlendiğinde kullanıcı için belirlenmiş şifrenin kullanıcı tarafından değiştirilmesini engeller. |
| Şifre her zaman geçerli olsun | İşaretlendiğinde şifre süresi olmaksızın aynı şifre ile sisteme giriş yapılmasına yarar. İşaretsiz ise aşağıda girilen gün sayısı dolduğunda kullanıcıyı şifre yenilemeye zorlar. |
| Kullanıcı sms ile giriş yapsın | İşaretli olduğunda kullanıcı adı ve şifre girildikten sonra kullanıcının cep telefonuna gelen kod bilgisi ile giriş yapılmasına yarar. |
| Cep Tel No | Kullanıcının cep telefonu bilgisi girilir. Kullanıcı sms ile giriş yapsın işaretli ise cep telefonu girilmesi zorunludur. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Ekranda bilgileri girilmiş kullanıcıyı kaydetmeye ve yeni kullanıcı açmak için boş bir ekran açmaya yarar. |
| Kaydet ve Kapat | Ekranda bilgileri girilmiş kullanıcıyı kaydetmeye ve ekranı kapatmaya yarar. |
| Kapat | Ekranı Kapatma işlemi yapar. |
