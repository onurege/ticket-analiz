# Müşteri Not

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → İşlemler → Müşteri Not`

## Ekran Tanımı

Müşteri notlarının tanımlandığı alandır. Öncelikle ekrana sistemde kayıtlı not tanımları listelenir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Kayıda verilen kod bilgisidir. Zorunlu alandır. |
| Distribütör Kodu | Tanımın hangi distribütör için geçerli olucağı kriterinin girildiği alandır. |
| Uygulama Yeri | Tanımın hangi ürün için geçerli olucağı bilgisinin girildiği alandır. Panorama / Quest / Panorama ve Quest / Münferit / Hepsi / Stokbar / CallDesk değerlerini alabilir. |
| Rut Dışı Neden | Rut dışı nedeni bilgisinin girildiği alandır. Evet / Hayır değerlerini alabilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Tanımı kaydeder. |
| İptal | Tanım ekranını kapatır. |
