# Müşteri No

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → Tanımlamalar → Müşteri No`

## Ekran Tanımı

Müşteriye belirli bir formatta numara tanımı yapılmasını sağlayan ekrandır. Tanımlanacak olan müşterilere bu formatta numara atanması sağlanır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Sistem tarafından verilen otomatik artışlı numaradır. |
| Kod Ön Eki | Verilecek numaranın ön eki tanımının yapıldığı alandır. |
| Son No | Verilecek numaranın son numarasının tanımının yapıldığı alandır. |
| Distribütör Kodu | Rehberden tanımın bağlı olucağı distribütör tanımının yapıldığı alandır. |
| Maks. Uzunluk | Verilecek numaranın maksimum uzunluğunun tanımlandığı alandır. |
| Sıfırlarla Doldur | Kod ön eki ve son numarası arasındaki kalan karakterlerin bu alandaki Evet / Hayır parametresine göre 0 larla doldurulup doldurulmayacağının seçildiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan tanımı kaydeder. |
| İptal | Ekranı kapatır. |
