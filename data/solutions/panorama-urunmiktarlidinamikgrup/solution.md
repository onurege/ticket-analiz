# Ürün Miktarlı Dinamik Grup

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Tanımlamalar → Ürün Miktarlı Dinamik Grup`

## Ekran Tanımı

Pratik Dinamik Gruplarının tanımlandığı ekrandır. Ürün Dinamik Grupdan farkı, detaya eklenen ürünlere miktar girişi yapılabilmektedir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Sistem tarafından verilen artışlı numaradır. |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Detayda yeni tanım yapılmasını sağlar. |
| İzle | Detayda yapılan tanımın görüntülenmesini sağlar. |
| Düzenle | Detayda yapılan tanımın düzenlenmesini sağlar. |
| Sil | Detayda yapılan tanımın silinmesini sağlar. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Tip | Tanımlanıcak olan ürün miktarlı pratik dinamik grubunun hangi ürün tipine göre tanımlanıcağının belirlendiği alandır. Kod / Grup / Ek Grup değerlerini alabilir. |
| Detay Kod | Belirtilen Tip alanı değerine göre ürünün ekleneceği alandır. |
| Miktar | Eklenen ürün miktarının girildiği alandır. |
| Kaydet | Detayda yapılan tanımın kaydedilmesini sağlar. |
| İptal | Detayda yapılan tanımın iptal edilmesini sağlar. |
| Kaydet | Yapılan tanımın kadedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
