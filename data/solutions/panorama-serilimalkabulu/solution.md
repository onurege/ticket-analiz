# Serili Mal Kabulu

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Alış İşlemleri → Serili Mal Kabul`

## Ekran Tanımı

Serili mal kabülü yapıldığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Belgenin hangi Distribütör için kesileceğinin tanımlandığı alandır. |
| Belge Kodu | Belge kodu bilgisidir. |
| Tip | Fatura tipi bilgisidir. |
| Kod | Ürün kod bilgisidir. |
| Ad | Ürün adı bilgisidir. |
| Toplam Miktar | Toplam miktar bilgisidir. |
| Tek Seri Girişi | Tek seri girişi yapılıcaksa seçilir. |
| Seri Numara Aralığı | Seri numara aralığı ile yapılıcaksa seçilir. |
| Seri Başlangıç | Seri başlangıç bilgisidir. |
| Seri Bitiş | Seri bitiş bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Seçimlere göre listeleme işlemi gerçekleştirir. |
| İşlemi Başlat | Yapılan değişiklikleri kaydedip işlemleri başlatmak için kullanılır. |
| Kapat | Belge ekranını kapatmak için kullanılır. |
| Ekle | Detaya seri giriş yapılmasını sağlar |
| Çıkar | Detayadaki seçili belgeyi çıkartmak için kullanılır |
| Temizle | Yapılan seçimler ve listeyi temizlemek için kullanılır. |
| Seç ve Kapat | Seri giriş yapıldıktan sonra seçme işlemini gerçekleştirip ekranı kapatır |
| Demirbaş Özelliği | Demirbaş Özelliklerini belirtildiği ekranı açmaktadır. |
| Kapat | Ekranı kapatmak için kullanılır. |
