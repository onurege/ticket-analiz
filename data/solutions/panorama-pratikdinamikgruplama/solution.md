# Pratik Dinamik Gruplama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım Kanalı → Tanımlamalar → Pratik Dinamik Gruplama`

## Ekran Tanımı

Distribütör pratik dinamik grubu bilgilerinin tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kriter Tipi | Tanımlanan işlem tipinin programda belirlenen Kod ya da Grup Koduna göre koşulun girildiği sahadır. |
| Grup | Distribütör Gruplarının seçildiği alandır. |
| Ek Grup | Distribütör Ek Gruplarının seçildiği alandır. |
| Kriter | Grup ve Ek Gruplar seçildikten sonra filtrelenen distribütörlerin listelendiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İşlemi Başlat | Değişiklikleri tamamlayarak işlemleri başatır. |
| Listele | Verilen değerlere bağlı olarak listeleme işlemi gerçekleştirir. |
| Temizle | Listeyi temizlemek için kullanılır |
| Kapat | Yapılan değişiklikleri kaydetmenden ekranı kapamaya çalışır. |
