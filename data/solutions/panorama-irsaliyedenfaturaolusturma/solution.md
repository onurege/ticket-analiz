# Irsaliyeden Fatura Oluşturma

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Alış İşlemleri → Irsaliyeden Fatura Oluşturma`

## Ekran Tanımı

Panorama programında dış ortamdan entegrasyon ile veya belge dönüşüm ekranlarında İş Bildirimi seçeneği işaretlendiğinde oluşan belgelerin listelendiği bölümdür.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yükleme Kodu | Yapılan işleme ait program tarafından otomatik verilen kayıt numarasıdır. |
| Distribütör Kodu | Belgenin hangi Distribütör için kesileceğinin tanımlandığı alandır. |
| Başlangıç Tarihi | Alış Irsaliye fatura belgelerinin hangi tarihten başlayacağı bilgisidir. |
| Bitiş Tarihi | Alış Irsaliye fatura belgelerinin hangi tarihte biteceği bilgisidir. |
| Başlangıç Sevk Tarihi | Alış Irsaliye fatura belgelerinin hangi Sevk tarihinden başlayacağı bilgisidir. |
| Bitiş Sevk Tarihi | Alış Irsaliye fatura belgelerinin hangi Sevk tarihinden biteceğinin bilgisidir. |
| Hareket Tipi | Alış Irsaliye fatura için tanımlanan hareket tipinin seçildiği alandır. Zorunlu bir alandır. |
| Satış Temsilcisi | Alış Irsaliye fatura hangi Satış Temsilcileri üzerinden listelendiği seçiminin yapıldığı alandır. |
| Bölge Kodları | Alış Irsaliye fatura hangi müşteri Bölge üzerinden listeleneceği seçiminin yapıldığı bilgisidir. |
| Seçilen Araç | İrsaliyelerin yükleme yapılacağı araç seçilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Temizle | Listedeki verileri temizlemek için kullanılır. |
| Sevk Tarihleri | Sevk tarhi ayarlamak için kullanılan ekrandır. |
| Listele | Girdilere göre listeleme işlemleri yapılmaktadır. |
| Belgeleştir | Sevk Tarihleri ekranını açmaktadır. |
| Kaydet | Belgeyi kaydetmek için kullanılır. |
| Kapat | Belge ekranını kapatmak için kullanılır. |
| Yenile | Belge detay listesini yenilemek için kullanılır. |
| Belge İzle | Belge detay satırında seçili belgeyi incelemek için kullanılır. |
| Transfer | Listelenen siparişlerin istenilen yere transfer işlemi gerçekleştirilir. |
| Yukarı | “Yukarı” – “Aşağı “ butonları ile oluşacak belgelerin öncelik sıralaması değiştirilebilir. |
| Aşağı | “Yukarı” – “Aşağı “ butonları ile oluşacak belgelerin öncelik sıralaması değiştirilebilir. |
