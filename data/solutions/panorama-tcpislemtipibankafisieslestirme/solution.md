# İşlem Tip- Banka Fişi Eşleştirme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → Ticari Paket → Eşleştirme → İşlem Tip Banka Fişi Eşleştirme`

## Ekran Tanımı

Panorama üzerinde kullanılan Havale-EFT tahsilat ve ödeme işlem tiplerine karşılık ; Ticari pakette kullanılan banka fişi bilgisinin girildiği menüdür.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Adı | İşlem yapılacak Distribütör bilgisi seçilir. |
| Tp Kod | Listelenen Havale – EFT işlem tip kayıtlarına karşılık Ticari Paket programında kullanılan banka fişi kod bilgisi girilir. Giriş yapıldığı anda kaydetme işlemi gerçekleşecektir. |
| Kod | Listelenen Havale – EFT işlem tip kayıtlarının Panoramada kullanılan kod bilgisi gösterilir. |
| Ad | Listelenen Havale – EFT işlem tip kayıtlarının Panoramada kullanılan ad bilgisi gösterilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kapat | Form kapatılır. |
| Yenile | Sayfanın yenilenmesini sağlar. |
| Düzenle | Seçilen satırda Tp Kod alanını aktif hale getirir ve değer girişi yapılmasını sağlar. |
| Yön Tuşları | Listelenen Havale – EFT işlem tip kayıtlarında ileri-geri sayfa geçişi yapılmasını sağlar. |
