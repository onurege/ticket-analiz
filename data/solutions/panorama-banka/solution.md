# Banka

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tanımlamalar → Banka`

## Ekran Tanımı

Banka bilgilerinin tanımlandığı ekrandır. Öncelikle ekrana sisteme girilmiş banka listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Banka tanımına program tarafından verilen artışlı numaradır. |
| Kod | Tanımlanan banka için kullanıcının belirlediği kod bilgisidir. |
| Ad | Bankanın adının girildiği metin sahasıdır. |
| Durum | Bankanın durum bilgisidir. Aktif ve Pasif değerlerini alır. |
| Referans 1 | Tanımlanan bankaya ait referans kod1 bilgisidir. Entegrasyonlar için kullanılır. |
| Referans 2 | Tanımlanan bankaya ait referans kod2 bilgisidir. Entegrasyonlar için kullanılır. |
| İl Kod | Bankanın bulunduğu il bilgisinin girilebileceği rehber alandır. |
| İlçe Kod | Bankanın bulunduğu ilçe bilgisinin girilebileceği rehber alandır. |
| Posta Kodu | Bankanın posta kodu bilgisidir. |
| İlgili Kişi | Bankadaki yetkili kişinin girilebileceği alandır. |
| Adres | Bankanın adres bilgisinin girilebileceği metin alanıdır. |
| Telefon | Bankanın telefon bilgisinin girilebileceği alandır. |
| Faks | Bankanın faks bilgisinin girilebileceği alandır. |
| Swift Kodu | Bankanın uluslararası para transferlerinde kullandığı kod bilgisidir. |
