# ST Yonetim Hiyerarsi

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Parametreler → ST Yönetim Hiyerarşi`

## Ekran Tanımı

Satış temsilcisi yönetim hiyerarşi bilgisidir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Seviye Sayısı | Hiyerarşi seviye bilgisidir. |
| Seviye Ayıracı | Seviye ayırıcı bilgisidir. |
| Sıra | Hiyerarşi sıra numarası bilgisidir. |
| Karakter Sayısı | Kararkter sayısı bilgisidir. |
| Seviye Adı | Personelin isim bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İşlemi Başlat | Yönetim hiyerarşisini başlatmaya yarayan butonudur. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
| Yenile | Listeyi yenilemek amaçlı kullanılır. |
| Yeni | Yeni kayıt ekleme amaçlı kullanılır. |
| Düzenle | Hiyerarşi bilgilerini düzenlemek için kullanılır. |
| İzle | Kayıtları incelemek için kullanılır. |
| Sıra | Hiyerarşi seviye bilgisidir. |
| Karakter Sayısı | Seviye ayırıcı bilgisidir. |
| Seviye Adı | Seviye adı bilgisidir. |
