# Depolar Arası Transfer

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `İş Akış → Onay İşlemleri → Depolar Arası Transfer`

## Ekran Tanımı

Onaylanacak/Reddedilen/Onaylanan alış siparişlerinin listelendiği ekrandır. Onaylama ve red işlemi bu ekrandan yapılmaktadır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Kullanıcı Grup | Listeleme yapılacak kullanıcı grubu seçilir. |
| Onay Bekleyenler | Sadece onay bekleyenlerin listelenmesi sağlanılır. |
| Onayladıklarım | Sadece onaylananların listelenmesi sağlanır. |
| Reddettiklerim | Sadece reddedilenlerin listelenmesi sağlanılır. |
| İşlem | İşlem tipi “Bekliyor” veya “Onay” seçilerek kriter belirlenmesi yapılıp “Kaydet” butonuna tıklanılarak işlem tipinin değiştirilmesi sağlanılır. |
| Ret Neden | Ret nedenini belirtildiği açıklama alanıdır. |
| Belge Kodu | Belge kaydedildiğinde Panorama tarafından verilen otomatik artışlı numaradır. |
| İşlem Tarihi | Depo transfer belgesinin işlem tarihi bilgisidir. Otomatik günün tarihi gelir, istenirse değiştirilebilir. |
| Hareket Tipi | Depo transfer belgesi hareket tipidir. Belge detayında gelen ürünler hareket tipine göre filtrelenir. |
| Giriş Depo | Depo girişin kaydının yapıldığı depo bilgisidir. |
| Çıkış Depo | Depo çıkışın kaydının yapıldığı depo bilgisidir. |
| Depo Hareket Nedeni | Depo hareket nedeni bilgisidir. |
| Bozuk İade Nedeni | Bozuk iadelerin hangi depoda bulunacağı seçilir. |
| Özel Kod | Depo transfer belgesinin özel kod bilgisidir. |
| Matbu No | Depo transfer belgesinin matbu no bilgisidir. |
| Onay | Onay durum bilgisidir. |
| Giriş Tipi | Belgenin oluşturulma yeri bilgisidir. Panorama, Panorama Mobile Aktarım değerlerini alabilir. |
| Belge Durumu | Durum bilgisi kriterinin belirlendiği sahadır. |
| Aktarım Durumu | Sadece Fatura belgeleri için kaydın ticari pakete aktarılıp aktarılmadığı bilgisidir. Aktarıldı ve ya aktarılmadığı değerlerini alır. |
| Net Tutar | Belge net tutar bilgisidir. (Brüt Tutar - İskonto Tutarı + KDV Tutarı) Formülü ile hesaplanır. |

| Alan | Açıklama |

| İzle | Tanımın izle ile açılmasını sağlar. |

| Kapat | Ekranı kaydetmeden kapatır. |
| Ret Neden | Ret Nedenin kod bilgisidir. |
| Geri Gelme Nedeni | Önceki adıma gönder adımından gelen açıklama bilgisidir. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| Distribütör Adı | Distribütör adı bilgisidir. |
| Hareket Tipi | Hareket tip bilgisidir. |
| İşlem Tarihi | İşlem tarihi bilgisidir. |
| Takip Kodu | Takip kodu bilgisidir. |
| Yıl | Yıl bilgisidir. |
| Son İşlem Tarihi | Son işlem tarihi bilgisidir. |
