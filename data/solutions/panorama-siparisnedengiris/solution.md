# Sipariş Neden Giriş

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri → Satış Kriterleri → Sipariş Neden Giriş`

## Ekran Tanımı

Sipariş Neden Giriş tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Kapak Uygulama Kriterleri tanımına program tarafından verilen artışlı numaradır. |
| Ad | Şipariş Neden Giriş bilgisidir. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Ekranda bilgileri girilmiş kullanıcıyı yarar. |
| İptal | Ekranı kaydetmeden Kapatma işlemi yapar. |
