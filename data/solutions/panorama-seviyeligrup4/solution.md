# Seviyeli Grup 1

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım Kanalı → Tanımlamalar → Seviyeli Grup 1`

## Ekran Tanımı

Distribütör Seviyeli Grup 1 tanımlamalarının tanımladığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Seviyeli Grup | Distribütör seviye listesini ağaç yapısında gösterir. |
| Grup Kodu | Grup Kodu bilgisidir. |
| Grup Adı | Grup adı bilgisidir. |
| Personel | Seviye gruba eklenecek personel bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yeni | Yeni Personel eklemek için kullanılır. |
| Düzenle | Personel bilgilerini düzenlemek için kullanılır. |
| Sil | Kayıtlı grup veya personel silmek amacıyla kullanılır. |
| Kaydet | Değişiklikleri kaydeder. |
| İptal | Yapılan değişiklikleri iptal eder. |
| Kapat | Ekranı kapatmak için kullanılır. |
