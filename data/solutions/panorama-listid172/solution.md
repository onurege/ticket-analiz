# Depo Hareket Nedeni

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Tanımlamalar → Depo Hareket Nedeni`

## Ekran Tanımı

Depo Hareket Nedenlerinin tanımlandığı alandır. Öncelikle ekrana, sisteme tanımlı olan Depo Hareket Nedenlerinin Listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Ad | Depo Hareket Nedenlerine verilen isimdir. Girilmesi zorunlu sahadır. |
| Distribütör Kodu | İlgili Distribütörün seçileceği sahadır. |
| Uygulama Yeri | Tanımlanan kayıtın hangi uygulamada kullanılacağı belirtilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemin kaydedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
