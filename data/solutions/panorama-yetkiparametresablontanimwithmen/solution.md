# Yetki ve Parametre Şablon

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Sistem Yönetimi→ Genel Parametreler → Yetki ve Parametre şablon`

## Ekran Tanımı

Panorama ve PanoramaMobile programları için gerekli parametrelerin ayarlandığı

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Tümü | Parametrelerin tamamını ekranda listeler. |
| Genel | Genel kullanılan parametrelerin bulunduğu başlıktır. |
| Mobil Satış Dağıtım | Backoffice parametrelerin bulunduğu başlıktır. |
| Panorama Mobile | Mobile parametrelerin bulunduğu başlıktır. |
| PM Bilgi Hazırlama | Bilgi hazırlamada kullanılan parametrelerin bulunduğu başlıktır. |
| PM Bilgi İşleme | Bilgi işlemede kullanılan parametrelerin bulunduğu başlıktır |
| Servis İşlemleri | EnrouteCalldesk Parametrelerinin bulunduğu başlıktır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Kapat | Değişiklikleri kaydettikten sonra kapatma işlemini yapar |
| Kapat | Açılan ekranı kapatır. |
