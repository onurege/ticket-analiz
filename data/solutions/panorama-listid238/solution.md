# Satış Grup Yöneticisi

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Tanımlamalar → Satış Grup Yöneticisi`

## Ekran Tanımı

Satış grup yöneticisi tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Satış Grup Yöneticisine program tarafından verilen artan matrisli numaradır . |
| Ad | Satış Grup Yöneticisinin isminin girildiği sahadır. |
| Üretici Yetkisi | Yetkili olduğu üretici firma bilgisidir. |
| Distribütör Kodu | Distribütör kodunun seçildiği sahadır. Satış Grup Yöneticisinin hangi distribütöre bağlı olduğu belirlenir. |
| Personel Kodu | Satış Grup Yöneticisine herhangi bir Personel tanımı yapıldıysa, seçildiği sahadır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemleri kaydetmek için kullanılan butondur. |
| İptal | Ekranı kaydetmeden kapatmaya yarayan butondur. |
