# Kontrat Tanımlama Onay

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `İş Akış → Onay İşlemleri → Kontrat Tanımlama Onay`

## Ekran Tanımı

Onaylanacak/Reddedilen/Onaylanan alış siparişlerinin listelendiği ekrandır. Onaylama ve red işlemi bu ekrandan yapılmaktadır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Kullanıcı Grup | Listeleme yapılacak kullanıcı grubu seçilir. |
| Onay Bekleyenler | Sadece onay bekleyenlerin listelenmesi sağlanılır. |
| Onayladıklarım | Sadece onaylananların listelenmesi sağlanır. |
| Reddettiklerim | Sadece reddedilenlerin listelenmesi sağlanılır. |
| İşlem | İşlem tipi “Bekliyor” veya “Onay” seçilerek kriter belirlenmesi yapılıp “Kaydet” butonuna tıklanılarak işlem tipinin değiştirilmesi sağlanılır. |
| Kod | Belge kaydedildiğinde Panorama tarafından verilen otomatik artışlı numaradır. |
| Durum | Kontrat durum bilgisidir. Aktif, Pasif değerlerini alır. |
| Distribütör Kodu | Kontratın hangi distribütör için tanımlandığı bilgisidir. |
| Onay Durumu | İş akışı kullanıldığı durumda tanımın onay durumunun görüntülendiği alandır. Onaylandı/Onaylanmadı değerlerini alabilir. |
| Müşteri Kodu | Müşterinin kod bilgisidir. |
| Ödeme Durumu | Kontratın ödeme durumunun belirtildiği alandır. |
| Satış Temsilcisi | Müşterini rutunda olduğu satış temsilcisi bilgisidir. |
| Kotalı Sözleşme | Kotalı Sözleşme bilgisidir. |
| Başlangıç Tarihi | Kontrat Tanımlama onayı başlangıç tarihinin girildiği alandır. |
| Bitiş Tarihi | Kontrat Tanımlama onayı bitiş tarihinin girildiği alandır. |
| Ödeme Türü | Ödemenin yapılma şekli bilgisidir |
| Tutar | Tutar bilgisidir. |
| Sözleşme Sınıfı | Sözleşme sınıfı bilgisidir. |
| Bakım Masrafı | Bakım masrafı bilgisidir. |
| ROI Değeri | Yatırımın size ne kadar getiri sağladığını belirten değer bilgisidir. |
| Ünite Bedeli | Ünite bedeli bilgisidir. |
| Tutar Öneri | Tutar Öneri bilgisidir. |
| Sökülecek Stant | Sökülecek Stant bilgisidir |
| Revize/Red Nedeni | Reddetme durumunda neden açıklama alanıdır. |

| Sözleşme Notu | Sözleşme notu bilgisidir. |

| Stant | Stant bilgisidir. |
| Raf Sayısı | Raf Sayısı bilgisidir. |
| Sutun Sayısı | Sutun Saıyısı bilgisidir. |
