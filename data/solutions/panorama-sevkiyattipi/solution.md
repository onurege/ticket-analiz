# Sevkiyat Tipi

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Tanımlamalar → Sevkiyat Tipi`

## Ekran Tanımı

Sevkiyat Tipinin tanımlandığı alandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Sevkiyat Tipi Kod | Sevkiyat tipi bilgisidir. |
| Sevkiyat Tipi Adı | Sevkiyat tipi adı bilgisidir. |
| Distribütör Kodu | İlgili Distribütörün seçileceği sahadır. |
| Uygulama Yeri | Tanımlanan kayıtın hangi uygulamada kullanılacağı belirtilir. |
| Merkez Planlı Sevkiyat | Merkez planlı sevkiyat yapılsın mı yapılmasın mı bilgisidir. |
| Ürün Girişinde Seçimli Özel Kod Alanı Pasif Olsun | Ürün girişinde seçimli özel kod alanı pasif olsun mu olmasın mı seçimidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Kapat | Yapılan işlemin kaydedilmesini ve kapatması sağanır. |
| Kapat | Ekranı kapatır. |
