# Ürün Sorgulama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Satış İşlemleri → Ürün Sorgulama`

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Ürün | Sorulanmak istenen ürün alanıdır. |
| Marka/Mevsim | Marka ve mevsim seçimi yapılıcak alandır. |
| Marka/Grup | Marka ve grup bilgisi alanıdır. |
| Stok | Stok seçimi yapılıcak alandır. |
| İsk 1/2/3/4/5 | İskonto 1-5 bilgisidir. |
| Temizle ve Listele | Kriterleri ve detayı temizlemek ve belgeleri detay ekranında listelemek için kullanılır. |
| Hesapla ve Listele | Girilen ürünlerin hesaplama işlemini gerçekleştirip listelenmesini sağlar. |
| Pos Sipariş Girişi | Müşteri, ST, Depo bilgilerinin girildiği ekranı açmaya yarayan Sipariş girişi butonudur. |
| Pos İrsaliye Girişi | Müşteri, ST, Depo bilgilerinin girildiği ekranı açmaya yarayan İrsaliye girişi butonudur. |
| Pos Fatura Girişi | Müşteri, ST, Depo bilgilerinin girildiği ekranı açmaya yarayan Fatura girişi butonudur. |
| Kapat | Ekranı Kapatma işlemi yapar. |
| İzle | Seçilen belgeyi gözlemlemek için kullanılır. |
