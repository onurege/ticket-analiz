# Rutları izleme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Harita İşlemleri → Rut İzleme`

## Ekran Tanımı

Rut izleme Ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| ST/SE Atanma | Satış temsilcisi Atandı/Atanmadı seçimidir. |
| ST/SE Elenmanları | Satış temsilcisi elemanları seçilir. |
| Tarih | Tarih bilgisidir. |
| Takip Kodu | Takip kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Ekranı ve bilgileri yenilemek için kullanılır |
| Ortala | Haritayı ortalamaya yarayan butondur. |
| Temizle | Yapılan işlemleri temizleme butonudur. |
| Arama | Rut noktası arama butonudur. |
| Kapat | Rut ekranını kapatma butonudur. |
