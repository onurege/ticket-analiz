# Ticari Paket Veri Çekme Yetkilendirme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → Ticari Paket → Ticari Paket Veri Çekme Yetkilendirme`

## Ekran Tanımı

Ticari pakete aktarılacak alanların kısıtlaması,zorunlu bırakılması gibi yetkilendirme tanımları yapılır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yetki Kodu | Sistem tarafından verilen otomatik artışlı numaradır. |
| Distribütör Kriteri | Yetki tanımı yapılan distribütör bilgisi seçilir. Çoklu seçim yapılabilir. |
| Saha Kod | Aktarım yapılacak alan adı bilgisidir. |
| Aktarım | Evet – Hayır değerini alabilir. Evet seçilir ise ilgili alan Smart Connect ekranında seçilmiş olarak , Hayır seçilir ise ilgili alan Smart Connect ekranında seçilmemiş olarak gelecektir. |
| İşlem Yetkisi | Değiştirilemez – Değiştirilebilir değerini alabilir. Değiştirilemez seçilir ise ilgili alan için ‘Aktarım’ seçeneği smart connect değiştirilemeyecektir. Değiştirilebilir seçilir ise ilgili alan için ‘Aktarım’ seçeneği smart connect değiştirilebilecektir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfanın yenilenmesini sağlar. |
| Düzenle | Seçilen satırın düzenle ile açılarak yeki değeri seçilmesini sağlar. |
| İzle | Seçilen satırı izle modunda açar. |
| Kaydet | Formun kaydedilmesini sağlar. |
| Kapat | Formun kapatılmasını sağlar. |
