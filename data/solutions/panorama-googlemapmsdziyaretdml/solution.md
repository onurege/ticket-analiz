# Ziyaret İzleme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Harita İşlemleri → Ziyaret İzleme`

## Ekran Tanımı

Ziyaret izleme Ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Tarih | Tarih bilgisidir. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| ST/SE Elenmanları | Satış temsilcisi elemanları seçilir. |
| Ziyaretleri Göster | Ziyaretleri gösterip gösterilmiceğinin seçeneğidir. |
| Rut Dışı Ziyaretler Gösterilmesin | Rut dışında ziyaretler gösterilip gösterilemeyeceğini seçebilirsiniz. |
| Beklemeleri Göster | Bekleyen Ziyaretleri gösterip göstermeyeceği seçeneğidir. |
| Gps Yolunu Göster | Gps yolunu gösterip gösterilmeyeceği seçeneğidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Ekranı ve bilgileri yenilemek için kullanılır |
| Ortala | Haritayı ortalamaya yarayan butondur. |
| Temizle | Yapılan işlemleri temizleme butonudur. |
| Arama | Ziyaret noktası arama butonudur. |
| Kapat | Ziyaret ekranını kapatma butonudur. |
