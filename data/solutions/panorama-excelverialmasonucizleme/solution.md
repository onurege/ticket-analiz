# Veri Alma Sonuç İzleme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ İşlemler → Veri Alma Sonuç İzleme`

## Ekran Tanımı

Excel üzerinden Panorama’ya veri alma işlemi sonuçları kontrol edilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Veri aktarım kontrolü yapılacak distribütör bilgisi seçilir. |
| Görev Kod | Veri aktarım kontrolü yapılacak görev kodu seçilir. |
| Başlangıç Tarihi | Aktarım kontrolü yapılacak başlangıç tarih bilgisi girilir. |
| Bitiş Tarihi | Aktarım kontrolü yapılacak bitiş tarih bilgisi girilir. |
| Başlama Tarihi | Veri aktarımı başlangıç tarih bilgisi gösterilir. |
| İşlem No | Veri aktarımı işlem no bilgisi gösterilir. |
| İşlem Durum | Veri aktarımı işlem durum bilgisi gösterilir. |
| İşlem Sonuç | Veri aktarımı işlem sonuç bilgisi gösterilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Giriş yapılan kriterlere göre veri aktarım sonuçları listelenir. |
| Detay Sonuç İzleme | Listelen sonuç kayırları için detay bilgi ekranı açılır. |
| Kapat | Ekran kapatılır. |
