# Pratik Fiyat Tanımlama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriteri → Fiyat İşlemleri → Pratik Fiyat Tanımlama`

## Ekran Tanımı

Sistemde tanımlı olan ürünlere fiyatların toplu bir şekilde tanımlanabileceği ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Başlangıç Tarihi | Yeni tanımlanacak fiyat bilgilerinin başlangıç tarihinin girildiği alandır. Buraya girilecek tarih ile tanımlanan fiyat bilgisi aktif olacaktır. |
| Bitiş Tarihi | Yeni tanımlanacak fiyat bilgilerinin bitiş tarihinin girildiği alandır. Buraya girilecek tarih ile tanımlanan fiyat bilgisi ilgili tarihi geçtiğinde tanımlanan fiyat hareket ekranlarında çıkmayacaktır. |
| Uygulama Yeri | Fiyat tanımının uygulama yeri bilgisidir. Panorama, Quest, Panorama, Quest, Münferit seçilebilir. |
| Distribütör Kriter Tipi | Distribütörlerin seçileceği kriterlerin girildiği alandır değer olarak (Kod, Grup, Ek Grup, Dinamik Grup) değerlerini almaktadır. |
| Distribütör Kriter | Distribütör Kriter Tipi ne göre değişkenlik gösteren bir alandır. Örneğin Distribütör Kriter Tipi “Kod” seçilirse sisteme kayıtlı tüm Distribütörler gelir ve tek bir Distribütör seçilebilir fakat Distribütör Kriter Tipi “Grup” seçilir ise Distribütör grupları seçilir. |
| Müşteri Kriter Tipi | Carilerin seçileceği kriterlerin girildiği alandır değer olarak (Kod, Grup, Ek Grup, Dinamik Gurup) değerlerini almaktadır. |
| Müşteri Kriteri | Müşteri Kriter Tipi ne göre değişkenlik gösteren bir alandır. Örneğin Müşteri Kriter Tipi “Kod” seçilirse sisteme kayıtlı tüm Müşteriler gelir ve tek bir Müşteri seçilebilir fakat Müşteri Kriter Tipi “Grup” seçilir ise Müşteri grupları seçilir ve o gruba dahil olan tüm müşterilere fiyat tanımlanmış olur. |
| Ek Saha Seçenek | Müşteri kriter tipi Ek saha seçilmesi durumunda bu saha aktif olur ve kriter olarak verilen ek saha seçilir. |
| Ürün | Tanımlanmış ürün bilgisidir. |
| Para Birimi | Toplu fiyat olarak tanımlanacak fiyatların hangi para birimine göre tanımlanacağının seçildiği alandır. |
| Satış Kdv Oranı | Satış kdv oranı bilgisidir. |
| Alış Kdv Oranı | Alış kdv oranı bilgisidir. |
| Satış | ÖTV/ÖİV Uygulanacak belge türleri seçilir. |
| Alış | ÖTV/ÖİV Uygulanacak belge türleri seçilir. |
| Satıcıya İade | ÖTV/ÖİV Uygulanacak belge türleri seçilir. |
| İade | ÖTV/ÖİV Uygulanacak belge türleri seçilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İşlemi Başlat | Yapılan işlemin kaydetmeyi ve başlatmayı sağlar. |
| Listele | Girilen verilere göre listeleme işlemi gerçekleştirir |
| Temizle | Listeyi ve girilen değerleri temizlemeye yarar. |
| Kapat | Ekranı kapatır. |
| Yenile | Detaydaki listeyi yeniler. |
| Düzenle | Detayda düzenleme yapmak için kullanılır. |
| İzle | Detaydaki kaydı görüntülemek için kullanılır. |
