# E-İskonto Tipi

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ve Pazarlama Kriterleri → İskonto Promosyon → E-İskonto Tipi`

## Ekran Tanımı

E-puan tip bilgilerinin tanımlandıpğı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | E-puan Tip tanımına program tarafından verilen artışlı numaradır. |
| Distribütör Kriter Tipi | Tanımın Hangi Distribütör için geçerli olacağını belirlemek için kullanılır. Kriter tipi Kod, Grup, Ek Grup, Dinamik Grup değerleri alır. |
| Distribütör Kriteri | Kriter tipinde yapılan seçime göre değerlerin seçildiği alandır. Örneğin Kriter tipi kod seçilir ise Distribütör kodu listesi üzerinden distribütör seçimi yapılır. Boş bırakıldığı durumda tüm distribütörler için geçerlidir. |
| Dağıtım Kriter Tipi | Tanımın Hangi Ürün Kriteri için geçerli olacağını belirlemek için kullanılır. Kriter tipi Kod, Grup, Ek Grup, Dinamik Grup değerleri alır. |
| Dağıtım Kriteri | Dağıtım Kriter Bilgisinin seçildiği alandır. |
| Durum | E-puan tip tanımının durum bilgisinin seçildiği alandır. Aktif ve Pasif değerini alabilir. |
| Dağıtım Sıra | Uygulanacak E Puan iskontosunun hangi iskonto sırasında uygulanacağının belirlendiği alandır. |
| Kullanım Yüzdesi | Noktanın biriken puanlarından yüzde kaçını kullanabileceği bilgisidir. |
