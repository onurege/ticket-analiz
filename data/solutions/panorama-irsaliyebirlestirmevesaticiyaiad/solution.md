# İrsaliye Birleştirme ve Satıcıya İade Faturası Oluşturma

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Satış İşlemleri → İrsaliye Birleştirme Ve Satıcıya İade Fatura Oluşturma`

## Ekran Tanımı

İrsaliye Birleştirme Ve Satıcıya İade Fatura Oluşturma oluşturulduğu ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yükleme Kodu | Yapılan işleme ait program tarafından otomatik verilen kayıt numarasıdır. |
| Distribütör Kodu | İrsaliye kayıtlarının bağlı olduğu Distiribütör kodunun seçildiği alandır. |
| Başlangıç Tarihi | Satış İrsaliye belgelerinin hangi tarihten başlayacağı bilgisidir. |
| Bitiş Tarihi | Satış İrsaliye belgelerinin hangi tarihte biteceği bilgisidir. |
| Başlangıç Sevk Tarihi | Satış İrsaliye belgelerinin hangi Sevk tarihinden başlayacağı bilgisidir. |
| Bitiş Sevk Tarihi | Satış İrsaliye belgelerinin hangi Sevk tarihinde biteceği bilgisidir. |
| Hareket Tip | Satış İrsaliyelerin hangi hareket tipinde geleceğinin seçiminin yapıldığı alandır. |
| Satış Temsilcileri | Satış İrsaliyelerinin hangi Satış Temsilcileri üzerinden listelendiği seçiminin yapıldığı alandır. |
| Bölge Kodları | Satış İrsaliyelerin hangi müşteri Bölge üzerinden listeleneceği seçiminin yapıldığı bilgisidir. |
| Uygulanan İskonto Promosyonları İptal Et ve Yeniden Hesapla | Uygulanan iskonto promosyonlari iptal et ve yeniden hesapla seçeneğidir. |
| Seçilen Araç | İrsaliyelerin yükleme yapılacağı araç seçilir. |
| Fatura Tarihi | Oluşacak Fatura belgelerinin tarihlerinin belirlendiği alandır. |
| Fatura Hareket Tipi | Oluşacak Fatura belgelerinin Hareket Tiplerinin belirlendiği alandır. |
| İade Nedeni | Oluşacak Fatura belgeleri için İade Nedeni belirlendiği alandır. |
| Vade Günü | Oluşacak Fatura belgeleri için Vade Günü belirlendiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Temizle | Sahaya girilen bilgilerin silmesini sağlar. |
| Sevk Tarihleri | Sevk Tarihleri ekranının açılmasını sağlamaktadır. |
| Listele | Girilen kriterlere göre ilgili kayıtların listelenmesini sağlamaktadır. |
| Sadakat | Sadakat İskontosu Seçimi ekranını açmaktadır. |
| Belgeleştir | Bu butona basıldığında “Yüklenemeyenler” hariç diğer tüm araçlardaki irsaliyeler araç sırası ve araç içerisindeki irsaliye sırasına göre Satıcıya iade faturaları oluşturulma işlemine girer. |
| Kaydet | Bu buton yardımı ile yükleme kaydedilebilir. Kaydet butonuna basıldığında “Kriterler” gruplamasındaki ve yüklenemeyenler aracı veritabanına kaydedilmez. |
| Kapat | Ekranı Kapatma işlemi yapar. |
| Yenile | Liste kayıtlarının güncellenmesini sağlamaktadır. |
| Belge İzle | Satış İrsaliye ürün detay bilgilerinin izlenmesini sağlar. |
| Transfer | Yüklenmemiş olan İrsaliyelerin seçilenlerin araca transferi sağlanır. |
| Yukarı | Belgele sıralamasını aşağı yönde yer değiştirmesini sağlar |
| Aşağı | Belgele sıralamasını yukarı yönde yer değiştirmesini sağlar |
