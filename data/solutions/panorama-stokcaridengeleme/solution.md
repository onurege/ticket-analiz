# Stok/Cari Sıfırlama İşlemleri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → İşlemler → Stok/Cari Sıfırlama İşlemleri`

## Ekran Tanımı

Stok/Cari sıfırlama işlemleri ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| Stok Sıfırlama | Stok Sıfırlama yapılacak veya yapılmayacak seçeneğidir. |
| Cari Sıfırlama | Cari Sıfırlama yapılacak veya yapılmayacak seçeneğidir. |
| Tüm Müşterilere Ait Carileri Sıfırla | Tüm müşteriler ait carileri sıfırlama yapılacak veya yapılmayacak seçeneğidir. |
| Depo Kodu | Depo kodu bilgisidir. |
| Satış Temsilcisi | Satış temsilcisi bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İşlemi Başlat | Yapılan işlemleri başlatma butonudur. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
