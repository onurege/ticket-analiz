# Nokta Devri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → İşlemler → Nokta Devri`

## Ekran Tanımı

Müşteri Nokta devir işlemi ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Devir Tarihi | Nokta devri yapılacak yerin devir tarihi girilir. |
| Distribütör Kodu | Nokta devriyle ilgili distribütör seçilir. |
| Müşteri | Müşteri kodunun seçildiği alandır. |
| İptal Nedeni | Nokta devri yapılacak yerin iptal nedeni yazılır. Zorunlu alandır. |
| Müşteri Bakiyesi Alt Limit | Müşteri bakiyesi alt limit bilgsidir. |
| Müşteri Bakiyesi Devredilsin | Müşteri bakiyesi devredilsin istenirse seçilebilir. |
| E-İskonto Bakiyesi | E-iskonto bakiyesi devrinin de dahil olması isteniliyorsa e-iskonto bakiyesi kutucuğu işaretlenmeli. |
| Rut Tanımı Devredilsin | Rut tanımı devredilsin istenirse seçilebilir. |
| İskonto Bilgileri | İskonto bilgileri gösterilsin istenirse seçilebilir. |
| Kontrat Bilgileri | Kontrat bilgileri gösterilsin istenirse seçilebilir. |
| Yeni Müşteri Kodu | Nokta devri yapılan yeni müşteri kodu yazılır. Zorunlu alandır. |
| Ünvan | Nokta devri yapılan unvanı yazılır. Zorunlu alandır. |
| Kısa Adı | Nokta devri yapılan kısa adı yazılır. |
| İlgili Kişi | Nokta devriyle ilgili kişi yazılır. Zorunlu alandır. |
| İlgili Kişi 2 | Nokta devriyle ilgili diğer kişi yazılır. |
| Vergi Tipi | Vergi tipi için KDV’den Muaf, Gerçek Kişi ya da Tüzel Kişi seçimi yapılır. |
| Vergi Dairesi | Nokta devri yapılan vergi dairesi seçilir. |
| Vergi Numarası | Nokta devri yapılan vergi numarası girilir. |
| TC Kimlik No | Nokta devri ile ilgili kişinin TC Kimlik No su yazılır. |
| Ruhsat Daire | Nokta Devrinin yeni ruhsat dairesi yazılır. |
| Ruhsat No | Nokta Devrinin yeni ruhsat No su yazılır. |
| Web Adresi | Nokta Devrinin yeni web dairesi yazılır. |
| Mail Adresi | Nokta Devrinin yeni mail dairesi yazılır. |
| Cep Tel No | Nokta Devri ile ilgili kişinin cep telefon numarası yazılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İşlemi Başlat | İşlemi başlatmak için kullanılır. |
| Kapat | Ekranı kapatır. |
