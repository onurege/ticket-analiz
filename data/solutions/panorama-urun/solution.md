# Ürün

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Tanımlamalar → Ürün`

## Ekran Tanımı

Ürün tanımlamasının yapıldığı ekrandır. Öncelikle ekrana, sisteme girilmiş olan ürün listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Program tarafından verilen artışlı numaradır. |
| Ürün Kodu | Ürün kodu bilgisidir. Programda bu kod üzerinden işlemler gerçekleştirilir. |
| Distribütör Kodu | Ürünün hangi distribütör için tanımlandığı bilgisidir. |
| Ad | Ürünün tanım bilgisidir. |
| Kısa Adı | Ürünün kısa tanım bilgisidir. |
| Durum | Ürün durum bilgisidir. Aktif, Pasif değerlerini alır. |
| Uygulama Yeri | Ürünün uygulama yerinin belirtildiği alandır. Panorama / Quest / Panorama ve Quest /Münferit / Hepsi / Stokbar değerlerini alabilir. |
| Onay | İş akışı kullanıldığı durumda tanımın onay durumunun görüntülendiği alandır. Onaylandı/Onaylanmadı değerlerini alabilir. |
| Ürün Grup Kodu | Ürün Grup Kodu bilgisidir. |
| Ürün Ek Grup Kodu | Ürün Ek Grup Kodu bilgisidir. |
| Üretici Kodu | Ürünün üretici bilgisidir. |
| Dinamik Grup | Dinamik Kod bilgisidir. |
| Seviyeli Ürün-1 | Ürün seviyeli grup 1 bilgisidir. |
| Seviyeli Ürün-2 | Ürün seviyeli grup 2 bilgisidir. |
| Merkez Kod | Ürnün merkez kod bilgisidir. |
| Özel Kod | Ürüne ait özel kod bilgisinin tutulduğu alandır. |
| Birimler | Birim açıklama bilgisidir. Bir ürün için beş farklı birim tanımlanabilir. |
| Birim Çarpanı | Tanımlanan birimin içeriğindeki ana birim miktarının tanımlandığı çarpan alanıdır. |
| Barkodlar | Tanımlanan birime ait barkod bilgisidir. |
| Max.Ağırlık | Tanımlanan birime ait maximum ağırlık bilgisidir. |
| Satış KDV Oranı | Satış belgelerinde kullanılacak KDV oran bilgisidir. |
| Alış KDV Oranı | Alış belgelerinde kullanılacak KDV oran bilgisidir. |
| Ürün Tipi | Ürün tipi bilgisidir. Ürünler Normal / Karma Koli / POP / Bedelsiz / Depozito / Demirbaş / E-Iskonto Hediye / Paket Ürün / Hizmet / Sayaç / Stand / Palye tipinde tanımlanabilir. |
| Alkol Oranı | Ürün alkol oran bilgisidir. |
| Maks.Isk.Oranı |  |
| Maks.Isk.Tutarı |  |
| Ürün Birimi | Birden fazla birime sahip ürünlerde satışlarda kullanılacak hareket bilgisidir. Seçilen değer faturalarda otomatik gelir. |
| Uygulama Yeri | Ürünün uygulama yerinin belirtildiği alandır. Panorama /Quest / Panorama ve Quest /Münferit değerlerini alabilir. |
| GTIP No | Ürüne ait Gümrük Tarife Numarası bilgisidir. |
| Ağırlık | Ürüne ait ağırlık bilgisidir. Değer gram olarak girilmelidir. |
| En | Ürüne ait en bilgisidir. |
| Hacim | Ürüne ait hacim bilgisidir. |
| Boy | Ürüne ait boy bilgisidir. |
| Litre | Ürüne ait litre bilgisidir. |
| Yükseklik | Ürüne ait yükseklik bilgisidir. |
| El Bilgisayarı Sıra No | Ürünün satış esnasında el bilgisayarında kaçıncı sırada çıkacağı bilgisidir. |
| Hesap Kapatma | Ürünün hesap kapatma ekranlarında kullanılıp kullanılmayacağı bilgisidir. |
| Özel Vergi Tip | Özel vergi tipinin seçileceği alandır. Bu alan da ÖTV/OİV Yok, ÖTV ve ÖİV seçimleri yapılabilir. |
| KDV Hesap Tip | KDV hesap tipinin seçileceği alandır.Net Fiyattan veya Brüt fiyattan seçimi yapılır. |
| Seri Numarası Tipi | Ürün seri numarası tipinin seçileceği alandır. Sayısal / Metin / Hepsi (Sayısal Öncelikli) / Hepsi (Metin Öncelikli) seçimleri yapılabilir. |
| Seri Takibi Var mı ? | Ürünün seri takibi yapılıp yapılmayacağının belirlendiği alandır. Yok / Var / Var - Kontrolsüz değerlerini alabilir. |
| Satış Sip.Kontrol Birimi | Parametreye bağlı olarak satış siparişlerinde kontrol edilecek ürün birimini belirleyen alandır. |
| Rakip | Ürün olarak tanımlanan rakip ürün kartlarının satış ekranında çıkmaması için kullanılacak parametredir. Rakip ürün tanımları Quest modülünde hazırlanacak Anket/Bulunurluk tanımlamalarında kullanılır. |
| Ağırlık | Ağırlık barkodu kullanılan ürünler için seçilecek parametredir. Bu parametre ile ürünün ağırlık değeri barkod üzerinden okutulur. |
| Satır tutarından miktar hesaplanabilsin mi? | St parametresine bağlı olarak tutardan miktar hesaplamayı sağlamaktadır. |
| Sözleşmeli Ürün | Görev tanımına bağlı olarak işaretli olan ürün siparişlerini mail ile gönderimini sağlamaktadır. |
| Komisyon Uygulanacak mı? | Kredi kartı ile yapılacak satışlarda banka bazında tanımlanan komisyon tanımlarının satışta hangi ürünlere yansıtılacağının belirtildiği alandır. |
| İsk-Pro Muaf | İşaretli ise iskonto promosyon işlemlerine dahil olmamaktadır. |
| Kapak | Kapak iskontosu olarak tanımlanan ürünler için işaretlenecek alandır. |
| Promosyon Para Kart | Hizmet ürün tiplerinde aktif olur. Promosyon para kart giriş ekranından tanımların yapılmasını sağlar. |
| Seride check digit kontrolü var mı? | Serilerde check digit kontrolü yapmayı sağlar. |
| Demirbaş Özellik | Demirbaş ürün tiplerinde aktif olur. Demirbaş özelliklerin girilmesini sağlar. |
| Raf Ömrü | Ürünün raf ömrünün belirlendiği alandır. |
| Video URL | Ürüne ait video linki verilmesini sağlar. |
| Min. Ağırlık | Ürünün minimum ağırlık değerinin girildiği alandır. |
| Max. Ağırlık | Ürünün maximum ağırlık değerinin girildiği alandır. |
| Alış Sip. Sıra Miktarı | Parametrik olarak alış siparişlerinde sıra miktarı kontrolü yapılmasını sağlamaktadır. |
| Alış Sip. Palet Miktarı | Parametrik olarak alış siparişlerinde palet miktarı kontrolü yapılmasını sağlamaktadır. |
| Kapak Kriter Tipi | Kapak ürün kriter tipini belirlemektedir. |
| Kapak Kriteri | Kapak kriter tipine bağlı olarak ilgili rehber ekranını açmaktadır. |
| Seri ön ek karakter sayısı | Seri takibi var ve metin seçiliyse alan aktif olur. Seriye verilen ön ek karakter sayısını belirlemektedir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Ek Sahalar | Ürün için tanımlanan ek sahaların girildiği alandır. |
| Ürün Detayı | Karma koli ürün detayının tanımlandığı alandır. Ürün Tipi Karma Koli olarak tanımlanan ürünlerde ürün Detayı aktif hale gelir. |
| Depozito | Ürüne ait depozito ürün tanımlarının yapıldığı alandır. |
| Stand Bilgisi | Ürüne ait stand bilgisinin tanımlandığı alandır. Ürün tipinin Stand olduğu durumda kullanılabilmektedir. |
| Kaydet ve Yeni | Girilen verileri kaydeder ve yeni ürün girişi yapılır. |
| Kaydet ve Kapat | Girilen verileri kaydeder ve ekranı kapatır. |
| Kapat | Ekranı kapatır. |
