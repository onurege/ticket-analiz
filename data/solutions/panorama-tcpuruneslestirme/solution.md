# Ürün

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → Ticari Paket → Eşleştirme → Ürün`

## Ekran Tanımı

Panorama ile ticari pakette kullanılan ürün kodlarının farklı olduğu durumlarda, bu menü üzerinden ticari pakette kullanılan ürün kodları girilerek aktarım sağlanabilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Adı | İşlem yapılacak Distribütör bilgisi seçilir. |
| Tp Kod | Listelenen ürünlerin Ticari Paket programında kullanılan kod bilgisi girilir. Giriş yapıldığı anda kaydetme işlemi gerçekleşecektir. |
| Takip Kodu | Listelenen ürünlerin Panoramada kullanılan takip kod bilgisi gösterilir. |
| Ürün Kodu | Listelenen ürünlerin Panoramada kullanılan kod bilgisi gösterilir. |
| Ürün Adı | Listelenen ürünlerin Panoramada kullanılan ad bilgisi gösterilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kapat | Form kapatılır. |
| Yenile | Sayfanın yenilenmesini sağlar. |
| Düzenle | Seçilen satırda Tp Kod alanını aktif hale getirir ve değer girişi yapılmasını sağlar. |
| Yön Tuşları | Listelenen ürün kayıtlarında ileri-geri sayfa geçişi yapılmasını sağlar. |
