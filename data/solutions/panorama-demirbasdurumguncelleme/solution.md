# Demirbaş Durum Güncelleme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Demirbaş İşlemleri → Demirbaş Durum Güncelleme`

## Ekran Tanımı

Demirbaş durum tanımlarının güncellenmesi için kullanılan ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | İlgili distribütörün seçildiği alandır. |
| Lokasyon | Demirbaş durum güncellemesinin müşteri üzerinden mi depo üzerinden mi olacağının belirtildiği alandır. |
| Seri No | Seri detayı listesinden seri no seçimi yapılır. |
| Müşteri Kodu | Müşteri listesinden müşteri seçiminin yapıldığı alandır. |
| Ürün | Hangi demirbaş ürününün durum güncellemesi yapılacağının seçildiği alandır. |
| Düzenle | Detaydaki tanımı düzenlemek için kullanılır. |
| İzle | Detaydaki tanımın detaylarını görüntülemek için kullanılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Seçilen kriterlere göre demirbaş ürünlerin listelenmesini sağlar. |
| İşlemi Başlat | Durum güncelleme işlemini başlatır. |
| Kapat | Ekranı kapatır. |
