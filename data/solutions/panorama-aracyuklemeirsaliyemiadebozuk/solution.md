# Irsaliyeden Fatura Oluşturma(Bozuk)

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Müşteriden İade İşlemleri → İrsaliyeden Fatura Oluşturma(Bozuk)`

## Ekran Tanımı

Distribütörün ürün alımı yaptığı firma veya genel merkezinden talep etmiş olduğu İrsaliyeden fatura oluşturma (bozuk) ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yükleme Kodu | Yapılan işleme ait program tarafından otomatik verilen kayıt numarasıdır. |
| Distribütör Kodu | Belgenin hangi Distribütör için kesileceğinin tanımlandığı alandır. |
| İş Bildirim | İş birliği bilgisidir. |
| Başlangıç Tarihi | Fatura (bozuk) belgelerinin hangi tarihten başlayacağı bilgisidir. |
| Bitiş Tarihi | Fatura (bozuk) belgelerinin hangi tarihte biteceği bilgisidir. |
| Başlangıç Sevk Tarihi | İrsaliyeden Fatura belgelerinin hangi Sevk tarihinden başlayacağı bilgisidir. |
| Bitiş Sevk Tarihi | İrsaliyeden Fatura belgelerinin hangi Sevk tarihinden biteceğinin bilgisidir. |
| Satış Temsilcisi | Fatura (bozuk) hangi Satış Temsilcileri üzerinden listelendiği seçiminin yapıldığı alandır. |
| Bölge Kodları | Fatura (bozuk) hangi müşteri Bölge üzerinden listeleneceği seçiminin yapıldığı bilgisidir. |
| Başlangıç Sevk Tarihi | Fatura (bozuk) belgelerinin hangi Sevk tarihinden başlayacağı bilgisidir. |
| Hareket Tipi | Fatura (bozuk) için tanımlanan hareket tipinin seçildiği alandır. Zorunlu bir alandır. |
| Seçilen Araç | Fatura (bozuk) yükleme yapılacağı araç seçilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Temizle | Listedeki verileri temizlemek için kullanılır. |
| Sevk Tarihleri | Sevk tarhi ayarlamak için kullanılan ekrandır. |
| Listele | Girdilere göre listeleme işlemleri yapılmaktadır. |
| Belgeleştir | Sevk Tarihleri ekranını açmaktadır. |
| Kaydet | Belgeyi kaydetmek için kullanılır. |
| Kapat | Belge ekranını kapatmak için kullanılır. |
| Yenile | Belge detay listesini yenilemek için kullanılır. |
| Düzenle | Belge detay satırında düzenleme yapmak için kullanılır. |
| Belge İzle | Belge detay satırında seçili belgeyi incelemek için kullanılır. |
| Transfer | Listelenen siparişlerin istenilen yere transfer işlemi gerçekleştirilir. |
| Fatura/İrsaliye | “Fatura/İrsaliye” butonu ile oluşacak belgelerin hareket tipinin fatura veya irsaliye olmasına karar verilebilir. |
| Kısmi Sevk | İstenilen belgelerin parçalı bir şekilde sevk edilebilmesini sağlar. |
| Yukarı | “Yukarı” – “Aşağı “ butonları ile oluşacak belgelerin öncelik sıralaması değiştirilebilir. |
| Aşağı | “Yukarı” – “Aşağı “ butonları ile oluşacak belgelerin öncelik sıralaması değiştirilebilir. |
