# Prim TL

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri → Hedef Prim İşlemleri → Prim TL`

## Ekran Tanımı

Prim TL tanımlamasının yapıldığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Durum | Prim tanımının durum bilgisini gösterir. Aktif ve Pasif değerlerini alır. |
| Başlangıç Tarihi | Tanımlanacak primin hangi başlangıç tarihi seçilir. |
| Bitiş Tarihi | Tanımlanacak primin bitiş tarihi seçilir. |
| Distribütör Kriter Tipi | Distribütör kriter tipi Kod/Grup/Ek Grup/Dinamik Grup değerlerini alabilir. |
| Distribütör Kriteri | Distribütör kriterinin seçildiği alandır. |
| Müşteri Kriter Tipi | Müşteri Kriter Tipi Kod/Grup/Ek Grup/Dinamik Grup/Bölge değerlerini alabilir. |
| Müşteri Kriteri | Müşteri kriter tipinde seçilen değere göre listelenen kayıtlardan birisi seçilir. |
| Uygulama Tipi | Primin Satış Yöneticisi, Satış Temsilcisi ya da Distribütörden biri seçilerek uygulama tipi belirlenir. |
| Uygulama Kriteri | Dist. Müfettiş listesinden uygulama kriteri belirlenir. |
| Değerlendirme Tipi | Primin Satış ya da Alış üzerinden birisi seçilerek değerlendirme tipi belirlenir. |
| Satış Koçu hesaplamalarında kullanılsın | Satış koçu hesaplamalarında kullanılması veya kullanılmaması belirtilir. |
| Hedef Ürün Kriter | Prim tanımının hangi üründe geçerli olacağını belirlemek için kullanılır. Kriter tipi Kod, Grup, Ek Grup ve Dinamik Grup tiplerinden seçilir. |
| Hedef Ürün Birimi | Hedef kriter tipi miktar seçildiği durumda dikkate alınacak birim değeridir. |
| Hedef Kriter Tipi | Seçilecek hedefin tipi belirlenir, miktar/tutar/ağırlık/hacim/litre/SKU çeşidi değerlerini alır. |
| İade Faturası | Hareket tipi olarak satıcıya ya da alıcıya iade faturası bastırılması istenilirse “Dikkate Al”, istenilmezse “Dikkate Alma” seçilir. |
| Değerlendirilecek Kayıtları | Primin uygulanacağı kayıtlar ya Fatura ya İrsaliye-Fatura ya da Sipariş-İrsaliye-Fatura olarak seçilir. |
| Hedef Zaman Birimi | Hedef Zaman Birimi olarak Gün, Hafta ya da Ay seçilir. Gün seçilirse sadece işaretli olan günlerde yapılır. Hafta seçilirse sadece haftada işlem yapılır. Ay olarak seçilirse sadece o ay içerisinde seçilen günlerde işlem yapılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Hedef Girişi | Seçilen kriterlere göre otomatik olarak gelen listedir. |
| Prim Girişi | Prim girişinin bulunduğu ekrandır. |
| Kaydet ve Kapat | Fiyatı kaydetme ve ekranı kapatma işlemi yapar. |
| Kapat | Ekranı kapatma işlemi yapar. |
