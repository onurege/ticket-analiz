# Alışveriş Faturası İş Bildirimi

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Alış İşlemleri → Irsaliyeden Fatura Oluşturma`

## Ekran Tanımı

Alış Faturası İş bildirimi ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Belge Kodu | Fatura belge kodu bilgisidir. |
| İşlem Tarihi | Belge işlem tarihi bilgisidir. |
| Distribütör Adı | Distribütör Adı bilgisidir. |
| Müşteri Ünvanı | Müşteri ünvanı bilgisidir. |
| Tür | Belge tür bilgisidir. |
| Matbu No | Matbu No bilgisidir. |
| Sipariş Kod | Sipariş Kod bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İşlemi Başlat | İşlem başlatmak için kullanılır. |
| Yenile | Listeyi yenilemek için kullanılır. |
| İzle | Listedeki seçili belgeyi izlemek için kullanılır. |
| Kapat | Belge ekranını kapatmak için kullanılır. |
