# Ürün Birleştirme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Tanımlamalar → Ürün Birleştirme`

## Ekran Tanımı

Farklı seri numarasına sahip belirli sayıdaki üründen bir paket olarak farklı bir ürün oluşturmak için kullanılır. Öncelikli olarak TBLURUNPATLATMA tablosuna LNGPAKETURUNKOD kısmına yeni oluşacak paket urun kodunu ve LNGADETURUNKOD kısmına da paket içerisinde olucak ürün kodunun tanımlanması gerekmektedir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Ürün | Paket içeriğindeki ürünün girildiği alandır. |
| Paket Ürün Kodu | Ürün kısmına paket içeriğindeki ürün tanımlandığında TBLURUNPATLATMA tablosundan otomatik olarak oluşacak paket ürün kodunun geldiği alandır. |
| Başlangıç Seri | Paket içeriğindeki ürünlerin seri başlangıç numarasının girildiği alandır. |
| Bitiş Seri | Paket içeriğindeki ürünlerin seri bitiş numarasının girildiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İşlemi Başlat | Gerekli bütün alanlar doldurulduktan sonra bu butona tıklanır ve işlem gerçekleştirilir. |
| Kapat | Ekranı kapatır. |
