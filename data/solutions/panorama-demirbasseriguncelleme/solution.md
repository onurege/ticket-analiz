# Demirbaş Seri Güncelleme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Demirbaş İşlemleri → Demirbaş Seri Güncelleme`

## Ekran Tanımı

Demirbaş seri bilgisinin güncellenmesi için kullanılan ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | İlgili distribütörün seçildiği alandır. |
| Seri No | Seri detayı listesinden seri no seçimi yapılır. |
| Müşteri Kodu | Müşteri listesinden müşteri seçiminin yapıldığı alandır. |
| Ürün Kodu | Hangi demirbaş ürününün seri durum güncellemesinin yapılacağının seçildiği alandır. |
| Düzenle | Detaydaki tanımı düzenlemek için kullanılır. |
| İzle | Detaydaki tanımın detaylarını görüntülemek için kullanılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Seçilen kriterlere göre demirbaş ürünlerin listelenmesini sağlar. |
| İşlemi Başlat | Seri durum güncelleme işlemini başlatır. |
| Kapat | Ekranı kapatır. |
