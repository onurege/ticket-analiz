# Toplu Müşteriye Özel Net Fiyat İptal

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriteri → Fiyat İşlemleri → Toplu Müşteriye Özel Net Fiyat İptal`

## Ekran Tanımı

Toplu müşteriye özel net fiyat iptal bilgilerinin tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Uygulama Yeri | Uygulama yeri bilgisidir. |
| Başlangıç Tarihi | Başlangıç tarihi bilgisidir. |
| Bitiş Tarihi | Bitiş tarihi bilgisidir. |
| Distribütör Kriter Tipi | Distribütör kriter tipi bilgisidir. |
| Distribütör Kriteri | Distribütör kriteri bilgisidir. |
| Müşteri Kriter Tipi | Müşteri kriter tipi bilgisidir. |
| Müşteri Kriter | Müşteri kriteri bilgisidir. |
| Ek Saha Seçenek | Ek saha seçenek bilgisidir. |
| Ürün Kriter Tipi | Ürünün kriter tipi seçilir, kod/grup/ek grup/dinamik grup değerlerini alır. |
| Ürün Kriteri | Ürün kriteri bilgisidir. |
| Özel Kod | Özel kod bilgisidir. |
| Para Birimi | Para birimi bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan değişiklikleri kaydetmek için kullanılır. |
| Kapat | Yapılan ekranı kapatmak içi kullanılır. |
| Yenile | Yenilemek için kullanılır. |
| İzle | Detaydaki seçili belgeyi incelemek için kullanılır. |
