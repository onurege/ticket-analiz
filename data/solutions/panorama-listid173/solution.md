# Demirbaş Durum

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Demirbaş İşlemleri → Demirbaş Durum`

## Ekran Tanımı

Demirbaş durumlarının tanımlandığı ekrandır. Öncelikle ekrana, sisteme girilmiş olan demirbaş durumlarının listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Tanıma verilecek olan takip kodudur. Zorunlu alandır. |
| Ad | Tanıma verilecek olan isimdir. Zorunlu alandır. |
| Distribütör Kodu | İlgili distribütörün seçileceği alandır. |
| Uygulama Yeri | Tanımın hangi modülde kullanılacağının belirtildiği alandır. Panorama / Quest / Panorama ve Quest / Münferit / Hepsi / Stokbar değerlerini alabilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Tanımı kaydeder. |
| İptal | Yapılan tanımın kaydedilmeden iptal edilmesini sağlar. |
