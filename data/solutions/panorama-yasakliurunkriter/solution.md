# Yasaklı Ürün Uygulama Kriteri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Ürün Yönetimi → Yasaklı Ürün Uygulama Kriterleri`

## Ekran Tanımı

Yasaklı ürün uygulama kriter tanımının yapıldığı ekrandır. Öncelikle ekrana, sisteme girmiş olan yasaklı ürün kriter tanımları gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Distribütör Kriter Tipi | Tanımlanacak yasaklı ürün uygulama kriter tanımının hangi distribütör kriter tipine uygulanacağının belirlendiği alandır. Kod / Grup / Ek Grup / Dinamik Grup değerlerini alabilir. |
| Distribütör Kriteri | Distribütör Kriter Tipi seçimine bağlı olarak distribütör kriter seçiminin yapıldığı alandır. |
| Bağlantı Tipi | Bağlantı tipi değerinin girildiği alandır. Ve / Veya değerlerini alabilir. |
| Kaydet | Tanımı kaydeder. |
| Kapat | Ekranı kapatır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Detayda yeni tanım yapılması için kullanılır. |
| İzle | Detaydaki kaydın detaylarını görüntülemek için kullanılır. |
| Sil | Detaydaki kaydı silmek için kullanılır. |
| Grup | Yasaklı ürün kriter grup detay ekranından tanım yapılmasını sağlar. |
