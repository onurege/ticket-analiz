# Para Birimi

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Sistem Yönetimi 🡪 Genel Tanımlamalar 🡪 Para Birimi`

## Ekran Tanımı

Para birimlerinin tanımlarının yapıldığı ekrandır. Öncelikle sisteme girilmiş olan kayıtların listesi gelir. Kullanılmak istenen para birimlerinin tanımlaması yapılır. Bu tanımlar, fiyat tanımı yapılırken seçilir. Tek bir para birimi ile işlem yapılacaksa 1 adet kaydın yapılması yeterlidir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Yapılan tanıma verilen numaradır. |
| Simge | Para birimi simgesidir. |
| Alt Birim | Para birimi için bir alt birimin ismidir. |
