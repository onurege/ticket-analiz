# Sipariş Geç Sevk İptal Nedenleri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri → Tanımlamalar → Sipariş Geç Sevk İptal Nedenleri`

## Ekran Tanımı

Panorama üzerinde yer alan siparişlerin belge üzerinde yazan tarihten daha geç sevk edilmesinde, sevk edilmemesinde ya da iptal edilmesi durumunda raporlama amaçlı kullanım için nedenlerin girildiği ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Başlangıç Tarihi | Listelenecek siparişlerin başlangıç tarih bilgisidir. Ön değer olarak günün tarihi getirilir. |
| Bitiş Tarihi | Listelenecek siparişlerin Bitiş tarih bilgisidir. Ön değer olarak yılın son günü gelir. |
| Distribütör Kodu | Kaydın hangi distribütöre ait olduğunu belirten, distribütör kod bilgisidir. |
| Müşteri Kriter Tipi | Müşteri kriterinin hangi tipte olduğu bilgisidir. Kod, Grup, Ek Grup Dinamik Grup seçilebilir. |
| Müşteri Kriter | Hangi müşterilere ait siparişlerin listeleneceği bilgisidir. |
| Satış Temsilcisi | Hangi Satış Temsilcisine ait siparişlerin listeleneceği bilgisidir. |
| İlk Sipariş No | Seçilen sipariş ve daha büyük takip kodlu siparişlerin listelenmesini sağlar. |
| Son Sipariş No | Seçilen siparişin ve daha küçük takip kodlu siparişlerin listelenmesini sağlar. |
| Sipariş Durum | Hangi durumdaki siparişlerin listeleneceği bilgisidir. “Hepsi”, “İptal”, “Geç Sevk” ya da Faturalaşmamış seçilebilir. |
| İşlem Yapılmış Siparişleri | Önceden neden kodu girilmiş siparişlerinde listelenmesi istendiğinde değiştirilmesi gereken parametredir. “Listelenmesin”, “Listelensin” ya da “Eksik Detaylı siparişler listelensin” seçilebilir. |
| Belge Kodu | Belge kodu bilgisidir. |
| Matbu No | Matbu no bilgisidir. |
| İşlem Tarihi | İşlem Tarihi bilgisidir. |
| Müşteri | Müşteri bilgisidir. |
| Brüt Tutar | Brüt tutar bilgisidir. |
| Sipariş Durum | Sipariş Durum bilgisidir. |
| Neden Kod | Neden Kod bilgisidir. |
| Neden | Neden bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Belge detay listesini yenilemek için kullanılır. |
| Düzenle | Belge detay satırında düzenleme yapmak için kullanılır. |
| İzle | Detaya eklenen satırı izle modunda açar. |
| Sil | Detaya eklenen satırı silmek için kullanılır. |
| Detaylar | Detaylar bilgisidir. |
| İşlemi Başlat | İşlemi Başlatma butonudur. |
| Listele | Listele butonuna bastığımızda kriter alanına girilen kriterlere bağlı olarak İrsaliyeler detay alanında listelenir. |
| Temizle | Temizle butonuna bastığımızda kriter alanına girilen kriterler ve bunlara bağlı olarak detay alanında listelenen irsaliyeler temizlenir. |
| Kapat | Sipariş ekranını kapatır. |
