# Fatura Mütabakat

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → DBS İşlemleri → Fatura Mutabakat`

## Ekran Tanımı

Ö deme tipi DBS olarak kayıt edilen alış-satış faturaları listelenmesi sağlanmaktadır. Rapor Türü Panoramadaki Faturalar ve Bankadaki Faturalar seçimine göre listelenmektedir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Rapor Türü | Nerede bulunan faturaların listeleneceğinin seçildiği alandır. |
| Banka | Banka bilgisinin seçildiği alandır. |
| Distribütör Bölge | Merkez kullanıcı için distribütör bölge kriterinin verilebildiği alandır. |
| Distribütör Kodu | Distribütör kriterinin verildiği alandır. |
| Tarih | Belge üzerinde yer alan hangi tarihe göre listelenmesi istenilen alandır |
| Başlangıç Tarihi | Belli bir tarih aralığı verilmek isteniliyor ise başlangıç tarihinin girildiği alandır. |
| Bitiş Tarihi | Belli bir tarih aralığı verilmek isteniliyor ise bitiş tarihinin girildiği alandır. |
| Fatura Durum | Listelenmesi istenilen faturanın durum bilgisi kriterinin verildiği alandır. |
| Pan.Fatura Durum | Listelenecek belgelerde geçerli yada iptal belgelerin listelenmesi için kullanılan kriterdir. |
| Fatura Akıbet | Detay bilginin listelendiği alandır. |
| Bölge Toplam | Distribütör bölge bazında özet bilginin bulunduğu alandır. |
| Distribütör Bölge | Listelenen kayıtların hangi bölgede olduğu bilgisinin olduğu alandır. |
| Distribütör Adı | Listelenen kayıdın hangi distribütöre ait olduğu bilgisinin olduğu alandır. |
| Banka Adı | Listelenen kayıt bankada ise banka bilgisinin bulunduğu alandır. |
| İşlem Tarihi | Listelenen belgenin işlem tarihinin bilgisidir. |
| Fatura Tarihi | Listelenen belgenin fatura tarihinin bilgisidir. |
| Vade Tarihi | Listelenen kayıdın vade tarihi bilgisidir. |
| Ödeme Türü | Listelenen kayıdın ödeme türünün bilgisidir. |
| Tutar | Listelenen belgenin tutar bilgisidir. |
| Fatura No | Listelenen kayıdının sistem takip numarasıdır. |
| Fatura Durum | Listelenen kayıdın durum bilgisidir. |
| Pan.Fatura Durum | Listelenen kayıdın geçerlilik durum bilgisidir. |
| Banka Durum | Listelenen kayıdın bankaya gönderildi bilgisidir. |
| Kayıt İşlem No | Bankadaki kayıt işlem numarasıdır. |
| İptal İşlem No | Bankadaki iptal işlem numarasıdır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Girilen kriterlere göre verilen listelenmesini sağlar. |
| Excel’e Aktar | Ekranda listelenen verilerin excel formatında kayıt işleminin yapılmasını sağlar |
| Pdf’e Aktar | Ekranda listelenen verilerin pdf formatında kayıt işleminin yapılmasını sağlar |
| Kapat | Açılan ekran kapanır. |
