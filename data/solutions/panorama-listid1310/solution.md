# Dist. Onay İptal Neden

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri → Tanımlamalar → Dist Onay İptal Neden`

## Ekran Tanımı

Dist Onay Nedenlerinin tanımlandığı alandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Ad | Distribütör Onay İptal Nedenlerine verilen isimdir. Girilmesi zorunlu sahadır. |
| Distribütör Kodu | İlgili Distribütörün seçileceği sahadır. |
| Düzenlemede Kullanılsın | Düzenlemede kullanılsın mı kullanılmasın mı seçeneğidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemin kaydedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
