# E-İskonto Değeri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ve Pazarlama Kriterleri → İskonto Promosyon → E-İskonto Değeri`

## Ekran Tanımı

E-Puan değer bilgilerinin tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | E-puan değer tanımına program tarafından verilen artışlı numaradır. |
| Başlangıç Tarihi | E-puan değer tanımının başlayacağı tarih bilgisini içerir. |
| E-İskonto Tipi | E-Puan değer tanımında baz alınacak e-puan tipi seçilir. |
| Distribütör Kodu | E-puan değer tanımının distribütör kodudur. E-puan değer tanımının hangi distribütöre ait olduğu bu alan üzerinden izlenir. |
| Uygulama Yeri | E-puan değer tanımının kullanılacağı modül bilgisinin tutulduğu alandır. |
| Bitiş Tarihi | E-puan değer tanımının biteceği tarih bilgisini içerir. |
| Değer | E-puan hakkediş sonrasında harcama aşamasında kullanılacak hesap değerinin tanımlandığı alandır. |
