# Müşteri Filo Tanımlama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → Tanımlamalar → Müşteri Filo Tanımlama`

## Ekran Tanımı

Müşteriye ait filo bilgilerinin tanımlandığı ekrandır. Öncelikle ekrana, sisteme girilmiş olan müşteri filosu listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Sistem tarafından tanıma verilen artışlı numaradır. |
| Filo Adı | Müşteri filosuna verilen isimdir. |
| Distribütör Kodu | İlgili distribütörün seçildiği rehber alanıdır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan tanımın kaydedilmesini sağlar. |
| İptal | Ekranı kapatır. |
