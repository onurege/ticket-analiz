# Satış İrs. S.İade Fatura Oluşturma

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Satış İşlemleri → Satış İrs`

## Ekran Tanımı

Satış İrsaliye belgelerinden Satıcıya İade Faturası oluşturulduğu ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yükleme Kodu | Yapılan işleme ait program tarafından otomatik verilen kayıt numarasıdır. |
| Distribütör Kodu | İrsaliye kayıtlarının bağlı olduğu Distiribütör kodunun seçildiği alandır. |
| Güncel Fiyatlardan hesaplansın mı? | İşaretli olduğunda; İrsaliye kayıtlarına uygulanmış olan Fiyat tanımlarının yeniden hesaplanması sağlamaktadır. |
| Otm. İsk./Pro. İptal Et? | İşaretli olduğunda; İrsaliye kayıtlarına uygulanmış olan İskonto/Promosyonların yeniden hesaplanması sağlamaktadır. |
| Başlangıç Tarihi | Satış İrsaliye belgelerinin hangi tarihten başlayacağı bilgisidir. |
| Bitiş Tarihi | Satış İrsaliye belgelerinin hangi tarihte biteceği bilgisidir. |
| Başlangıç Sevk Tarihi | Satış İrsaliye belgelerinin hangi Sevk tarihinden başlayacağı bilgisidir. |
| Bitiş Sevk Tarihi | Satış İrsaliye belgelerinin hangi Sevk tarihinde biteceği bilgisidir. |
| Araçlar | Satış İrsaliyelerin yükleneceği araç ya da araçların seçiminin yapıldığı alandır. |
| Hareket Tip | Satış İrsaliyelerin hangi hareket tipinde geleceğinin seçiminin yapıldığı alandır. |
| Satış Temsilcileri | Satış İrsaliyelerinin hangi Satış Temsilcileri üzerinden listelendiği seçiminin yapıldığı alandır. |
| Bölge Kodları | Satış İrsaliyelerin hangi müşteri Bölge üzerinden listeleneceği seçiminin yapıldığı bilgisidir. |
| Detay Kriterler | Daha detaylı kriter girilmek isteniyorsa “Detay Kriterler” butonuna basılarak yeni bir ekranda, Müşteri Kodu (Rehber), Müşteri Grup (Çoklu seçim), Müşteri Ek Grup (Çoklu seçim), Depo Kodları (Çoklu Seçim), Özel Kod (Text Alan), İrsaliye Tipi (Açık, Kapalı), Ödeme tipi (Peşin, Çek, Senet, Kredi kartı, Açık Hesap, Ticariş kart, DBS, Havale/EFT) |
| Seçilen Araç | İrsaliyelerin yükleme yapılacağı araç seçilir. |
| Araç Nokta Sayısı | Araç Tanımı üzerindeki Nokta Sayısı değerini göstermektedir. |
| Nokta Sayısı | Seçilen araç bilgisi içerisindeki müşteri sayısını göstermektedir. |
| Belge Sayısı | Seçilen araç bilgisi içerisindeki belge sayısını göstermektedir. |
| Dağıtıcı | “Dağıtıcı Kodu” ve “Dağıtıcı Adı” seçilen araç için seçilen dağıtıcının kodunu ve adını göstermektedir. |
| Araç Hacim | Seçilen aracın maksimum alabileceği Hacim miktar bilgisidir. |
| Toplam Hacim | Seçilen araca yüklenmiş belgelerin depozito iade kalemleri hariç diğer ürünlerinin toplam hacim miktarını belirtir. |
| Hacim Oran (%) | Araç Hacim ile Toplam Hacim arasındaki yüzdesel oranı gösterir. |
| İade Hacim | “İade Hacim” seçilen araca yüklenmiş belgelerin tüm depozito iade kalemlerinin toplam hacim miktarını gösterir. |
| İade Hacim Oran (%) | İade Hacim alanının yüzdesel Oranını gösterir. |
| Araç Ağırlık | Seçilen aracın maksimum alabileceği Ağırlık miktar bilgisidir. |
| Toplam Ağırlık | Seçilen araca yüklenmiş belgelerin depozito iade kalemleri hariç diğer ürünlerinin toplam ağırlık miktarını gösterir. |
| Ağırlık Oran (%) | Araç Ağırlık ile Toplam Ağırlık arasındaki yüzdesel oranı gösterir. |
| İade Ağırlık | “İade Ağırlık” seçilen araca yüklenmiş belgelerin tüm depozito iade kalemlerinin toplam ağırlık miktarını gösterir. |
| İade Ağırlık Oran (%) | İade Ağırlık alanının yüzdesel Oranını gösterir. |
| Fatura Tarihi | Oluşacak Fatura belgelerinin tarihlerinin belirlendiği alandır. |
| Fatura Hareket Tipi | Oluşacak Fatura belgelerinin Hareket Tiplerinin belirlendiği alandır. |
| İade Nedeni | Oluşacak Fatura belgeleri için İade Nedeni belirlendiği alandır. |
| Vade Günü | Oluşacak Fatura belgeleri için Vade Günü belirlendiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Temizle | Sahaya girilen bilgilerin silmesini sağlar. |
| Sevk Tarihleri | Sevk Tarihleri ekranının açılmasını sağlamaktadır. |
| Listele | Girilen kriterlere göre ilgili kayıtların listelenmesini sağlamaktadır. |
| Sadakat | Sadakat İskontosu Seçimi ekranını açmaktadır. |
| Belgeleştir | Bu butona basıldığında “Yüklenemeyenler” hariç diğer tüm araçlardaki irsaliyeler araç sırası ve araç içerisindeki irsaliye sırasına göre Satıcıya iade faturaları oluşturulma işlemine girer. |
| Kaydet | Bu buton yardımı ile yükleme kaydedilebilir. Kaydet butonuna basıldığında “Kriterler” gruplamasındaki ve yüklenemeyenler aracı veritabanına kaydedilmez. |
| Kapat | Ekranı Kapatma işlemi yapar. |
| Yenile | Liste kayıtlarının güncellenmesini sağlamaktadır. |
| Belge İzle | Satış İrsaliye ürün detay bilgilerinin izlenmesini sağlar. |
| Transfer | Yüklenmemiş olan İrsaliyelerin seçilenlerin araca transferi sağlanır. |
| Yukarı | Belgele sıralamasını aşağı yönde yer değiştirmesini sağlar |
| Aşağı | Belgele sıralamasını yukarı yönde yer değiştirmesini sağlar |
