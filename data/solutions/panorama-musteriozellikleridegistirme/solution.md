# Müşteri Özellikleri Değiştirme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → İşlemler → Müşteri Özellikleri Değiştirme`

## Ekran Tanımı

Belirtilen kriterlere göre müşteri/müşterilerin toplu olarak özelliklerinin değiştirilebildiği ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Özelliği değiştirilecek müşterileri filtrelemek için distribütör kriterinin bilgisinin girildiği alandır. |
| Grup Kodu | Özelliği değiştirilecek müşterileri filtrelemek için grup kodu bilgisinin girildiği alandır. |
| Ek Grup Kodu | Özelliği değiştirilecek müşterileri filtrelemek için ek grup kodu bilgisinin girildiği alandır. |
| Dinamik Grup | Özelliği değiştirilecek müşterileri filtrelemek için dinamik grup kodu bilgisinin girildiği alandır. |
| Bölge Kodu | Özelliği değiştirilecek müşterileri filtrelemek için bölge kodu bilgisinin girildiği alandır. |
| Satış Temsilcisi Adı | Özelliği değiştirilecek müşterileri filtrelemek için satış temsilcisi adının girildiği alandır. |
| Müşteri | Özelliği değiştirilecek müşterilerin müşteri listesinden seçim yapılarak kriter olarak girildiği alandır. |
| Durum | Müşterinin durum değerini değiştirmek için kullanılan alandır. Aktif / Pasif değerlerini alabilir. |
| Fiyat Yetki | Müşterinin fiyat yetki durumunun değiştirildiği alandır. İşaret Koy / İşaret Kaldır değerlerini alabilir. |
| Maksimum İskonto Kontrolü Yap | Müşteride maksimum iskonto kontrolünün yapılıp yapılmayacağının belirtildiği alandır. İşaret Koy / İşaret Kaldır değerlerini alabilir. |
| İsk. Pro. Çakışma Kontrolünden Muaf | Müşteride iskonto poromosyon çakışma kontrolünün yapılıp yapılmayacağının belirtildiği alandır. İşaret Koy / İşaret Kaldır değerlerini alabilir. |
| Merkeze Faturala | Müşterinin faturalarının merkeze fatura olarak oluşturulacağının kontrolünün belirtildiği alandır. İşaret Koy / İşaret Kaldır değerlerini alabilirler. |
| Mağaza Siparişleri Birleştirilsin | Müşterinin farklı mağazalarının siparişlerinin birleştirilebilmesi kontrolünün belirtildiği alandır. İşaret Koy / İşaret Kaldır değerlerini alabilir. |
| İrsaliyeler Mağaza Detayında Basılsın | İrsaliyeler mağaza detayında basılmasının kontrolünün belirtildiği alandır. İşaret Koy / İşaret Kaldır değerlerini alabilir. |
| Fatura Kapatma da Kullanılsın mı ? | Fatura kapatmanın kullanılabilmesi için kontrolün belirtildiği alandır. İşaret Koy / İşaret Kaldır değerlerini alabilir. |
| Ödeme Tipi | Müşterinin ödeme tipinin değiştirilebildiği alandır. Peşin / Çek / Senet / Kredi Kartı / Vadeli / Ticari Kart / DBS / Havale/EFT değerlerini alabilir. |
| Ödeme Tipleri | Müşteri kartı üzerinde bulunan ödeme tip yada tiplerinin değiştirildiği alandır. Peşin / Çek / Senet / Kredi Kartı / Vadeli / Ticari Kart / DBS / HavaleEFT değerlerini alabilir. |
| Ruhsat Daire | Müşterinin Ruhsat Daire bilgisinin değiştirildiği alandır. |
| Vergi Tipi | Müşterinin Vergi Tipinin değiştirildiği alandır. KDV'den Muaf / Gerçek Kişi / Tüzel Kişi / Yabancı Uyruk değerlerini alabilir. |
| Vergi Dairesi | Müşterinin Vergi Dairesi bilgisinin değiştirildiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yeni tanımda ön değer atansın mı ? |  |
| İşlemi Başlat | Belirtilen kriterlerdeki müşterilerin verilen özelliklere göre müşteri kartı üzerindeki bilgi/bilgilerin değiştirme işlemini başlatır. |
| Kapat | Ekranı kapatır. |
