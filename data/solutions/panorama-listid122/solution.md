# Müşteri Grubu

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → Tanımlamalar → Müşteri Grubu`

## Ekran Tanımı

Müşteri grup bilgilerinin tanımlandığı ekrandır. Öncelikle ekrana, sisteme girilmiş olan müşteri grubu listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Müşteri grubuna verilen artışlı numaradır. |
| Ad | Müşteri grubuna verilen isimdir. |
| Distribütör Kodu | Müşteri grubu distribütör kodudur. Müşteri grubunun hangi distribütöre ait olduğu bu alan üzerinden izlenir. |
| Uygulama Yeri | Müşteri grubunun kullanılacağı modül bilgisinin tutulduğu alandır. Panorama / Quest / Panorama ve Quest / Münferit / Hepsi / Stokbar değerlerini alabilir. |
| Müşteri Ek Grup | Müşteri ek grubunun seçildiği alandır. |
| Özel Kod Teklik Kontrolü | Belirlenen gruplara ait müşterilerde kullanılan özel kod bilgisinin farklı müşterilerde kullanılmasını engellemek için kullanılan alandır. Yapılmasın / Distribütör İçi / Genel değerlerini alabilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan tanımı kaydeder. |
| İptal | Ekranı kapatır. |
