# Road Net Bilgi Gönder

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ İşlemler → Road Net Bilgi Gönder`

## Ekran Tanımı

Road Net sevkiyat sistemine bilgi gönderme işlemidir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Aktarım yapılacak distribütör seçilir. |
| Belge Tipi | Aktarım yapılacak belge tipi seçilir. |
| Başlangıç Tarihi | Başlangıç yükleme tarihi seçilir. |
| Bitiş Tarihi | Bitiş yükleme tarihi seçilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Giriş yapılan kriterlere göre listeme işlemi yapılır. |
| Bilgi Gönder | Litelenen belgeler Road Net sistemine gönderilir. |
| Kapat | Ekran kapatılır. |
