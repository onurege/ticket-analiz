# Şube

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tanımlamalar → Şube`

## Ekran Tanımı

Bankalara bağlu şube bilgilerinin tanımlandığı ekrandır. Öncelikle ekrana sisteme girilmiş olan şubelerin listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Şube tanımına ait program tarafından verilen artışlı numaradır. |
| Kod | Tanımlanan şube için kullanıcının belirlediği kod bilgisidir. |
| Şube Ad | Şubenin adının girildiği metin sahasıdır. |
| İlgili Kişi | Şubedeki yetkili kişinin girilebileceği alandır. |
| Banka Kodu | Şubenin bağlı bulunduğu bankanın seçildiği rehber sahasıdır. |
| İl Kod | Şubenin bulunduğu il bilgisinin girilebileceği rehber alandır. |
| İlçe Kod | Şubenin bulunduğu ilçe bilgisinin girilebileceği rehber alandır. |
| Telefon | Bankanın telefon bilgisinin girilebileceği alandır. |
| Faks | Bankanın faks bilgisinin girilebileceği alandır. |
| Posta Kodu | Şubenin posta kodu bilgisidir. |
| Adres | Bankanın adres bilgisinin girilebileceği metin alanıdır. |
