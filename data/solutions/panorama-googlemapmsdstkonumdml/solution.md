# St Lokasyon

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Harita İşlemleri → ST Lokasyon`

## Ekran Tanımı

Satış temsilcisi lokasyon ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Tarih | Tarih bilgisidir. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| ST/SE Elenmanları | Satış temsilcisi elemanları seçilir. |
| SY Elenmanları | Satış yöneticisi elemanları seçilir. |
| ST/SE Canlı Konumları Göster | Satış Temsilcisi/Satış elemanı canlı konumunu gösterilip gösterilmeyeceğini belirleyen seçenektir. |
| Gps Yolunu Göster | Gps yolunu gösterip gösterilmeyeceği seçeneğidir. |
| Beklemeleri Göster | Bekleyen Ziyaretleri gösterip göstermeyeceği seçeneğidir. |
| ST/SE’nin Rutunu Göster | Satış Temsilcisi/Satış elemanı rutunun gösterilip gösterilmeyeceğini belirleyen seçenektir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Ekranı ve bilgileri yenilemek için kullanılır |
| Ortala | Haritayı ortalamaya yarayan butondur. |
| Temizle | Yapılan işlemleri temizleme butonudur. |
| Arama | St lokasyon arama butonudur. |
| Kapat | St lokasyon ekranını kapatma butonudur. |
