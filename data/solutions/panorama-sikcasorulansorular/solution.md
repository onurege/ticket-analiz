# Sıkça Sorulan Sorular

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Sistem yönetimi 🡪 Ana Sayfa İşlemleri 🡪 Sıkça Sorulan Sorular`

## Ekran Tanımı

Panorama programında sıkça yaşadığımız sorunlar ve çözümlerinin girildiği ekrandır. Bu ekrana merkez kullanıcılar tarafından bilgiler girilir. Öncelikle ekrana, sisteme girişmiş olan sıkça sorulan sorular listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artan numaradır. |
| Durum | Aktif durumda ise belge ana sayfada yayınlanır, pasif ise yayınlanmaz |
| Görünüm sırası | İstenilen bilginin hangi sırada sayfada yayınlanması gerektiği seçilebilir. |
| Kullanıcı Kriter Tip | Kullanıcı kriter tipi kod/grup olarak tercih edilir. |
| Kullanıcı Kriter | Yayınlanmak istenilen bilgiler isteğe bağlı kullanıcı kriterinden kullanıcı ile sınırlandırılabilir. |
| Soru | Kullanıcılardan gelen soruların yayınlandığı bölümdür. |
| Cevap | Soruların ardından cevapların yayınlandığı bölümdür. |
