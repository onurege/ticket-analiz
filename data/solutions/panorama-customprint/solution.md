# Standart Kağıt Basım

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Belge Basım İşlemleri →Standart Kağıt Basım`

## Ekran Tanımı

Standart Kağıt Basım Ekranından Tipi Fatura veya İrsaliye olan belgelerin seçimli olarak belli formatta toplu olarak yazdırılma işlemi bu ekrandan yapılır. ( Ekran 3 farklı formatta active report olarak çalışmaktadır)

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Matbu No | Belge basımı yapılacak irsaliye veya faturanın matbu no değeri girilir ve detayda bu matbu numarasına ait belge listelenir. |
| Distribütör Kodu | Basımı yapılacak belgelerin bağlı olduğu distribütör seçimi bu alandan yapılır. |
| İlk Belge No | Belge numarasına bağlı olarak irsaliye veya fatura listelenecek ise, yani yazdırılacak irsaliye veya faturanın belge numaraları sıralıysa ilk irsaliye veya faturanın belge no değeri bu alana yazılır |
| Son Belge No | Belge numarasına bağlı olarak irsaliye veya faturalar listelenecek ise, yani yazdırılacak irsaliye veya faturanın belge numaraları sıralıysa son irsaliye veya faturanın belge no değeri bu alana yazılır. |
| Başlangıç Tarihi | Yazdırılacak belgelerin işlem tarihlerini kapsayan tarih aralığı için Başlangıç Tarih değeri bu alana yazılır. |
| Bitiş Tarihi | Yazdırılacak belgelerin işlem tarihlerini kapsayan tarih aralığı için Bitiş Tarih değeri bu alana yazılır. |
| Müşteri Kriter Tipi | Müşteri kriteri vererek irsaliye veya faturaların detayda listelenmesi ve buna bağlı olarak yazdırılma işlemi yapılacak ise Müşteri Kriter Tip alanından Kod, Grup Ek grup değerlerinden biri seçilerek Müşteri Kriteri rehberinde listelenecek değerler belirlenir. |
| Müşteri Kriter | Müşteri kriteri vererek irsaliye veya faturaların detayda listelenmesini istiyorsak ve buna bağlı olarak yazdırma işlemi yapılacak ise, Müşteri Kriter Tip alanında seçmiş olduğumuz değere göre Müşteri Kriter seçimi yapılır ve seçilen müşterilere ait irsaliye veya faturalar detayda listelenir. |
| Satış Temsilcisi | Satış Temsilcisine bağlı irsaliye veya faturalar yazdırılacak ise; bu alandan Satış temsilcisi seçimi yapılarak detayda sadece seçilen satış temsilcisine ait irsaliye veya faturaların listelendiğini görürüz. |
| Hareket Tipi | Hareket Tipi kriterine bağlı olarak irsaliye veya faturalar yazdırılacak ise; bu alandan irsaliye veya fatura hareket tipi seçilerek detayda seçilen hareket tiplerine ait irsaliye veya faturaların listelendiğini görürüz. |
| Belge Tipi | Bu alanda Fatura, İrsaliye, Hepsi seçenekleri yer alır. Sadece yukarıda belirttiğimiz kriterlere de bağlı olarak irsaliyeler detayda listelensin isteniyorsa bu alanda İrsaliye seçimi yapılır. Sadece yukarıda belirttiğimiz kriterlere de bağlı olarak Faturaların detayda listelensin isteniyorsa bu alanda Fatura seçimi yapılır. Hem Fatura Hem de İrsaliyelerin listelenmesini istiyorsak Hepsi seçimi yapılır. |
| Ödeme Tipi | Ödeme Tipi alanında Hepsi, Peşin, Çek, Senet, Kredi Kartı, Açık Hesap, Ticari Kart, DBS, Havale/ EFT, Mail order değerleri listelenir ve bu alandan seçmiş olduğumuz ödeme tipindeki İrsaliye veya Faturalar detayda listelenir ve yazdırma işlemi yapılabilir. |
| Basım Tipi | Kapı Kağıdı, Yollama Kağıdı, Araç Yükleme seçenekleri çıktının hangi formatta yazılacağını belirler. |
| Belge Kodu | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen Belgelerin Kodu gösterir. |
| Müşteri Kodu | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen belgelerin Müşteri Kodu değerini gösterir. |
| Müşteri Adı | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen belgelerin Müşteri Adını gösterir. |
| ST Kodu | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen belgelerin Satış Temsilcisi Kodunu gösterir. |
| Satış Temsilcisi | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen belgelerin Satış Temsilcisi Adını gösterir. |
| İşlem Tarihi | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen belgelerin İşlem Tarihini gösterir. |
| Net Tutar | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen belgelerin Net Tutar değerini gösterir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Listele butonuna bastığımızda kriter alanına girilen kriterlere bağlı olarak belgeler detay alanında listelenir. |
| Temizle | Temizle butonuna bastığımızda kriter alanına girilen kriterler ve bunlara bağlı olarak detay alanında listelenen belgeler temizlenir. |
| Kapat | Standart Kağıt Basım ekranını kapatır. |
| Yenile | Detay listesinin yenilenmesini sağlar |
| Yazdır | Detay alanında listelenen ve daha sonra seçimi yapılan Belgelerin Yazdırılması sağlanır. Yazdır butonuna bastığımızda Basım Ekranı açılır. |
| İzle | Detay alanında listelenen ve seçimi yapılan belgeyi izle modunda açar. |
