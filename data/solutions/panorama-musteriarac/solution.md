# Müşteri Araç Tanım

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → Tanımlamalar → Müşteri Araç Tanım`

## Ekran Tanımı

Müşteriye ait araçların tanımının yapıldığı ekrandır. Öncelikle ekrana, sisteme girilmiş olan araç tanımlarının listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Sistem tarafından verilen artışlı numaradır. |
| Durum | Araç durumunu belirtir. Aktif / Pasif değerlerini alabilir. |
| Distribütör Kodu | Dsitribütör seçiminin yapıldığı rehber alanıdır. |
| Plaka 1 | Araca ait plakanın girildiği alandır. |
| Müşteriler | Aracın bağlı olduğu müşteri yada müşterilerin seçildiği alandır. |
| Grup | Aracın bağlı olduğu grubun seçildiği alandır. |
| Marka | Aracın markasının seçildiği alandır. |
| Model | Aracın modelinin seçildiği alandır. |
| Alt Tip | Aracın alt tipinin seçildiği alandır. |
| Yakıt Seviyesi | Aracın yakıt seviyesinin seçildiği alandır. |
| Şasi No | Aracın şasi numarasının belirtildiği alandır. |
| Renk | Aracın renginin belirtildiği alandır. Bej / Beyaz / Bordo / Füme / Gri / Gümüş Gri / Kahverengi / Kırmızı / Lacivert / Mavi / Mor / Pembe / Sarı / Siyah / Şampanya / Turkuaz / Turuncu / Yeşil değerlerini alabilir. |
| Model Yıl | Aracın model yılının seçildiği alandır. |
| Motor Gücü | Aracın motor gücünün seçildiği alandır. |
| Yakıt Türü | Aracın yakıt türünün seçildiği alandır. |
| Karoser | Aracın karoser bilgisinin seçildiği alandır. |
| Motor No | Aracın motor numara bilgisinin girildiği alandır. |
| KM | Aracın kilometresinin girildiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Yapılan tanımı kaydeder ve yeni tanım ekranını getirir. |
| Kaydet ve Kapat | Yapılan tanımı kaydeder ve ekranı kapatır. |
| Kapat | Ekranı kapatır. |
