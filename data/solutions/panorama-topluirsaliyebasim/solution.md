# Toplu Irsaliye Basım

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Belge Basım İşlemleri →Toplu İrsaliye Basım`

## Ekran Tanımı

Toplu İrsaliye Basım Ekranından Tipi İrsaliye olan belgelerin toplu olarak yazdırma işlemi yapılıyor.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Matbu No | Belge basımı yapılacak irsaliyenin matbu no değeri girilir ve detayda bu matbu numarasına ait irsaliye listelenir. |
| Distribütör Kodu | Basımı yapılacak belgelerin bağlı olduğu distribütör seçimi bu alandan yapılır. |
| İlk Belge No | Belge numarasına bağlı olarak İrsaliyeler listelenecek ise, yani yazdırılacak irsaliyenin belge numaraları sıralıysa ilk İrsaliyenin belge no değeri bu alana yazılır |
| Son Belge No | Belge numarasına bağlı olarak İrsaliyeler listelenecek ise, yani yazdırılacak irsaliyelerin belge numaraları sıralıysa son İrsaliyenin belge no değeri bu alana yazılır. |
| Başlangıç Tarihi | Yazdırılacak belgelerin işlem tarihlerini kapsayan tarih aralığı için Başlangıç Tarih değeri bu alana yazılır. |
| Bitiş Tarihi | Yazdırılacak belgelerin işlem tarihlerini kapsayan tarih aralığı için Bitiş Tarih değeri bu alana yazılır. |
| Müşteri Kriter Tipi | Müşteri kriteri vererek İrsaliyelerin detayda listelenmesi ve buna bağlı olarak yazdırılma işlemi yapılacak ise Müşteri Kriter Tip alanından Kod, Grup Ek grup değerlerinden biri seçilerek Müşteri Kriteri rehberinde listelenecek değerler belirlenir. |
| Müşteri Kriter | Müşteri kriteri vererek İrsaliyelerin detayda listelenmesini istiyorsak ve buna bağlı olarak yazdırma işlemi yapılacak ise, Müşteri Kriter Tip alanında seçmiş olduğumuz değere göre Müşteri Kriter seçimi yapılır ve seçilen müşterilere ait İrsaliyeler detayda listelenir. |
| Satış Temsilcisi | Satış Temsilcisine bağlı İrsaliyeler yazdırılacak ise; bu alandan Satış temsilcisi seçimi yapılarak detayda sadece seçilen satış temsilcisine ait İrsaliyelerin listelendiğini görürüz. |
| Bölge | Belli bölgelerdeki müşterilerin İrsaliyeleri yazdırılacak ise, bu alandan Bölge Kodu seçilerek detayda seçilen bölgeye ait müşterilerin İrsaliyelerin listelendiğini görürüz. |
| Depo | Depo Kodu kriterine bağlı olarak İrsaliyeler yazdırılacak ise, bu alandan Depo Kodu seçilerek detayda seçilen depolara ait İrsaliyelerin listelendiğini görürüz. |
| Yükleme Kodu | Belli bir yükleme koduna ait İrsaliyeler yazdırılacak ise; bu alandan yükleme kodu seçilerek detayda ilgili yüklemedeki İrsaliyelerin listelendiğini görürüz. |
| Araçlar | Araç Kodu kriterine bağlı olarak İrsaliyeler yazdırılacak ise; bu alandan Araç Kodu seçilerek detayda seçilen araçlara ait İrsaliyelerin listelendiğini görürüz. |
| Hareket Tipi | Hareket Tipi kriterine bağlı olarak İrsaliyeler yazdırılacak ise; bu alandan İrsaliye hareket tipi seçilerek detayda seçilen hareket tiplerine ait İrsaliyelerin listelendiğini görürüz. |
| Belge Tipi | Bu alanda Satış, Alış, Konsinye Satış, Konsinye Alış, İade İrsaliyesi(Bozuk) ve İade İrsaliyesi( Sağlam) belge tipleri listelenir ve yazdırılacak irsaliyelerin belge tipi bu alandan seçilir. |
| Satış Tip | Satış Tipi alanında Hepsi, Açık, Kapalı değerleri listelenir ve bu alandan seçmiş olduğumuz satış tipindeki irsaliyeler detayda listelenir ve yazdırma işlemi yapılabilir. |
| Giriş Tip | Giriş Tipi alanında Hepsi, Manuel, El Terminali, Aktarım değerleri listelenir ve bu alandan seçmiş olduğumuz giriş tipindeki irsaliyeler detayda listelenir ve yazdırma işlemi yapılabilir. |
| Ödeme Tipi | Ödeme Tipi alanında Hepsi, Peşin, Çek, Senet, Kredi Kartı, Açık Hesap, Ticari Kart, DBS, Havale/ EFT, Mail order değerleri listelenir ve bu alandan seçmiş olduğumuz ödeme tipindeki irsaliyeler detayda listelenir ve yazdırma işlemi yapılabilir. |
| E-Fatura Müşterisi | Bu alanda Hepsi, Evet, Hayır değerleri listelenir E- Fatura Müşterilerine ait İrsaliyelerin detayda listelenmesi ve toplu yazdırma işleminin yapılması isteniyorsa EVET değeri seçilmeli. E- Fatura Müşterilerine ait İrsaliyelerin toplu yazdırma işleminin yapılması istenmiyorsa HAYIR değeri seçilmeli. HEPSİ değerinin seçilmesi durumunda Tüm müşterilere ait irsaliyeler listelenir(diğer kriterlere bağlı olarak ) |
| Belge Basım Durumu | Bu alanda yazdırılmadı, yazdırıldı ve Hepsi seçenekleri listelenir. Daha önceden yazdırılmış İrsaliyelerin tekrar toplu belge basımından yazdırılması isteniyor ise Yazdırıldı değeri seçilerek detayda listelenmesi sağlanır, Hepsi değeri seçilir ise daha önceden yazdırılan ve yazdırılmayan İrsaliyelerin tümü diğer kriterlere bağlı olarak listelenir. Yazdırılmadı değerinin seçilmesi durumunda daha önceden yazdırılmayan irsaliyeler yine diğer kriterlere bağlı olarak detayda listelenir. |
| Belge Kodu | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen İrsaliyelerin Belge Kodu değerini gösterir. |
| Müşteri Kodu | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen İrsaliyelerin Müşteri Kodu değerini gösterir. |
| Müşteri Adı | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen İrsaliyelerin Müşteri Adı değerini gösterir. |
| ST Kodu | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen İrsaliyelerin Satış Temsilcisi Kodunu gösterir. |
| Satış Temsilcisi | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen İrsaliyelerin Satış Temsilcisi Adını gösterir. |
| Araç Kodu | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen İrsaliyelerin Araç Kodunu gösterir. |
| Dağıtım Sıra | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen İrsaliyelerin Dağıtım sırasıdır. |
| Plaka | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen İrsaliyelerin Araç Plaka Kodunu gösterir. |
| İşlem Tarihi | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen İrsaliyelerin İşlem Tarihini gösterir. |
| Net Tutar | Kriter alanında verilen kriterlere bağlı olarak detayda listelenen İrsaliyelerin Net Tutar değerini gösterir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Listele butonuna bastığımızda kriter alanına girilen kriterlere bağlı olarak İrsaliyeler detay alanında listelenir. |
| Temizle | Temizle butonuna bastığımızda kriter alanına girilen kriterler ve bunlara bağlı olarak detay alanında listelenen irsaliyeler temizlenir. |
| Kapat | İrsaliye Toplu Basım ekranını kapatır. |
| Yenile | Detay listesinin yenilenmesini sağlar |
| Yazdır | Detay alanında listelenen ve daha sonra seçimi yapılan İrsaliyelerin Yazdırılması sağlanır. Yazdır butonuna bastığımızda Basım Ekranı açılır. |
| İzle | Detay alanında listelenen ve seçimi yapılan İrsaliye izle modunda açar. |
