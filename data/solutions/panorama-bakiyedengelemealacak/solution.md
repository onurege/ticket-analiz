# Bakiye Dengeleme Alacak

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tahsilat İşlemleri → Bakiye Dengeleme Alacak`

## Ekran Tanımı

Bakiye Dengeleme Alacak işleminin yapıldığı adımdır. Ekrana daha önceden girilmiş olan Bakiye Dengeleme Alacakları gelir. Önceki girilen Bakiye Dengeleme Alacakları üzerinde iptal, düzenleme işlemi yapılabilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Bakiye Dengeleme Alacağa program tarafından verilen artışlı numaradır. |
| Belge No | Bakiye Dengeleme Alacağa verilecek olan belge kodudur. |
| Makbuz No | Tahsilat makbuzu üzerindeki girilecek olan makbuz numarasıdır. |
| İşlem Tarihi | Bakiye Dengeleme Alacağın yapıldığı tarih bilgisinin girildiği alandır. |
| Distribütör Adı | Distribütör bilgisinin girildiği alandır. |
| Müşteri | Bakiye Dengeleme Alacağının yapıldığı müşteri kodudur. |
| Satış Temsilcisi | Müşterini rutunda olduğu satış temsilcisi bilgisidir. |
| Tahsilat ST Kod | Tahsilatı yapan satış temsilcisi bilgisidir. |
| İşlem Tipi | Tahsilat işlem tipinin girildiği alandır. |
| Döviz Tipi | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsidir. |
| Dövizli Tutar | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsinden tutarıdır. |
| Kur | Döviz takibi yapılıyor ise işlem yapılan tahsilatın o günkü kur bilgisidir. |
| Tutar | Yapılan Bakiye Dengeleme Alacağın tutarının girildiği alandır. |
| Bakiye | Bakiye Dengeleme Alacak kaydı girilen müşterinin bakiye bilgisidir. |
| Basım | Bakiye Dengeleme Alacak kaydının basım durum bilgisidir. |
| Giriş Tip | Bakiye Dengeleme Alacak kaydının giriş tipi bilgisidir. |
| Durum | Bakiye Dengeleme Alacak kaydının belge durum bilgisidir. Aktif, silindi değerlerini alır. |
| Fatura Kodu | Bakiye Dengeleme Alacak kaydı için bu alan kullanılmamaktadır |
| Sipariş Kodu | Bakiye Dengeleme Alacak kaydı için bu alan kullanılmamaktadır. |
| Aktarım Durumu | Bakiye Dengeleme Alacak kaydının ticari paket aktarım durum bilgisidir. Aktarıldı, Aktarılmadı değerlerini alır. |
| Tahsilat İptal Nedeni | Tahsilat iptal nedeninin girilebildiği alandır. |
| Proforma Fatura Kodu | Bakiye Dengeleme Alacak kaydı için bu alan kullanılmamaktadır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Yeni | Yeni kayıt girmek için kullanılan butondur. |
| Düzenle | Mevcut kayıtlarda düzenleme amacıyla kullanılan butondur. |
| İzle | Mevcut kayıtlarda izleme yapabilmek için kullanılan butondur. |
| Kopyala | Listede seçili olan tahsilatı kopyalamak için kullanılan butondur. |
| Sil | İptali istenen kaydı silmek için kullanılan butondur. |
| Yazdır | Tanımlı dizayna göre çıktı alınmak istendiğinde kullanılan butondur. |
| Fatura Kapat | Fatura kapatma ekranını açmaktadır. |
