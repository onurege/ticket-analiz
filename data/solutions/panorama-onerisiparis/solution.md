## Ekran Tanımı

Öneri Sipariş ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Durum | Sipariş durum bilgisidir. Aktif/Pasif değerlerini alır. |
| Başlangıç Tarihi | Kaydın(Dönemin) başlayacağı zorunlu Tarih alanıdır. |
| Bitiş Tarihi | Kaydın(Dönemin) biteceği zorunlu Tarih alanıdır. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| ST Grup Kodu | Satış temsilcisi grup kodu bilgisidir. |
| Satış Tamsilcisi | Satış temsilcisi bilgisidir. |
| Müşteri Kriter Tipi | Müşteri kriter tipi bilgisidir. |
| Müşteri Kriter | Müşteri kriter bilgisidir. |
