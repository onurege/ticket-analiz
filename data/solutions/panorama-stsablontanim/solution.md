# Satış Temsilcisi Parametre şablon

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Parametreler → Satış Temsilcisi Parametre Şablon`

## Ekran Tanımı

Satış temsilcisi parametre şablon ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Satış temsilcisi şablon kodu bilgisidir. |
| Ad | Saha adı bilgisidir. |
| Menü Grubu | Menü grubu bilgisidir. “Dizayn”, “GPS”, “İletişim”, “İskonto/Promosyon”, “Parametreler”, “Risk Kontrol”, “Uygulama Kriteri”, “Yetkiler” değerlerini alır. |
| Saha Adı |  |
| İşlem Yetkisi |  |
| Değer |  |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Değişiklikleri kaydet ve yeni belge tanımlama butonudur. |
| Kaydet ve Kapat | Değişiklikleri kaydet ve ekranı kapatmaya yarar. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
| Yenile | Listeyi yenilemek amaçlı kullanılır. |
| Düzenle | Hiyerarşi bilgilerini düzenlemek için kullanılır. |
| İzle | Kayıtları incelemek için kullanılır. |
| Parametre Detay | Parametre detay bilgisidir. |
| Saha Adı | Saha adı bilgisidir. |
| İşlem Yetkisi | İşlem yetkisi bilgisidir. |
| Değer | Dizayn değer bilgisidir. |
