# Toplu Müşteri Hiyerarşi Değiştirme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → İşlemler → Toplu Müşteri Hiyerarşi Değiştirme`

## Ekran Tanımı

Müşteri hiyerarşilerinin toplu olarak değiştirilmesini sağlayan ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Müşterilerin distribütör koduna göre filtrelenmesini sağlar. Zorunlu alandır. |
| Grup Kodu | Müşterilerin grup koduna göre filtrelenmesini sağlar. |
| Ek Grup Kodu | Müşterilerin Ek Grup Koduna göre filtrelenmesini sağlar. |
| Yönetim HY-1 | Müşterilerin Yönetim Hiyerarşi-1 bilgisine göre filtrelenmesini sağlar. |
| Yönetim HY-2 | Müşterilerin Yönetim Hiyerarşi-2 bilgisine göre filtrelenmesini sağlar. |
| Grup Kırılımı | Müşterilerin Grup Kırılım bilgisine göre filtrelenmesini sağlar. |
| Yönetim HY-1 | Müşterinin Yönetim Hiyerarşi-1 bilgisinin hangi tanıma güncelleneceğinin bilgisinin girildiği alandır. |
| Yönetim HY-2 | Müşterinin Yönetim Hiyerarşi-2 bilgisinin hangi tanıma güncelleneceğinin bilgisinin girildiği alandır. |
| Toplu Müşteri Ekle | Müşteri LNGKOD bilgilerinin aralarında virgülle girilip toplu eklenmesi için kullanılan alandır. |
| Eklenemeyen | Listele yapıldıktan sonra Toplu Müşteri Ekle alanına girilen Müşteri LNGKOD larından listeye Eklenemeyen kodların belirtildiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İşlemi Başlat | Müşterilerin hiyerarşi değiştirme işleminin başlaması için kullanılır. |
| Listele | Girilen kriter bilgilerine göre detayda müşterilerin listelenmesini sağlar. |
| Temizle | Girilen kriter bilgilerini ekrandan kaldırır. |
| Kapat | Ekranı kapatır. |
| \|< | Listede en baş sayfaya gelmek için kullanılır. |
| >\| | Listede en son sayfaya geçmek için kullanılır. |
| Yenile | Listeyi yeniler. |
| İzle | Müşteri kaydının detayını görüntülemek için kullanılır. |
| Çıkar | Müşteri güncellenilecek listeden çıkarmak için kullanılır. |
| SAP Aktarım Sonuç İzleme | Müşteri Hiyerarşi bilgileri değiştirilip SAP ye aktarıldıktan sonra burada sonuçların görüntülenmesi için kullanılır. |
