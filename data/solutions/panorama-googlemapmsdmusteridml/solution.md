# Müşteri İzleme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → Harita İşlemleri → Müşteri İzleme`

## Ekran Tanımı

Müşteri izleme ekranıdır. Belirtilen kriterlere göre harita üzerinde müşterileri izleme ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Distribütör Kodu bilgisidir. |
| Müşteri Grup | Müşteri Grup bilgisidir. |
| Müşteri Ek Grup | Müşteri Ek Grup bilgisidir. |
| Müşteri | Müşteri kodu bilgisidir. |
| İl | İl kodu bilgisidir. |
| İlçe | İlçe bilgisidir. |
| Semt | Semt bilgisidir. |
| ST/SE Elemanları | Satış Temsilcisi/Satış Elemanları bilgisidir. |
| Durum | Müşteri durum bilgisidir. Aktif, Pasif değerlerini alır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Ekrandaki bilgileri yenilemek için kullanılır. |
| Ortala | Haritayı müşterilere göre ortalamak amaçlı kullanılır. |
| Temizle | Filtreleri temizlemek için kullanılır. |
| Arama | Müşteri aramak için kullanılır. |
| Kapat | Ekranı kapatmak için kullanılır. |
| Filtre Uygula | Girilen bilgileri bazalarak filtreleme uygulanır. |
| Temizle | Girilen kriterleri temizlemek için kullanılır. |
| Müşteri Tanımı | Seçilen müşteriyi haritada göstermeye ve bilgilendirmeye yarar. |
| Müşteri Ara | Müşteri kodu girilerek ekran üzerinde müşteri araması yapılır. |
