# İzin Nedenleri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Parametreler → İzin Nedenleri`

## Ekran Tanımı

İzin nedenleri listesidir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | İzin nedeni kod bilgisidir. |
| Ad | İzin nedeni isim bilgisidir. |
| Distibütör Kodu | Tanımın hangi distribütöre ait olduğunu belirten distribütör kod bilgisidir. |
| Durum | Hangi durumdaki izin bilgilerinin aktarılacağı bilgisidir. (Aktif, Pasif, İptal) |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Bilgileri kaydetmeye yarar. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
