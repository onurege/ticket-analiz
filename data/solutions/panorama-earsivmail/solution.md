# E-Arşiv Mail

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Kart Takip → E-Arşiv Mail`

## Ekran Tanımı

E-Arşiv Mail ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Listleme işlemi için kriter olan Distribütör kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Girdilere göre listeleme işlemleri yapılmaktadır. |
| Mail Gönder | Mail gönderme işlemini gerçekleştirir. |
| Kapat | Belge ekranını kapatmak için kullanılır. |
| Yenile | Belge detay listesini yenilemek için kullanılır. |
| Hepsini Göster | Tümünü listelemek için kullanılır. |
| Mail Gönderilmeyenler | Mail gönderilmeyenleri listemek için kullanılır. |
