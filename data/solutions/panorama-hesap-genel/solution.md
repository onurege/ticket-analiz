# Hesap-Genel

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tanımlamalar → Hesap - Genel`

## Ekran Tanımı

Genel hesap bilgilerinin tanımlandığı ekrandır. Öncelikle ekrana sisteme girilmiş olanlar genel hesaplar listelenir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Genel - Hesap tanımına ait program tarafından verilen artışlı numaradır. |
| Durum | Tanımlanan hesaba ait durum bilgisidir. Aktif ve Pasif değerlerini alır. |
| Ad | Tanımlanan hesabın adının girildiği metin sahasıdır. |
| Hesap Tipi | Tanımlanan hesabın tip bilgisidir. Mevduat veya DBS değerlerini alır. |
| Distribütör Kodu | Tanımlanan hesabın bağlı olduğu distribütör kodudur. |
| Hesap No | Tanımlanan hesaba ait bankanın verdiği hesap no bilgisidir. |
| Banka Kodu | Hesabın bağlı olduğu banka bilgisidir ve rehber alandır. |
| Ödeme Tipi | Hesabın ödeme tipi bilgisidir. Peşin veya Vadeli seçilebilir. |
| Şube | Hesabın bağlı olduğu şube bilgisidir ve rehber alandır. |
| Vade Günü | Hesaba ait vade gün bilgisidir. |
| IBAN No | Hesaba ait IBAN numarası bilgisidir. |
| Abone No | Hesaba ait abone numarası bilgisidir. |
| Müşteri Kodu | Hesapla ilintili müşteri bilgisi var ise otomatik olarak gösterildiği pasif alandır. |
| Merkez DBS | DBS entegrasyonu yapılacak ise DBS entegrasyonunun seçildiği rehber alanıdır. |
| Döviz Tipi | Hesap dövizli ise para biriminin seçildiği rehber alanıdır. |
