# Müşteri Grubu Alt Kırılım

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → Tanımlamalar → Müşteri Grubu Alt Kırılımı`

## Ekran Tanımı

Müşteri grubuna ait alt kırılım bilgilerinin tanımlandığı ekrandır. Öncelikle ekrana, sisteme girilmiş olan müşteri grubu alt kırılımı listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Grup Kodu | Yapılacak alt kırılım tanımının hangi gruba ait olucağının belirtildiği alandır. |
| Kod | Müşteri grup alt kırılımın tanımına verilen numaradır. |
| Ad | Müşteri grup alt kırılım tanımına verilen isimdir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemin kaydedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
