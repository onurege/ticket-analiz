# Kota Şablon

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriteri → Satış Kriterleri → Kota Şablon`

## Ekran Tanımı

Kota Şablon bilgilerinin tanımlandığı ekrandır. Öncelikle ekrana, sisteme girilmiş olan kota şablon listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Kota Şablon grubuna verilen artışlı numaradır. |
| Durum | Kota Şablon Durum bilgisidir, aktif/pasif değerlerini alabilir. |
| Konrol Tip | Kota Şablon Kontrol bilgisidir, uyarı ver/ işlemi kes değerlerini alabilir. |
| Başlangıç Yıl | Tanımlanan Kota Şablon ayarlarının hangi ayda başlayacağı bilgisi girilir. |
| Başlangıç Ay | Tanımlanan limit ayarlarının hangi ayda başlayacağı bilgisi girilir. |
| Bitiş Yıl | Tanımlanan Kota Şablon ayarlarının hangi yıldan itibaren geçersiz olacağı bilgisi girilir. |
| Bitiş Ay | Tanımlanan Kota Şablon ayarlarının hangi aydan itibaren geçersiz olacağı bilgisi girilir. |
| Belge Tür | Satış veya alış işlemi olduğunu belirtir. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| Dist Grup | Distribütör grup bilgisidir. |
| Dist Ek Grup | Distribütör ek grup bilgisidir. |
| Dist İl Kod | Distribütör il bilgisidir. |
| Müşteri Kodu | Müşterinin kod bilgisidir. |
| Müşteri Grup Kod | Müşterinin grup kod bilgisidir. |
| Müşteri Ek Grup Kod | Müşterinin ek grup kod bilgisidir. |
| Ürün Kriter Tipi | Ürünün kriter tipi seçilir, kod/grup değerlerini alır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemin kaydedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
| Hedef Kontrol Tipi | Hedef tip miktar ya da tutar şeklinde seçilebilir. Miktar seçildiği zaman belirlenen ürün miktarı üzerinden limitleme yapılır. Tutar seçildiği zaman belirlenen ürünün tutarı üzerinden limitleme yapılır. |
| Ürün Kriteri | Ürün kriteri ürün listesinden seçilir. |
| Günlük | Günlük en az kaç adede kota şablon uygulanacağı sınırı belirtilir. |
| Haftalık | Haftalık en fazla kaç adede kota şablon uygulanacağı sınırı belirtilir. |
| Aylık | Aylık en fazla kaç adede kota şablon uygulanacağı sınırı belirtilir. |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Detayda yeni tanım yapmak için kullanılır. |
| İzle | Detaydaki kaydı görüntülemek için kullanılır. |
| Sil | Detaydaki kaydın silinmesi için kullanılır. |
