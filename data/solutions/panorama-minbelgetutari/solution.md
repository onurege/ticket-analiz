# Minimum Belge Tutarı

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriteri → Satış Kriterleri → Minimum Belge Tutarı`

## Ekran Tanımı

Minimum belge tutarı ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Minimum Belge Tutar grubuna verilen artışlı numaradır. |
| Durum | Minimum Belge Tutar Durum bilgisidir, aktif/pasif değerlerini alabilir. |
| Başlangıç Tarihi | Tanımlanan Minimum belge tutarı ayarlarının hangi tarihte bağlıcağı bilgisi girilir. |
| Bitiş Tarihi | Tanımlanan Minimum belge tutarı ayarlarının hangi tarihte biticeği bilgisi girilir. |
| ST Grup Kodu | Satış temsilcisi grup kodu bilgisidir. |
| Satış Temsilcisi | Satış temsilcisi bilgisidir. |
| Ürün Kriter | Ürünün kriteri seçilir, “Grup”, “Ek Grup”, “Dinamik Grup” değerlerini alır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemin kaydedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
| Ürün | Minimum belge tutarı girilicek ürün bilgisidir. |
| Tutar | Minimum tutar bilgisidir. |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Detayda yeni tanım yapmak için kullanılır. |
| İzle | Detaydaki kaydı görüntülemek için kullanılır. |
| Sil | Detaydaki kaydın silinmesi için kullanılır. |
