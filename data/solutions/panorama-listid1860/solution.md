# Rut Grup Tanım

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Rut Ve İş Emri Planlama → Rut Grup Tanım`

## Ekran Tanımı

Rut grup tanımlama işlemidir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Önceden tanımlanmış müşteri listesinden seçilen müşterilerin kod bilgilerinin otomatik olarak geldiği ekrandır. |
| Durum | Aktif/Pasif durumunu belirleyen seçenektir |
| Rut Grup Adı | Rut grup isim bilgisidir. |
| Distribütör Kodu | Distribütör kodunun seçildiği sahadır. Rut grup tanımının hangi distribütöre bağlı olduğu belirlenir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemlerin kayıt edilmesini sağlar. |
| İptal | Yapılan işlemlerin kayıt edilmeden, ekranın kapanmasını sağlar. |
