# Müşteri Stoğuna Göre Fiyat İndirim Tanımlama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri 🡪 Fiyat İşlemleri 🡪 Müşteri Stoğuna Göre Fiyat İndirim Tanımlama`

## Ekran Tanımı

Öneri Sipariş ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Program tarafından verilen artışlı numaradır. |
| Durum | Sipariş durum bilgisidir. Aktif/Pasif değerlerini alır. |
| Distribütör Kriter Tipi | Distribütörlerin seçileceği kriterlerin girildiği alandır değer olarak (Kod, Grup, Ek Grup, Dinamik Grup) değerlerini almaktadır. |
| Distribütör Kriteri | Distribütör Kriter Tipi ne göre değişkenlik gösteren bir alandır. Örneğin Distribütör Kriter Tipi “Kod” seçilirse sisteme kayıtlı tüm Distribütörler gelir ve tek bir Distribütör seçilebilir fakat Distribütör Kriter Tipi “Grup” seçilir ise Distribütör grupları seçilir. |
| Para Birimi | Fiyat indirimi olarak tanımlanacak fiyatların hangi para birimine göre tanımlanacağının seçildiği alandır. |
| Başlangıç Tarihi | Kaydın(Dönemin) başlayacağı zorunlu Tarih alanıdır. |
| Bitiş Tarihi | Kaydın(Dönemin) biteceği zorunlu Tarih alanıdır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Yapılan işlemin kaydedilmesini sağlar ve yeni belge açar. |
| Kaydet ve Kapat | Yapılan işlemin kaydedilmesini sağlar ve ekranın kapatılması işlemini yapar. |
| Kapat | Ekranı kapatır. |
| Yenile | Listeyi yenilemeye yarar. |
| Sil | Listeden seçili fiyat indirimi silmek için kullanılır. |
| Yeni | Yeni fiyat indirim eklemek için kullanılır. |
