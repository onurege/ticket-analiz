# Kasa İşlemleri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Kasa İşlemleri → Kasa İşlemleri`

## Ekran Tanımı

Kasa İşlemleri ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Tip | Listeleme seçenekleri tip bilgisidir. |
| Başlangıç Tarihi | Başlangıç tarihi bilgisidir. |
| Bitiş Tarihi | Bitiş tarihi bilgisidir. |
| Kasa Kod | Kasa kodu bilgisidir. |
| Satış Temsilcisi | Satış Temsilcisi bilgisidir. |
| Yeni işlem Tipi | Yeni işlem tipi bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Listele | Girilen detaylara göre listeleme işlemi yapmaktadır. |
| Temizle | Ekrandaki değişiklikleri temizlemek için kullanılır. |
| Kapat | Ekranı kaydetmeden kapatır. |
| Yeni | Yeni kayıt girmek için kullanılan butondur. |
| Düzenle | Mevcut kayıtlarda düzenleme amacıyla kullanılan butondur. |
| İzle | Mevcut kayıtlarda izleme yapabilmek için kullanılan butondur. |
| Sil | İptali istenen kaydı silmek için kullanılan butondur. |
