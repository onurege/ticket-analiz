# Distribütör Nokta Devri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → İşlemler → Distribütör Nokta Devri`

## Ekran Tanımı

Distribütör nokta devri ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Devralacak Distribütör | Distribütör listesinden devralacak distribütör seçilir. |
| Devralacak Satış Temsilcisi | Distribütör listesinden distribütör kodu seçilir. |
| Distribütör Kodu | Müşterilerin distribütör koduna göre filtrelenmesini sağlar. Zorunlu alandır. |
| İl | İl listesinden il seçimi yapılır. |
| İlçe | İlçe listesinden ilçe seçimi yapılır. |
| Semt | Semt listesinden semt seçimi yapılır. |
| Satış Temsilcisi | Satış temsilcisi bilgisidir. |
| Müşteri Grup | Müşteri grubu listesinden müşteri grubu seçimi yapılır. |
| Müşteri Ek Grup | Müşteri ek grup listesinden müşteri ek grubu seçimi yapılır. |
| Bölge | Müşteri bölge MSD listesinden bölge seçimi yapılır. |
| Müşteri İptal Nedeni | Önceden tanımlanmış olan iptal nedenleri listesinden müşteri iptal nedeni seçimi yapılır. |
| Rut | Rut tanımları devredilmek isteniliyorsa kutucuk işaretlenmeli. |
| Müşteri Genel Kodu | Müşteri genel kodu bilgisinin devredilmesi isteniliyorsa kutucuk işaretlenmeli. |
| Ek Saha İçerik Bilgileri | Ek saha içerik bilgileri devredilmesi isteniliyorsa kutucuk işaretlenmeli. |
| E-İskonto Bakiyesi | E-İskonto bakiyesi devredilmek isteniliyorsa kutucuk işaretlenmeli. |
| Demirbaş Bilgileri | Demirbaş bilgileri devredilmek isteniliyorsa kutucuk işaretlenmeli. |
| İskonto Bilgileri | İskonto bilgisidir gösterilsin istenirse seçilebilir. |
| Cari Bakiye | Cari bakiye bilgisi gösterilsin istenirse seçilebilir. |
| Sözleşme Bilgileri | Sözleşme bilgisi gösterilsin istenirse seçilebilir. |
| Kontrat Bilgileri | Kontrat Bilgileri bilgisidir. |
| Ticari Pazarlama Bilgileri | Ticari pazarlama bilgisdir. |
| Müşteri Merkez Kod | Müşteri merkez koduna göre belge düzeltilmek istenirse seçilir. |
| Müşteri Kodu | Müşteri koduna göre belge düzeltilmek istenirse seçilir. |
| Toplu Ekle/Çıkar | Müşteri listesinden toplu ekleme yapılır. |
| Eklenemeyen | Eklenemeyen bilgi alanıdır. |
| Müşteri Kodu | Müşteri listesinden müşteri seçimi yapılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Belgeyi kaydetmek için kullanılır |
| Toplu Ekleme | Toplu müşteri ekleme ekranını açar |
| Temizle | Girilen kıstasları temizler |
| Kapat | Ekranı kapatır. |
| Yenile | Listeyi yeniler. |
| Yeni | Yeni kayıt eklemek için kullanılır. |
| İzle | Seçilen kaydı incelemek için kullanılır. |
| Sil | Seçilen kaydı temizlemek için kullanılır. |
