# Belge Basım Düzeltme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Sistem yönetimi 🡪 Ek İşlemler 🡪 Belge Basım Düzeltme`

## Ekran Tanımı

Belgelerin, Belge basım durumlarının değiştirildiği ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Belge Tipi | Belge tipinin seçildiği ekrandır. Örneğin: Alış faturası-Satış faturası vb. |
| Belge Tür | Belge türünün seçildiği ekrandır. Örneğin: Alış-Satış-Müşteriden İade vb. |
| Distribütör Kodu | Distribütör seçildiği alandır. Distribütör kullanıcılarında bu bilgi otomatik olarak gelmektedir. Alan zorunludur |
| Satış Temsilcisi | Satış temsilcisinin seçildiği alandır. |
| Başlangıç Tarihi | Belgelerin başlangıç tarihlerinin girildiği alandır. |
| Bitiş Tarihi | Belgelerin bitiş tarihlerinin girildiği alandır. |
| Başlangıç Takip No | Başlangıç takip girilen numeric bir alandır. |
| Bitiş Takip No | Bitiş takip girilen numeric bir alandır. |
| Matbu No | Belgenin Matbu no sunun yazıldığı alandır. |
| Toplu Kod Giriş | Birden fazla belge girişi yapılmak istendiğinde toplu olarak belge kodları bu alana yazılabilir. |
| Başarısız | Belge durumu değiştirilmek istenen belgelerin işlemi başarısız ise , sistem bu alana belge kodlarını gösterir. |
| Belge Kodu | Belge durumu değiştirilmek istenen belgelerin seçiminin yapıldığı alandır. |
| Basım durumu | Belgenin basım durumunun hangi basım şekline alınmak istendiğinin seçildiği alandır. Yazdırıldı ve Yazdırılmadı olarak 2 seçenek vardır. |
