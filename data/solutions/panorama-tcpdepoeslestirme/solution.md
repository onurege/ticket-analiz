# Depo

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → Ticari Paket → Eşleştirme → Depo`

## Ekran Tanımı

Panorama ile ticari pakette kullanılan depo kodlarının farklı olduğu durumlarda, bu menü üzerinden ticari pakette kullanılan depo kodları girilerek aktarım sağlanabilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Adı | İşlem yapılacak Distribütör bilgisi seçilir. |
| Tp Kod | Listelenen depo kayıtlarının Ticari Paket programında kullanılan kod bilgisi girilir. Giriş yapıldığı anda kaydetme işlemi gerçekleşecektir. |
| Kod | Listelenen depo kayıtlarının Panoramada kullanılan kod bilgisi gösterilir. |
| Ad | Listelenen depo kayıtlarının Panoramada kullanılan ad bilgisi gösterilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kapat | Form kapatılır. |
| Yenile | Sayfanın yenilenmesini sağlar. |
| Düzenle | Seçilen satırda Tp Kod alanını aktif hale getirir ve değer girişi yapılmasını sağlar. |
| Yön Tuşları | Listelenen depo kayıtlarında ileri-geri sayfa geçişi yapılmasını sağlar. |
