# Üretim Yeri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri → Tanımlamalar → Üretim Yeri`

## Ekran Tanımı

Üretim yerinin tanımlandığı alandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Takip kodu bilgisidir. |
| Kod | Program tarafından verilen artışlı numaradır. |
| Ad | Üretim Yeri isim bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemin kaydedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
