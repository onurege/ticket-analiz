# Irsaliyeden Fatura Oluşturma(Sağlam)

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri →Müşteriden İade İşlemleri → İrsaliyeden Fatura Oluşturma(Sağlam)`

## Ekran Tanımı

Distribütörün ürün alımı yaptığı firma veya genel merkezinden talep etmiş olduğu İrsaliyeden fatura oluşturma (sağlam) ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yükleme Kodu | Yapılan işleme ait program tarafından otomatik verilen kayıt numarasıdır. |
| Distribütör Kodu | Belgenin hangi Distribütör için kesileceğinin tanımlandığı alandır. |
| İş Bildirim | İş birliği bilgisidir. |
| Başlangıç Tarihi | Fatura (sağlam) belgelerinin hangi tarihten başlayacağı bilgisidir. |
| Bitiş Tarihi | Fatura (sağlam) belgelerinin hangi tarihte biteceği bilgisidir. |
| Başlangıç Sevk Tarihi | İrsaliyeden Fatura belgelerinin hangi Sevk tarihinden başlayacağı bilgisidir. |
| Bitiş Sevk Tarihi | İrsaliyeden Fatura belgelerinin hangi Sevk tarihinden biteceğinin bilgisidir. |
| Satış Temsilcisi | Fatura (sağlam) hangi Satış Temsilcileri üzerinden listelendiği seçiminin yapıldığı alandır. |
| Bölge Kodları | Fatura (sağlam) hangi müşteri Bölge üzerinden listeleneceği seçiminin yapıldığı bilgisidir. |
| Başlangıç Sevk Tarihi | Fatura (sağlam) belgelerinin hangi Sevk tarihinden başlayacağı bilgisidir. |
| Hareket Tipi | Fatura (sağlam) için tanımlanan hareket tipinin seçildiği alandır. Zorunlu bir alandır. |
| Seçilen Araç | Fatura (sağlam) yükleme yapılacağı araç seçilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Temizle | Listedeki verileri temizlemek için kullanılır. |
| Sevk Tarihleri | Sevk tarhi ayarlamak için kullanılan ekrandır. |
| Listele | Girdilere göre listeleme işlemleri yapılmaktadır. |
| Belgeleştir | Sevk Tarihleri ekranını açmaktadır. |
| Kaydet | Belgeyi kaydetmek için kullanılır. |
| Kapat | Belge ekranını kapatmak için kullanılır. |
| Yenile | Belge detay listesini yenilemek için kullanılır. |
| Düzenle | Belge detay satırında düzenleme yapmak için kullanılır. |
| Belge İzle | Belge detay satırında seçili belgeyi incelemek için kullanılır. |
| Transfer | Listelenen siparişlerin istenilen yere transfer işlemi gerçekleştirilir. |
| Yukarı | “Yukarı” – “Aşağı “ butonları ile oluşacak belgelerin öncelik sıralaması değiştirilebilir. |
| Aşağı | “Yukarı” – “Aşağı “ butonları ile oluşacak belgelerin öncelik sıralaması değiştirilebilir. |
