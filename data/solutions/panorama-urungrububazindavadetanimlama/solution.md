## Ekran Tanımı

Ürün grubu bazında vade tanımlama ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Durum | İşlem durum bilgisidir. Aktif/Pasif değerlerini alır. |
| Dist.Kodu | Distribütör kodu bilgisidir. |
| Müşteri Kriter Tipi | Müşteri kriter tipi bilgisidir. |
| Müşteri Kriter | Müşteri kriter bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Kapat | Ekranı kaydetmek ve kapatmak için kullanılan butondur. |
| Kapat | Ekranı Kapatma işlemi yapar. |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Detayda yeni tanım yapmak için kullanılır. |
| İzle | Detaydaki kaydı görüntülemek için kullanılır. |
| Sil | Detaydaki kaydın silinmesi için kullanılır. |
