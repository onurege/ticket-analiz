# Fatura Gecikme Günü Tanımları

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → İşlemler → Fatura Gecikme Günü Tanımları`

## Ekran Tanımı

Gecikmiş fatura işlemlerini tanımlamak ve hesaplamaya hazır hale getirye yarayan ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Takip Kodu | Tanımlanacak Fatura Gecikme Günü hangi takip koduyla belirtileceğini gösteren kod bilgisidir. |
| Başlangıç Tarihi | Tanımlanacak Fatura Gecikme Günü Hesaplama ayarlarının hangi tarihte geçerli olacağı bilgisi girilir. |
| Bitiş Tarihi | Tanımlanacak Fatura Gecikme Günü Hesaplama ayarlarının hangi tarihten itibaren geçersiz olacağı bilgisi girilir. |
| Distribütör Kodu | Tanımın hangi distribütöre ait olduğunu belirten distribütör kod bilgisidir. Zorunlu alandır. |
| Satış Temsilcisi | Satış Temsilcisi bilgisinin seçildiği ekrandır. Zorunlu alandır. |
| Esas Alınacak Tarih | Fatura gecikme günü için esas alınacak tarih bilgisi girilir. |
| İşlem Tarihi | Fatura gecikme gününden itibaren hangi gün işlem yapılacağı girilir. |
| Param Esas Trh | Fatura gecikmedeki parametre başlangış tarihibilgisidir. |
| Param Bitiş Trh | Fatura gecikmedeki parametre bitiş tarihi bilgisidir. |
| Kullanılacak Tarih | Kullanılacak tarih seçiminin yapıldığı alandır. Vade ya da işlem seçimi yapılır. |
| Max. Tolerans | Fatura gecikme gününden sonra maksimum tolerans gösterilecek gün sayısı girilir. |
| Gecikmişler | Sadece faturası gecikmişlerin gösterilmesi istenilirse kutucuk işaretlenir. |
| Rut Dışı | Tanımlanacak Fatura Gecikme Günü için sadece Rut Dışı bilgilerin görünmesi için işaretlenir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Değişiklikleri kaydet ve yeni belge tanımlama butonudur. |
| Kaydet ve Kapat | Değişiklikleri kaydet ve ekranı kapatmaya yarar. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
