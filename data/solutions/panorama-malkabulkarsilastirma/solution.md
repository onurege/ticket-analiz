# Irsaliye Mal Kabul Karşılaştırma

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Alış İşlemleri → Irsaliye Mal Kabul Karşılaştırma`

## Ekran Tanımı

İrsaliye mal kabul kartılaştırma işleminin gerçekleştiği ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kriter Tipi | Distribütör kriteri bilgisidir. |
| Distribütör Kodu | Belgenin hangi Distribütör için kesileceğinin tanımlandığı alandır. |
| Müşteri Kodu | Müşteri kodu bilgisidir. |
| Başlangıç Tarihi | Alış Irsaliye fatura belgelerinin hangi tarihten başlayacağı bilgisidir. |
| Bitiş Tarihi | Alış Irsaliye fatura belgelerinin hangi tarihte biteceği bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Girdilere göre listeleme işlemleri yapılmaktadır. |
| Kapat | Belge ekranını kapatmak için kullanılır. |
| Detaylar | Listede seçilen belgenin detay bilgisini görüntülemeye yarar |
| Belge İzle | Belge detay satırında seçili belgeyi incelemek için kullanılır. |
