# Ürün Bağlantı

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Parametreler → Ürün Bağlantı`

## Ekran Tanımı

Ürün ek saha tanımının başka bir ürün ek saha tanımına bağlantısının yapıldığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Ek sahanın tanımının rehberden seçildiği alandır. |
| Seçenek | Ek saha seçeneğinin rehberden seçildiği alandır. |
| Bağlantı Ek Saha Kod | Bağlantı yapılacak ek sahanın rehberden seçildiği alandır. |
| Bağlantı Ek Saha Seçenek | Bağlantı yapılacak ek sahanın rehberden seçildiği alandır. Birden fazla seçim yapılabilmektedir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan tanımı kaydeder. |
| Kapat | Ekranı kapatır. |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Yeni detay tanımı ekranını açar. |
| İzle | Detaydaki tanımın detaylarının görüntülenmesini sağlar. |
| Düzenle | Detaydaki tanımın detaylarının düzenlenmesini sağlar. |
| Sil | Detaydaki tanımın silinmesini sağlar. |
