# Pm Şablon

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Parametreler → PM Şablon`

## Ekran Tanımı

Satış temsilcisi şablon atama ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Şablon kod bilgisidir. |
| Menü Grubu | Hangi menü başlığında olduğunu belirten bilgi alanıdır. |
| Menü Kod | Menü kodu bilgisidir. |
| Menü Kısayolu | Kısayol tuşuları ile etkileşim için belirtilen kısayolu bilgisidir. |
| Görünürlük | Görünürlük bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Kapat | Değişiklikleri kaydet ve ekranı kapatmaya yarar. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
| Yenile | Listeyi yenilemek amaçlı kullanılır. |
| Düzenle | Şablonları düzenlemek için kullanılır. |
| İzle | Kayıtları incelemek için kullanılır. |
| Yukarı | Listede seçili alanın sırasını değiştirmek için kullanılır. |
| Aşağı | Listede seçili alanın sırasını değiştirmek için kullanılır. |
