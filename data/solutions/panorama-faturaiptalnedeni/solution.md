# Fatura İptal Nedeni

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriteri → Satış Kriterleri → Fatura İptal Nedeni`

## Ekran Tanımı

Fatura iptal nedeni ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Başlangıç Tarihi | Tanımlanan Fatura İptal Nedeni ayarlarının hangi tarihte bağlıcağı bilgisi girilir. |
| Bitiş Tarihi | Tanımlanan Fatura İptal Nedeni ayarlarının hangi tarihte biticeği bilgisi girilir. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| Müşteri Kriter Tipi | Müşteri kriter tipi bilgisidir. “Kod”, “Grup”, “Ek Grup”, “Dinamik Grup” değerleri alabilir. |
| Müşteri Kriter | Müşteri kriter bilgisidir. |
| Satış Temsilcisi | Satış temsilcisi bilgisidir. |
| İlk Fatura No | İlk Fatura numarası bilgisidir. |
| Son Fatura No | Son Fatura numarası bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İşlemi Başlat | Yapılan değişiklikleri kaydetmeye ve işlemi başlatmaya yarar. |
| Listele | Distribütör kodu bilgisine göre iptal edilen faturaları listeler. |
| Temizle | Listeyi ve girilen değerleri temizlemeye yarar. |
| Kapat | Ekranı kapatır. |
| Yenile | Detaydaki listeyi yeniler. |
| Düzenle | Detayda seçili belgeyi düzenlemek için kullanılır. |
| İzle | Detaydaki kaydı görüntülemek için kullanılır. |
| Detaylar | Detaydaki kaydın detaylarını görmek için kullanılır. |
