# Bütçe Dönem

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Fon ve Bütçe Yönetimi 🡪 Tanımlar 🡪 Bütçe Dönem`

## Ekran Tanımı

Fon ve Bütçe Yönetimi işlemlerinde kullanılacak olan Bütçe dönemlerinin tanımlandığı Ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Durum | Deponun durum bilgisidir. Aktif/Pasif değerlerini alır. |
| Başlangıç Tarihi | Kaydın(Dönemin) başlayacağı zorunlu Tarih alanıdır |
| Bitiş Tarihi | Kaydın(Dönemin) biteceği zorunlu Tarih alanıdır |
| Harcama Tipi | Harcamanın ne biçimde yapılacağını gösteren kombo box alandır. Değerleri Oran ve Tutardır. |
| Harcama Değeri | Harcamanın biçimine göre ne kadar yapılacağının girildiği alandır. |
| Yıl | Yıl bilgisinin girildiği zorunlu alandır |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Ekranı kaydedip yeni bir ekran açmak için kullanılacak olan butondur. |
| Kaydet ve Kapat | Ekranı kaydedip kapatmak için kullanılacak olan butondur. |
| Kapat | Ekranı Kapatma işlemi yapar. |
