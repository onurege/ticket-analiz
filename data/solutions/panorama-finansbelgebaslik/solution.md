# Çek Giriş

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Çek İşlemleri → Çek Giriş`

## Ekran Tanımı

Çek girişi yapılan ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Kod | Çek giriş işlemleri belge kodu bilgisidir. |
| İşlem Tarihi | İşlem tarihi bilgisidir. |
| Distribütör Kodu | Distribütör Kodu bilgisidir. |
| Müşteri Kodu | Müşteri Kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan değişiklikleri kaydetmeye yarar. |
| Kapat | Ekranı kapatmak için kullanılır. |
| Yenile | Detayı yenilemek için kullanılan butondur. |
| Yeni | Yeni kayıt girmek için kullanılan butondur. |
| Düzenle | Detay ekranında seçilen belgeleri düzenlemek için kullanılır. |
| İzle | Detay ekranında seçilen belgeleri izlemek için kullanılır. |
| Kopyala | Detay ekranında seçilen belgelerin kopyalarını almak kullanılır. |
| Sil | İptali istenen kaydı silmek için kullanılan butondur. |
