# Çek

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tahsilat İşlemleri → Çek`

## Ekran Tanımı

Çek tahsilat işleminin yapıldığı adımdır. Ekrana daha önceden girilmiş olan çek tahsilatları gelir. Önceki girilen çek tahsilatları üzerinde iptal, düzenleme işlemi yapılabilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Çek tahsilata program tarafından verilen artışlı numaradır. |
| Belge No | Çek tahsila ta verilecek olan belge kodudur. |
| Makbuz No | Tahsilat makbuzu üzerindeki girilecek olan makbuz numarasıdır. |
| Ödendi | Çekin ödeme durumunun belirtildiği alandır. |
| İşlem Tarihi | Çek tahsilatının yapıldığı tarih bilgisinin girildiği alandır. |
| Keşide | Çekin keşide edilip edilmediğinin seçildiği alandır. |
| Distribütör Adı | Distribütör bilgisinin girildiği alandır. |
| Müşteri | Çek tahsilatının yapıldığı müşteri kodudur. |
| Satış Temsilcisi | Müşterini rutunda olduğu satış temsilcisi bilgisidir. |
| Tahsilat ST Kod | Tahsilatı yapan satış temsilcisi bilgisidir. |
| İşlem Tipi | Tahsilat işlem tipinin girildiği alandır. |
| Döviz Tipi | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsidir. |
| Dövizli Tutar | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsinden tutarıdır. |
| Kur | Döviz takibi yapılıyor ise işlem yapılan tahsilatın o günkü kur bilgisidir. |
| Tutar | Yapılan çek tahsilatın tutarının girildiği alandır. |
| Ödeme Planı | Tanımlı ödeme planı var ise seçildiği alandır. |
| Vade Tarihi | Alınan çek tahsilatının vade günün girildiği alandır. |
| Ödeme Tarihi | Alınan çekin ödemesinin yapılacağı tarih bilgisidir. |
| Banka | Çekin ödemesinin yapılacağı banka bilgisidir. Zorunlu alandır. |
| Şube | Çekin ödemesinin yapılacağı bankanın şube bilgisidir. Zorunlu alandır. |
| Keşide Yeri | Çekin keşide yeri var ise yazıldığı alandır. |
| Hesap No. | Takas çek hesap numarasının yazıldığı alandır. |
| Çek No | Çek üzerinde yazan çek numarasıdır. Zorunlu alandır. |
| Esas Borçlu | Çekin üzerinde yazan esas borçlu bilgisidir. |
| Vergi Numarası | Müşterinin vergi numarası bilgisidir. |
| Vergi Dairesi | Müşterinin vergi dairesi bilgisidir. |
| Bakiye | Çek Tahsilat kaydı girilen müşterinin bakiye bilgisidir. |
| Basım | Çek Tahsilatı kaydının basım durum bilgisidir. |
| Giriş Tip | Çek Tahsilatı kaydının giriş tipi bilgisidir. |
| Durum | Çek Tahsilat kaydının belge durum bilgisidir. Aktif, silindi değerlerini alır. |
| Fatura Kodu | Çek kaydı için bu alan kullanılmamaktadır |
| Sipariş Kodu | Çek kaydı için bu alan kullanılmamaktadır. |
| Aktarım Durumu | Çek Tahsilat kaydının ticari paket aktarım durum bilgisidir. Aktarıldı, Aktarılmadı değerlerini alır. |
| Tahsilat İptal Nedeni | Tahsilat iptal nedeninin girilebildiği alandır. |
| Proforma Fatura Kodu | Çek kaydı için bu alan kullanılmamaktadır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Yeni | Yeni kayıt girmek için kullanılan butondur. |
| Düzenle | Mevcut kayıtlarda düzenleme amacıyla kullanılan butondur. |
| İzle | Mevcut kayıtlarda izleme yapabilmek için kullanılan butondur. |
| Kopyala | Listede seçili olan tahsilatı kopyalamak için kullanılan butondur. |
| Sil | İptali istenen kaydı silmek için kullanılan butondur. |
| Yazdır | Tanımlı dizayna göre çıktı alınmak istendiğinde kullanılan butondur. |
| Fatura Kapat | Fatura kapatma ekranını açmaktadır. |
