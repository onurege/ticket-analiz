# Senet Çıkış

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Senet İşlemleri → Senet Çıkış`

## Ekran Tanımı

Finans çek çıkış ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Kod | Senet çıkış işlemleri belge kodu bilgisidir. |
| İşlem Tarihi | İşlem tarihi bilgisidir. |
| Nereye | Senetin nereye verileceği bilgisidir. “Banka”, “Müşteri” değerleri alabilir. |
| Distribütör Kodu | Distribütör Kodu bilgisidir. |
| Banka Kodu | Banka kodu bilgisidir. |
| Banka Hesap | Banka hesap numarası bilgisidir. |
| Müşteri Kodu | Müşteri kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan değişiklikleri kaydetmeye yarar. |
| Kapat | Ekranı kapatmak için kullanılır. |
| Yenile | Detayı yenilemek için kullanılan butondur. |
| Yeni | Yeni kayıt girmek için kullanılan butondur. |
| Sil | İptali istenen kaydı silmek için kullanılan butondur. |
| Kendi Çeki | Kendine Çek senet girişi yapma ekranını açmaktadır. |
