# Kıymetli Evrak Takibi

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Tanımlamalar → Kıymetli Evrak Takibi`

## Ekran Tanımı

Kıymetli Evrak Takibi yapılıcak ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Kıymetli Evrak Kod bilgisidir. |
| Durum | Tanımlanan evraka ait durum bilgisidir. Aktif, Pasif tanımlanabilir. |
| Distribütör Kodu | Distribütör kod bilgisidir. Evrak Listeleme işlemi için kullanılan kriterdir. |
| Belge Tipi | Belge Tipi bilgisidir. “Fatura”, “Tahsilat”, “İrsaliye” girdilerini alabilir |
| Satış Temsilcisi | Satış temsilcisi bilgisidir. |
| ST Kodu | Kıymetli evrak takibi için satış temsilcisi kodu bilgisidir. |
| Satış Temsilcisi Adı | Satış temsilcisi adı bilgisidir. |
| Seri No | Seri no bilgisidir. |
| Başlangıç Seri No | Başlangıç seri numarası bilgisidir. |
| Bitiş Seri No | Bitiş seri numarası bilgisidir. |
| Son Seri No | Son seri numarası bilgisidir. |
| Durum | Evrak durum bilgisidir. “Aktif”, “Pasif” değer alabilir |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Değişiklikleri kaydet ve yeni belge tanımlama butonudur. |
| Kaydet ve Kapat | Değişiklikleri kaydet ve ekranı kapatmaya yarar. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
| Yenile | Ekranda bulunan kayıtların yenilenmesini sağlayan butondur. |
| Yeni | Yeni kayıt eklemek için kullanılır |
| Düzenle | Listeden seçilen kayıtları düzenlemek için kullanılır. |
| İzle | Listede bulunan kayıtları incelemek için kullanılır. |
| Sil | Listeden seçilen kayıdı silmek için kullanılır. |
