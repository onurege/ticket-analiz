# Personel

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Tanımlamalar → Personel`

## Ekran Tanımı

Personeli sisteme tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Personel Kodu | Personele program tarafından verilen artan matrisli numaradır. |
| Durum | Aktif/Pasif değer alabilen durum bilgisidir. |
| Ad | Personele verilen adın yazıldığı alandır. |
| Soyad | Personele verilen soyadın yazıldığı alandır. |
| Tip | ST Gruplarından girilen tanımların seçildiği ekrandır. |
| Distribütör Kodu | Distribütör kodunun seçildiği sahadır. Personelin hangi distribütöre bağlı olduğu belirlenir. |
| Kullanıcı | Yetkilendirme için kullanıcının seçildiği sahadır. |
| Sürücü Belgesi | Hangi tip sürücü belgesi olduğunun bilgisidir. |
| Bayi Şube | Bayi şube bilgisidir. |
| TC Kimlik No | Personelin TC Kimlik numarasının girildiği sahadır. |
| Cinsiyet | Personelin cinsiyetinin girildiği sahadır. |
| Medeni Hal | Medeni durum bilgisidir. |
| Askerlik Durumu | Askerlik durum bilgisidir. |
| Ünvan | Ünvan bilgisidir. |
| Departman | Çalışacağı departman bilgisidir. |
| Yönetici | Personelin yönetici bilgisidir. |
| Öğrenim Durumu | Öğrenim durumu bilgisidir. |
| Çalışma Durumu | Çalışma durumu(Part time vb.) bilgisidir. |
| Pozisyon | Pozisyon bilgisidir. |
| İşe Giriş Tarihi | İşe giriş tarihi bilgisidir. |
| İşten Çıkış Tarihi | İşten çıkış tarihi bilgisidir. |
| Doğum Tarihi | Personel doğum tarihi bilgisidir. |
| Oryantasyon Baş.Tarihi | Oryantasyon başlangıç tarihidir. |
| Oryantasyon Bit. Tarihi | Oryantasyon bitiş tarihidir. |
| Sosyal Ağ | Sosyal Ağ bilgisidir. |
| Mail Adresi | Mail adresi bilgisidir. |
| Posta Kodu | Personelin ikamet ettiği yerin posta kodunun girildiği sahadır. |
| Ülke | Personelin bağlı olduğu ülkenin girildiği sahadır. |
| İl | Personelin ikamet ettiği ilin girildiği sahadır. |
| İlçe | Personelin ikamet ettiği ilçenin girildiği sahadır. |
| Telefon | Personelin ikamet ettiği yerin telefon numarasıdır. |
| Cep Telefonu | Personele ulaşılabilecek cep telefonu numarasıdır. |
| Adres | Personelin ikametgâh adresinin girildiği sahadır. |
| Koordinat X | Personel İkametgah adresinin kordinasyon X bilgisidir. |
| Koordinat Y | Personel İkametgah adresinin kordinasyon Y bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Yeni personel tanımlamak ve kaydetmek için kullanılan butondur. |
| Kaydet ve Kapat | Yeni personel tanımlamak ve kaydedip kapatmak için kullanılan butondur. |
| Kapat | Ekranı kaydetmeden kapatmaya yarayan butondur. |
