# Kontrat Faturalarından Verilen Hizmet Oluşturma

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri 🡪 Kontrat İşlemleri 🡪 Kontrat Faturalarında Hizmet Oluşturma`

## Ekran Tanımı

Kontrat faturalarından verilen hizmet oluşturma ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Durum | İşlem durum bilgisidir. Aktif/Pasif değerlerini alır. |
| Fatura Kodu | Fatura kodu bilgisidir. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| Satış Temsilcisi | Satış temsilcisi bilgisidir. |
| İşlem Tarihi | İşlem tarihi bilgisidir. |
| Alınan Fatura Kod | Alınan fatura kodu bilgisidir. |
| Tutar | Seçilen fatura tutar bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Ekranı kaydetmek için kullanılan butondur. |
| Kapat | Ekranı Kapatma işlemi yapar. |
| Yenile | Listeyi yenilemeye yarar. |
| İzle | Listede seçilen faturayı incelemek için kullanılır. |
| Ekle | Yeni işlem eklemek için kullanılır. |
| Sil | Listeden seçili işlemi silmek için kullanılır. |
