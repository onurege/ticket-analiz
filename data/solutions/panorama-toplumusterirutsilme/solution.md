# Toplu Müşteri Rut Silme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → İşlemler → Toplu Müşteri Rut Silme`

## Ekran Tanımı

Satış ekibinin mesajlarını özelleştirebildiği ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Toplu müşteri rut silme işleminde müşterilerin hangi distribütör kodundan olacağı seçilir. |
| Müşteri Kriter | Müşteri kriterinin seçildiği alandır. |
| Müşterilerin Tüm Rutlarını Sil | Müşterilerin tüm rutlarını sil kutucuğu işaretlenirse müşterilerin tüm rutları silinir. |
| Rut | Rut seçilir. |
