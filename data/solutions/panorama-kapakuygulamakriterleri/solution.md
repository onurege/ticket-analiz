# Kapak Uygulama Kriterleri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri → Satış Kriterleri → Kapak Uygulama Kriteri`

## Ekran Tanımı

Kapak Uygulama Kriterlerinin tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Kapak Uygulama Kriterleri tanımına program tarafından verilen artışlı numaradır. |
| Durum | Tercihli Ürün Satamama Nedeninin Adının girildiği mecburi alandır |
| Distribütör Kriter Tipi | Tanımın Hangi Distribütör için geçerli olacağını belirlemek için kullanılır. Kriter tipi Kod, Grup, Ek Grup, Dinamik Grup değerleri alır. |
| Distribütör Kodu | Hangi Distribütör veya Distribütör gruplarına Kapak Uygulama Kriterinin uygulanacağı seçimin yapılacağı alandır. Distribütör Kriter Tipinde ki seçime göre yansımaktadır. |
| Başlangıç Tarihi | Kapak Uygulama Kriterleri başlangıç tarihidir. |
| Bitiş Tarihi | Kapak Uygulama Kriterleri bitiş tarihidir. |
| Ürün | Kapak Uygulama Kriterinin hangi ürün için uygulanacağının seçildiği alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Ekranda bilgileri girilmiş kullanıcıyı yarar. |
| İptal | Ekranı kaydetmeden Kapatma işlemi yapar. |
