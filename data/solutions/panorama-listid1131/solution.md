# İptal Nedenleri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → İşlemler → İptal Nedenleri`

## Ekran Tanımı

Müşteri üzerinde yapılacak işlem tipine göre iptal nedeni tanımının yapıldığı ekrandır. Öncesinde ekrana

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Sistem tarafından verilen otomatik artışlı numaradır. |
| Durum | Tanıma ait durum bilgisinin girildiği alandır. Aktif / Pasif değerlerini alabilir. |
| Distribütör Kodu | Tanımın hangi distribütör için geçerli olucağı kriterinin girildiği alandır. |
| Tip | Tanımın hangi iptal nedeni için kullanılacağı bilgisinin girildiği alandır. Müşteri İptal Nedeni / Nokta Devri İptal Nedeni / Herikisi için iptal nedeni / Pasif Nedeni / Potansiyel müşteri iptal nedeni değerlerini alabilir. |
| Uygulama Yeri | Tanımın hangi ürün için geçerli olucağı bilgisinin girildiği alandır. Panorama / Quest / Panorama ve Quest / Münferit / Hepsi / Stokbar / CallDesk değerlerini alabilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Tanımı kaydeder. |
| İptal | Tanım ekranını kapatır. |
