# Anket Uygulama Kriterleri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri → Anket ve Bulunurluk → Anket Uygulama Kriterleri`

## Ekran Tanımı

Anket tanımında kullanılması için kriter tanımının yapıldığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Distribütör Kriter Tipi | Kod,Grup,Ek Grup,Dinmaik Grup değerini alabilir. Bu alanda seçilen değere göre Distribütör kriteri girilir. |
| Distribütör Kriteri | Distribütör kriter değeri seçilir. |
| Bağlantı Tipi | Brden fazla kriter tanımı yapıldığında ; Bağlantı tipi ‘Ve’ seçilir ise bütün kriterlerin kapsadığı müşteride anket uygulanacağı anlamına gelir. Bağlantı tipi ‘Veya’ seçilir ise tanımlanan kriterlerden bir tanesine uyan müşteride anket uygulanır. |
| Bağlantı Tipi | Bu ekranda tanımlanan kriterlerin birbirleri arasındaki bağlantıyı belirler. Birden fazla kriter tanımı yapılarak tamam butonu ile kaydedildiğinde bir sonraki yapılarak kriter tanımlarının birbirleri arasındaki bağlantı bilgisidir. Ve: Girilen kriterlerin tümünün karşılanması durumunda uygulanacağı belirtilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfanın yenilenmesini sağlar. |
| Yeni | Yeni kriter tip tanımlanması için ekranı açar. |
| İzle | Seçilen satırı izle modunda açar. |
| Sil | Seçilen satırı siler |
| Grup | Girilen kriter tip satırı için detay giriş ekranını açar. |
