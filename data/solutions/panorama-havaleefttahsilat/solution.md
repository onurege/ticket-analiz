# Havale/EFT

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tahsilat İşlemleri → Havale/EFT`

## Ekran Tanımı

Havale/EFT tahsilat işleminin yapıldığı adımdır. Ekrana daha önceden girilmiş olan Havale/EFT tahsilatları gelir. Önceki girilen Havale/EFT tahsilatları üzerinde iptal, düzenleme işlemi yapılabilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Havale/EFT tahsilatına program tarafından verilen artışlı numaradır. |
| Belge No | Havale/EFT tahsila tına verilecek olan belge kodudur. |
| Makbuz No | Tahsilat makbuzu üzerindeki girilecek olan makbuz numarasıdır. |
| İşlem Tarihi | Havale/EFT tahsilatının yapıldığı tarih bilgisinin girildiği alandır. |
| Distribütör Adı | Distribütör bilgisinin girildiği alandır. |
| Müşteri | Havale/EFT tahsilatının yapıldığı müşteri kodudur. |
| Satış Temsilcisi | Müşterini rutunda olduğu satış temsilcisi bilgisidir. |
| Tahsilat ST Kod | Tahsilatı yapan satış temsilcisi bilgisidir. |
| İşlem Tipi | Tahsilat işlem tipinin girildiği alandır. |
| Döviz Tipi | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsidir. |
| Dövizli Tutar | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsinden tutarıdır. |
| Kur | Döviz takibi yapılıyor ise işlem yapılan tahsilatın o günkü kur bilgisidir. |
| Tutar | Yapılan DBS tahsilatının tutarının girildiği alandır. |
| Banka Kodu | Havale/EFT kaydı yapılan banka bilgisidir. |
| Şube | Havale/EFT kaydı yapılan bankanın şube bilgisidir. |
| Hesap Numarası | Havale/EFT kaydı yapılan bankanın hesap numarasıdır. |
| Bakiye | Havale/EFT Tahsilat kaydı girilen müşterinin bakiye bilgisidir. |
| Basım | Havale/EFT Tahsilatı kaydının basım durum bilgisidir. |
| Giriş Tip | Havale/EFT Tahsilatı kaydının giriş tipi bilgisidir. |
| Durum | Havale/EFT Tahsilat kaydının belge durum bilgisidir. Aktif, silindi değerlerini alır. |
| Fatura Kodu | Havale/EFT kaydının fatura kodu bilgisidir. Girilen faturaya karşılık oluşan takip kodu bilgisidir. |
| Sipariş Kodu | Havale/EFT kaydının sipariş kodu bilgisidir. Girilen siparişe karşılık oluşan takip kodu bilgisidir. |
| Aktarım Durumu | Havale/EFT Tahsilat kaydının ticari paket aktarım durum bilgisidir. Aktarıldı, Aktarılmadı değerlerini alır. |
| Proforma Fatura Kodu | Havale/EFT kaydının proforma fatura kodu bilgisidir. Girilen proforma faturaya karşılık oluşan takip kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Yeni | Yeni kayıt girmek için kullanılan butondur. |
| Düzenle | Mevcut kayıtlarda düzenleme amacıyla kullanılan butondur. |
| İzle | Mevcut kayıtlarda izleme yapabilmek için kullanılan butondur. |
| Kopyala | Listede seçili olan tahsilatı kopyalamak için kullanılan butondur. |
| Sil | İptali istenen kaydı silmek için kullanılan butondur. |
| Yazdır | Tanımlı dizayna göre çıktı alınmak istendiğinde kullanılan butondur. |
| Fatura Kapat | Fatura kapatma ekranını açmaktadır. |
