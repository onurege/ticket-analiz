# Tahsilat Dizaynı

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Menü Adımı : Sistem Yöneticisi 🡪 Matbu Dizayn 🡪 Tahsilat Dizaynı`

## Ekran Tanımı

Tahsilat dizaynının tanımlandığı ekrandır. Öncelikler ekrana, daha önceden kayıtlı belge varsa, bu bilgiler gelecektir. Yeni tahsilat tanımı yapılacaksa Yeni butonuna tıklanır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Müşteri bilgileri | Genelde Müşteri sabit bilgilerinin tutulduğu ekrandır. |
| Tarihler | Belge tarih bilgilerinin tutulduğu ekrandır. |
| Ürün Bilgileri | Belgede geçen ürün bilgilerinin getirildiği ekrandır. |
| Miktar/Tutar Bilgileri | Programda hesaplanan miktar ve fiyat bilgilerinin girildiği ekrandır. |
| Belge Bilgileri | Belgeye ait bilgilerin tutulduğu ekrandır. |
| Kullanıcı Tanımlı Sahalar | Programda yazmasını istediğimiz sabit değerler varsa (çizgi, açıklama vs.) girildiği ekrandır. Yukarıda yer alan Saha Ekle ya da Saha sil ile gerçekleşir. |
| Kaydet ve Kapat | Hazırlanan Matbu dizaynı kayıt eder ve ekranı kapatır. |
| Kapat | Ekranı kapatır. |
| Ayarlar | Matbu Dizaynın Ayarlarının yapıldığı ekrandır. |
| Saha Ekle | Programda yazmasını istediğimiz sabit değerler varsa (çizgi, açıklama vs.), Saha ekle butonundan eklenir. |
| Saha Sil | Kullanıcı tanımlı sahalardan eklenmiş olan Sahayı Siler. |
| Şablon Yükle | Var olan bir şablonu bilgisayardan dosya seçerek yükleme işlemi yapar. |
| Toplam Sayfa Sayısı | Sürekli form üzerinde kaç satır olduğunun girildiği alandır. |
| Yazdırılabilir Sayfa Satır Sayısı | Toplam sayfanın ne kadarının yazdırılabilir değerinin girildiği ekrandır. |
| Sayfa Kolon Sayısı | Belgenin kaç kolondan oluştuğunun girildiği ekrandır. |
| Sonuç Satır Sayısı | Sonuç bilgilerinin kaç satır kullanılacağının bilgisidir. |
| Yazıcı Tipi | Yazıcı Tipinin girildiği ekrandır. |
| Yazı Tipi ve boyutu | Yazıcı tipi Windows Printing seçili ise aktif olur ve Windows’un yazım font değerinin girildiği kısımdır. |
| Taslak olarak Yazılsın | Taslak olarak yazılması isteniyor ise işaretlenmesi gereken checkbox dır. |
| Tamamını sıkıştırılmış olarak bas | Belgenin tamamını sığdırmak maksadıyla işaretlenmesi gereken parametredir. |
| Yazıcı sayfa başına otomatik Konumlanıyor | Yazıcı sayfa başına otomatik olarak konumlanıyorsa işaretlenmesi gereken parametredir. |
| Kopya Sayısı | Kaç adet kopya istendiğinin yazıldığı alandır. |
