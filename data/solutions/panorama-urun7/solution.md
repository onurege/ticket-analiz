# Ürün

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `İş Akış → Onay İşlemleri → Ürün`

## Ekran Tanımı

Onaylanacak/Reddedilen/Onaylanan alış siparişlerinin listelendiği ekrandır. Onaylama ve red işlemi bu ekrandan yapılmaktadır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Kullanıcı Grup | Listeleme yapılacak kullanıcı grubu seçilir. |
| Onay Bekleyenler | Sadece onay bekleyenlerin listelenmesi sağlanır. |
| Onayladıklarım | Sadece onaylananların listelenmesi sağlanır. |
| Reddettiklerim | Sadece reddedilenlerin listelenmesi sağlanılır. |
| İşlem | İşlem tipi “Bekliyor” veya “Onay” seçilerek kriter belirlenmesi yapılıp “Kaydet” butonuna tıklanılarak işlem tipinin değiştirilmesi sağlanılır. |
| Ret Neden | Ret nedenini belirtildiği açıklama alanıdır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Kaydet | Yapılan değişikliklerin kaydedilmesini sağlar. |
| İzle | Tanımın izle ile açılmasını sağlar. |
| Tarihçe | Onay geçmişini gösterir. |
| Düzenle | İzin verilen yetki dahilinde belgenin düzenleme butonudur. |
| Kapat | Ekranı kaydetmeden kapatır. |
| Ret Neden | Ret Nedenin kod bilgisidir. |
| Geri Gelme Nedeni | Önceki adıma gönder adımından gelen açıklama bilgisidir. |
| Kod | Sistem tarafından takip edilen artışlı numara bilgisidir. |
| Ürün | Hangi ürün oladuğunu belirtir. |
| Ad | Ürünün tanım bilgisidir. |
| Kısa Adı | Ürünün kısa tanım bilgisidir. |
| Distribütör Kodu | Ürünün hangi distribütör için tanımlandığı bilgisidir. |
| Durum | Ürün durum bilgisidir. Aktif, Pasif değerlerini alır. |
| Grup Kodu | Satış temsilcisi Grup bilgisidir. |
| Ek Grup Kodu | Ürün Ek Grup Kodu bilgisidir. |
| Onay | Onay durum bilgisidir. Onaylanmadı, Onaylandı değerlerini alır. |
| Uygulama Yeri | Ürünün uygulama yerinin belirtildiği alandır. Panorama /Quest / Panorama ve Quest /Münferit değerlerini alabilir. |
| Son İşlem Tarihi | Belgede yapılan son değişiklik tarihidir. |
