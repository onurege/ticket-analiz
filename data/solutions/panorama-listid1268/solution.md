# Ürün Kodu Tanımlama Şablonu

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Parametreler → Ürün Kodu Tanımlama Şablonu`

## Ekran Tanımı

Ürün kodunun belirtilen kriterler ile kontrole girerek tanım yaparken bu kriterler kontrol edilerek tanımın yapılmasını sağlar.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Ürün kod tanımlama şablonuna verilen artışlı numaradır. |
| Kullanıcı Kriter Tipi | Kullanıcı kriter tipinin belirtildiği alandır. Kod / Grup değerlerini alabilir. |
| Kullanıcı Kriteri | Kriter tipi tanımına göre kulanıcının rehberden seçildiği alandır. |
| Ürün Kod Kriteri | Ürün kod kriterinin tanımlandığı alandır. |
| Kontrol Tipi | Kontrol tipinin tanımlandığı alandır. İle başlayamaz / ile başlamalı / içeremez değerlerini alabilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan tanımı kaydeder. |
| İptal | Ekranı kapatır. |
