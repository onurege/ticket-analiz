# Belge Dönüşüm Kriter

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Satış İşlemleri → Belge Dönüşüm Kriter`

## Ekran Tanımı

Belge dönüşüm kriteri ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Belge kod bilgisidir. |
| Durum | Belge durumu bilgisidir. |
| Distribütör | Kaynak belgesinin distribütör bilgisidir. |
| Müşteri | Kaynak belgesinin müşteri bilgisidir. |
| Belge Tipi | Belge tipi bilgisidir. |
| Hareket Tip | Hareket tipi bilgisidir. |
| Giriş Tip | Giriş tipi bilgisidir. |
| Sipariş Bölme Ve Yönlendirmede Kullanılsın mı? | Sipariş bölme ve yönlendirmede kullanılsın istenirse seçilir. |
| Belge Dönüşüme Giren Belgeler Birebir Aktarılsın Mı? | Belge dönüşüme giren belgeler birebir aktarılsın istenirse seçilir. |
| Distribütörün Tüm Müşterileri | Distribütörün tüm müşterileri görülsün istenirse seçilir. |
| Distribütör | Hedef belge distribütör bilgisidir. |
| Müşteri | Hedef belge müşteri bilgisidir. |
| ST | Hedef belge satış temsilcisi bilgisidir. |
| Depo | Hedef belgenin depo bilgisidir. |
| Belge Tipi | Belge tipi bilgisidir. |
| Hareket Tip | Hareket Tipi bilgisidir. |
| Banka Kodu | Banka kodu bilgisidir. |
| Banka Hesap | Banka hesap bilgisidir. |
| Stok Kontrol | Stok kontrol bilgisidir. |
| Risk Kontrol | Risk kontrol bilgisidir. |
| Hedef Belgeye Ait Belge Bulunsun Mu? | Hedef belgeye ait belge bulunsun istenirse seçilir. |
| Kaynak Belge Adresi Sevk Adresine Atansın | Kaynak belge adresi sevk adresine atansın istenirse seçilir. |
| Bulunacak Bağlı Belge | Bulunacak bağlı belge seçimi yapılabilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Listele butonuna bastığımızda kriter alanına girilen kriterlere bağlı olarak siparişler detay alanında listelenir. |
| Temizle | Temizle butonuna bastığımızda kriter alanına girilen kriterler ve bunlara bağlı olarak detay alanında listelenen siparişler temizlenir. |
| Kapat | Toplu Sipariş İptali ekranını kapatır. |
