# Tahsilat İptal Nedeni

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tahsilat İşlemleri → Tahsilat İptal Nedeni`

## Ekran Tanımı

Tahsilat iptal nedenlerinin tanımlandığı ekrandır. Öncelikle ekrana sisteme girilmiş tahsilat iptal nedenleri listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Tahsilat İptal Nedeni tanımına program tarafından verilen artışlı numaradır. |
| Distribütör Kodu | Distribütöre özel tanım yapılacaksa seçildiği rehber alanıdır. |
| Uygulama Yeri | Uygulama Yerinin seçildiği alandır. Panorama, Quest, Panorama ve Quest, Münferit, Hepsi, Stokbar, CallDesk değerlerini alır. |
