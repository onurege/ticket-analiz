# İskonto Promosyon Yetkileri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Parametreler → İskonto Promosyon Yetkileri`

## Ekran Tanımı

Satış temsilcisi parametrelerinden Yetkiler kısmında “Serbest İskonto” parametresi evet verilir ise Satış temsilcisinin hangi iskontodan ne kadar verebileceğinin belirlendiği ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | ST İskonto/Promosyon Kod bilgisidir. |
| Yetki Tip | Yetki tipi “Serbest”/”Serbest Tanım” olarak belirlenir. |
| Durum | İskonto/Promosyon’nun Aktif/Pasif durumunu belirtir |
| Başlangıç Tarihi | Başlangıç Tarihi bilgisidir. |
| Bitiş Tarihi | Bitiş Tarihi bilgisidir. |
| Distibütör Kriter Tipi | Distribütör Kriter Tipi bilgisidir. Değer olarak “Kod”, ”Grup Kodu”, ”Ek Grup Kodu” ve “Dinamik Grup” olarak değer alır. |
| Distibütör Kriteri | Distribütör Kriter bilgisidir. |
| Satış Temsilcisi Adı | Satış Temsilcisi isim bilgisidir. |
| Müşteri Kriteri Tipi | Eğer iskontolar müşteri tipine göre verilecek ise Müşteri Kriter Tipi değer almalıdır. Değer olarak “Kod”, ”Grup Kodu”, ”Ek Grup Kodu” ve “Dinamik Grup” olarak değer alır. |
| Müşteri | “Müşteri Kriteri” ne bağlı olarak değer alabilir “Müşteri Kriteri” nasıl seçildiyse “Müşteri” alanı bağlı olarak değer alırlar. |
| Ürün Kriter Tipi | Eğer iskontolar müşteri tipine göre verilecek ise Ürün Kriter Tipi değer almalıdır. Değer olarak “Kod”, ”Grup Kodu”, ”Ek Grup Kodu” ve “Dinamik Grup” olarak değer alır. |
| Ürün | “Ürün Kriteri” ne bağlı olarak değer alabilir “Ürün Kriteri” nasıl seçildiyse “Ürün” alanı bağlı olarak değer alırlar. |
| Ek Saha Kriteri | Ek Saha Kriter bilgisidir. |
| Bakiye İsk/Pro Uygulama | Bakiyeye iskonto/promosyon uygulansın veya uygulanmasın seçeneğidir. |
| Tanımlı İskonto Promosyon Uygula | Satış temsilcilerine iskonto yetkisi verildiğinde işaretlenmesi gereken önemli bir parametredir. Bu parametrenin anlamı sistemde kayıtlı ve güncel iskonto promosyonları satış temsilcileri hareket kayıtlarında uygulaması demektir. Eğer parametre seçili değil ise sisteme tanımlanmış hiç bir iskonto promosyonu uygulamayacaktır. |
| İsk1/../İsk8 | Satış temsilcisinin hareket ekranlarında hangi iskontolara değer verebileceği yüzdesel olarak belirlenir. Değer olarak numerik değer alır. |
| Promosyon Maksimum | Promosyonun maksimum değeridir. |
| Fatura Altı İskonto 1 | Satış temsilcisinin hareket ekranlarında hangi fatura altı iskontolara değer verebileceği yüzdesel olarak belirlenir. Değer olarak numerik değer alır. |
| Fatura Altı İskonto 2 | Satış temsilcisinin hareket ekranlarında hangi fatura altı iskontolara değer verebileceği yüzdesel olarak belirlenir. Değer olarak numerik değer alır. |
| Fatura Altı İskonto 3 | Satış temsilcisinin hareket ekranlarında hangi fatura altı iskontolara değer verebileceği yüzdesel olarak belirlenir. Değer olarak numerik değer alır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Rutu kaydet ve yeni rut tanımlama butonudur. |
| Kaydet ve Kapat | Rutu kaydet ve ekranı kapatmaya yarar. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
