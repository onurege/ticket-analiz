# Bağlantı Profili

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → Ticari Paket → Bağlantı Profili`

## Ekran Tanımı

Ticari paket bağlantı bilgilerinin profil bazında kaydedilerek farklı kullanıcıların aktarım yapma işleminin daha kolay hale getirilmesidir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Sistem tarafından verilen otomatik artışlı numaradır. |
| Paket Tip | Kullanılan ticari paket tipi seçilir. Seçilen tipe göre formdaki ilgili alanlar aktif , pasif olur. |
| Dist | Aktarım yapılacak distribütör bilgisi seçilir. |
| Kullanıcı Kodu | Aktarım yapılacak kullanıcı bilgisi seçilir. Dist kullanıcısı ile login olunmuşsa otomatik olarak ilgili alana set edilir. |
| Panorama Web Servis URL | Giriş yapılan panorama adresi otomatik olarak set edilir. |
| Ldap Kullanılsın | Login tipi olarak LDAP kullanıyor ise seçilir. |
| Ürün / Müşteri Aktarımında Tarihi Dikkate Al | Ürün ve müşteri aktarımında son aktarım tarihinin dikkate alınması isteniyorsa seçilir. |
| Fatura Kesilmiş Müşterileri Dikkate Al | Writer Müşteri aktarımında sadece fatura kesilmiş müşterilerin dikkate alınması isteniyorsa işretlenir. |
| Yalnızca Fatura Kesilmiş Müşteriler Aktarılsın | Reader Müşteri aktarımında sadece fatura kesilmiş müşterilerin dikkate alınması isteniyorsa işretlenir. |
| Firma | Ticari paket firma kodu bilgisi girilir . |
| İşyeri | Ticari paket işyeri bilgisi girilir. |
| Dönem | Ticari paket dönem bilgisi girilir. |
| Bölüm Kodu | Ticari paket bölüm kodu bilgisi girilir. |
| Şube | Ticari paket şube bilgisi girilir. |
| İşletme | Ticari Paket işletme bilgisi girilir. |
| Proje Kodu | Ticari paket proje kodu bilgisi girilir. |
| ST | Aktarım yapılacak st kriteri girilir. Zorunlu alan değildir boş bırakılırsa bütün satış temsilcilerine ait hareketler aktarılır. |
| Depo Kodu | Aktarım yapılacak depo kriteri girilir. Zorunlu alan değildir boş bırakılırsa bütün depolara ait hareketler aktarılır. |
| Kullanıcı | Ticari paket kullanıcı adı bilgisi girilir. |
| Şifre | Ticari paket şifre bilgisi girilir. |
| Versiyon | Ticari paket versiyon bilgisi girilir. |
| Onadalık Ayraç | Fatura aktarımında kullanılacak ondalık ayraç değeri girilir. |
| Sunucu | Ticari paket sistem sunucu bilgisi girilir. |
| DB Adı | Ticari paket database adı bilgisi girilir. |
| DB Kullanıcı Adı | Ticari paket database kullanıcı adı bilgisi girilir. |
| DB Şifre | Ticari paket database şifre bilgisi girilir. |
| Uluslararası Dil Desteği | Txt aktarımda kullanılan dis desteği seçimi yapılır. |
| Ürün Kriter Tipi | Ürün aktarımında kullanılacak kriter tipi seçilir. Zorunlu değildir boş bırakılması halinde bütün ürünler aktarılacaktır. |
| Ürün Kriteri | Ürün kriter tipi seçimi yapıldı ise ürün kriteri seçimi yapılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Form kaydedilir ve yeni bir form açılır. |
| Kaydet ve Kapat | Form kaydedilir ve kapanır. |
| Kapat | Form kaydedilmeden kapanır. |
