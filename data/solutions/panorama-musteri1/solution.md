# Müşteri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → Ticari Paket → Eşleştirme → Müşteri`

## Ekran Tanımı

Panorama ile ticari pakette kullanılan müşteri kodlarının farklı olduğu durumlarda, bu menü üzerinden ticari pakette kullanılan müşteri kodları girilerek aktarım sağlanabilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Adı | İşlem yapılacak Distribütör bilgisi seçilir. |
| Tp Kod | Listelenen müşterilerin Ticari Paket programında kullanılan kod bilgisi girilir. Giriş yapıldığı anda kaydetme işlemi gerçekleşecektir. |
| Kod | Listelenen müşterilerin Panoramada kullanılan kod bilgisi gösterilir. |
| Ünvan | Listelenen müşterilerin Panoramada kullanılan ünvan bilgisi gösterilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kapat | Form kapatılır. |
| Yenile | Sayfanın yenilenmesini sağlar. |
| Düzenle | Seçilen satırda Tp Kod alanını aktif hale getirir ve değer girişi yapılmasını sağlar. |
| Yön Tuşları | Listelenen müşteri kayıtlarında ileri-geri sayfa geçişi yapılmasını sağlar. |
