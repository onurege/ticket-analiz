# Teslimat Şekli

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri → Tanımlamalar → İhracat Tanımları → Teslimat Şekli`

## Ekran Tanımı

Teslimat şeklinin tanımlandığı alandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Ad | Teslimat Şekli isim bilgisidir. |
| Teslimat Fiyatı | Teslimat Fiyatı bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemin kaydedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
