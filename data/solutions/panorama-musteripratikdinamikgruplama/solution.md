# Pratik Dinamik Gruplama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → Tanımlamalar → Pratik Dinamik Gruplama`

## Ekran Tanımı

Müşteri Pratik Dinamik Gruplarının tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Dinamik Grup | Dinamik grup tanımının seçildği alandır. |
| Kriter Tipi | Müşteri kriter tipinin belirlendiği alandır. Kod / Grup / Ek Grup değerlerini alabilir. |
| Uygulama Yeri | Müşteri grubunun kullanılacağı modül bilgisinin tutulduğu alandır. Panorama / Quest / Panorama ve Quest / Münferit / Hepsi / Stokbar değerlerini alabilir. |
| Grup | Müşteri Gruplarına göre filtrelenmesini sağlar. |
| Ek Grup | Müşteri Ek Gruplarına göre filtrelenmesini sağlar. |
| Dist | Müşteri Distlerine göre filtrelenmesini sağlar. |
| Bölge | Müşteri Bölgelerine göre filtrelenmesini sağlar. |
| İl | Müşteri İllerine göre filtrelenmesini sağlar. |
| İlçe | Müşteri İlçelerine göre filtrelenmesini sağlar. |
| Toplu Ekle | Müşteri Kodu veya Müşteri Takip Kodu kutucuklarından uygun olanı seçilerek toplu ekleme yapılacak ürünlerin girildiği alandır. |
| Eklenemeyen | Toplu ekleme alanına girilen değerlerden işlemi başlat a bastıktan sonra eklenemeyen değerlerin gösterildiği alandır. |
| Kriter | Toplu seçim kriter seçim alanıdır. |
| Yenile | Detaydaki listeyi yeniler. |
| İzle | Detaydaki kaydın görüntülenmesini sağlar. |
| Sil | Detaydaki kaydın silinmesini sağlar. |
| Detayları Temizle | Detaydaki yapılan tanım yada tanımların listeden silinmesi için kullanılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İşlemi Başlat | Bütün alanlar doldurulduktan sonra belirlenen kriterlere göre pratik dinamik grup tanımını oluşturur. |
| Listele | Belirtilen kriterlere göre detayda kayıtların gösterilmesi için kullanılır. |
| Temizle | Belirtilen kriterlerin ekrandan temizlenmesi için kullanılır. |
| Kapat | Ekranı kapatır. |
