# Nakit

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tahsilat İşlemleri → Nakit`

## Ekran Tanımı

Nakit tahsilat işleminin yapıldığı adımdır. Ekrana daha önceden girilmiş olan tahsilatlar gelir. Önceki girilen nakit tahsilatlar üzerinde iptal, düzenleme işlemi yapılabilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Nakit tahsilata program tarafından verilen artışlı numaradır. |
| Belge No | Nakit tahsila ta verilecek olan belge kodudur. |
| Makbuz No | Tahsilat makbuzu üzerindeki girilecek olan makbuz numarasıdır. |
| İşlem Tarihi | Nakit tahsilatın yapıldığı tarih bilgisinin girildiği alandır. |
| Distribütör Adı | Distribütör bilgisinin girildiği alandır. |
| Müşteri | Nakit tahsilatın yapıldığı müşteri kodudur. |
| Satış Temsilcisi | Müşterini rutunda olduğu satış temsilcisi bilgisidir. |
| Tahsilat ST Kod | Tahsilatı yapan satış temsilcisi bilgisidir. |
| İşlem Tipi | Tahsilat işlem tipinin girildiği alandır. |
| Döviz Tipi | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsidir. |
| Dövizli Tutar | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsinden tutarıdır. |
| Kur | Döviz takibi yapılıyor ise işlem yapılan tahsilatın o günkü kur bilgisidir. |
| Tutar | Yapılan nakit tahsilat tutarının girildiği alandır. |
| Bakiye | Nakit Tahsilat kaydı girilen müşterinin bakiye bilgisidir. |
| Basım | Nakit Tahsilat kaydının basım durum bilgisidir. |
| Giriş Tip | Nakit Tahsilat kaydının giriş tipi bilgisidir. |
| Durum | Nakit Tahsilat kaydının belge durum bilgisidir. Aktif, silindi değerlerini alır. |
| Fatura Kodu | Nakit Tahsilat kaydının fatura kodu bilgisidir. Girilen faturaya karşılık oluşan fatura takip kodu bilgisidir. |
| Sipariş Kodu | Nakit Tahsilat kaydının sipariş kodu bilgisidir. Girilen siparişe karşılık oluşan sipariş takip kodu bilgisidir. |
| Aktarım Durumu | Nakit Tahsilat kaydının ticari paket aktarım durum bilgisidir. Aktarıldı, Aktarılmadı değerlerini alır. |
| Tahsilat İptal Nedeni | Tahsilat iptal nedeninin girilebildiği alandır. |
| Proforma Fatura Kodu | Nakit Tahsilat kaydının proforma fatura kodu bilgisidir. Girilen proforma faturaya karşılık oluşan proforma fatura takip kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Yeni | Yeni kayıt girmek için kullanılan butondur. |
| Düzenle | Mevcut kayıtlarda düzenleme amacıyla kullanılan butondur. |
| İzle | Mevcut kayıtlarda izleme yapabilmek için kullanılan butondur. |
| Kopyala | Listede seçili olan tahsilatı kopyalamak için kullanılan butondur. |
| Sil | İptali istenen kaydı silmek için kullanılan butondur. |
| Yazdır | Tanımlı dizayna göre çıktı alınmak istendiğinde kullanılan butondur. |
| Fatura Kapat | Fatura kapatma ekranını açmaktadır. |
