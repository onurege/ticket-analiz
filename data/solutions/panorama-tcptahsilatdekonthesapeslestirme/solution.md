# Tahsilat İşlem Tipi - Dekont Hesap Eşleştirme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → Ticari Paket → Eşleştirme → Tahsilat İşlem Tipi - Dekont Hesap Eşleştirme`

## Ekran Tanımı

Kredi kartı tahsilatları için ticari paket programında kullanılan dekont bilgisi girilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Adı | İşlem yapılacak distribütör seçimi yapılır. |
| TP HESAP KOD | Ticari paket kredi kartı hesap kod değeri girilir. |
| TP SERİ NO / TP ÖDEME PLANI KODU | Ticari paket kredi kartı ödeme planı kodu girilir. |
| KOD | Panorama kredi kartı hareket tipi kodu gösterilir. |
| AÇIKLAMA | Panorama kredi kartı hareket tipi açıklaması gösterilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kapat | Form kapatılır. |
| Yenile | Sayfanın yenilenmesini sağlar. |
| Düzenle | Seçilen alanda değer girişi yapılmasını sağlar. |
| Yön Tuşları | Listelenen kredi kartı hareket tipi kayıtlarında ileri-geri sayfa geçişi yapılmasını sağlar. |
