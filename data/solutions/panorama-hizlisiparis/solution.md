# Hızlı Sipariş

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Satış İşlemleri → Hızlı Sipariş`

## Ekran Tanımı

Hızlı sipariş ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Belgenin hangi Distribütör için kesileceğinin tanımlandığı alandır. |
| Hareket Tip | Hareket Tipi bilgisidir. |
| Depo Kodu | Depo kodu bilgisidir. |
| İşlem Tarihi | İşlem tarihi bilgisidir. |
| Sevk Tarihi | Sipariş sevk tarihi bilgisidir. |
| İsk./Pro. | İskonto/Promosyon bilgisidir. |
| Sıra | Belge detayına eklenen ürünün hangi sırada eklendiğini gösterir. |
| Referans | Listelenecek ürünlerin referans numarasının belirtildiği alandır. |
| Müşteri Kodu | Müşteri kodu bilgisidir. |
| Müşteri Ünvanı | Müşteri ünvanı bilgisidir. |
| İlgili Kişi | İlgili kişi bilgisidir. |
| Müşteri Adres | Müşteri adres bilgisidir. |
| ST | Satış temsilcisi bilgisidir. |
| ST/SE Adı | Satış temsilcisi/Satış elemanı isim bilgisidir. |
| Ürün | Seçilen Ürünün ürün kodu değeridir. |
| Ürün Adı | Seçilen ürünün isim bilgisidir. |
| Miktar | Ürünün miktar değerini gösterir. |
| Birim | Ürünün birimini belirtir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İşlemi Başlat | Tüm değişiklikleri kaydedip işlem başlatmaya yarar. |
| Kapat | iskonto Promosyon Seçim ekranını kapatır. |
