# Satış Temsilcisi

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → Ticari Paket → Eşleştirme → Satış Temsilcisi`

## Ekran Tanımı

Panorama ile ticari pakette kullanılan satış temsilcisi kodlarının farklı olduğu durumlarda, bu menü üzerinden ticari pakette kullanılan satış temsilcisi kodları girilerek aktarım sağlanabilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Adı | İşlem yapılacak Distribütör bilgisi seçilir. |
| Tp Kod | Listelenen satış temsilcisi kayıtlarının Ticari Paket programında kullanılan kod bilgisi girilir. Giriş yapıldığı anda kaydetme işlemi gerçekleşecektir. |
| Kod | Listelenen satış temsilcisi kayıtlarının Panoramada kullanılan kod bilgisi gösterilir. |
| Ad | Listelenen satış temsilcisi kayıtlarının Panoramada kullanılan ad bilgisi gösterilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kapat | Form kapatılır. |
| Yenile | Sayfanın yenilenmesini sağlar. |
| Düzenle | Seçilen satırda Tp Kod alanını aktif hale getirir ve değer girişi yapılmasını sağlar. |
| Yön Tuşları | Listelenen satış temsilcisi kayıtlarında ileri-geri sayfa geçişi yapılmasını sağlar. |
