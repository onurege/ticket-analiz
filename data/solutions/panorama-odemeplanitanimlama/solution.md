# Ödeme Planı Tanımlama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri → Fiyat İşlemleri → Ödeme Planı Tanımlama`

## Ekran Tanımı

El terminallerinde satış yapılamadığı durumlarda neden satış yapılamadığını göstermek için girilecek olan nedenin tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Ödemi planı kod bilgisidir. |
| Distribütör Kodu | İşlem yapılacak Distribütörün seçildiği rehber alandır. |
| Vade Günü | Vade Günü bilgisidir. |
| Durum | Durum bilgisidir. “Aktif”, “Pasif” değerler alabilir. |
| Haftasonu Dikkate Alınsın | Hafta sonu alınıp alınmayacağı bilgisidir. “Evet”, “Hayır” değerleri alabilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Ekranda bilgileri girilmiş kullanıcıyı yarar. |
| İptal | Ekranı kaydetmeden Kapatma işlemi yapar. |
