# Nakit

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Ödeme İşlemleri → Nakit`

## Ekran Tanımı

Nakit ödeme işleminin yapıldığı adımdır. Öncelikle ekrana, sisteme girilmiş olan Nakit Ödeme listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Kod | Nakit ödemede program tarafından verilen artışlı numaradır. |
| Belge No | Nakit ödeme verilecek olan belge kodudur. |
| Makbuz No | Ödeme makbuzu üzerindeki girilecek olan makbuz numarasıdır. |
| İşlem Tarihi | Nakit ödeme yapıldığı tarih bilgisinin girildiği alandır. |
| Distribütör Adı | Distribütör bilgisinin girildiği alandır. |
| Müşteri | Nakit ödemenin yapıldığı müşteri kodudur. |
| Satış Temsilcisi | Müşterini rutunda olduğu satış temsilcisi bilgisidir. |
| Tahsilat ST Kod | Ödemeyi yapan satış temsilcisi bilgisidir. |
| İşlem Tipi | Ödeme işlem tipinin girildiği alandır. |
| Döviz Tipi | Döviz takibi yapılıyor ise işlem yapılan ödemenin döviz cinsidir. |
| Dövizli Tutar | Döviz takibi yapılıyor ise işlem yapılan ödemenin döviz cinsinden tutarıdır. |
| Kur | Döviz takibi yapılıyor ise işlem yapılan ödemenin o günkü kur bilgisidir. |
| Tutar | Yapılan nakit ödeme tutarının girildiği alandır. |
| Bakiye | Nakit ödeme kaydı girilen müşterinin bakiye bilgisidir. |
| Basım | Nakit ödeme kaydının basım durum bilgisidir. |
| Giriş Tip | Nakit ödeme kaydının giriş tipi bilgisidir. |
| Durum | Nakit ödeme kaydının belge durum bilgisidir. Aktif, silindi değerlerini alır. |
| Fatura Kodu | Nakit ödeme kaydının fatura kodu bilgisidir. Girilen faturaya karşılık oluşan fatura takip kodu bilgisidir. |
| Sipariş Kodu | Nakit ödeme kaydının sipariş kodu bilgisidir. Girilen siparişe karşılık oluşan sipariş takip kodu bilgisidir. |
| Aktarım Durumu | Nakit ödeme kaydının ticari paket aktarım durum bilgisidir. Aktarıldı, Aktarılmadı değerlerini alır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Yeni | Yeni kayıt girmek için kullanılan butondur. |
| Düzenle | Mevcut kayıtlarda düzenleme amacıyla kullanılan butondur. |
| İzle | Mevcut kayıtlarda izleme yapabilmek için kullanılan butondur. |
| Kopyala | Listede seçili olan tahsilatı kopyalamak için kullanılan butondur. |
| Sil | İptali istenen kaydı silmek için kullanılan butondur. |
| Yazdır | Tanımlı dizayna göre çıktı alınmak istendiğinde kullanılan butondur. |
