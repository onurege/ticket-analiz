# Prim Faturalarının SAP'ye Aktarımı

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → İşlemler → Prim Faturalarının SAP'ye Aktarımı`

## Ekran Tanımı

Prim faturaların merkez entegrasyon ile SAP’ye aktarım arayüzüdür.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Aktarım yapılacak distribütör seçimi yapılır. |
| İşlem Tipi | Aktarım yapılacak işlem tipi seçimi yapılır. |
| Yıl | Aktarım yapılacak yıl seçimi yapılır. |
| Ay | Aktarım yapılacak ay değeri girilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Aktar | Seçilen kriterlere göre kayıtların aktarılmasını sağlar. |
| Kapat | Formun kapatılmasını sağlar. |
