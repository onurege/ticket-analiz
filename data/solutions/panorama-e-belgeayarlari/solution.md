# E-Belge Ayarları

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Sistem Yönetimi→ Genel Parametreler → E-Arşiv Parametreleri`

## Ekran Tanımı

E-Arşiv entegratör firma servis bağlantı bilgilerinin girildiği ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Sistemin otomatik olarak attığı koddur. |
| Distribütör Kodu | Entegratörü kullanak distribütör seçildiği alandır. |
| Entegratör | Entegratör firmanın seçildiği alandır. |
| Kullanıcı Adı | Entegratör firmanın verdiği kullanıcı adıdır. |
| Şifre | Entegratör firmanın verdiği kullanıcı şifresidir. |
| Web Servis Adresi | Entegratör firmadan gelen webservice adresi girilir. |
| XSLT uuid | Fatura görsel şablonu, entegratör firmadan alınır. |
| Mükellef Olma Tarihi | Mükellef olunan tarih bilgisi girilir. |
| Kurum Kodu | Seçilen müşteri üzerinde sevk adresi var ise sevk adresinin seçildiği alandır. |
| E-Arşiv | E-arşiv kullanılacak ise checkbox işaretlenir. |
| E-Fatura | E-fatura kullanılacak ise checkbox işaretlenir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Yapılan tanımları kaydedip, boş bir ekran açar. |
| Kaydet ve Kapat | Yapılan işlemleri kaydedip, ekranı kapatır. |
| Kapat | Açılan ekranı kapatır. |
