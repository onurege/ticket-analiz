# İrsaliye İş Bildirimi

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Alış İşlemleri → Irsaliyeden Fatura Oluşturma`

## Ekran Tanımı

Panorama programında dış ortamdan entegrasyon ile veya belge dönüşüm ekranlarında İş Bildirimi seçeneği işaretlendiğinde oluşan belgelerin listelendiği bölümdür.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | İrsaliye belge kod bilgisidir. |
| İşlem Tarihi | Belge işlem tarihi bilgisidir. |
| Distribütör Adı | Distribütör Adı bilgisidir. |
| Müşteri Ünvanı | Müşteri ünvanı bilgisidir. |
| Tür | Belge tür bilgisidir. |
| Matbu No | Matbu No bilgisidir. |
| Referans | Belge referans bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Teslim Zamanı Girişi | Teslim zamanı giriş bilgisidir. |
| İzle | Listede seçilen irsaliyeler arasında gözlemleme yapmak için kullanılır |
| Kaydet | Yapılan değişiklikleri kaydetmek için kullanılır. |
| Kargo/Nakliye Bilgisi Sorgula | Kargo/Nakliye bilgisi sorgulamak için kullanılır. |
| Kapat | Belge ekranını kapatmak için kullanılır. |
| Yenile | Belge detay listesini yenilemek için kullanılır. |
| Şerhli Kabul | Şerhli belge kabul işlemi için kullanılır. |
