# Kontrat Tanımlama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ve Pazarlama Kriterleri → Kontrat İşlemleri → Kontrat Tanımlama`

## Ekran Tanımı

Kontrat tanımlandığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Kod bilgisidir. |
| Durum | Kontrat durum bilgisidir. “Aktif”, “Pasif”, “İptal” değerleri almaktadır. |
| Distribütör Kodu | Müşterinin hangi distribütör olduğunun bilgisidir. |
| Onay Durumu | Onay durumu bilgisidir. |
| Müşteri Kodu | Müşteri kodu bilgisidir. |
| Ödeme Durumu | Ödeme durumu bilgisidir. “Ödendi”, “Ödenmedi” değerleri almaktadır. |
| Satış Temsilcisi | Satış temsilcisi bilgisidir. |
| Kotalı Sözleşme | Kotalı sözleşme seçeneğidir. |
| Başlangıç Tarihi | Kontrat başlangıç tarihi bilgisidir. |
| Bitiş Tarihi | Kontrat bitiş tarihi bilgisidir. |
| Ödeme Türü | Ödeme Türü bilgisidir. “Banka Havalesi”, “Yükleme” değerleri alabilir. |
| Tutar | Tutar bilgisidir. |
| Sözleşme Sınıfı | Sözleşme Sınıfı bilgisidir. |
| Bakım Masrafı | Bakım masrafı bilgisidir. |
| ROI Değeri | ROI değeri bilgisidir. |
| Ünite Bedeli | Ünite değeri bilgisidir. |
| Tutar Öneri | Tutar Öneri bilgisidir. |
| Sökülecek Stant | Sökülecek Stant bilgisidir. |
| Rakip Stant | Rakip Stant bilgisidir. |
| Sözleşme Notu | Sözleşme notu bilgisidir. |
| Revize/Red Nedeni | Revize/Red nedeni bilgisidir. |
| Revize/Reddeden Kullanıcı | Revize/Reddeden kullanıcı bilgisidir. |
| Revize/Reddeden Grup | Revize/Red grup bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Kapat | Tanımı kaydeder ve ekranı kapatır. |
| Hizmet Faturası Oluşturma | Hizmet Faturasının oluşturulduğu alandır. |
| Stant Bilgisi | Stant bilgileri oluşturmak için kullanılır. |
| Kota | Kota ekranını açmaktadır. |
| Ödeme | Ödeme işlemleri için kullanılır. |
| ROI Hesapla | Hesaplama işlemleri için kullanılır. |
| Kapat | Ekranı kapatmak için kullanılır. |
