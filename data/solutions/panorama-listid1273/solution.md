# Nakliye Gün Limit Kontrol

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım ve Stok İşlemleri → Dağıtım İşlemleri → Nakliye Gün Limit Konrtol`

## Ekran Tanımı

Nakliye Gün Limit Konrtol ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kod | Takip kodu bilgisidir. |
| Seçimli Özel Kod | Seçimli özel kod bilgisidir. |
| İl Kod | İl kodu bilgisidir. |
| Min. Gün Sayısı | Minimum gün sayısı bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemin kaydedilmesini sağlar. |
| Kapat | Ekranı kapatır. |
