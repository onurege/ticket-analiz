# Rut Tanım

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Rut Ve İş Emri Planlama → Rut Tanım`

## Ekran Tanımı

Rut tanımlaya yarar.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Rut Tanım için program tarafından verilen artan matrisli numaradır. |
| Distribütör Kodu | Distribütör kodunun seçildiği sahadır. Rut tanımının hangi distribütöre bağlı olduğu belirlenir. |
| Kod | Önceden tanımlanmış müşteri listesinden seçilen müşterilerin kod bilgilerinin otomatik olarak geldiği ekrandır. |
| Toplu Ekle/Çıkar | Rut tanımına eklenmek istenilen müşterilerin kodlarının toplu olarak eklenmesinin sağlandığı ekrandır. |
| Ziyaret Başlangıç | Tanımlanmak istenilen rut güzergâhı için ziyaret başlangıcı belirlenir. |
| Müşteri Kodu | Müşteri kodu bilgisine göre işlem yapılması için kullanılan seçenektir. |
| Müşteri Takip Kodu | Müşteri takip kodu bilgisine göre işlem yapılması için kullanılan seçenektir. |
| Müşteri Merkez Kod | Müşteri merkez kodu bilgisine göre işlem yapılması için kullanılan seçenektir. |
| Tip | Rut Tanım için tanımlanacak ürün tipi seçilir. |
| Onay | Onay durumu bilgisi Onaylandı veya Onaylanmadı olarak belirtilir. |
| Eklenemeyen | Toplu ekleme ekranından eklenmek istenilen müşteri kodlarından eklenemeyenlerin listelendiği sahadır. |
| Ziyaret Artış | Tanımlanmak istenilen rut güzergâhı için ziyaretin artış sayısı belirlenir. |
| Harita Sıralama Kriteri | Harita sıralama kriteri, Sıra veya Ziyaret sırasına göre belirtilir. |
| Toplam KM | Toplam km bilgisi girilen alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Yeni | Rutu kaydet ve yeni rut tanımlama butonudur. |
| Kaydet ve Kapat | Rutu kaydet ve ekranı kapatmaya yarar. |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
| Düzenle | Ziyaret Sırasını ve Giriş/Çıkış Saatini düzenlemeye yarar. |
| Ekle | Yeni ziyaret alanı eklemeye yarar. |
| Sil | Ziyaret alanı silmeye yarar. |
| Yukarı | Seçili lokasyona öncelik vermeye yarar. |
| Aşağı | Seçili lokasyonun önceliğini ayarlamaya yarar. |
