# Müşteri Ek Saha

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Müşteriler → Parametreler → Müşteri Ek Saha`

## Ekran Tanımı

Müşteri tanımı yapılırken ek bir açıklama girilmesi için daha önceden hazırlanan saha ekleme ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Ek saha Takip Kodu bilgisidir. |
| Distribütör Kriter Tipi | Distribütörlerin seçileceği kriterlerin girildiği alandır değer olarak (Kod, Grup, Ek Grup, Dinamik Grup) değerlerini almaktadır. |
| Distribütör Kriteri | Distribütör Kriter Tipi ne göre değişkenlik gösteren bir alandır. Örneğin Distribütör Kriter Tipi “Kod” seçilirse sisteme kayıtlı tüm Distribütörler gelir ve tek bir Distribütör seçilebilir fakat Distribütör Kriter Tipi “Grup” seçilir ise Distribütör grupları seçilir. |
| Zorunlu | Ek sahanın zorunlu olması istenirse seçilir. |
| Ek Saha Zorunlu Değil | Ek sahanın zorunlu olmaması istenirse seçilir. |
| Ek Saha Özelliği | Ek sahaya girilecek olan bilginin çoktan seçmeli mi, metin mi, sayısal mı ya da tarih mi olacağı seçilir. |
| Min. Uzunluk | Ek sahaya girilecek metinin mininum uzunluğunu belirtir. |
| Maks. Uzunluk | Ek sahaya girilecek metinin maksimum uzunluğunu belirtir. |
| Müşteri Kriter Tipi | Ek sahanın çıkacağı müşteri kriteridir. |
| Müşteri Kriter | Ek sahanın çıkacağı müşteri bilgisidir. |
| Müşteri Kriter Tipi 2 | Ek sahanın çıkacağı müşteri kriteridir. |
| Müşteri Kriteri 2 | Ek sahanın çıkacağı müşteri bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Seçenek | Ek saha seçenekleri oluşturmak ve düzenlemek için kullanılır. |
| Kaydet ve Kapat | Değişiklikleri kaydet ve ekranı kapatmaya yarar. |
| Kapat | Ekranı kapatmak için kullanılır. |
