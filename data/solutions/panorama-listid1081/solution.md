# St Eşleştirme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi→ El Terminal İşlemleri → St Eşleştirme`

## Ekran Tanımı

El terminallerinin Panorama üzerinden bilgi alis-verisi yapabilmesi için yapilan El Terminal tanimlarinin temsilci ile eslestirildigi ekrandir. Bir el terminali tanimi hangi temsilci ile eslestirildi ise o el terminalinde yapilan bilgi alis-verisi bu temsilci üzerinden yapilir. Öncelikle ekrana, sistemde tanimli olan ST Eslestirme listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafindan verilen artan numaradir. |
| Seri No | El terminali seri no bilgisinin girildigi alandir. |
| Kriter | Satis Temsilcisi ya da Saha Elemani seçimi yapilir. Mobil satis dagitim uygulamasi için Satis Temsilcisi seçilir. |
| ST / SE | El terminalini kullanacak satis temsilcisinin girilecegi alandir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Yapılan işlemlerin kayıt edilmesini sağlar. |
| İptal | Yapılan işlemlerin kayıt edilmeden, ekranın kapanmasını sağlar. |
