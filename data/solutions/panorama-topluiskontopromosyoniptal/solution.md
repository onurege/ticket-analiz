# Toplu İskonto/Promosyon İptali

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ve Pazarlama Kriterleri → İskonto Promosyon → Toplu İskonto/Promosyon İptali`

## Ekran Tanımı

Toplu iskonto/promosyon iptali ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kriter Tipi | Distribütör kirter tipi bilgisidir. |
| Distribütör Kriteri | Distribütör kriteridir. |
| Müşteri Kriter Tipi | Müşteri kriter tipi bilgisidir. |
| Müşteri Kriter | Müşteri kriter bilgisidir. |
| Tarih kriteri kayıt tarihi için kullanılsın | Tarih kriteri kayıt tarihi için kullanılsın istenirse seçilebilir. |
| Başlangıç Tarihi | Başlangıç tarihi bilgisidir. |
| Bitiş Tarihi | Bitiş tarihi bilgisidir. |
| Kullanılmamış iskontoları promosyonları listele | Kullanılmamış iskontoları ve promosyonları listelenmesini istenirse seçilebilir. |
| İskonto/Promosyon | İskonto/Promosyon bilgisidir. |
| İptal işleminde müşteri kodları kullanılsın | İptal işleminde müşteri kodları kullanılsın istenirse seçilir. |
| İptal işleminde iskonto kodları kullanılsın | İptal işleminde iskonto kodları kullanılsın istenirse seçilir. |
| Müşteri Kodu | Müşteri kodu bilgisidir. |
| Müşteri Takip Kodu | Müşteri Takip Kodu bilgisidir. |
| İskonto Kodu | İskonto kodu bilgisidir. |
| Müşteri Merkez Kodu | Müşteri merkez kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| İşlemi Başlat | İşlemi başlatmak için kullanılır. |
| Kapat | Ekranı kaydetmeden kapatır. |
