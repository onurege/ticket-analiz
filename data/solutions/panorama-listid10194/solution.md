# Online İsk/Pro Güncelleme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ve Pazarlama Kriterleri → İskonto Promosyon →Online İsk/Pro Güncelleme`

## Ekran Tanımı

Mobil platform üzerinden online güncellenecek iskonto tanımlarının girildiği ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Distribütör Kodu | Distribütör seçimi yapılan rehberdir. |
| İsk/Pro Kod | Online güncelleme tanımı yapılacak iskonto kaydının seçildiği rehberdir. |
| İşlem Tarihi | Güncelleme tanımı tarihi girildiği alandır. |
| Durum | Durum alanıdır. Pasif olan kayıtlar mobil’de güncellenmez. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet | Tanım kaydedilir form kapanır. |
| İptal | Tanım kaydedilmez form kapanır. |
