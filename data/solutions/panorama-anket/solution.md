# Anket

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriterleri → Anket ve Bulunurluk → Anket`

## Ekran Tanımı

Sahada uygulanmak üzere Anket tanımlarının yapıldığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Program tarafından verilen artışlı numaradır. |
| Durum | Durum bilgisidir. Aktif olan kayıtlar sahada uygulanır. |
| Kriter Kodu | Anket Uygulama Kriterinde tanımlanan kriter listesinden seçim yapılır. |
| Başlangıç Tarihi | Anket tanımının uygulanacağı başlangıç tarihi seçilir. |
| Bitiş Tarihi | Anket tanımının bitiş tarihi seçilir. |
| Uygulama Tipi | Anketin el terminalinde tüm müşteriler için hangi sıklıkla sorulacağı detayının belirlendiği alandır. Sürekli, birkez, tekrarlı ve cevaba göre seçenekleri mevcuttur. • Sürekli: Tanımlanan anket, tüm müşteriler için her ziyarette sürekli sorulacak demektir. |
| Soru | Anket soru bilgisi girilir. |
| Soru Tipi | Anket soru tipi seçilir. Çoktan Seçmeli , Metin , Sayısal , Ek Saha , Barkod Okuma seçeneklerini alabilir |
| Maksimum Cevap Sayısı | Anket sorusu için maksimum cevap sayısı değeri girilir. |
| Cevap | Soru olarak tanımlanan ankete verilebilecek cevap yazılır. |
| Katsayı | Verilen cevaba göre belirlenen değerdir. (Genellikle kullanılmayıp “1” olarak yazılır.) |
| Cevap Tip | Ankete verilen cevabın “Geçerli” ya da “Geçersiz” olduğu seçilir. Eğer verilen cevap “Geçersiz” ise cevap “Geçerli” olana kadar ankete devam edilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfanın yenilenmesini sağlar. |
| Yeni | Yeni soru tanımlama ekranını açar. |
| Düzenle | Kaydedilen soru kaydını düzenle modunda açar. |
| İzle | Kaydedilen soru kaydını izle modunda açar. |
| Sil | Kaydedilen soru kaydını siler. |
| Anket Cevapları | Kaydedilen soru kaydı için Cevap giriş ekranını açar. |
