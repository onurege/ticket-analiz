# KA Müşterilerin SAP'den Alınması

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → İşlemler → KA Müşterilerinin SAP'tan Alınması`

## Ekran Tanımı

Merkez entegrasyon ile SAP’den Key Account müşterilerinin aktarım menüsü arayüzüdür.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Başlangıç Tarihi | Aktarım yapılacak başlangıç tarihi seçimi yapılır. |
| Bitiş Tarihi | Aktarım yapılacak bitiş tarihi seçimi yapılır. |
| Distribütör Kodu | Aktarım yapılacak distribütör seçimi yapılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Aktar | Seçilen kriterlere göre kayıtların aktarılmasını sağlar. |
| Kapat | Formun kapatılmasını sağlar. |
