# Distribütör

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım Kanalı → Tanımlamalar → Distribütör`

## Ekran Tanımı

Distribütör Tanımlarının yapıldığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Sistem tarafından takip edilen artışlı numara bilgisidir. |
| Distribütör Kodu | Distribütör kodu bilgisidir. |
| Ünvan | Distribütör ünvan bilgisidir. |
| Ad | Distribütör ad bilgisidir. |
| Uygulama Yeri | Distribütör tanımına ait uygulama yeri bilgisidir. |
| İlgili Kişi | Distribütörde yer alan ilgili kişi bilgisidir. |
| Grup | Distribütör Grup bilgisidir. |
| Ek Grup | Distribütör Ek Grup bilgisidir. |
| Seviyeli Grup | Seviyeli Grup 1 bilgisidir. Raporlama amaçlı kullanılır. |
| Seviyeli Grup 2 | Seviyeli Grup2 bilgisidir. Raporlama amaçlı kullanılır. |
| Yönetim HY-1 | Yönetim hiyerarşi 1 bilgisidir. |
| Yönetim HY-2 | Yönetim hiyerarşi 2 bilgisidir. |
| Yetki Şablonu | Distribütör yetki şablonudur. Distribütörün çalışma şekli bu şablondan belirlenmektedir. |
| Üretici Yetkisi | Belirli Üretici listesinden seçilen üreticiye verilen yetkidir. |
| Tedarikçi Müşt. | Distribütörün ürün aldığı müşteri bilgisidir. |
| TC Kimlik No | Distribütör ilgili kişisi TC Kimlik No bilgisidir. |
| Şipariş Yönlendirme Distribütörleri | Sipariş yönlendirme distribütörü bilgisidir. |
| Adres 1-2 | Distribütör Adres bilgisidir. 2 satırdan oluşur. |
| İl Kod | Distribütör il bilgisidir. |
| İlçe Kod | Distribütör ilçe bilgisidir. |
| Posta Kodu | Distribütör Posta kodu bilgisidir. |
| Ülke | Distribütör Ülke bilgisidir. |
| Telefon 1-2-3 | Distribütör Telefon numarası bilgisidir. |
| Özel Kod 1 | Distribütör Özel kod bilgisidir. |
| Faks 1-2-3 | Distribütör Faks numarası bilgisidir. |
| Vergi Dairesi | Distribütör Vergi Dairesi bilgisidir. |
| Vergi Numarası | Distribütör Vergi Numarası bilgisidir. |
| Web Adresi | Distribütör WEB adresi bilgisidir. |
| Mail Adresi 1-2-3 | Distribütör Mail adresi bilgisidir. |
| Görev Sonuç Maili | Distribütöre tanımlanan görevlerin sonuç mailinin hangi durumlarda gönderileceği bilgisidir. Başarılı, Başarısız, Hepsi değerlerini alır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Ek Sahalar | Distribütör için tanımlanan ek sahaların girildiği alandır. |
| ST Parametre Şablon Yetkileri | Distribütöre bağlı olan Satış temsilcilerinin bağlı olduğu şablon yetki tanım bilglsidir. |
| Personel | Distribütör Personel ekranıdır. |
| IP Bilgileri | IP bilgisi ekranıdır. |
| Alt Bayiler | Distribütör Alt Bayileri ekranıdır. |
| Vade | Distribütör vadesi belirtildiği ekrandır. |
| Kaydet ve Yeni | Girilen yeni verileri kaydeder ve ekranı kapatır. |
| Kaydet ve Kapat | Değişiklikleri kaydet ve ekranı kapatmaya yarar. |
| Kapat | Ekranı kapatmak için kullanılır. |
