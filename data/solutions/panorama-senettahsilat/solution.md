# Senet

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tahsilat İşlemleri → Senet`

## Ekran Tanımı

Senet tahsilat işleminin yapıldığı adımdır. Ekrana daha önceden girilmiş olan Senet tahsilatları gelir. Önceki girilen Senet tahsilatları üzerinde iptal, düzenleme işlemi yapılabilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Senet tahsilata program tarafından verilen artışlı numaradır. |
| Belge No | Senet tahsila ta verilecek olan belge kodudur. |
| Makbuz No | Tahsilat makbuzu üzerindeki girilecek olan makbuz numarasıdır. |
| Ödendi | Senetin ödeme durumunun belirtildiği alandır. |
| İşlem Tarihi | Senet tahsilatının yapıldığı tarih bilgisinin girildiği alandır. |
| Keşide | Senetin keşide edilip edilmediğinin seçildiği alandır. |
| Distribütör Adı | Distribütör bilgisinin girildiği alandır. |
| Müşteri | Senet tahsilatının yapıldığı müşteri kodudur. |
| Satış Temsilcisi | Müşterini rutunda olduğu satış temsilcisi bilgisidir. |
| Tahsilat ST Kod | Tahsilatı yapan satış temsilcisi bilgisidir. |
| İşlem Tipi | Tahsilat işlem tipinin girildiği alandır. |
| Döviz Tipi | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsidir. |
| Dövizli Tutar | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsinden tutarıdır. |
| Kur | Döviz takibi yapılıyor ise işlem yapılan tahsilatın o günkü kur bilgisidir. |
| Tutar | Yapılan Senet tahsilatın tutarının girildiği alandır. |
| Ödeme Planı | Tanımlı ödeme planı var ise seçildiği alandır. |
| Vade Tarihi | Alınan Senet tahsilatının vade günün girildiği alandır. |
| Ödeme Tarihi | Alınan Senetin ödemesinin yapılacağı tarih bilgisidir. |
| Esas Borçlu | Senetin üzerinde yazan esas borçlu bilgisidir. |
| Vergi Numarası | Müşterinin vergi numarası bilgisidir. |
| Vergi Dairesi | Müşterinin vergi dairesi bilgisidir. |
| Bakiye | Senet Tahsilat kaydı girilen müşterinin bakiye bilgisidir. |
| Basım | Senet Tahsilatı kaydının basım durum bilgisidir. |
| Giriş Tip | Senet Tahsilatı kaydının giriş tipi bilgisidir. |
| Durum | Senet Tahsilat kaydının belge durum bilgisidir. Aktif, silindi değerlerini alır. |
| Fatura Kodu | Senet kaydı için bu alan kullanılmamaktadır |
| Sipariş Kodu | Senet kaydı için bu alan kullanılmamaktadır. |
| Aktarım Durumu | Senet Tahsilat kaydının ticari paket aktarım durum bilgisidir. Aktarıldı, Aktarılmadı değerlerini alır. |
| Tahsilat İptal Nedeni | Tahsilat iptal nedeninin girilebildiği alandır. |
| Proforma Fatura Kodu | Senet kaydı için bu alan kullanılmamaktadır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Yeni | Yeni kayıt girmek için kullanılan butondur. |
| Düzenle | Mevcut kayıtlarda düzenleme amacıyla kullanılan butondur. |
| İzle | Mevcut kayıtlarda izleme yapabilmek için kullanılan butondur. |
| Kopyala | Listede seçili olan tahsilatı kopyalamak için kullanılan butondur. |
| Sil | İptali istenen kaydı silmek için kullanılan butondur. |
| Yazdır | Tanımlı dizayna göre çıktı alınmak istendiğinde kullanılan butondur. |
| Fatura Kapat | Fatura kapatma ekranını açmaktadır. |
