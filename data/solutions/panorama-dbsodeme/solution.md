# DBS

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Ödeme İşlemleri → DBS`

## Ekran Tanımı

DBS ödeme işlemlerinin yapıldığı ekrandır. Öncelikle ekrana, sisteme girilmiş olan DBS Ödeme listesi gelir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Kod | Ödeme belgesine Program tarafından verilen otomatik artışlı sayıdır. |
| Belge No | Belge numarası bilgisidir. |
| Makbuz No | Ödeme Belge numarasıdır. Panorama üzerinden girişlerde kullanıcı tarafından yazılır. Zorunlu alandır. |
| İşlem Tarihi | Ödemenin yapıldığı tarihtir. Zorunlu alandır. |
| Distribütör Adı | Ödeme yapan distribütör bilgisidir. Zorunlu alandır. |
| Müşteri | Ödeme yapılan müşteri bilgisidir. Zorunlu alandır. |
| Satış Temsilcisi | Müşteri satış temsilcisidir. Zorunlu alandır. |
| Tahsilat ST Kod | Ödeme yapan satış temsilcisidir. Panorama üzerinden girişlerde ödeme farklı bir satış temsilcisi tarafından yapıldığı durumda bilgi değiştirilecektir. Alana standart olarak müşteri satış temsilcisi gelmektedir. Zorunlu alandır. |
| İşlem Tipi | Ödeme kaydı hareket tipidir. Bu alana bilgi otomatik olarak gelmektedir. Zorunlu alandır. |
| Döviz Tipi | Döviz takibi yapılıyor ise işlem yapılan ödemenin döviz cinsidir. |
| Dövizli Tutar | Döviz takibi yapılıyor ise işlem yapılan ödemenin döviz cinsinden tutarıdır. |
| Kur | Döviz takibi yapılıyor ise işlem yapılan ödemenin o günkü kur bilgisidir. |
| Tutar | DBS tutar bilgisidir. Zorunlu alandır. |
| Ödeme Planı | Ödeme işleminin nasıl gerçekleştiriliceğini belirtildiği alandır. |
| Vade Tarihi | DBS vade bilgisidir. |
| Banka | DBS banka bilgisidir. |
| Hesap Numarası | DBS banka hesap numarası bilgisidir. |
| Bakiye | Bakiye Dengeleme Borç girişi yapılan müşterinin bakiye bilgisidir. |
| Basım | Belgenin basım durumudur. Yazdırıldı veya yazdırılmadı değerlerini alabilir. |
| Giriş Tip | Belgenin giriş yapıldığı modül bilgisidir. EnroutePanorama ve Panorama Mobile değerlerini alır. |
| Durum | Belgenin durum bilgisidir. Aktif ve Pasif değerlerini alabilir. |
| Fatura Kodu | Belge Ödeme bilgileri giriş ekranından oluşturulduğu durumda belgenin bağlı olduğu fatura numarasının yazıldığı alandır. |
| Sipariş Kodu | Senet ödeme işlemlerinde bu alan kullanılmamaktadır. |
| Aktarım Durumu | Ticari paket aktarım durumudur. Aktarıldı ve aktarılmadı değerlerini alır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Yeni | Yeni kayıt girmek için kullanılan butondur. |
| Düzenle | Mevcut kayıtlarda düzenleme amacıyla kullanılan butondur. |
| İzle | Mevcut kayıtlarda izleme yapabilmek için kullanılan butondur. |
| Kopyala | Listede seçili olan tahsilatı kopyalamak için kullanılan butondur. |
| Sil | İptali istenen kaydı silmek için kullanılan butondur. |
| Yazdır | Tanımlı dizayna göre çıktı alınmak istendiğinde kullanılan butondur. |
