# Fiyat Onaylama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Pazarlama Kriteri → Fiyat İşlemleri → Fiyat Onaylama`

## Ekran Tanımı

Sistemde tanımlı olan ürünlere fiyatların toplu bir şekilde onaylanabileceği ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Kodu | Tanımlanması yapılmak istenilen fiyat onaylamanın hangi distribütöre ait olduğunun bilgisidir. |
| Ürün | Tanımlanması yapılmak istenilen fiyat onaylamanın hangi ürün için geçerli olacağının bilgisidir. |
| Geçerlilik Tarihi | Fiyat onaylama tanımının geçerlilik süresi belirtilir. |
| Onay Durumu | Tanımlanan fiyat onaylama durumunun belirlendiği alandır. Onaylandı ya da onaylanmadı seçenekleri vardır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Listele | Girilen verilere göre listeleme işlemi gerçekleştirir |
| Kapat | Ekranı kapatır. |
| Yenile | Detaydaki listeyi yeniler. |
| Onay | Fiyat Onaylamak için kullanılır. |
| İzle | Detaydaki kaydı görüntülemek için kullanılır. |
| İptal | Fiyat İptali işlemi için kullanılır. |
