# Rut Tanım Bilgileri Alma

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ İşlemler → Rut Tanım Bilgileri Alma`

## Ekran Tanımı

Excel üzerinden Panoramaya rut tanımı alma işlemidir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Görev Kod | Servis üzerinden tanımlanan Rut alma görev kodu seçilir. |
| İşlem No | Sistem tarafından otomatik olarak atanan işlem numarası gösterilir. |
| Uyarıya rağmen alınsın | Veri alma işlemi sırasında sistem tarafından rut kontrolü ile ilgili uyarı verilse dahi veri alma işleminin devam etmesi sağlanır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Dosya Yükle | Panorama’ya aktarılacak excel dosyası seçimi için kullanılır. |
| Dosya Aktar | Seçilen excel dosyasının sisteme aktarılması için kullanılır. |
| Rut Tanım Alma | Seçilen excel dosyasındaki rut tanım bilgileri sisteme aktarılır. |
| Rut Plan Alma | Seçilen excel dosyasındaki rut plan bilgileri sisteme aktarılır. |
| Kapat | Ekran kapatılır. |
| Seç | Sistem üzerinde yaratılan , Panorama’ya aktarılacak excel doyası seçimi yapılır. |
| Yükle | Seçilen excel dosyasının sisteme aktarılması için kullanılır. |
| Kapat | Ekran kapatılır. |
