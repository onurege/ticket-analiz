# Satış Temsilcisi İşlem Tipi

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Tanımlamalar → Satış Temsilcisi İşlem Tipi`

## Ekran Tanımı

Satış Temsilcisi işlem tipi bilgisi düzenle ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Satış temsilcisi kod bilgisidir. |
| Durum | Satış temsilcisi durum bilgisidir. |
| Distribütör Kodu | Satış Temsilcisi işlem tipi için distribütör kod bilgisidir. |
| Satış Temsilcisi | Satış Temsilcisi kodudur. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Ekranda bulunan kayıtların yenilenmesini sağlayan butondur. |
| Yeni | Yeni işlem tipi belirlemek için kullanılar. |
| İzle | Kayıtlı olan kriterleri izlemek için kullanılır. |
| Sil | Kayıtlı olan kriterleri silmek için kullanılar. |
