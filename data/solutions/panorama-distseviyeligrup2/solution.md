# Distribütör Seviyeli Grup 2

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Dağıtım Kanalı → Parametreler → Distribütör Seviyeli Grup 2`

## Ekran Tanımı

Distribütör Seviyeli Grup 2 tanımlamalarının tanımladığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Seviye Sayısı | Distribütör gruplarına ait kaç seviye olacağının tanımlandığı alandır. |
| Seviye Ayracı | Distribütör grup seviyeleri ve alt seviyeleri birbirleriyle ayraç aracılığıyla ilişkilendirilir. Bu ayraç tanımının yapıldığı alandır. |
| Sıra | Seviyeli Grup sıra tanımının yapıldığı alandır. |
| Karakter Sayısı | Seviyeye ait numaranın kaç karakterli olduğunun tanımının yapıldığı alandır. |
| Seviye Adı | Seviyeye ait isim tanımının yapıldığı alandır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Detaydaki listeyi yeniler. |
| Yeni | Detayda yeni tanım yapılmasını sağlar. |
| Düzenle | Detaydaki tanımın düzenlenmesini sağlar. |
| İzle | Detaydaki tanıma ait detayların görüntülenmesini sağlar. |
| İşlemi Başlat | Distribütör Seviyeli Grup 2 tanımını oluşturur. |
| Kapat | Ekranı kapatır. |
