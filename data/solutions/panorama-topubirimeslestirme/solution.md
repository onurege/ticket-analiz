# Toplu Birim Eşleştirme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → Ticari Paket → Eşleştirme → Toplu Birim Eşleştirme`

## Ekran Tanımı

Panorama kullanılan birim adlarının ticari paket karşılıklarının girildiği ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Otomatik artışlı numaradır. Sistem tarafından verilir. |
| Distribütör Kodu | İşlem yapılacak dsitribütör bilgisi seçilir. |
| Tip | Veri Okuma ve Veri Yazma değerlerini alabilir. Yapılan eşleştirme bilgisinin hangi aktarım tipinde olacağı seçimi yapılır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfanın yenilenmesini sağlar. |
| Yeni | Yeni birim açıklama kaydı girilmesini sağlar. |
| Düzenle | Varolan birim açıklama kaydını düzenle |
| İzle | Varolan birim açıklama kaydını izle modunda açar. |
| Sil | Varolan birim açıklama kaydını siler. |
| Kaydet | Form kaydedilir. |
| Kapat | Farm kapatılır. |
