# İrsaliyeden Fatura Oluşturma

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış ve Alış İşlemleri → Satıcıya İade İşlemleri → Irsaliyeden Fatura Oluşturma`

## Ekran Tanımı

İrsaliyeden fatura oluşturma ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yükleme Kodu | Yapılan işleme ait program tarafından otomatik verilen kayıt numarasıdır. |
| Distribütör Kodu | Belgenin hangi Distribütör için kesileceğinin tanımlandığı alandır. |
| Başlangıç Tarihi | Satış Irsaliye fatura belgelerinin hangi tarihten başlayacağı bilgisidir. |
| Bitiş Tarihi | Satış Irsaliye fatura belgelerinin hangi tarihte biteceği bilgisidir. |
| Başlangıç Sevk Tarihi | Satış Irsaliye fatura belgelerinin hangi Sevk tarihinden başlayacağı bilgisidir. |
| Bitiş Sevk Tarihi | Satış Irsaliye fatura belgelerinin hangi Sevk tarihinden biteceğinin bilgisidir. |
| Araç | Araç bilgisidir. |
| Hareket Tipi | Satış Irsaliye fatura için tanımlanan hareket tipinin seçildiği alandır. Zorunlu bir alandır. |
| Satış Temsilcisi | Satış Irsaliye fatura hangi Satış Temsilcileri üzerinden listelendiği seçiminin yapıldığı alandır. |
| Bölge Kodları | Satış Irsaliye fatura hangi müşteri Bölge üzerinden listeleneceği seçiminin yapıldığı bilgisidir. |
| Seçilen Araç | Siparişin yükleme yapılacağı araç seçilir. |
| Araç Hacim | Araç hacim bilgisidir. |
| Toplam Hacim | Toplam hacim bilgisidir. |
| Hacim Oran | Hacim oran bilgisidir. |
| İade Hacim | İade hacim bilgisidir. |
| İade Hacim Oran | İade hacim oran bilgisidir. |
| Araç Nokta Sayısı | Araç nokta sayısı bilgisidir. |
| Nokta Sayısı | Nokta sayısı bilgisidir. |
| Araç Ağırlık | Araç ağırlık bilgisidir. |
| Toplam Ağırlık | Toplam ağırlık bilgisidir. |
| Ağırlık Oran | Ağırlık oran bilgisidir. |
| İade Ağırlık | İade ağırlık bilgisidir. |
| İade Ağırlık Oran | İade ağırlık oran bilgisidir. |
| Dağıtıcı | Dağıtıcı bilgisidir. |
| Belge Sayısı | Belge sayısı bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Temizle | Listedeki verileri temizlemek için kullanılır. |
| Sevk Tarihleri | Sevk tarhi ayarlamak için kullanılan ekrandır. |
| Listele | Girdilere göre listeleme işlemleri yapılmaktadır. |
| Belgeleştir | Sevk Tarihleri ekranını açmaktadır. |
| Kaydet | Belgeyi kaydetmek için kullanılır. |
| Kapat | Belge ekranını kapatmak için kullanılır. |
| Yenile | Belge detay listesini yenilemek için kullanılır. |
| Belge İzle | Belge detay satırında seçili belgeyi incelemek için kullanılır. |
| Transfer | Listelenen siparişlerin istenilen yere transfer işlemi gerçekleştirilir. |
| Yukarı | “Yukarı” – “Aşağı “ butonları ile oluşacak belgelerin öncelik sıralaması değiştirilebilir. |
| Aşağı | “Yukarı” – “Aşağı “ butonları ile oluşacak belgelerin öncelik sıralaması değiştirilebilir. |
