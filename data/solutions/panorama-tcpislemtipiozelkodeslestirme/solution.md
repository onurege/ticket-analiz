# İşlem Tip - Özel Kod(LKS2) Eşleştirme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Entegrasyon→ Smart Connect → Ticari Paket → Eşleştirme → İşlem Tip Özel Kod (LKS-2) Eşleştirme`

## Ekran Tanımı

LOGO ticari paket programına Xml ile aktarımlarda kullanılır. Satış faturası işlem tipi bazında logo’da kullanılmak istenen Özel Kod girişi yapılır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Distribütör Adı | İşlem yapılacak Distribütör bilgisi seçilir. |
| Tp Kod | Listelenen satış faturası işlem tip kayıtlarının karşılığında Ticari Paket programında kullanılacak özel kod bilgisi girilir. Giriş yapıldığı anda kaydetme işlemi gerçekleşecektir. |
| Kod | Listelenen satış faturası işlem tip kayıtlarının Panoramada kullanılan kod bilgisi gösterilir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kapat | Form kapatılır. |
| Yenile | Sayfanın yenilenmesini sağlar. |
| Düzenle | Seçilen satırda Tp Kod alanını aktif hale getirir ve değer girişi yapılmasını sağlar. |
| Yön Tuşları | Listelenen satış faturası işlem tip kayıtlarında ileri-geri sayfa geçişi yapılmasını sağlar. |
