# Ürün Ek Saha

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Parametreler → Ürün Ek Saha`

## Ekran Tanımı

Ürün ek saha tanımının yapıldığı ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kodu | Ürün ek saha tanımına verilen artışlı numaradır. |
| Distribütör Kriter Tipi | Burada seçilen değere göre distribütör kriterde distribütör listesi gelir. Kod / Grup / Ek Grup / Dinamik Grup değerlerini alabilir. |
| Ek Saha Özelliği | Kullanılacak ek saha tanımının ürün kartında hangi alanda bulunacağının belirtildiği alandır. Ek Sahalar / Dinamik Sahalar değerlerini alabilir. |
| Distribütör Kriteri | Distribütör kriter tipi olarak kod seçilmiş ise bu alanda distribütör listesinden distribütör seçimi yapılır. Eğer ki distribütör kriter tipi Grup seçilmiş ise distribütör grup listesinden grup seçimi yapılır. |
| Zorunlu | Zorunlu alan değeri seçildiğinde ürün kartı üzerinde bu ek sahaya ait cevap yada seçimin yapılması zorunlu hale gelir. |
| Ek Saha Özelliği | Ek sahanın hangi özellikte olucağının tanımı yapılır. Çoktan Seçmeli / Metin / Sayısal / Tarih değerlerini alabilir. |
| Firma | Firma tanımının yapıldığı alandır. Firma yada Rakip değerlerini alır. |
| Min. Uzunluk | Ek Saha Özelliği Metin olduğunda aktif hale gelir. Girilecek metinin minimum kaç karakterden oluşacağının tanımı yapılır. |
| Maks. Uzunluk | Ek Saha Özelliği Metin olduğunda aktif hale gelir. Girilecek metinin maksimum kaç karakterden oluşacağının tanımı yapılır. |
| Maksimum Cevap Sayısı | Ek Saha Özelliği Çoktan Seçmeli olduğunda aktif olur. Maksimum verilecek cevap sayısı tanımlanır. |
| Kod | Ek saha seçenek detayına verilen numaradır. |
| Kaydet | Yapılan tanımı kaydeder. |
| Kapat | Tanım ekranını kapatır. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Seçenek | Ürün ek saha seçenek detay ekranını açar. |
| Kaydet ve Yeni | Yapılan tanımı kaydeder ve yeni tanım ekranını getirir. |
| Kaydet ve Kapat | Yapılan tanımı kaydeder ve ekranı kapatır. |
| Kapat | Ekranı kapatır. |
