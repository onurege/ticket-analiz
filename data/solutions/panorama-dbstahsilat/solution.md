# DBS

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → Tahsilat İşlemleri → DBS`

## Ekran Tanımı

DBS (Doğrudan Borçlandırma Sistemi) tahsilat işleminin yapıldığı adımdır. DBS; ana firmanın bankaya ilettiği tutarların belirtilen vade sonunda müşterinin banka hesabından ya da kredi limitinden çekilmesidir. Ekrana daha önceden girilmiş olan DBS tahsilatları gelir. Önceki girilen DBS tahsilatları üzerinde iptal, düzenleme işlemi yapılabilir.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | DBS tahsilatına program tarafından verilen artışlı numaradır. |
| Belge No | DBS tahsila tına verilecek olan belge kodudur. |
| Makbuz No | Tahsilat makbuzu üzerindeki girilecek olan makbuz numarasıdır. |
| İşlem Tarihi | DBS tahsilatının yapıldığı tarih bilgisinin girildiği alandır. |
| Distribütör Adı | Distribütör bilgisinin girildiği alandır. |
| Müşteri | DBS tahsilatının yapıldığı müşteri kodudur. |
| Satış Temsilcisi | Müşterini rutunda olduğu satış temsilcisi bilgisidir. |
| Tahsilat ST Kod | Tahsilatı yapan satış temsilcisi bilgisidir. |
| İşlem Tipi | Tahsilat işlem tipinin girildiği alandır. |
| Döviz Tipi | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsidir. |
| Dövizli Tutar | Döviz takibi yapılıyor ise işlem yapılan tahsilatın döviz cinsinden tutarıdır. |
| Kur | Döviz takibi yapılıyor ise işlem yapılan tahsilatın o günkü kur bilgisidir. |
| Tutar | Yapılan DBS tahsilatının tutarının girildiği alandır. |
| Ödeme Planı | Tanımlı ödeme planı var ise seçildiği alandır. |
| Vade Tarihi | Alınan DBS tahsilatının vade günün girildiği alandır. |
| Banka | DBS kaydı yapılan banka bilgisidir. |
| Hesap Numarası | DBS kaydı yapılan bankanın hesap numarasıdır. |
| Bakiye | DBS Tahsilat kaydı girilen müşterinin bakiye bilgisidir. |
| Basım | DBS Tahsilatı kaydının basım durum bilgisidir. |
| Giriş Tip | DBS Tahsilatı kaydının giriş tipi bilgisidir. |
| Durum | DBS Tahsilat kaydının belge durum bilgisidir. Aktif, silindi değerlerini alır. |
| Fatura Kodu | DBS kaydının fatura kodu bilgisidir. Girilen faturaya karşılık oluşan takip kodu bilgisidir. |
| Sipariş Kodu | DBS kaydının sipariş kodu bilgisidir. Girilen siparişe karşılık oluşan takip kodu bilgisidir. |
| Aktarım Durumu | DBS Tahsilat kaydının ticari paket aktarım durum bilgisidir. Aktarıldı, Aktarılmadı değerlerini alır. |
| Proforma Fatura Kodu | DBS kaydının proforma fatura kodu bilgisidir. Girilen proforma faturaya karşılık oluşan takip kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Sayfayı yenilemek için kullanılan butondur. |
| Yeni | Yeni kayıt girmek için kullanılan butondur. |
| Düzenle | Mevcut kayıtlarda düzenleme amacıyla kullanılan butondur. |
| İzle | Mevcut kayıtlarda izleme yapabilmek için kullanılan butondur. |
| Kopyala | Listede seçili olan tahsilatı kopyalamak için kullanılan butondur. |
| Sil | İptali istenen kaydı silmek için kullanılan butondur. |
| Yazdır | Tanımlı dizayna göre çıktı alınmak istendiğinde kullanılan butondur. |
| Fatura Kapat | Fatura kapatma ekranını açmaktadır. |
