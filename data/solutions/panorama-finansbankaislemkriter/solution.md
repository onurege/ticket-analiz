# Banka İşlemleri

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Finans İşlemleri → İşlemler → Banka İşlemleri`

## Ekran Tanımı

Banka İşlemleri ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| Kod | Banka işlemleri belge kodu bilgisidir. |
| İşlem Tipi | İşlem tipi bilgisidir. |
| İşlem Tarihi | İşlem tarihi bilgisidir. |
| Distribütör Kodu | Distribütör Kodu bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| ALAN | AÇIKLAMALAR |
| İşlem Başlat | Bütün alanlar doldurulduktan sonra belirlenen kriterlere göre listelemenin başlatılması için İşlemi Başlat butonuna tıklanılır. |
| Kapat | Ekranı kaydetmeden kapatır. |
