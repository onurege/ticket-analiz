# Saha İzleme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → İşlemler → Saha İzleme`

## Ekran Tanımı

Sahadaki temsilcilerin el bilgisayarında yaptığı işlemlerin görüntülendiği ekrandır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Aktif Filtre | Sıkça kullanılacak Filtreler kaydedilebilir ve çağırılabilir. |
| Dist | Listelenecek Distribütör bilgisidir. |
| ST Kriter Tip | ST kriter tipi kod, grup yöneticisi ya da grup seçilir. |
| ST Kriter | ST kriter bilgisidir. |
| İşlem Tarihi | İşlem tarihi bilgisidir. |
| Birim | Hangi birimde olduğunu belirten alandır. |
| Fatura | Listelenecek belgelerin seçiminin yapıldığı alandır. |
| Sipariş | Listelenecek belgelerin seçiminin yapıldığı alandır. |
| Tahsilat | Listelenecek belgelerin seçiminin yapıldığı alandır. |
| İrsaliye | Listelenecek belgelerin seçiminin yapıldığı alandır. |
| Konsinye | Listelenecek belgelerin seçiminin yapıldığı alandır. |
| Quest | Listelenecek belgelerin seçiminin yapıldığı alandır. |
| Diğer | Listelenecek belgelerin seçiminin yapıldığı alandır. |
| Depo Hareket | Listelenecek belgelerin seçiminin yapıldığı alandır. |
| Dist | Distribütör isim bilgisidir. |
| ST/SE Adı | Satış temsilcisi/Satış elemanı adı bilgisidir. |
| Müşteri | Müşteri adı bilgisidir. |
| Rut | Rut bilgisidir. |
| Tarih | Listelenecek kayıtların tarih bilgisidir. |
| Saat | İşlemin yapıldığı saati gösterir. |
| İşlem | Sahada yapılan işlem adımlarını gösterir. |
| Belge | İşlem yapılan belge numarasını gösterir. |
| Tutar | Yapılan işlemlerin tutarını gösterir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Filter | Önceden kaydedilmiş olan filtreleri seçerek filtreleme işlemi gerçekleştirilir. |
| Filtreleri Temizle | Kullanılan filtreyi temizlemek için kullanılır |
| Kapat | Yapılan değişiklikleri kaydetmeden kapatmaya yarar. |
| Listele | Belirtilen kriterlere bağlı olarak listeleme işlemini gerçekleştirir. |
| Excel | Ekrandaki bilgileri Excel dosyasına çevirmek için kullanılır. |
| İşlemler | İşlem bilgilerinin bulunduğu tabloyu ekrana getirir. |
| Dashboard | Yüzdeler tablo halinde ekrana getirilir. |
