# Satış Temsilcisi Hareket Tipi Eşleştirme

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Satış Ekibi → Tanımlamalar → Satış Temsilcisi Hareket Tipi Eşleştirme`

## Ekran Tanımı

Satış Temsilcisi hareket tipi bilgisi düzenle ekranıdır.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Takip Kod | Teslimat takip kod bilgisidir. |
| Durum | Satış temsilcisi durum bilgisidir. |
| Satış Temsilcisi | Satış Temsilcisi kodudur. |
| Hareket Tip | Hareket tipi bilgisidir. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Yenile | Ekranda bulunan kayıtların yenilenmesini sağlayan butondur. |
| Yeni | Yeni hareket tipi belirlemek için kullanılar. |
| İzle | Kayıtlı olan kriterleri izlemek için kullanılır. |
| Sil | Kayıtlı olan kriterleri silmek için kullanılar. |
