# Ürün Sıralama

_Kaynak: Panorama Kullanım Kılavuzu_

**Menü Adımı:** `Ürünler → Tanımlamalar → Ürün Sıralama İşlemi`

## Ekran Tanımı

Android platformunda veri giriş işlemlerinde ürün sıralamasının, veri giriş uygulama kriterinde seçilmiş olan ürün şablonundaki sırayla yapılmasını sağlar.

## Alan Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kod | Sistem tarafından verilen artışlı numaradır. |
| Durum | Tanımın durumunun belirtildiği alandır. Aktif / Pasif değerleri alabilir. |
| Düzenle | Ürün sıra numarasını düzenlemek için kullanılır. |
| İzle | Ürün sıra numarasını gösterir. |
| \|< | Listede en başa gidilmesini sağlar. |
| >\| | Listede en son sayfaya gidilmesini sağlar. |
| Yukarı | Kayıdın sırasını bir satır yukarı taşır. |
| Aşağı | Kayıdın sırasını bir satır aşağı taşır. |
| Temizle | Listedeki kayıtları ekrandan siler. |

## Buton Açıklamaları

| Alan | Açıklama |
| --- | --- |
| Kaydet ve Kapat | Yapılan tanımı kaydeder ve ekranı kapatır. |
| Listele | Ürünleri listeler. |
| Kapat | Ekranı kapatır. |
